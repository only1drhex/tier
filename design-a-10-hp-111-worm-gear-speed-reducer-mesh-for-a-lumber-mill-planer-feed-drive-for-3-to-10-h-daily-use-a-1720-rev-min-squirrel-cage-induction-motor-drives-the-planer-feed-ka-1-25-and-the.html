Guest Question.

Design a 10-hp 11:1 worm-gear speed-reducer mesh for a lumber mill planer feed drive for 3- to 10-h daily use. A 1720-rev/min squirrel-cage induction motor drives the planer feed (\(K_{a}=1.25\)), and the ambient temperature is 70°F.

Admin
Solution

Function: \(H_{0}=10 hp , m_{G}=11, n_{W}=1720 \text { rev } / min .\)

 

Design factor: \(n_{d}=1.2\)

 

Materials and processes: case-hardened alloy steel worm, sand-cast bronze gear.
Worm threads: double, \(N_{W}=2, N_{G}=m_{G} N_{W}=11(2)=22\) gear teeth acceptable for \(\phi_{n}=20^{\circ}\), according to Table 15–10.
Decision 1: Choose an axial pitch of worm\(p_{x}=1.5\) in. Then,

 

\(P_{t}=\pi / p_{x}=\pi / 1.5=2.0944\)

 

\(D=N_{G} / P_{t}=22 / 2.0944=10.504 \text { in }\)

 

Eq. (15–39):

 

\(a=\frac{p_{x}}{\pi}=0.3183 p_{x}\) (15–39)

 

\(a=0.3183 p_{x}=0.3183(1.5)=0.4775 \text { in (addendum) }\)

 

Eq. (15–40):

 

\(b=\frac{1.157 p_{x}}{\pi}=0.3683 p_{x}\) (15–40)

 

\(b=0.3683(1.5)=0.5525 \text { in (dedendum) }\)

 

Eq. (15–41):

 

\(h_{t}=\left\{\begin{array}{ll}\frac{2.157 p_{x}}{\pi}=0.6866 p_{x} & p_{x} \geq 0.16 \text { in } \\\frac{2.200 p_{x}}{\pi}+0.002=0.7003 p_{x}+0.002 & p_{x}<0.16 \text { in }\end{array}\right.\) (15–41)

 

\(h_{t}=0.6866(1.5)=1.030 \text { in }\)

 

Decision 2: Choose a mean worm diameter d = 2.000 in. Then

 

\(C=(d+D) / 2=(2.000+10.504) / 2=6.252 \text { in }\)

 

\((d)_{ lo }=6.252^{0.875} / 3=1.657 \text { in }\)

 

\((d)_{ hi }=6.252^{0.875} / 1.6=3.107 in\)

 

The range, given by Eq. (15–27), is \(1.657 \leq d \leq 3.107 \text { in, }\) which is satisfactory. Try d = 2.500 in. Recompute C:

 

\(C=(2.5+10.504) / 2=6.502 \text { in }\)

 

\(\frac{C^{0.875}}{3} \leq d \leq \frac{C^{0.875}}{1.6}\) (15–27)

 

The range is now \(1.715 \leq d \leq 3.216\) in, which is still satisfactory. Decision: d = 2.500 in. Then

 

Eq. (13–27): \(L=p_{x} N_{W}=1.5(2)=3.000 \text { in }\)

 

\(\tan \lambda=\frac{L}{\pi d_{W}}\) (13–28)

 

Eq. (13–28): \(\lambda=\tan ^{-1}[L /(\pi d)]=\tan ^{-1}[3 /(\pi 2.5)]=20.905^{\circ} \quad(\text { from Table } 15-9 \text { lead angle } OK )\)

 

Eq. (15–62): \(V_{s}=\frac{\pi d n_{W}}{12 \cos \lambda}=\frac{\pi(2.5) 1720}{12 \cos 20.905^{\circ}}=1205.1 ft / min\)

 

\(V_{W}=\frac{\pi d n_{W}}{12}=\frac{\pi(2.5) 1720}{12}=1125.7 ft / min\)

 

\(V_{G}=\frac{\pi D n_{G}}{12}=\frac{\pi(10.504) 1720 / 11}{12}=430.0 ft / min\)

 

Eq. (15–33): \(C_{s}=\left\{\begin{array}{lll}1000 & C>3 & D_{m} \leq 2.5 in \\1190-477 \log D_{m} & C>3 & D_{m}>2.5 in\end{array}\right.\)

 

\(C_{s} = 1190 – 477 log 10.504 = 702.8\)

 

Eq. (15–36):

 

\(C_{m}=\left\{\begin{array}{ll}0.02 \sqrt{-m_{G}^{2}+40 m_{G}-76}+0.46 & 3<m_{G} \leq 20 \\0.0107 \sqrt{-m_{G}^{2}+56 m_{G}+5145} & 20<m_{G} \leq 76 \\1.1483-0.00658 m_{G} & m_{G}>76\end{array}\right.\) (15–36)

 

\(C_{m} = 0.02\sqrt{-11^{2}+40\left(11\right)-76 } + 0.46 = 0.772 \)

 

Eq. (15–37):

 

\(C_{v}=\left\{\begin{array}{ll}0.659 \exp \left(-0.0011 V_{s}\right) & V_{s}<700 ft / min \\13.31 V_{s}^{-0.571} & 700 \leq V_{s}<3000 ft / min \\65.52 V_{s}^{-0.774} & V_{s}>3000 ft / min\end{array}\right.\) (15–37)

 

\(C_{v}=13.31(1205.1)^{-0.571}=0.232\)

 

Eq. (15–38):

 

\(f=\left\{\begin{array}{ll}0.15 & V_{s}=0 \\0.124 \exp \left(-0.074 V_{s}^{0.645}\right) & 0<V_{s} \leq 10 ft / min \\0.103 \exp \left(-0.110 V_{s}^{0.450}\right)+0.012 &V_{s}>10 ft / min\end{array}\right.\) (15–38)

 

\(f = 0.103 exp[-0.11(1205.1)^{0.45}] + 0.012 = 0.0191\)

 

Eq. (15–54):

 

\(e_{W}=\frac{\cos \phi_{n}-f \tan \lambda}{\cos \phi_{n}+f \cot \lambda}\) (15–54)

 

\(e_{W}=\frac{\cos 20^{\circ}-0.0191 \tan 20.905^{\circ}}{\cos 20^{\circ}+0.0191 \cot 20.905^{\circ}}=0.942\)

 

(If the worm gear drives, \(e_{G}=0.939\).) To ensure nominal 10-hp output, with adjustments for \(K_{a}, n_{d}, \text { and } e\),

 

Eq. (15–57):

 

\(W^{t}_{W} = W^{t}_{G} \frac{cos \phi _{n}sin \lambda + f cos \lambda }{cos \phi _{n}cos \lambda – f sin \lambda}\) (15–57)

 

\(W^{t}_{W} = 1222 \frac{cos 20° sin 20.905° + 0.0191 cos 20.905°}{cos 20° cos 20.905° – 0.0191 sin 20.905°} = 495.4 lbf\)

 

Eq. (15–58):

 

\(W_{G}^{t}=\frac{33000 n_{d} H_{0} K_{a}}{V_{G} e}\) (15–58)

 

\(W^{t}_{G} = \frac{33000 \left(1.2\right)10\left(1.25\right)}{430 \left(0.942\right) } = 1222 lbf \)

 

Eq. (15–59):

 

\(H_{W}=\frac{W_{W}^{t} V_{W}}{33000}=\frac{\pi d_{W} n_{W} W_{W}^{t}}{12(33000)}\) (15–59)

 

\(H_{W}=\frac{\pi(2.5) 1720(495.4)}{12(33000)}=16.9 hp\)

 

Eq. (15–60):

 

\(H_{G}=\frac{W_{G}^{t} V_{G}}{33000}=\frac{\pi d_{G} n_{G} W_{G}^{t}}{12(33000)}\) (15–60)

 

\(H_{G}=\frac{\pi(10.504) 1720 / 11(1222)}{12(33000)}=15.92 hp\)

 

Eq. (15–61):

 

\(W_{f}=\frac{f W_{G}^{t}}{f \sin \lambda-\cos \phi_{n} \cos \lambda}\) (15–61)

 

\(W_{f} = \frac{0.0191\left(1222\right) }{0.0191 \sin 20.905^{\circ} – \cos 20^{\circ} \sin 20.905^{\circ}} = -26.8 lbf\)

 

Eq. (15–63):

 

\(H_{f}=\frac{\left|W_{f}\right| V_{s}}{33000} hp\) (15–63)

 

\(H_{f}=\frac{|-26.8| 1205.1}{33000}=0.979 hp\)

 

From Eq. (15–28), with \(C_{s}=702.8, C_{m}=0.772 \text { , and } C_{v}=0.232 \text { , }\)

 

\(\left(F_{e}\right)_{ req }=\frac{W_{G}^{t}}{C_{s} D^{0.8} C_{m} C_{v}}=\frac{1222}{702.8(10.504)^{0.8} 0.772(0.232)}=1.479 in\) (15–28)

 

Decision 3: The available range of \(\left(F_{e}\right)_{G} \text { is } 1.479 \leq\left(F_{e}\right)_{G} \leq 2 d / 3 \text { or } 1.479 \leq\left(F_{e}\right)_{G} \leq\)\(1.667 \text { in. Set }\left(F_{e}\right)_{G}=1.5 \text { in. }\)

 

Eq. (15–28):

 

\(\left(W^{t}\right)_{\text {all }}=C_{s} D_{m}^{0.8} F_{e} C_{m} C_{v}\)

 

\(W_{\text {all }}^{t}=702.8(10.504)^{0.8} 1.5(0.772) 0.232=1239 lbf\)

 

This is greater than 1222 lbf. There is a little excess capacity. The force analysis stands.
Decision 4:

 

Eq. (15–50):

 

\(\hbar_{ CR }=\left\{\begin{array}{ll}\frac{n_{W}}{6494}+0.13 & \text { no fan on worm shaft } \\\frac{n_{W}}{3939}+0.13 & \text { fan on worm shaft }\end{array}\right.\) (15–50)

 

\(\hbar_{ CR }=\frac{n_{W}}{6494}+0.13=\frac{1720}{6494}+0.13=0.395 ft \cdot lbf /\left( min \cdot in ^{2} \cdot{ }^{\circ} F \right)\)

 

Eq. (15–49):

 

\(H_{\text {loss }}=33000(1-e) H_{\text {in }}\) (15–49)

 

\(H_{ loss }=33000(1-e) H_{W}=33000(1-0.942) 16.9=32347 ft \cdot lbf / min\)

 

The AGMA area, from Eq. (15–52), is Amin \(A_{\min }=43.2 C^{1.7}=43.2(6.502)^{1.7}=1041.5 in ^{2}\).
A rough estimate of the lateral area for 6-in clearances:

 

Vertical: \(d+D+6=2.5+10.5+6=19 \text { in }\)

 

Width: \(D+6=10.5+6=16.5 \text { in }\)

 

Thickness: \(d+6=2.5+6=8.5 \text { in }\)

 

Area: \(2(19) 16.5+2(8.5) 19+16.5(8.5) \approx 1090 in ^{2}\)

 

Expect an area of \(1100 in ^{2}\). Choose: Air-cooled, no fan on worm, with an ambient
temperature of 70°F.

 

\(t_{s}=t_{a}+\frac{H_{\text {loss }}}{\hbar_{ CR } A}=70+\frac{32350}{0.395(1100)}=70+74.5=144.5^{\circ} F\)

 

Lubricant is safe with some margin for smaller area.

 

Eq. (13–18):

 

\(P_{n}=\frac{P_{t}}{\cos \psi}\) (13–18)

 

\(P_{n}=\frac{P_{t}}{\cos \lambda}=\frac{2.094}{\cos 20.905^{\circ}}=2.242\)

 

\(p_{n}=\frac{\pi}{P_{n}}=\frac{\pi}{2.242}=1.401 \text { in }\)

 

Gear bending stress, for reference, is

 

Eq. (15–53):

 

\(\sigma_{a}=\frac{W_{G}^{t}}{p_{n} F_{e} y}\) (15–53)

 

\(\sigma=\frac{W_{G}^{t}}{p_{n} F_{e} y}=\frac{1222}{1.401(1.5) 0.125}=4652 psi\)

 

The risk is from wear, which is addressed by the AGMA method that provides \(\left(W_{G}^{t}\right)_{\text {all }}\)
