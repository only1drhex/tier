Guest Question.

In 1938 Howarth proposed a linearly decelerating external velocity distribution

\(U(x) = U_0 \left(1 - \frac{x}{L}\right)\) (1)

as a theoretical model for laminar-boundary-layer study. (a) Use Thwaites’s method to compute the separation point \(x_{sep} for θ_0 = 0\), and compare with the exact computer solution \(x_{sep}/L = 0.119863\) given by H. Wipperman in 1966. (b) Also compute the value of \(c_f = 2\tau_w/(ρU^2)\) at x/L = 0.1.

Admin
Solution

Part (a)

First note that dU/dx = – \(U_0\)/L = constant: Velocity decreases, pressure increases, and the pressure gradient is adverse throughout. Now integrate Eq. (7.54):

\(\theta^2 = \theta_0^2 \left(\frac{U_0}{U}\right)^6 + \frac{0.45 \nu}{U^6} \int^x_0 {U^5 dx}\) (7.54)

\(\theta^2 = \frac{0.45 \nu}{U_0^6 (1 – x/L)^6} \int^x_0{U_0^5 \left(1 – \frac{x}{L}\right)^5 dx} = 0.075 \frac{\nu L}{U_0} \left[\left(1 – \frac{x}{L}\right)^{-6} – 1\right]\) (2)

Then the dimensionless factor λ is given by

\(\lambda = \frac{\theta^2}{\nu}\frac{dU}{dx} = – \frac{\theta^2 U_0}{\nu L} = -0.075 \left[\left(1 – \frac{x}{L}\right)^{-6} – 1\right]\) (3)

From Eq. (7.55) we set this equal to -0.09 for separation:

λ = -0.09 (7.55)

\(\lambda_{sep} = -0.09 = -0.075 \left[\left(1 – \frac{x_{sep}}{L}\right)^{-6} – 1\right]\)

or \(\frac{x_{sep}}{L} = 1 – (2.2)^{-1/6} = 0.123\)

This is less than 3 percent higher than Wipperman’s exact solution, and the computational effort is very modest.

Part (b)

To compute \(c_f\) at x/L = 0.1 (just before separation), we first compute λ at this point, using Eq. (3):

λ(x = 0.1L) = -0.075[(1 – 0.1)\(^{-6}\) – 1] = -0.0661

Then from Eq. (7.56) the shear parameter is

\(S(\lambda) = \frac{\tau_w \theta}{\mu U} \approx (\lambda + 0.09)^{0.62}\) (7.56)

S(x = 0.1L) = (-0.0661 + 0.09)\(^{0.62}\) = 0.099 = \(\frac{1}{2}c_f Re_{\theta}\) (4)

We can compute \(Re_{\theta}\) in terms of \(Re_L\) from Eq. (2) or (3):

\(\frac{\theta^2}{L^2} = \frac{0.0661}{UL/ \nu} = \frac{0.0661}{Re_L}\)

or \(Re_{\theta} = 0.257 Re_L^{1/2} at \frac{x}{L} = 0.1\)

Substitute into Eq. (4):

\(0.099 = \frac{1}{2}c_f (0.257 Re_L^{1/2})\)

or \(c_f = \frac{0.77}{Re_L^{1/2}} Re_L = \frac{UL}{\nu}\)

We cannot actually compute \(c_f\) without the value of, say, \(U_0 L/ \nu\).
