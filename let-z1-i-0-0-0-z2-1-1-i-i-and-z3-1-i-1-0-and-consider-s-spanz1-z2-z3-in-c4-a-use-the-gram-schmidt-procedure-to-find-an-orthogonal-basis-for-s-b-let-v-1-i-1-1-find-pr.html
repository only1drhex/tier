Guest Question.

Let \(\mathbf{z}_1 = \left [ \begin{matrix} i \\ 0 \\ 0 \\ 0 \end{matrix} \right ] , \mathbf{z}_2 = \left [ \begin{matrix} 1 \\ 1 \\ -i \\ i \end{matrix} \right ]\), and \(\mathbf{z}_3 = \left [ \begin{matrix} 1 \\ i \\ 1 \\ 0 \end{matrix} \right ]\) and consider \(\mathbb{S} = Span \left\{\mathbf{z}_1, \mathbf{z}_2, \mathbf{z}_3\right\}\) in \(\mathbb{C} ^4\).

(a) Use the Gram-Schmidt Procedure to find an orthogonal basis for \(\mathbb{S}\).

(b) Let \(\mathbf{v} = \left [ \begin{matrix} 1 \\ -i \\ 1 \\ -1 \end{matrix} \right ]\). Find \(proj_{\mathbb{S}} (\mathbf{v})\).

Admin
Solution

(a) First Step: Let \( \mathbf{w}_1 = \mathbf{z}_1\) and \(\mathbb{S} = Span \left\{\mathbf{w}_1\right\}\).

Second Step: Determine \(proj_{\mathbb{S}_{1}} (\mathbf{z}_2)\).

\(\mathbf{w}_2 = \mathbf{z}_2 – \frac{\left\langle \mathbf{w}_1, \mathbf{z}_2\right\rangle }{\left\|\mathbf{w}_1\right\|^2 } \mathbf{w}_1 = \left [ \begin{matrix} 1 \\ 1 \\ -i \\ i \end{matrix} \right ] – \frac{-i}{1} \left [ \begin{matrix} i \\ 0 \\ 0 \\ 0 \end{matrix} \right ] = \left [ \begin{matrix} 0 \\ 1 \\ -i \\ i \end{matrix} \right ]\)

Third Step: Determine \(proj_{\mathbb{S}_{2}} (\mathbf{z}_3)\) where \(\mathbb{S}_2 = Span \left\{\mathbf{w}_1, \mathbf{w}_2 \right\}\).

\(proj_{\mathbb{S}_{2}} (\mathbf{z}_3) = \mathbf{z}_3 – \frac{\left\langle \mathbf{w}_1, \mathbf{z}_3\right\rangle }{\left\|\mathbf{w}_1\right\|^2 } \mathbf{w}_1 – \frac{\left\langle \mathbf{w}_2, \mathbf{z}_3\right\rangle }{\left\|\mathbf{w}_2\right\|^2 } \mathbf{w}_2 = \left [ \begin{matrix} 1 \\ i \\ 1 \\ 0 \end{matrix} \right ] + i \left [ \begin{matrix} i \\ 0 \\ 0 \\ 0 \end{matrix} \right ] – \frac{2i}{3} \left [ \begin{matrix} 0 \\ 1 \\ -i \\ i \end{matrix} \right ] = \left [ \begin{matrix} 0 \\ i/3 \\ 1/3 \\ 2/3 \end{matrix} \right ]\)

We take \(\mathbf{w}_3 = \left [ \begin{matrix} 0 \\ i \\ 1 \\ 2 \end{matrix} \right ]\) and get that \(\left\{\mathbf{w}_1, \mathbf{w}_2, \mathbf{w}_3\right\}\) is an orthogonal basis for \(\mathbb{S}\).

(b) Using the orthogonal basis we found in (a), we get

\(proj_{\mathbb{S}} (\mathbf{v}) = \frac{\left\langle \mathbf{w}_1, \mathbf{v}\right\rangle }{\left\|\mathbf{w}_1\right\|^2 } \mathbf{w}_1 + \frac{\left\langle \mathbf{w}_2, \mathbf{v}\right\rangle }{\left\|\mathbf{w}_2\right\|^2 } \mathbf{w}_2 + \frac{\left\langle \mathbf{w}_3, \mathbf{v}\right\rangle }{\left\|\mathbf{w}_3\right\|^2 } \mathbf{w}_3\)

 

\(= -i \left [ \begin{matrix} i \\ 0 \\ 0 \\ 0 \end{matrix} \right ] + \frac {i} {3} \left [ \begin{matrix} 0 \\ 1 \\ -i \\ i \end{matrix} \right ] – \frac {2} {6} \left [ \begin{matrix} 0 \\ i \\ 1 \\ 2 \end{matrix} \right ] = \left [ \begin{matrix} 1 \\ 0 \\ 0 \\ -1 \end{matrix} \right ] \)
