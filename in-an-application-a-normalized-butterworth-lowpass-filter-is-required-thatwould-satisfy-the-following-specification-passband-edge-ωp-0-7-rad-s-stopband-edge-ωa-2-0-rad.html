Guest Question.

In an application a normalized Butterworth lowpass filter is required thatwould satisfy the following specification:

• Passband edge \(\omega_{p}: 0.7\) rad/s

• Stopband edge \(\omega_{a}: 2.0\) rad/s

• Maximum passband loss \(A_{p}: 0.5\) dB

• Minimum stopband loss \(A_{a}: 30.0\) dB

(a) Find the minimum filter order that would satisfy the specifications. (b) Calculate the actual maximum passband loss and minimum stopband loss. (c) Obtain the required transfer function.

Admin
Solution

(a) To ensure that the passband loss is equal to or greater than \(A_{p}: 0.5\) dB the inequality in Eq. (10.20) must be satisfied, i.e.,

 

\(n \geq \frac{\left[-\log \left(10^{0.1 A_{p}}-1\right)\right]}{\left(-2 \log \omega_{p}\right)}\) (10.20)

 

\(n \geq \frac{\left[-\log \left(10^{0.1 A_{p}}-1\right)\right]}{\left(-2 \log \omega_{p}\right)}\)

 

\(\geq \frac{\left[-\log \left(10^{0.1 * 0.5}-1\right)\right]}{(-2 \log 0.7)}\)

 

\(\geq 2.9489 \rightarrow 3\)

 

To ensure that the stopband loss is equal to or greater than dB, \(A_{a}: 30.0\) Eq. (10.21) must be satisfied, i.e.,

 

\(n \geq \frac{\log \left(10^{0.1 A_{a}}-1\right)}{2 \log \omega_{a}}\) (10.21)

 

\(n \geq \frac{\log \left(10^{0.1 A_{a}}-1\right)}{2 \log \omega_{a}}\)

 

\(\geq \frac{\log \left(10^{0.1 * 30}-1\right)}{2 \log 2.0}\)

 

\(\geq 4.9822 \rightarrow 5\)

 

In order to satisfy the passband as well as the stopband specifications, we choose the order n to be the larger of 3 and 5, that is, n = 5.

 

(b) Because of the monotonic increasing nature of the loss of the Butterworth approximation, the actual maximum passband loss occurs at the passband edge. Hence, Eq. (10.18) gives

 

\(A(\omega)=10 \log \left(1+\omega^{2 n}\right)\) (10.18)

 

\(A\left(\omega_{p}\right)=10 \log \left(1+\omega_{p}^{2 n}\right)=10 \log \left(1+0.7^{10}\right)=0.1210 dB\)

 

Similarly, the actual minimum stopband loss occurs at the stopband edge and thus

 

\(A\left(\omega_{a}\right)=10 \log \left(1+\omega_{a}^{2 n}\right)=10 \log \left(1+2.0^{10}\right)=30.11 d B\)

 

The Butterworth method, like the Bessel-Thomson method to follow, yields only one approximation for each filter order and, therefore, the required transfer function is the one found in , part (a).
