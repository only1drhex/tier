Guest Question.

Derive the equations of motion for steady flow of any fluid.

Admin
Solution

Consider as a free body the elementary mass of fluid dM shown in Fig. 7-10(a) and (b). Motion is in the plane of the paper, and the x axis is chosen in the direction of motion. The forces normal to the direction of motion are not shown acting on the free body dM. Forces acting in the x direction are due to (7) the pressures acting on the end areas, (2) the component of the weight, and (3) the shearing forces (\(d F_{s}\) in pounds) exerted by adjacent fluid particles.

\(\left[+p d A-(p+d p) d A-\gamma d A d l \sin \theta_{x}-d F_{s}\right]=\left(\frac{\gamma d A d l}{g}\right)\left(\frac{d V}{d t}\right)\) (1).

\(\text { Dividing }(1) \text { by } \gamma d A \text { and replacing } d l / d t \text { by velocity } V \),

\(\left[\frac{p}{\gamma}-\frac{p}{\gamma}-\frac{d p}{\gamma}-d l \sin \theta_{x}-\frac{d F_{s}}{\gamma d A}\right]=\frac{V d V}{g}\). (2).

\(\text { The term } \frac{d F_{s}}{\gamma d A} \text { represents the resistance to flow in length } d l \text {. The shearing forces } d F_{s} \text { can be replaced }\)\(\text { by the intensity of the shear } \tau \text { times the area over which it acts (perimeter } \times \text { length), or } d F_{s}=\tau d P d l \).

\(\text { Then } \frac{d F_{s}}{\gamma d A}=\frac{\tau d P d l}{\gamma d A}=\frac{\tau d l}{\gamma R} \text {, }\) where R is called the hydraulic radius, which is defined as the crosssectional area divided by the wetted perimeter or, in this case dA/dP. The sum of all the shearing forces is the measure of energy lost due to the flow, and, in ft-lb/lb, is

\(\text { lost head } d h_{L}=\frac{\tau d l}{\gamma R}=\frac{ lb / ft ^{2} \times ft }{ lb / ft ^{3} \times ft ^{2} / ft }= ft\)

For future reference, \(\tau=\gamma R\left(\frac{d h_{L}}{d l}\right)\). (3).

\(\text { Returning to expression (2), since } d l \sin \theta_{x}=d z \text {, it is written in final form as }\)

\(\frac{d p}{\gamma}+\frac{V d V}{g}+d z+d h_{L}=0\) (4).

This expression is known as Euler’s equation when applied to an ideal fluid (lost head = 0). When integrated for fluids of constant density, it is known as the Bernoulli equation. This differential equation (4) for steady flow is a fundamental fluid flow equation.

CASE 1: Flow of Incompressible Fluids

For incompressible fluids, the integration is simple, as follows:

\(\int_{p_{1}}^{p_{2}} \frac{d p}{\gamma}+\int_{V_{1}}^{V_{2}} \frac{V d V}{g}+\int_{z_{1}}^{z_{2}} d z+\int_{1}^{2} d h_{L}=0\) (A).

Methods of evaluating the last term will be discussed in Chapter 8. The total lost head term will be called \(H_{L}\). Integrating and substituting limits,

\(\left(\frac{p_{2}}{\gamma}-\frac{p_{1}}{\gamma}\right)+\left(\frac{V_{2}^{2}}{2 g}-\frac{V_{1}^{2}}{2 g}\right)+\left(z_{2}-z_{1}\right)+H_{L}=0\).

\(\left(\frac{p_{1}}{\gamma}+\frac{V_{1}^{2}}{2 g}+z_{1}\right)-H_{L}=\frac{p_{2}}{\gamma}+\frac{V_{2}^{2}}{2 g}+z_{2}\).

which is the customary form in which the Bernoulli theorem is applied to the flow of incompressible fluids (no external energy added).

CASE 2: Flow of Compressible Fluids

\(\text { For compressible fluids, the term } \int_{p_{1}}^{p_{2}} \frac{d p}{\gamma} \text { cannot be integrated until } \gamma \text { is expressed in terms of variable }\)p. The relationship between y and p will depend upon the thermodynamic conditions involved.

(a) For isothermal (constant temperature) conditions, the general gas law can be expressed as

\(p_{1} / \gamma_{1}=p / \gamma=\text { constant } \quad \text { or } \quad \gamma=\left(\gamma_{1} / p_{1}\right) p\).

\(\text { where } \gamma_{1} / p_{1} \text { is a constant and } p \text { must be in } lb / ft ^{2} \text { or } P a \text { absolute. Substituting in }(A) \text { above. }\)

\(\int_{p_{1}}^{p_{2}} \frac{d p}{\left(\gamma_{1} / p_{1}\right) p}+\int_{V_{1}}^{V_{2}} \frac{V d V}{g}+\int_{z_{1}}^{z_{2}} d z+\int_{1}^{2} d h_{L}=0\).

\(\text { Integrating and substituting limits, } \frac{p_{1}}{\gamma_{1}} \ln \frac{p_{2}}{p_{1}}+\left(\frac{V_{2}^{2}}{2 g}-\frac{V_{1}^{2}}{2 g}\right)+\left(z_{2}-z_{1}\right)+H_{L}=0\),or , converting to the customary form,

\(\frac{p_{1}}{\gamma_{1}} \ln p_{1}+\frac{V_{1}^{2}}{2 g}+z_{1}-H_{L}=\frac{p_{1}}{\gamma_{1}} \ln p_{2}+\frac{V_{2}^{2}}{2 g}+z_{2}\). (B).

Combining this equation with the equation of continuity and the gas law for isothermal conditions yields an expression with only one unknown velocity. Thus, for steady flow,

\(\gamma_{1} A_{1} V_{1}=\gamma_{2} A_{2} V_{2} \text { and } \frac{p_{1}}{\gamma_{1}}=\frac{p_{2}}{\gamma_{2}}=R T, \text { from which } V_{1}=\frac{\gamma_{2} A_{2} V_{2}}{\left(\gamma_{2} / p_{2}\right) p_{1} A_{1}}=\left(\frac{A_{2}}{A_{1}}\right)\left(\frac{p_{2}}{p_{1}}\right) V_{2}\).

Substituting in the Bernoulli form (B) above,

\(\left[\frac{p_{1}}{\gamma_{1}} \ln p_{1}+\left(\frac{A_{2}}{A_{1}}\right)^{2}\left(\frac{p_{2}}{p_{1}}\right)^{2}\left(\frac{V_{2}^{2}}{2 g}\right)+z_{1}\right]-H_{L}=\left[\frac{p_{1}}{\gamma_{1}} \ln p_{2}+\frac{V_{2}^{2}}{2 g}+z_{2}\right]\) (C).

(b) For adiabatic (no heat gained or lost) conditions, the general gas law reduces to

\(\left(\frac{\gamma}{\gamma_{1}}\right)^{k}=\frac{p}{p_{1}} \text { or } \frac{p_{1}^{1 / k}}{\gamma_{1}}=\frac{p^{1 / k}}{\gamma}=\text { constant, } \quad \text { and thus } \gamma=\gamma_{1}\left(\frac{p}{p_{1}}\right)^{1 / k}\).

where k is the adiabatic exponent.

\(\text { Establishing and integrating the term } d p / \gamma \text { separately, we obtain }\).

\(\int_{p_{1}}^{p_{2}} \frac{d p}{\gamma_{1}\left(p / p_{1}\right)^{1 / k}}=\frac{p_{1}^{1 / k}}{\gamma_{1}} \int_{p_{1}}^{p_{2}} \frac{d p}{p^{1 / k}}=\left(\frac{k}{k-1}\right)\left(\frac{p_{1}}{\gamma_{1}}\right)\left[\left(\frac{p_{2}}{p_{1}}\right)^{(k-1) / k}-1\right]\).

and the Bernoulli equation in customary form becomes

\(\left[\left(\frac{k}{k-1}\right)\left(\frac{p_{1}}{\gamma_{1}}\right)+\frac{V_{1}^{2}}{2 g}+z_{1}\right]-H_{L}=\left[\left(\frac{k}{k-1}\right)\left(\frac{p_{1}}{\gamma_{1}}\right)\left(\frac{p_{2}}{p_{1}}\right)^{(k-1) / k}+\frac{V_{2}^{2}}{2 g}+z_{2}\right]\) (D).

Combining this equation with the equation of continuity and the gas law for adiabatic conditions yields an expression with only one unknown velocity.

\(\text { Using } \gamma_{1} A_{1} V_{1}=\gamma_{2} A_{2} V_{2} \text { and } \frac{p_{1}^{1 / k}}{\gamma_{1}}=\frac{p_{2}^{1 / k}}{\gamma_{2}}=\text { constant, } V_{1}=\frac{\gamma_{2} A_{2} V_{2}}{\gamma_{1} A_{1}}=\left(\frac{p_{2}}{p_{1}}\right)^{1 / k}\left(\frac{A_{2}}{A_{1}}\right) V_{2} \text {, }\) and the Bernoulli equation becomes

\(\left[\left(\frac{k}{k-1}\right)\left(\frac{p_{1}}{\gamma_{1}}\right)+\left(\frac{p_{2}}{p_{1}}\right)^{2 / k}\left(\frac{A_{2}}{A_{1}}\right)^{2}\left(\frac{V_{2}^{2}}{2 g}\right)+z_{1}\right]-H_{L}\)\(=\left[\left(\frac{k}{k-1}\right)\left(\frac{p_{1}}{\gamma_{1}}\right)\left(\frac{p_{2}}{p_{1}}\right)^{(k-1) / k}+\frac{V_{2}^{2}}{2g}+z_{2}\right] \) (E).
