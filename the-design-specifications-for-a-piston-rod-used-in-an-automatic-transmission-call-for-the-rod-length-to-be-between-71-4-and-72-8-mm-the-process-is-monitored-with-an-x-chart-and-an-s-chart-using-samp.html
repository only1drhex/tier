Guest Question.

The design specifications for a piston rod used in an automatic transmission call for the rod length to be between \(71.4\) and \(72.8 \mathrm{~mm}\). The process is monitored with an \(\bar{X}\) chart and an \(S\) chart, using samples of size \(n=5\). These show the process to be in control. The values of \(\overline{\bar{X}}\) and \(\bar{s}\) are \(\overline{\bar{X}}=71.8 \mathrm{~mm}\) and \(\bar{s}=0.20 \mathrm{~mm}\). Compute the value of \(C_{p k}\). Is the process capability acceptable?

Admin
Solution

We estimate \(\widehat{\mu}=\overline{\bar{X}}=71.8\). To compute \(\widehat{\sigma}\), we find, from Table A.10, that \(c_{4}=\) \(0.9400\) when the sample size is 5 . Therefore \(\widehat{\sigma}=\bar{s} / c_{4}=0.20 / 0.9400=0.2128\). The specification limits are LSL \(=71.4 \mathrm{~mm}\) and USL \(=72.8 \mathrm{~mm}\). The value \(\widehat{\mu}\) is closer to the LSL than to the USL. Therefore

\(\begin{aligned}C_{p k} &=\frac{\widehat{\mu}-\mathrm{LSL}}{3 \widehat{\sigma}}=\frac{71.8-71.4}{(3)(0.2128)} \\ &=0.6266 \end{aligned}\)

Since \(C_{p k}<1\), the process capability is not acceptable.
