Guest Question.

Consider the following circuit. Find the power of the top 5 Œ© resistor if \(v_{y} = 20V\).

Admin
Solution

Using mesh analysis, we define five mesh currents.

With large circuits of this type, it can be easy to get lost. In order to analyze the circuit systematically, we first consider the given real currents to find relationships between the mesh currents.We have
‚Ä¢ \(i_{x}= i_{b}‚àí i_{a}\),
‚Ä¢ \(i_{e}= 4i_{x} = 4i_{b} ‚àí 4i_{a}\),
‚Ä¢ \(i_{b} ‚àíi_{c} = 2\).
Then, applying KVL in mesh a, we derive
‚Ä¢ KVL(a): \(‚àí20 + 5(i_{a}‚àí i_{d}) + 5(i_{a}‚àí i_{b}) = 0\longrightarrow 2i_{a} ‚àí i_{b} ‚àí i_{d}= 4\).
None of the pairs of equations above are solvable. As usual, the required information is available in a supermesh:
‚Ä¢ KVL(b&c): \(5(i_{b}‚àí i_{a}) + 5(i_{c}‚àí i_{e}) + 20 + 5i_{c} + 5i_{b} = 0 \longrightarrow i_{a} ‚àí 2i_{b} ‚àí 2i_{c} + i_{e}= 4\).

Substituting \(i_{e}\) in the above, we get
\(i_{a}‚àí 2i_{b} ‚àí 2i_{c} + 4i_{b} ‚àí 4i_{a} = ‚àí3i_{a} + 2i_{b} ‚àí 2i_{c} = 4\).
Furthermore, using \(i_{b}‚àí i_{c} = 2\), we obtain \(i_{a}= 0\). In order to obtain \(i_{d}\), we consider KVL in mesh d,
‚Ä¢ KVL(d): \(5i_{d} ‚àí v_{y} + 5(i_{d}‚àí i_{a}) = 0 \longrightarrow i_{d}= 2A\).
Hence, \(i_{b}= 2i_{a} ‚àí i_{d} ‚àí 4 = ‚àí6A and i_{c} = i_{b}‚àí 2 = ‚àí8A\).

We note that finding \(i_{b} and i_{c} \)requires extra calculations, since the power of the top 5 Œ© resistor can be found via \(i_{d}\)as
\(p_{5 Œ©} = 5i^{2}_{d} = 20W\).
