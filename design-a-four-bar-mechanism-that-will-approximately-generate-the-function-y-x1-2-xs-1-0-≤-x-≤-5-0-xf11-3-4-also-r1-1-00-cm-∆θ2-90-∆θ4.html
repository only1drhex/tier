Guest Question.

Design a four-bar mechanism that will approximately generate the function

\(Y = X^{1/2}, X_{s} = 1.0 ≤ X ≤ 5.0 = X_{f}\) (11.3-4)

Also

\(r_{1} = 1.00 cm; ∆θ_{2} = -90° ; ∆θ_{4} = -40°\) (11.3-5)

Use Chebyshev spacing of three precision points.

Admin
Solution

This example specifies the motions of links 2 and 4. However, there is no stipulation of the starting angular configurations of these links. Therefore, we are free to select

\( (θ_{2})_{s} = 120°; (θ_{4})_{s} = 100°\) (11.3-6)

Using Equations (11.2-11) \(X_{1} = \frac{X_{f} + X_{s}}{2} – \frac{X_{f} – X_{s}}{2} \cos 30 = 0.933 X_{s} + 0.0670 X_{f}\), \(X_{2} = \frac{X_{f} + X_{s}}{2} = 0.500 X_{s} + 0.500 X_{f}\), \(X_{3} = \frac{X_{f} + X_{s}}{2} + \frac{X_{f} – X_{s}}{2} \cos 30 = 0.0670 X_{s} + 0.933 X_{f}\), we obtain

\( X_{1} = 0.933 X_{s} + 0.0670 X_{f} \)

 

\(= 0.933 × 1.0 + 0.0670 ×5.0 = 1.268\)

 

\( X_{2} = 3.000; X_{3} = 4.732 \) (11.3-7)

Also, from Equation (11.2-6)\(∆X = X_{f} – X_{s}; \Delta \theta _{2} = (\theta _{2})_{f} – (\theta _{2})_{s})\) ; \(∆Y = Y_{f} – Y_{s}; \Delta \theta _{4} = (\theta _{4})_{f} – (\theta _{4})_{s}\) we get

\(∆X = X_{f} – X_{s} = 4.000\) (11.3-8)

Then from Equations (11.2-7) \(\frac{\theta _{2} – (\theta _{2})_{s}}{X – X_{s}} = \frac{\Delta \theta _{2}}{∆X} = r_{X}\), (11.2-12) \((\theta _{2})_{j} = r_{X}(X_{j} – X_{s}) + (\theta _{2})_{s}\), \( j = 1,2,3 \), (11.3-7), and (11.3-8) we have

\( (θ_{2})_{1} = (θ_{2})_{s} + \frac{X_{1} – X_{s}}{∆X} ∆θ_{2} \)

 

\(= 120° + \frac{1.268 – 1.000}{4.000} (- 90°) = 113.97°\)

 

\( (θ_{2})_{2} = 75.00° ; (θ_{2})_{3} = 36.03° \) (11.3-9)

Using Equation (11.3-4) \(Y = X^{1/2}, X_{s} = 1.0 ≤ X ≤ 5.0 = X_{f}\) , the values of Y at the precision points are

\(Y_{j} = X_{j}^{1/2} , j =1 , 2 , 3\)

 

\(Y_{1} = X_{1}^{1/2} = 1.126; Y_{2} = 1.732; Y_{3} = 2.175\) (11.3-10)

Also

\(∆Y = Y_{f} – Y_{s} = X_{f}^{1/2} – X_{s}^{1/2} = 1.236\) (11.3-11)

Combining Equations (11.2-13) \((\theta _{4})_{j} = r_{Y}(Y_{j} – Y_{s}) + (\theta _{4})_{s}, j = 1,2,3\)
, (11.3-10), and (11.3-11), we obtain

\( (θ_{4})_{1} = (θ_{4})_{s} + \frac{Y_{1} – Y_{s}}{∆Y} ∆θ_{4} \)

 

\(= 100° + \frac{1.126 – 1.000}{1.236} (- 40°) = 95.92°\)

 

\( (θ_{4})_{2} = 76.31° ; (θ_{4})_{3} = 61.97° \) (11.3-12)

Substituting the values from Equations (11.3-9) and (11.3-12) in Equation (11.3-2) \(h_{1} \cos (\theta _{4})_{i} – h_{3} \cos (\theta _{2})_{i} + h_{5} = \cos ((\theta _{2})_{i} – (\theta _{4})_{i} )\) , \( i = 1 , 2 , 3\) , we get

\(\left[\begin{matrix} -0.103 & 0.406 & 1.000 \\ 0.237 & -0.259 & 1.000 \\ 0.470 & -0.809 & 1.000 \end{matrix}\right] \left\{\begin{matrix} h_{1} \\ h_{3} \\ h_{5} \end{matrix} \right\} = \left\{\begin{matrix} 0.951 \\ 1.000 \\ 0.899 \end{matrix} \right\}\) (11.3-13)

Solving for the unknowns, we obtain

\(h_{1} = 2.959; h_{3} = 1.438; h_{5} = 0.672\) (11.3-14)

Substituting Equations (11.3-14) in Equations (11.3-3) \(r_{2} = \frac{r_{1}}{h_{1}}\) ; \(r_{4} = \frac{r_{1}}{h_{3}}\) ; \( r_{3} = (r_{1}^{2} + r_{2}^{2} + r_{4}^{2} – 2 r_{2} r_{4} h_{5})^{1/2}\), the lengths of the links are

\(r_{1} = 0.338 cm; r_{3} = 1.132 cm; r_{4} = 0.695\)

Figure 11.10 shows the mechanism in three configurations corresponding to the precision points.
Having the link lengths of the four-bar mechanism, we may now evaluate the function that would actually be produced by the mechanism and then determine the structural error as a function of the rotation of link 2. For instance, consider the mechanism with

\( θ_{2} = (θ_{2})_{s} = 120°\)

Using Equations (4.3-56) \( d= -h_{1} + (1 – h_{3}) \cos \theta _{2} + h_{5} \) –(4.3-59) \(\theta _{4} = 2 \tan ^{-1} \left[\frac{-b \pm (b^{2} – 4 d e)^{1/2}}{2d} \right]\)
, gives

\((θ_{4})_{@ θ_{2} = 120° } = 98.902°\)

which does not match the prescribed functional value of

\( (θ_{4})_{s} = 100°\)

That is, in this configuration, using Equation (11.2-3) \(e(\theta _{2} ,r_{1},r_{2},r_{3},r_{4}) = g(\theta _{2} ,r_{1},r_{2},r_{3},r_{4}) – f_{\theta _{2}}(\theta _{2})\), the structural error of the angular dis-placement of link 4 is

\( e_{@ θ_{2} = 120° } = ( θ_{4} – (θ_{4})_{s})_{@ θ_{2} = 120° } = 98.902° – 100° = -1.098° \)

The non-dimensional structural error at \(X = X_{s} = 1.0\) is then

\((e(X))_{@ X = 1} = ( θ_{4} – (θ_{4})_{s})_{@ θ_{2} = 120° } \frac{∆Y}{ ∆θ_{4} } = -1.098° \frac{1.236}{-40° } = 0.034\)

Similar calculations may be completed for other configurations of the mechanism. Combining these results, Figure 11.11(a) shows plots of the nondimensional desired function and that produced by the mechanism in the required range of motion. A plot of the nondimensional structural error is provided in Figure 11.11(b).

Other synthesized mechanisms can be produced by selecting alternative starting angular configurations of links 2 and 4. If we select

\( (θ_{2})_{s} = 160°; (θ_{4})_{s} = 90°\) (11.3-15)

and reproduce the above procedure, the results are

\(r_{2} = 1.685 cm; r_{3} =6.375 cm; r_{4} = -5.279\)

The corresponding mechanism and function graph are shown in Figures 11.12 and 11.13. Note in this case that the value of structural error has changed. Although the structural error is reduced compared to the first solution, a designer may not wish to incorporate the second solution because the link lengths are greater and the transmission angle is smaller.
