Guest Question.

What flow can be expected in a 4-ft-wide rectangular cement-lined channel laid on a slope of 4 ft in 10,000 ft, if the water flows 2 ft deep? Use both Kutter’s C and Manning’s C.

Admin
Solution

(a) Using Kutter’s C: From , n = 0.015. Hydraulic radius R = (4)(2)/8 = 1.00 ft.

From , for S = 0.0004, R = 1.00, and n = 0.015, the value of C = 98.

\(Q=A V=A C \sqrt{R S}=(4 \times 2)(98) \sqrt{1.00 \times 0.0004}=15.7 cfs\).

(b) Using Manning’s C:

\(Q=A V=A \frac{1.486}{n} R^{2 / 3} S^{1 / 2}=(4 \times 2)\left(\frac{1.486}{0.015}\right)(1.00)^{2 / 3}(0.0004)^{1 / 2}=15.9 cfs\)
