Guest Question.

Consider the same two alternatives of as given below:

Alternative A Alternative B
First cost \((₹)\) 60,000 2,00,000
Annual maintenance & operation cost \((₹ / \text { year })\) 11,000 5,000
Salvage value at the end of life \((₹)\) 0 50,000
Useful life (years) 10 25
M.A.R.R. 15% 15%
Admin
Solution

Solution of Example 6.3 by the Present Worth Cost (P.W.C.) Method Since the study period is specified as 10 years, the lives of both alternatives will be taken as 10 years. The P.W.C. of the alternatives is calculated as:

P.W.C. of alternative A is determined as:

 

\(\begin{array}{l}\text { (P.W.C.) }_{A}=-₹ 60,000-₹ 11,000(P / A, 15 \%, 10) \\\text { (P.W.C.) }_{A}=-₹ 60,000-₹ 11,000(5.0188) \\\text { (P.W.C.) }_{A}=-₹ 1.15,206.8\end{array}\)

 

P.W.C. of alternative B is determined as:

 

\(\begin{array}{l}\text { (P.W.C.) }_{ B }=-₹ 2,00,000-₹ 5,000(P / A, 15 \%, 10)+₹ 1,25,000(P / F, 15 \%, 10) \\\text { (P.W.C.) }_{ B }=-₹ 2,00,000-₹ 5,000(5.0118)+₹ 1,25,000(0.2472) \\\text { (P.W.C.) }_{ B }=-₹ 1,94,194\end{array}\)

 

On comparing \(\text { (P.W.C.) }_{ A } \text { with (P.W.C.) }_{ B }\), it is found that \(\text { (P.W.C.) }_{A}\) is smaller than \(\text { (P.W.C.) }_{ B }\) and, therefore, alternative A should be chosen.

Solution of Example 6.3 by the Future Worth Cost (F.W.C.) Method F.W.C. of alternative A is determined as:

 

\(\begin{array}{l}\text { (F.W.C.) }_{A}=-₹ 60,000(F / P, 15 \%, 10)-₹ 11,000(F / A, 15 \%, 10) \\\text { (F.W.C.) }_{A}=-₹ 60,000(4.0456)-₹ 11,000(20.3037) \\\text { (F.W.C.) }_{A}=-₹ 4,66,076.7\end{array}\)

 

F.W.C. of alternative B is determined as:

 

\(\text { (F.W.C.) }_{\text {B }}=-₹ 2,00,000(F / P, 15 \%, 10)-₹ 5,000(F / A, 15 \%, 10)+₹ 1,25,000\)

 

\(\text { (F.W.C.) }_{ B }=-₹ 2,00,000(4.0456)-₹ 5,000(20.3037)+₹ 1,25,000\)

 

\(\text { (F.W.C.) }_{ B }=-₹ 7,85,638.5\)

 

On comparing \(\text { (F.W.C.) }_{A}\) with \(\text { (F.W.C.) }_{B}\) it is found that \(\text { (F.W.C.) }_{A}\) is smaller than \(\text { (F.W.C.) }_{B}\) therefore, alternative A should be chosen.

Solution of Example 6.3 by the Annual Cost (A.C.) Method A.C. of alternative A is determined as:

 

\(\begin{array}{l}\text { (A.C.) }_{A}=-₹ 60,000(A / P, 15 \%, 10)-₹ 11,000 \\\text { (A.C.) }_{A}=-₹ 60,000(0.19925)-₹ 11,000 \\\text { (A.C.) }_{A}=-₹ 22,955\end{array}\)

 

A.C. of alternative B is determined as:

 

\(\text { (A.C.) }_{ B }=-₹ 2,00,000(A / P, 15 \%, 10)-₹ 5,000+₹ 1,25,000(A / F, 15 \%, 10)\)

 

\(\begin{array}{l}\text { (A.C.) }_{ B }=-₹ 2,00,000(0.1993)-₹ 5,000+₹ 1,25,000(0.0493) \\\text { (A.C.) }_{B}=-₹ 51,022.5\end{array}\)

 

On comparing \(\text { (A.C.) }_{A}\) with \((\text { A.C. })_{B}\), it is found that \(\text { (A.C.) }_{A}\) is smaller than \(\text { (A.C.) }_{B}\) and therefore, alternative A should be chosen.

Solution of Example 6.3 by the Internal Rate of Return (I.R.R.) Method Let us compute I.R.R. by finding A.C. for the two alternatives at an unknown\(i^{\prime}\) and equating them.

 

\(-₹ 60,000\left(A / P, i^{\prime}, 10\right)-₹ 11,000=-₹ 2,00,000\left(A / P, i^{\prime}, 10\right)-₹ 5,000+₹ 1,25,000\left(A / F, i^{\prime}, 10\right)\)

 

Linear interpolation between 3% and 4% results in \(i^{\prime}\) = 3.37%, which is the I.R.R. Since 3.37% < 15%, the increment \(A \rightarrow B\) is not justified. Thus, alternative A is better than alternative B and it should be selected.

Solution of Example 6.3 by the External Rate of Return (E.R.R.) Method with e = 15%

 

Alternative A Alternative B
Investment \((₹)\) 60,000 2,00,000
FW of annual disbursements \((₹)\):\(\begin{array}{l}₹ 11,000(F / A, 15 \%, 10) \\₹ 5,000(F / A, 15 \%, 10)\end{array}\) 2,23,340.7 1,01,518.5
FW of Salvage \((₹)\) 1,25,000
Increment considered:
Δ Investment \((₹)\) 1,40,000
Δ FW of savings \((₹)\) (2,23,340.7 – 1,01,518.5) + 1,25,000 = 2,46,822.2
E.R.R. on Δ Investment 1,40,000(F/P, i’, 10) = 2,46,822.2 which yields i’ = 5.83%

 

Since E.R.R. (5.83%) < 15%, the increment \(A \rightarrow B \) is not justified. Thus, alternative A is better than alternative B and it should be selected.

Solution of Example 6.3 by the Explicit Reinvestment Rate of Return (E.R.R.R.) Method with e = 15%

 

Alternative A Alternative B
Annual expenses \(\text { (₹): }\) Disbursements 11,000 5,000
Depreciation \(\text { (₹): }\) \(\begin{array}{l}₹ 60,000(A / F, 15 \%, 10) \\₹(2,00,000-1,25,000)(A / F , 15 \%, 10)\end{array}\) 2,955 3,698
Total annual expenses \(\text { (₹): }\) 11,000 + 2955 = 13,955 5,000 + 3,697.5 = 8,697.5
Δ Annual expenses (annual savings) \(\text { (₹): }\) 13,955 – 8,697.5 = 5,257.5
Δ Investment \(\text { (₹): }\) 2,00,000 – 60,000 = 1,40,000
E.R.R.R. on Δ Investment 5,257.5/1,40,000 = 3.75%

 

Since E.R.R.R. (3.75%) < 15%, the increment \(A \rightarrow B \) is not justified. Thus, alternative A is better than alternative B and it should be selected.
