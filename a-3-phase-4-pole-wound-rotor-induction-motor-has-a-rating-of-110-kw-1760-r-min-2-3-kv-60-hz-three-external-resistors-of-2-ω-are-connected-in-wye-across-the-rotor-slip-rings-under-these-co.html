Guest Question.

A 3-phase, 4-pole, wound-rotor induction motor has a rating of 110 kW, 1760 r/min, 2.3 kV, 60 Hz. Three external resistors of 2 Ω are connected in wye across the rotor slip-rings. Under these conditions the motor develops a torque of 300 Nm at a speed of 1000 r/min.
a. Calculate the speed for a torque of 400 Nm.
b. Calculate the value of the external resistors so that the motor develops 10 kW at 200 r/min.

Admin
Solution

a. The given conditions are

 

\(\begin{aligned}T_{ n } &=300 N \cdot m \\s_{ n } &=(1800-1000) / 1800=0.444\end{aligned}\)

 

All other conditions being fixed, we have for a torque of 400 Nm the following:

 

\(\begin{aligned}s_{ x } &=s_{ n }\left(T_{ x } / T_{ n }\right)=0.444(400 / 300) \\&=0.592\end{aligned}\)

 

The slip speed \(=0.592 \times 1800=1066 r / min\)

Consequently, the motor speed is

 

\(\begin{aligned}n &=1800-1066 \\&=734 r / min\end{aligned}\)

 

Note that the speed drops from 1000 r/min to 734 r/min with increasing load.

 

 

b. The torque corresponding to 10 kW at 200 r/min is

 

\(\begin{aligned}T_{ x } &=9.55 P / n \\&=9.55 \times 10000 / 200 \\&=478 N \cdot m\end{aligned}\)

 

The rated torque is

 

\(\begin{aligned}T_{\text {rated }} &=9.55 P / n \\&=9.55 \times 110000 / 1760 \\&=597 N \cdot m\end{aligned}\)

 

Because \(T_{ x } \text { is less than } T_{\text {rated }} \text {, }\) we can apply Eq. 2. The slip is

 

\(s_{ x }=s_{ n }\left[\frac{T_{ x }}{T_{ n }}\right]\left[\frac{R_{ x }}{R_{ n }}\right]\left[\frac{E_{ n }}{E_{ x }}\right]^{2}\) (Equation 2)

 

\(s_{ x }=(1800-200) / 1800=0.89\)

 

All other conditions being fixed, we have, from Eq. 2

 

\(\begin{aligned}s_{ x } &=s_{ n }\left(T_{ x } / T_{ n }\right)\left(R_{ x } / R_{ n }\right) \\0.89 &=0.44(478 / 300)\left(R_{ x } / 2\right)\end{aligned}\)

 

and so

 

\(R_{ x }=2.5 \Omega\)

 

Three 2.5 Ω , wye-connected resistors in the rotor circuit will enable the motor to develop 10 kW at 200 r/min.
