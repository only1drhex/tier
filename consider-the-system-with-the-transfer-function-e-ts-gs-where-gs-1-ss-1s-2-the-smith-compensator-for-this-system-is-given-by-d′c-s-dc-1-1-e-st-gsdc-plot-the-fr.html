Guest Question.

\(\blacktriangle \) Consider the system with the transfer function \(e^{-T s} G(s)\), where,

\(G(s)=\frac{1}{s(s+1)(s+2)}\) .

The Smith compensator for this system is given by

\(D_{c}^{\prime}(s)=\frac{D_{c}}{1+\left(1-e^{-s T}\right) G(s) D_{c}}\) .

Plot the frequency response of the compensator for T = 5 and \(D_{c} = 1\), and draw a Bode plot that shows the gain and phase margins of the system.²

²This problem was given by Åström (1977).

Admin
Solution

This problem can be solved using a few different approaches. A computer tool such as MATLAB (see the MATLAB command pade) can be used to calculate the exact magnitude and phase of \(D_{c}^{\prime}(s)\), or a pade approximation can be made for the \(e^{-s T}\) terms,

\(e^{-s T} \approx \frac{2-s T+(-s T)^{2} / 2 !+(-s T)^{3} / 3 !+\ldots}{2+s T+(s T)^{2} / 2 !+(s T)^{3} / 3 !+\ldots}\).

We will show both the exact calculation and Bode plots using a fourth-order pade approximation. The Bode plot of the compensator

\(D_{c}(s)=\frac{1}{1+\left[1-e^{-s T}\right] G(s)}\),

appears in the figures, with,

\(G(s)=\frac{1}{s(s+1)(s+2)}, \text { and } T=5\).

The Bode plot of the closed-loop system is also shown.

Remark: Note that the Smith compensator, \(D_{c}^{\prime}(s)\), is structured such that the closed-loop transfer function is,

\(\mathcal T(s)=\frac{G(s)}{1+G(s) D_{c}(s)} e^{-T s}\).
