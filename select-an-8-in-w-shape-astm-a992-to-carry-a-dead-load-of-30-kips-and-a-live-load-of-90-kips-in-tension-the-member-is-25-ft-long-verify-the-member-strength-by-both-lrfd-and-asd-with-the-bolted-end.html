Guest Question.

Select an 8 in. W-shape, ASTM A992, to carry a dead load of 30 kips and a live load of 90 kips in tension. The member is 25 ft long. Verify the member strength by both LRFD and ASD with the bolted end connection shown. Verify that the member satisfies the recommended slenderness limit.

Admin
Solution

Calculate the required tensile strength

LRFD ASD
\(P_{u}=1.2(30 kips )+1.6(90 kips )\)

= 180 kips

 \(P_{a}=30 \text { kips }+90 \text { kips }\)

= 120 kips

Try a W8×21

Material Properties:

\(\begin{array}{llll} W 8 \times 21 & \text { ASTM A992 } & F_{y}=50 ksi & F_{u}=65 ksi\end{array}\) Manual Table 2-3

Geometric Properties:

\(\begin{array}{llll} W 8 \times 21 & A_{g}=6.16 in .{ }^{2} & b_{f}=5.27 in . & t_{f}=0.400 \text { in. } & d=8.28 in .\end{array}\) \(r_{y}=1.26 in\)

\(\bar{y}=0.831 \text { in. }(\text { for } W T 4 \times 10.5)\) Manual Table 1-1 Table 1-8

Check tensile yield limit state using tabulated values

LRFD ASD
277 kips > 180 kips o.k. 184 kips > 120 kips o.k.

Manual Table 5-1

Check the available tensile rupture strength at the end connection

Verify the table assumption that \(A_{e} / A_{g} \geq 0.75\) for this connection

Calculate U as the larger of the values from Table D3.1 case 2 or case 7

Case 2 – Check as 2 WT-shapes Commentary Fig. C-D3.1

\(U=1-\frac{\bar{x}}{l} \quad=1-\frac{0.831 in .}{9.00 in .}=0.908\) Table D3.1 Case 2

Case 7

\(b_{f}=5.27 \text { in. } \quad d=8.28 \text { in. } b_{f}<2 / 3 d\)

\(U=0.85\) Table D3.1 Case 7

Use U = 0.908

Calculate \(A_{n}\) Section D3.2

\(A_{n}=A_{g}-4\left(d_{h}+1 / 16 \text { in. }\right) t_{f}\) \(=6.16 in ^{2}-4(13 / 16 in .+1 / 16 in .)(0.400 in .)=4.76 in .^{2}\)

\(\text { Calculate } A_{e}\) Section D3.3

\(A_{e}=A_{n} U\)

\(=4.76 in .^{2}(0.908)=4.32 in .^{2}\) Eqn. D3-1

\(A_{e} / A_{g}=4.32 in ^{2} / 6.16 in .^{2}=0.701<0.75\) tabulated values for rupture n.a.

\(P_{n}=F_{u} A_{e}=(65 ksi )\left(4.32 in ^{2}\right)=281 \text { kips }\) Eqn. D2-2

LRFD ASD
\(\phi_{t}=0.75\) \(\Omega_{t}=2.00\)
\(\phi_{t} P_{n}=0.75(281 kips )=211 \text { kips }\) \(P_{n} / \Omega_{t}=(281 \text { kips }) / 2.00=141 \text { kips }\)
211 kips > 180 kips o.k. 141 kips > 120 kips o.k.
