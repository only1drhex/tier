Guest Question.

GEAR SPECIFICATION

Continue the case study by specifying appropriate gears, including pitch diameter, diametral pitch, face width, and material. Achieve safety factors of at least 1.2 for wear and bending.

Admin
Solution

Estimate the minimum diametral pitch for overall gearbox height = 22 in.

From Eq. (18–3) and Fig. 18–1,

 

\(P=\left(N_{3}+N_{2} / 2+N_{5} / 2+2\right) /(Y-\text { clearances }-\text { wall thicknesses })\) (18–3)

 

\(P_{\min }=\frac{\left(N_{3}+\frac{N_{2}}{2}+\frac{N_{5}}{2}+2\right)}{(Y-\text { clearances }-\text { wall thickness })}\)

 

Allow 1.5 in for clearances and wall thicknesses:

 

\(P_{\min }=\frac{\left(72+\frac{16}{2}+\frac{72}{2}+2\right)}{(22-1.5)}=5.76 \text { teeth } / \text { in }\)

 

Start with P = 6 teeth/in

 

\(d_{2}=d_{4}=N_{2} / P=16 / 6=2.67 \text { in }\)

 

\(d_{3}=d_{5}=72 / 6=12.0 \text { in }\)

 

Shaft speeds were previously determined to be

 

\(\omega_{2}=1750 rev / min \quad \omega_{3}=\omega_{4}=388.9 rev / min \quad \omega_{5}=86.4 rev / min\)

 

Get pitch-line velocities and transmitted loads for later use.

Eq. (13–34), p. 707

 

\(V=\pi d n / 12\) (13–34)

 

\(V_{23}=\frac{\pi d_{2} \omega_{2}}{12}=\frac{\pi(2.67)(1750)}{12}=1223 ft / min\)

 

\(V_{45}=\frac{\pi d_{5} \omega_{5}}{12}=271.5 ft / min\)

 

Eq. (13–35), p. 707

 

\(W_{t}=33000 \frac{H}{V}\) (13–35)

 

\(W_{23}^{t}=33000 \frac{H}{V_{23}}=33000\left(\frac{20}{1223}\right)=\underline{540.0 lbf }\)

 

\(W_{45}^{t}=33000 \frac{H}{V_{45}}=\underline{2431 lbf }\)

 

Start with gear 4, since it is the smallest gear, transmitting the largest load. It will likely be critical. Start with wear by contact stress, since it is often the limiting factor.

Gear 4 Wear

Eq. (14–23), p. 755

 

\(I=\left\{\begin{array}{ll}\frac{\cos \phi_{t} \sin \phi_{t}}{2 m_{N}} \frac{m_{G}}{m_{G}+1} & \text { external gears } \\\frac{\cos \phi_{t} \sin \phi_{t}}{2 m_{N}} \frac{m_{G}}{m_{G}-1} & \text { internal gears }\end{array}\right.\) (14–23)

 

\(I=\frac{\cos 20^{\circ} \sin 20^{\circ}}{2(1)}\left(\frac{4.5}{4.5+1}\right)=0.1315\)

 

Eq. (14–29), p. 756

 

\(\left(V_{t}\right)_{\max }=\left\{\begin{array}{ll}{\left[A+\left(Q_{v}-3\right)\right]^{2}} & ft / min \\\frac{\left[A+\left(Q_{v}-3\right)\right]^{2}}{200} & m / s\end{array}\right.\) (14–29)

 

For \(K_{v} \text {, assume } Q_{v}\) = 7. B = 0.731, A = 65.1

Eq. (14–27), p. 756

 

\(K_{v}=\left\{\begin{array}{ll}\left(\frac{A+\sqrt{V}}{A}\right)^{B} & V \text { in } ft / min \\\left(\frac{A+\sqrt{200 V}}{A}\right)^{B} & V \text { in } m / s\end{array}\right.\) (14–27)

 

\(K_{v}=\left(\frac{65.1+\sqrt{271.5}}{65.1}\right)^{0.731}=1.18\)

 

Face width F is typically from 3 to 5 times circular pitch. Try

 

\(F=4\left(\frac{\pi}{P}\right)=4\left(\frac{\pi}{6}\right)=2.09 \text { in }\)

 

Since gear specifications are readily available on the Internet, we might as well check for commonly available face widths. On www.globalspec.com, entering P = 6 teeth/in and d = 2.67 in, stock spur gears from several sources have face widths of 1.5 in or 2.0 in. These are also available for the meshing gear 5 with d = 12 in.

Choose F = 2.0 in.

Eq. (14–32), p. 760

 

\(C_{p f}=\left\{\begin{array}{ll}\frac{F}{10 d}-0.025 & F \leq 1 \text { in } \\\frac{F}{10 d}-0.0375+0.0125 F & 1<F \leq 17 \text { in } \\\frac{F}{10 d}-0.1109+0.0207 F-0.000228 F^{2} & 17<F \leq 40 \text { in }\end{array}\right.\) (14–32)

 

\(\text { For } K_{m}, \quad C_{p f}=0.0624\)

 

Eq. (14–31), p. 760

 

\(C_{m c}=\left\{\begin{array}{ll}1 & \text { for uncrowned teeth } \\0.8 & \text { for crowned teeth }\end{array}\right.\) (14–31)

 

\(C_{m c}=1 \text { uncrowned teeth }\)

 

Eq. (14–33), p. 760

 

\(C_{p m}=\left\{\begin{array}{ll}1 & \text { for straddle-mounted pinion with } S_{1} / S<0.175 \\1.1 & \text { for straddle-mounted pinion with } S_{1} / S \geq 0.175\end{array}\right.\) (14–33)

 

\(C_{p m}=1 \text { straddle-mounted }\)

\(\sigma_{c, \text { all }}=\left\{\begin{array}{ll}\frac{S_{c}}{S_{H}} \frac{Z_{N} C_{H}}{K_{T} K_{R}} & \text { (U.S. customary units) } \\\frac{S_{c}}{S_{H}} \frac{Z_{N} Z_{W}}{Y_{\theta} Y_{Z}} & \text { (SI units) }\end{array}\right.\) (14–18)

\(S_{c}=\frac{S_{H} \sigma_{c}}{Z_{N}}=\frac{1.2(161700)}{0.9}=215600 psi\)
