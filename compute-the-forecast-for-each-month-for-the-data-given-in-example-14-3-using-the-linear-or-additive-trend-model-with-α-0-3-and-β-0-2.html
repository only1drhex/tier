Guest Question.

Compute the forecast for each month for the data given in using the linear or additive trend model with \(\alpha=0.3 \text { and } \beta=0.2\).

Admin
Solution

In order to forecast the demand for Jan i.e. FJan, we need to know the values of the base St and trend Tt for one period before (Initial) i.e. \(S_{\text {Initial }} \text { and } T_{\text {Initial }}\). Let us take \(S_{\text {Initial }}=220\) and \(T_{\text {Initial }}\) = 0. The computation of forecast for other months is shown below:

 

\(\begin{aligned}S_{J a n} &=\alpha D_{J a n}+(1 \alpha)\left(S_{\text {Initial}}+T_{\text {Intial }}\right) \\&=0.3 \times 200+0.7 \times(220+0) \\S_{\text {Jan }} &=214\end{aligned}\)

 

\(\begin{aligned}T_{ Jan } &=\beta\left(S_{\text {Jan }}-S_{\text {Initial }}\right)+(1-\beta) T_{\text {Initial }} \\&=0.2 \times(214-220)+0.8 \times 0 \\T_{\text {Jan }} &=-1.2\end{aligned}\)

 

\(\begin{aligned}F_{ Feb } &=S_{ Jan }+T_{ Jan } \\&=214-1.2 \\F_{F e b} &=212.8\end{aligned}\)

 

\(\begin{aligned}S_{ Feb } &=\alpha D_{ Feb }+(1 \alpha)\left(S_{ Jan }+T_{ Jan }\right) \\&=0.3 \times 260+0.7 \times(214-1.2) \\S_{ Feb } &=226.96\end{aligned}\)

 

\(\begin{aligned}T_{ Feb } &=\beta\left(S_{ Feb }-S_{ Jan }\right)+(1-\beta) T_{ Jan } \\&=0.2 \times(226.96-214)+0.8 \times(-1.2) \\T_{ Feb } &=1.63\end{aligned}\)

 

\(\begin{aligned}F_{ Mar } &=S_{ Feb }+T_{ Feb } \\ &=226.96+1.63 \\F_{ Mar } &=228.59\end{aligned}\)

 

\(\begin{aligned}S_{ Mar } &=\alpha D_{ Mar }+(1 \alpha)\left(S_{ Feb }+T_{ Feb }\right) \\&=0.3 \times 210+0.7 \times(226.96+1.63) \\S_{ Mar } &=223.01\end{aligned}\)

 

\(T_{ Mar }=\beta\left(S_{ Mar }-S_{ Feb }\right)+(1-\beta) T_{ Feb }\)

 

\(\begin{aligned}&=0.2 \times(223.01-226.96)+0.8 \times 1.63 \\ T_{ Mar } &=0.51\end{aligned}\)

 

\(\begin{aligned}F_{ Apr } &=S_{ Mar }+T_{ Mar } \\ &=223.01+0.51 \\F_{ Apr } &=223.52\end{aligned}\)

 

\(\begin{aligned}S_{ Apr } &=\alpha D_{ Apr }+(1 \alpha)\left(S_{ Mar }+T_{ Mar }\right) \\&=0.3 \times 220+0.7 \times(223.01+0.51) \\S_{ Apr } &=222.46\end{aligned}\)

 

\(\begin{aligned}T_{ Apr } &=\beta\left(S_{ Apr }-S_{ Mar }\right)+(1-\beta) T_{ Mar } \\&=0.2 \times(222.46-223.01)+0.8 \times 0.51 \\T_{A p r} &=0.30\end{aligned}\)

 

\(\begin{aligned}F_{ May } &=S_{ Apr }+T_{ Apr } \\ &=222.46+0.30 \\F_{\text {May }} &=222.76\end{aligned}\)

 

\(\begin{aligned}S_{\text {May }} &=\alpha D_{\text {May }}+(1 \alpha)\left(S_{ Apr }+T_{ Apr }\right) \\&=0.3 \times 230+0.7 \times(222.46+0.30) \\S_{\text {May }} &=224.93\end{aligned}\)

 

\(\begin{aligned}T_{\text {May }} &=\beta\left(S_{\text {May }} S_{\text {Apr }}\right)+(1-\beta) T_{\text {Apr }} \\&=0.2 \times(224.93-222.46)+0.8 \times 0.30 \\T_{\text {May }} &=0.73\end{aligned}\)

 

\(\begin{aligned}F_{ Jun } &=S_{\text {May }}+T_{\text {May }} \\&=224.93+0.73 \\F_{\text {Jun }} &=225.76\end{aligned}\)

 

\(\begin{aligned}S_{\text {Jun }} &=\alpha D_{\text {Jun }}+(1-\alpha)\left(S_{\text {May }}+T_{ May }\right) \\&=0.3 \times 270+0.7 \times(224.93+0.730) \\S_{\text {Jun }} &=238.96\end{aligned}\)

 

\(\begin{aligned}T_{\text {Jun }} &=\beta\left(S_{\text {Jun }}-S_{\text {May }}\right)+(1-\beta) T_{\text {May }} \\&=0.2 \times(238.96-224.93)+0.8 \times 0.73 \\T_{\text {Jun }} &=3.39\end{aligned}\)

 

\(\begin{aligned}F_{ Jul } &=S_{ Jun }+T_{ Jun } \\&=238.96+3.39 \\F_{ Jul } &=242.35\end{aligned}\)

 

\(\begin{aligned}S_{\text {Jul }} &=\alpha D_{\text {Jul }}+(1-\alpha)\left(S_{\text {Jun }}+T_{\text {Jun }}\right) \\&=0.3 \times 290+0.7 \times(238.96+3.39) \\S_{\text {Jul }} &=256.65\end{aligned}\)

 

\(\begin{aligned}T_{ Jul } &=\beta\left(S_{ Jul }-S_{ Jun }\right)+(1-\beta) T_{ Jun } \\&=0.2 \times(256.65-238.96)+0.8 \times 3.39 \\T_{ Jul } &=6.25\end{aligned}\)

 

\(\begin{aligned}F_{\text {Aug }} &=S_{ Jul }+T_{ Jul } \\&=256.65+6.25 \\F_{\text {Aug }} &=262.90\end{aligned}\)
