Guest Question.

The circuit shown in Fig. 7.10a has reached steady state with the switch in position 1. At time t = 0 the switch moves from position 1 to position 2. We want to calculate \(v_{o}(t) \text { for } t>0\).

Admin
Solution

Step 1. \( v_{o}(t)\) is of the form \(K_{1}+K_{2} e^{-t / \tau}\).

Step 2. Using the circuit in Fig. 7.10b, we can calculate \(i_{L}(0-):\)

\(i_{A}=\frac{12}{4}=3 A\)

Then

\(i_{L}(0-)=\frac{12+2 i_{A}}{6}=\frac{18}{6}=3 A\)

Step 3. The new circuit, valid only for t = 0+, is shown in Fig. 7.10c. The value of the current source that replaces the inductor is \( i_{L}(0-)=i_{L}(0+)=3 A.\) Because of the current source

\( v_{o}(0+)=(3)(6)=18 V\)

Step 4. The equivalent circuit, for the steady-state condition after switch closure, is given in Fig. 7.10d. Using the voltages and currents defined in the figure, we can compute \( v_{o}(\infty)\) in a variety of ways. For example, using node equations, we can find \( v_{o}(\infty)\) from

\(\frac{v_{B}-36}{2}+\frac{v_{B}}{4}+\frac{v_{B}+2 i_{A}^{\prime}}{6}=0\)
\(i_{A}^{\prime}=\frac{v_{B}}{4}\)
\(v_{o}(\infty)=v_{B}+2 i_{A}^{\prime}\)
or, using loop equations,
\(36=2\left(i_{1}+i_{2}\right)+4 i_{1}\)
\(36=2\left(i_{1}+i_{2}\right)+6 i_{2}-2 i_{1}\)
\(v_{o}(\infty)=6 i_{2}\)
Using either approach, we find that \(v_{o}(\infty)=27 V\).
Step 5. The Thévenin equivalent resistance can be obtained via \( v_{ oc } \text { and } i_{ sc }\) because of the presence of the dependent source. From Fig. 7.10e we note that
\(i_{A}^{\prime \prime}=\frac{36}{2+4}=6 A\)
Therefore,
\(v_{ oc }=(4)(6)+2(6)\)
= 36 V
From Fig. 7.10f we can write the following loop equations (identical to those in step 4).
\(36=2\left(i_{A}^{\prime \prime \prime}+i_{ sc }\right)+4 i_{A}^{\prime \prime \prime}\)
\(36=2\left(i_{A}^{\prime \prime \prime}+i_{ sc }\right)+6 i_{ sc }-2 i_{A}^{\prime \prime \prime}\)
Solving these equations for \( \dot{i}_{ sc }\) yields
\( i_{ sc }=\frac{9}{2} A\)
Therefore,
\( R_{ Th }=\frac{v_{ oc }}{i_{ sc }}=\frac{36}{9 / 2}=8 \Omega \)
Hence, the circuit time constant is
\( \tau=\frac{L}{R_{ Th }}=\frac{3}{8} s\)
Step 6. Using the information just computed, we can derive the final equation for \( v_{o}(t)\):
\(K_{1}=v_{o}(\infty)=27\)
\(K_{2}=v_{o}(0+)-v_{o}(\infty)=18-27=-9\)
Therefore,
\( v_{o}(t)=27-9 e^{-t /(3 / 8)} V\)
