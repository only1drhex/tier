Guest Question.

The grain size distribution data of the soil in an embankment are given:

Size (mm) 0.02 0.04 0.075 0.15 0.30 0.425 1.18 2.36 4.75
% finer 1 6 23 47 70 80 98 100 100
Admin
Solution
\(D_{15, \text { soil }}=0.06 mm , D_{50 \text {, soil }}=0.16 mm , D _{85, \text { soil }}=0.55 mm\)

 

\(\text { Permeability Criteria } 1: D _{15, \text { filter }}>4 D _{15, \text { soil }} \rightarrow D _{15, \text { filter }}>0.24 mm\)

 

\(\text { Retention Criteria 2: } D _{15, \text { filter }}<5 D _{85 \text {, soil }} \rightarrow D _{15 \text {, filter }}<2.75 mm\)

 

\(\text { Retention Criteria 3: } D _{15 \text {, filter }} \leq 20 D _{15 \text {, soil }} \quad \rightarrow D _{15 \text {, filter }} \leq 1.2 mm\)

 

\(\text { Retention Criteria } 4: D _{50 \text {, filer }} \leq 25 D _{50 \text {, soil }} \rightarrow D _{50 \text {, filter }} \leq 4.0 mm\)

 

These four values are shown in the grain size distribution plot.
