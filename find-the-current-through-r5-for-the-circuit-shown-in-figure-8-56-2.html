Guest Question.

Find the current through \(R_{5}\) for the circuit shown in Figure 8–56.

Admin
Solution

By inspection we see that this circuit is not balanced, since

\(\frac{R_{1}}{R_{3}} \neq \frac{R_{2}}{R_{4}} \)

Therefore, the current through \(R_{5}\) cannot be zero. Notice, also, that the circuit contains two possible Δ configurations. If we choose to convert the top Δ to its equivalent Y, we get the circuit shown in Figure 8–57.

By combining resistors, it is possible to reduce the complicated circuit to the simple series circuit shown in Figure 8–58.
The circuit of Figure 8–58 is easily analyzed to give a total circuit current of

\(I=\frac{30V}{6\Omega +2\Omega +3.6\Omega } =2.59A\)

Using the calculated current, it is possible to work back to the original circuit. The currents in the resistors \(R_{3}\) and \(R_{4}\) are found by using the current divider rule for the corresponding resistor branches, as shown in Figure 8–57.

\(I_{R_{3}}=\frac{(6\Omega +3\Omega )}{(6\Omega +3\Omega )+(3\Omega +3\Omega )} (2.59A)=1.55A\)

 

\(I_{R_{4}}=\frac{(3\Omega +3\Omega )}{(6\Omega +3\Omega )+(3\Omega +3\Omega )} (2.59A)=1.03A\)

These results are exactly the same as those found in and . Using these currents, it is now possible to determine the voltage \(V_{bc}\) as

\(V_{bc}=-(3\Omega )I_{R_{4}}+(3\Omega )I_{R_{3}}\)

 

\(=(-3\Omega )(1.034A)+(3\Omega )(3.103A)\)

 

\(=1.55V\)

The current through \(R_{5}\) is determined to be

\(I_{R_{5}}=\frac{1.55V}{18\Omega } =0.086A\) (to the right)
