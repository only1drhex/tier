Guest Question.

For the motor drive system shown in Fig. 11.2 if the motor current is assumed to be constant, do the following:

a) Show that the maximum current ripple of is occurs when the duty cycle \(D = \frac{t_{ON}}{T_s}=0.5\).

b) For D = 0.5 , calculate the values of \(L_i\) and \(C_i\), when:

\(V_{in} = 120 V f_s = 400 Hz \overline{I}_o = 100 A\)

The rms input current fundamental component shall not exceed the 10% of the dc component. For the design, use the relation: \(f_s \geq 2f_R\).

c) For the values of Li and Ci, found in part (b), calculate the dc component and the rms values of the three first input current harmonic components.

d) Calculate the input filter apparent power.

Admin
Solution

a) The waveforms of the chopper of Fig. 11.2 in this case are:

From the above figure the following results are obtained:

\(i_{s,dc}\) = dc component of is =\(\overline{I}_o D\)

Furthermore, the rms value of \(i_s\) is given by:

\(\widetilde{I}_s\) = rms of \(i_s = (\int_{0}^{D}{\overline{I}^2_o dD} )^{1/2} = \overline{I}_o \sqrt{D}\)

 

\(\widetilde{I}_{s(ripple)}\) = rms ripple of \(i_s =[(\overline{I}_o \sqrt{D})^2 – (\overline{I}_o D)^2]^{1/2} = \overline{I}_o (D – D^2)^{1/2}\)

The converter exhibits maximum rms ripple of \(i_s\) when:

\( \frac{d\widetilde{I}_{s(ripple)}}{dD} = 0 = \frac{\overline{I}_o (1 – 2D)}{2(D – D^2)^{1/2}}\) or \(D = 0.5\)

b) The selection of input filter components must be done under worst operating condition from unwanted harmonics point of view which happens at D = 0.5. Using the above waveforms the current \(i_s\) can be represented by the following Fourier series:

\(i_s = \overline{I}_s + \frac{2\overline{I}_o}{n \pi}(\sin \omega t + \sin 3\omega t + \sin 5\omega t + . . .)\)

where

\(\overline{I}_s\) = dc component of \(i_s = \frac{100}{2} = 50 A\)

 

\(\widetilde{I}_{s,1} = \frac{2 \times 100}{\pi \sqrt{2}} = 45 A \widetilde{I}_{s,3} = \frac{2 \times 100}{3\pi \sqrt{2}} = 15 A \widetilde{I}_{s,5} = \frac{2 \times 100}{5\pi \sqrt{2}} = 9 A\)

Since the input filter is not altering the value of the dc current:

\(\overline{I}_i = \overline{I}_s = 50 A\)

In the specification of the input filter it was given that the rms input current fundamental harmonic component should not exceed 10% of the respective dc component. Therefore:

\(\widetilde{I}_{i,1} = 0.1\overline{I}_i = 5 A\)

Using Eq. (\(\widehat{I}_{i,n} = \frac{X_{C_i}/n}{nX_{L_i} – X_{C_i}/n}\widehat{I}_{s,n}= \frac{1}{\frac{n^2X_{L_i}}{X_{C_i}} – 1} \widehat{I}_{s,n}\)), the rms input current fundamental component is:

\(\widetilde{I}_{i,1} =\frac{X_{C_i}}{X_{L_i} – X_{C_i}}\widetilde{I}_{s,1}\) or \(5 = \frac{X_{C_i}}{X_{L_i} – X_{C_i}}\widetilde 45\) or \(X_{L_i} = 10X_{C_i}\)

The rms value of the capacitor current is given by:

\(\widetilde{I}_C =\sqrt{\widetilde{I}^2_{C,1} + \widetilde{I}^2_{C,2} + \widetilde{I}^2_{C,3} + . . .} \approx \widetilde{I}_{C,1} = \frac{X_{L_i}}{X_{L_i}-X_{C_i}} \widetilde{I}_{s,1} = \frac{10X_{L_i}}{10X_{L_i} – X_{C_i}} 45 = 50 A\)

Selecting an input filter capacitor \(C_i = 10,000 \mu F\), the following results are obtained:

\(X_{C_i} = \frac{1}{2\pi \times 400 \times 10^4 \times 10^{-6}} = 3.98 \times 10^{-2}\Omega\) \(X_{L_i} = 10X_{C_i} = 3.8 \times 10^{-1} \Omega\)

 

\(L = \frac{0.398}{2\pi \times 400} = 158 \mu H\)

 

\(f_R = \frac{1}{2\pi \sqrt{L_iC_i}} = 127 Hz\) \(f_s = 400 Hz\) \(\frac{f_s}{f_R}=\frac{400}{127}=3.15\)

Therefore, the input filter is not oscillating.

c) Using Eq. (\(\widehat{I}_{i,n} = \frac{X_{C_i}/n}{nX_{L_i} – X_{C_i}/n}\widehat{I}_{s,n}= \frac{1}{\frac{n^2X_{L_i}}{X_{C_i}} – 1} \widehat{I}_{s,n}\)), the following results are founds:

\(\widetilde{I}_{i,1}\)=rms input current fundamental component = \(\frac{45}{(3.15)^2 – 1} = 5A\)

 

\(\widetilde{I}_{i,3}\)=rms input current 3rd harmonic component=\(\frac{45}{(3 \times 3.15)^2 – 1} = 0.17A\)

 

\(\widetilde{I}_{i,5}\)=rms input current 5th harmonic component=\(\frac{45}{(5 \times 3.15)^2 – 1} = 0.036A\)

 

\(\overline{I}_i\) = input current dc component = 50 A

d) \(S_{L_i}\) = inductor apparent power =\(\sum\limits_{n=1}^{\infty}{\widetilde{I}^{2}_{i,n}(nX_{L_i})}\approx \widetilde{I}^{2}_{i,1}X_{L_i} + \widetilde{I}^{2}_{i,2}(2X_{L_i}) + \widetilde{I}^{2}_{i,3}(3X_{L_i})\)

 

\(S_{C_i}\) = capacitor apparent power = \(\sum\limits_{n=1}^{\infty}{\widetilde{I}^{2}_{C,n}(\frac{X_{C_i}}{n})}\approx \widetilde{I}^{2}_{C,1} X_{C_i} + \widetilde{I}^{2}_{C,2}(\frac{X_{C_i}}{2}) + \widetilde{I}^{2}_{C,3}(\frac{X_{C_i}}{3})\)

where \(X_{L_i} = 0.398 \Omega\) and \(X_{C_i} = 0.0398 \Omega\).

Therefore: \(S_{L_i} = 5^2(0.398) + (0.17)^2(3)(0.0398) = 9.98 VA\)

 

\(S_{C_i} = (35.35)^2(0.0398) + (15.17)^2(\frac{0.0398}{3}) = 52.78 VA\)

Finally, the input filter total apparent power is given by:

\(S_{filter} = S_{L_i} + S_{C_i} = 9.98 + 52.78 = 62.76 VA\)
