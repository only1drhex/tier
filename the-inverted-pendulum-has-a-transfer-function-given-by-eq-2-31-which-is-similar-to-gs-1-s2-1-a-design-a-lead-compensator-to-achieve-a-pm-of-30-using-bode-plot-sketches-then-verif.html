Guest Question.

The inverted pendulum has a transfer function given by Eq. (2.31) \( \frac{\theta^{\prime}(s)}{U(s)}=\frac{m_{p} l}{\left(\left(I+m_{p} l^{2}\right)-m_{p}^{2} l^{2}\right) s^{2}-m_{p} gl\left(m_{t}+m_{p}\right)}\), which is similar to

\(G(s)=\frac{1}{s^{2}-1}\)

(a) Design a lead compensator to achieve a PM of 30° using Bode plot sketches, then verify and refine your design using MATLAB.

(b) Sketch a root locus and correlate it with the Bode plot of the system.

(c) Could you obtain the frequency response of this system experimentally?

Admin
Solution

(a) Design the lead compensator :

\(D(s)=K \frac{T s+1}{\alpha T s+1}\)

such that the compensated system has \(PM \simeq 30^{\circ} \& \omega_{c} \simeq 1 rad / sec\). (Actually, the bandwidth or speed of response was not specified, so any crossover frequency would satisfy the problem statement.)

\(\alpha=\frac{1-\sin \left(30^{\circ}\right)}{1+\sin \left(30^{\circ}\right)}=0.32\)

To apply maximum phase lead at ω = 1 rad/sec,

\(\omega=\frac{1}{\sqrt{\alpha} T}=1 \Longrightarrow \frac{1}{T}=0.57, \frac{1}{\alpha T}=1.77\)

Therefore by applying the lead compensator :

\(D(s)=K \frac{\frac{s}{0.57}+1}{\frac{s}{1.77}+1}\)

By adjusting the gain K so that the crossover frequency is around 1 rad/sec, K = 1.13 results in :

\(PM=30.8^{\circ}\)

(c) No, because the sinusoid input will cause the system to blow up because the open loop system is unstable. In fact, the system will ”blow up” even without the sinusoid applied. Or, a better description would be that the pendulum will fall over until it hits the table.
