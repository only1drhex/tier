Guest Question.

The space between two parallel conducting plate each having an area \(S\) is filled with an inhomogeneous ohmic medium who e conductivity varies linearly from \({ \sigma }_{ 1 }\) at one plate \(( y=o)\) to \({ \sigma }_{ 2 }\) at the other plate \((y = d)\). A \(d-c\) voltage \({ V }_{ 0 }\) is applied across the plate Determine

\( a)\) the total resistance between the plates, and

\(b)\) the surface charge densities on the plate.

Admin
Solution
\(\sigma \left( y \right) ={ \sigma }_{ 1 }+\left( { \sigma }_{ 2 }-{ \sigma }_{ 1 } \right) \frac { y }{ d }\)

 

\(a)\) Neglecting fringing effect and assuming a current density:

\(\overline { J } =-\overline { { a }_{ y } } { J }_{ 0 }\rightarrow \overline { E } =\frac { \overline { J } }{ \sigma } =–\overline { { a }_{ y } } \frac { { J }_{ 0 } }{ \sigma \left( y \right) }\)

 

\({ V }_{ 0 }=-\int _{ 0 }^{ d }{ \overline { E } .\overline { { a }_{ y } } } dy=\int _{ 0 }^{ d }{ \frac { { J }_{ 0 }dy }{ { \sigma }_{ 1 }+\left( { \sigma }_{ 2 }-{ \sigma }_{ 1 } \right) \frac { y }{ d } } } =\frac { { J }_{ 0 }dy }{ { \sigma }_{ 2 }-{ \sigma }_{ 1 } } \ln { \frac { { \sigma }_{ 2 } }{ { \sigma }_{ 1 } } }\)

 

\(R=\frac { { V }_{ 0 } }{ I } =\frac { { V }_{ 0 } }{ { J }_{ 0 }S } =\frac { d }{ \left( { \sigma }_{ 2 }-{ \sigma }_{ 1 } \right) S } \ln { \frac { { \sigma }_{ 2 } }{ { \sigma }_{ 1 } } } \)

 

\(a)\) \({ \left( { \rho }_{ s } \right) }_{ u }={ \epsilon }_{ 0 }{ E }_{ y }\left( d \right) =\frac { { \epsilon }_{ 0 }{ J }_{ 0 } }{ { \sigma }_{ 2 } } =\frac { { \epsilon }_{ 0 }\left( { \sigma }_{ 2 }-{ \sigma }_{ 1 } \right) { V }_{ 0 } }{ { \sigma }_{ 2 }d\ln { \left( { \sigma }_{ 2 }/{ \sigma }_{ 1 } \right) } } \) on upper plate.

 

\(\quad { \left( { \rho }_{ s } \right) }_{ l }=-{ \epsilon }_{ 0 }{ E }_{ y }\left( 0 \right) =-\frac { { \epsilon }_{ 0 }{ J }_{ 0 } }{ { \sigma }_{ 1 } } =-\frac { { \epsilon }_{ 0 }\left( { \sigma }_{ 2 }-{ \sigma }_{ 1 } \right) { V }_{ 0 } }{ { \sigma }_{ 2 }d\ln { \left( { \sigma }_{ 2 }/{ \sigma }_{ 1 } \right) } } \) on lower plate.
