Determining the Order of a Butterworth Filter
a) Determine the order of a Butterworth filter that has a cutoff frequency of 1000 Hz and a gain of no more than -50 dB at 6000 Hz.
b) What is the actual gain in dB at 6000 Hz?

Admin
Solution

a) Because the cutoff frequency is given, we know \(ﾏダp = 1 \).We also note from the specification that\(10^{-0.1(-50)}\) is much greater than 1. Hence, we can use Eq. 15.46 with confidence:

\(n =\frac{(-0.05)(A_s)}{\log _{10}(\omega _s/\omega _p)}. \) (15.46)

\(n =\frac{(-0.05)(-50)}{\log _{10}(6000/1000)}= 3.21. \)

Therefore, we need a fourth-order Butterworth filter.

b) We can use Eq. 15.25 to calculate the actual gain at 6000 Hz. The gain in decibels will be

\(\left|H(j\omega )\right| =\frac{1}{\sqrt{1+\left(\omega /\omega _c\right)^{2n} } } \) (15.25)

\(K =20\log _{10}\left(\frac{1}{\sqrt{1+6^8 } }\right) = -62.25 \ dB. \)
