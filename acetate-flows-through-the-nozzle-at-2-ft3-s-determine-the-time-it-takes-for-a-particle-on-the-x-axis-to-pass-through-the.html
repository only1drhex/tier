Guest Question.

Acetate flows through the nozzle at \( 2 ft^3/s \). Determine the time it takes for a particle on the x axis to pass through the nozzle, from x = 0 to x = 6 in. if x = 0 at t = 0 Plot the distance-versus-time graph for the particle

Admin
Solution

Since the flow is assumed to be one dimensional and incompressible, its velocity can be determined using

\(u = \frac{Q }{A} \)

From the geometry shown in Fig. a, the radius r of the nozzleâ€™s cross-section as a function of x is

\(r = \frac{0.5}{12} ft + (\frac{0.5 – x}{0.5}) (\frac{1.5}{12} ft) = \frac{1}{24} (4-6x)ft \)

Thus, the cross-sectional area of the nozzle is

\(A = \pi r^2 = \pi [\frac{1}{24} (4-6x)]^2 = \frac{\pi }{576} (4- 6x)^2 ft^2 \)

Then

\(u = \frac{Q }{A} = \frac{2ft^3/s}{\frac{\pi }{576} (4 – 6x )^2 ft^2} = [\frac{1152}{\pi (4-6x)^2}] ft/s \)

Using the definition of velocity and the initial condition of x = 0 at t = 0,

\(\frac{dx}{dt}=u=\frac{1152}{\pi (4-6x)^2}\)

 

\(\int_{0}^{t}dt=\frac{\pi }{1152}\int_{0}^{x}(4-6x)^2dx\)

 

\(t=\frac{\pi }{1152}\int_{0}^{x}(36x^2-48x+16)dx\)

 

\(t=\frac{\pi }{1152}(12x^3-24x^2+16x)\)

 

\(t=\frac{\pi x}{288}(3x^2-6x+4)\)

When \(x=\frac{6}{12}ft=0.5ft\)

\(t=\frac{\pi (0.5)}{288}[3(0.5^{2})-6(0.5)+4]=9.5448(10^{-3})=9.54 ms\)

Using Eq (1), the following tabulation can be computed and the plot of x vs t is shown in Fig. b.
