Guest Question.

Use the method of Frobenius to find one solution near \(x=0 \text { of } x^{2} y^{\prime \prime}-x y^{\prime}+y=0\).

Admin
Solution

Here \(P(x)=-1 / x \text { and } Q(x)=1 / x^{2}, \text { so } x=0\) is a regular singular point and the method of Frobenius is applicable.
Substituting Eqs. (28.2) through (28.4) into the left side of the differential equation, as given, and combining coefficients of like powers of x, we obtain

\(x^{\lambda}(\lambda-1)^{2} a_{0}+x^{\lambda+1}\left[\lambda^{2} a_{1}\right]+\cdots+x^{\lambda+n}\left[(\lambda+n)^{2}-2(\lambda+n)+1\right] a_{n}+\cdots=0\)

Thus, \((\lambda-1)^{2} a_{0}=0\) (1)

and, in general, \(\left[(\lambda+n)^{2}-2(\lambda+n)+1\right] a_{n}=0\) (2)

From (1), the indicial equation is \((\lambda-1)^{2}=0\), which has roots \(\lambda_{1}=\lambda_{2}=1\). Substituting \(\lambda=1\) into (2), we obtain \(n^{2} a_{n}=0\), which implies that \(a_{n}=0, n \geq 1\). Thus, \(y_{1}(x)=a_{0} x\).

Eq. (28.2) :

\(\begin{aligned}y &=x^{\lambda} \sum_{n=0}^{\infty} a_{n} x^{n}=\sum_{n=0}^{\infty} a_{n} x^{\lambda+n} \\&=a_{0} x^{\lambda}+a_{1} x^{\lambda+1}+a_{2} x^{\lambda+2}+\cdots+a_{n-1} x^{\lambda+n-1}+a_{n} x^{\lambda+n}+a_{n+1} x^{\lambda+n+1}+\cdots\end{aligned}\)

Eq. (28.3) :

\(\begin{aligned}y^{\prime}=\lambda & a_{0} x^{\lambda-1}+(\lambda+1) a_{1} x^{\lambda}+(\lambda+2) a_{2} x^{\lambda+1}+\cdots \\&+(\lambda+n-1) a_{n-1} x^{\lambda+n-2}+(\lambda+n) a_{n} x^{\lambda+n-1}+(\lambda+n+1) a_{n+1} x^{\lambda+n}+\cdots\end{aligned}\)

Eq. (28.4) :

\(\begin{aligned}y^{\prime \prime}=\lambda(\lambda&-1) a_{0} x^{\lambda-2}+(\lambda+1)(\lambda) a_{1} x^{\lambda-1}+(\lambda+2)(\lambda+1) a_{2} x^{\lambda}+\cdots \\&+(\lambda+n-1)(\lambda+n-2) a_{n-1} x^{\lambda+n-3}+(\lambda+n)(\lambda+n-1) a_{n} x^{\lambda+n-2} \\&+(\lambda+n+1)(\lambda+n) a_{n+1} x^{\lambda+n-1}+\cdots\end{aligned}\)
