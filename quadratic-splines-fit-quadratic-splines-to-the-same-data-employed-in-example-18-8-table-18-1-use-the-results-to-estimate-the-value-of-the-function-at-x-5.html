Guest Question.

Quadratic Splines Fit quadratic splines to the same data employed in (Table 18.1). Use the results to estimate the value of the function at x = 5.

TABLE 18.1 Data to be fit with spline functions.
i \(X_{i}\) \(f_{i}\)
1 3.0 2.5
2 4.5 1.0
3 7.0 2.5
4 9.0 0.5
Admin
Solution

For the present problem, we have four data points and \(n = 3\) intervals. Therefore, after applying the continuity condition and the zero second-derivative condition, this means that \(2(4 − 1) − 1 = 5\) conditions are required. Equation (18.34) is written for \(i = 1\) through 3 (with \(c_1 = 0\)) to give

\(\begin{matrix} f_1 + b_1h_1 = f_2\\ f_2 + b_2h_2 + c_2h_2^2 = f3 \\f_3 + b_3 h_3 + c_3h^2_3 = f_4\end{matrix} \)

 

\(f_i + b_ih_i + c_ih_i^2 = f_{i+1} \) (18.34)

Continuity of derivatives, Eq. (18.35), creates an additional 3 − 1 = 2 conditions (again, recall that \(c_1 = 0\)):

\(\begin{matrix} b_1 = b_2\\ b_2 + 2c_2h_2 = b_3\end{matrix} \)

 

\(b_i + 2c_ih_i = b_{i+1} \) (18.35)

The necessary function and interval width values are

\( \begin{matrix}f_1 = 2.5&&& h_1 = 4.5 − 3.0 = 1.5\\ f_2 = 1.0 &&&h_2 = 7.0−4.5 =2.5\\ f_3 = 2.5 &&&h_3 = 9.0 − 7.0 = 2.0\\ f_4 = 0.5 \end{matrix} \)

These values can be substituted into the conditions, which can be expressed in matrix form as

\(\begin{bmatrix} 1.5& 0 &0 &0 &0\\ 0& 2.5 &6.25& 0 &0 \\0 &0 &0 &2 &4\\ 1 &−1 &0 &0 &0\\0& 1& 5& −1& 0 \end{bmatrix} \begin{Bmatrix} b_{1}\\b_{2}\\c_{3}\\b_{3}\\c_{3} \end{Bmatrix} =\begin{Bmatrix} −1.5\\1.5\\-2\\0\\0 \end{Bmatrix} \)

These equations can be solved with the results:

\( \begin{matrix} b_1 = −1\\b_{2}=-1 && c_{2}=0.64\\b_{3}=2.2&& c_{3}=-1.6 \end{matrix} \)

These results, along with the values for the a’s [Eq. (18.32)], can be substituted into the original quadratic equations to develop the following quadratic splines for each interval:

\(\begin{matrix}s_1(x) = 2.5 − (x − 3)\\ \\s_2(x) = 1.0 − (x − 4.5) + 0.64(x − 4.5)^2\\ \\s_3(x) = 2.5 + 2.2(x − 7.0) − 1.6(x − 7.0)^2 \end{matrix} \)

 

\(a_i = f_i\) (18.32)

Because x = 5 lies in the second interval, we use \(s_2\) to make the prediction

\(s_2(5) = 1.0 − (5 − 4.5) + 0.64(5 − 4.5)^2= 0.66\)

The total quadratic spline fit is depicted in Fig. 18.17b. Notice that there are two shortcomings that detract from the fit: (1) the straight line connecting the first two points and (2) the spline for the last interval seems to swing too high. The cubic splines in the next section do not exhibit these shortcomings and, as a consequence, are better methods for spline interpolation.
