Isentropic Unchoked Flow in a Converging–Diverging Duct

GIVEN Air flows subsonically and isentropically through the converging–diverging duct of .

FIND Graph the variation of Mach number, Ma, static temperature to stagnation temperature ratio, \(T / T_{0}\), and the static pressure to stagnation pressure ratio, \(p / p_{0}\), through the duct from \(x=-0.5 m \text { to } x=+0.5 m \text { for } Ma =0.48 \text { at } x=0 m\). Show the corresponding temperature–entropy diagram.

Admin
Solution

Since for this example, Ma = 0.48 at x = 0 m, the isentropic flow through the converging-diverging duct will be entirely subsonic and not choked. For air (k=1.4) flowing isentropically through the duct, we can use Fig. D.1 for flow field quantities. Entering Fig. D. 1 with Ma = 0.48 we read off $p / \(p / p_{0}=0.85, T / T_{0}=0.96, \text { and } A / A^{*}=1.4\). Even though the duct flow is not choked in this example and \(A^{*}\) does not therefore exist physically, it still represents a valid reference. For a given isentropic flow, \(p_{0}, T_{0}\), and \(A^{*}\) are constants. Since A at x=0 m is equal to \(0.10 m ^{2}\) (from Eq. 2 of ), \(A^{*}\) for this example is

 

\(A^{*}=\frac{A}{\left(A / A^{*}\right)}=\frac{0.10 m ^{2}}{1.4}=0.07 m ^{2}\) (1)

 

With known values of duct area at different axial locations, we can calculate corresponding area ratios, \(A / A^{*}\), knowing \(A^{*}=0.07 m ^{2}\). Having values of the area ratio, we can use Fig. D.1 and obtain related values of Ma, \(T / T_{0}\), and \(p / p_{0}\). The following table summarizes flow quantities obtained in this manner. The results are graphed in Fig. E11.10.

 

A more precise solution for the flow of this example could have been obtained with isentropic flow equations by following the steps outlined below.

1. Use Eq. 11.59 to get \(p / p_{0}\) at x=0 knowing k and Ma =0.48.

2. From Eq. 11.71, obtain value of \(A / A^{*}\) at x=0 knowing k and Ma.

3. Determine \(A^{*}\) knowing A and \(A / A^{*}\) at x=0.

4. Determine \(A / A^{*}\) at different axial locations, x.

5. Use Eq. 11.71 and \(A / A^{*}\) from step 4 above to get values of Mach numbers at different axial locations.

6. Use Eqs. 11.56 and 11.59 and Ma from step 5 above to obtain \(T / T_{0}\) and \(p / p_{0}\) at different axial locations, x.

 

\(\frac{T}{T_{0}}=\frac{1}{1+[(k-1) / 2] Ma ^{2}}\) (11.56)

 

\(\frac{p}{p_{0}}=\left\{\frac{1}{1+[(k-1) / 2] Ma ^{2}}\right\}^{k /(k-1)}\) (11.59)

 

\(\frac{A}{A^{*}}=\frac{1}{ Ma }\left\{\frac{1+[(k-1) / 2] Ma ^{2}}{1+[(k-1) / 2]}\right\}^{(k+1) /[2(k-1)]}\) (11.71)

 

COMMENT There are an infinite number of subsonic, isentropic flow solutions for the converging–diverging duct considered in this example one for any given \(Ma <1 \text { at } x=0\)).
