Guest Question.

A steel ball \(\frac{1}{8}\) inch in diameter and weighing 0.284 lb/in³ is falling in an oil of specific gravity 0.908 and kinematic viscosity 0.00157 ft²/sec. What is the terminal velocity of the ball?

Admin
Solution

The forces acting on the steel ball are the weight down, the buoyant force up, and the drag force up.
For constant velocity, ∑Y = 0, and, transposing,

weight of sphere — buoyant force = drag force.

or \(\gamma_{s}(\text { volume })-\gamma_{o}(\text { volume })=C_{D} \rho A V^{2} / 2\).

\(\text { Using } lb / \text { in }^{3} \times \text { in }^{3}=\text { weight, }\)

\(\frac{4}{3} \pi\left(\frac{1}{16}\right)^{3}\left(0.284-\frac{0.908 \times 62.4}{1728}\right)=C_{D}\left(\frac{0.908 \times 62.4}{32.2}\right) \pi\left(\frac{1}{12 \times 16}\right)^{2}\left(\frac{V^{2}}{2}\right)\).

\(\text { Assuming a value of } C_{D}=3.00 \text { (see Diagram } F \text {, spheres) and solving, }\)

\(V^{2}=3.43 / C_{D}=1.143 \text { and } V=1.069 ft / sec\).

\(\text { Check } C_{D} \text { assumed by calculating Reynolds number and using Diagram } F \).

\(\operatorname{Re}=\frac{V d}{v}=\frac{1.069 \times 1 /(8 \times 12)}{0.00157}=7.09 \quad \text { and } \quad C_{D}=5.6 \text { (increase } C_{D} \text { ) }\).

\(\text { Recalculating and checking, using } C_{D}=6.5 \text { to anticipate the effect of increased value of } C_{D} \text {, }\)

\(V^{2}=3.43 / 6.5=0.528, \quad V=0.726, \quad \operatorname{Re}=4.82, \quad C_{D}=7.2 \text { (increase } C_{D} \text { ) }\).

\(\text { Trying } C_{D}=7.8 \text {, }\)

\(V^{2}=3.43 / 7.8=0.440, \quad V=0.663, \quad \operatorname{Re}=4.40, \quad C_{D}=7.8(\text { checks })\).

Therefore, the terminal velocity = 0.66 ft/sec.
Had the Reynolds number been less than 0.60, the equation for the drag force could be written

\(C_{D} \rho A V^{2} / 2=(24 / \operatorname{Re}) \rho A V^{2} / 2=(24 v / V d) \rho\left(\pi d^{2} / 4\right) V^{2} / 2\).

\(\text { Since } \mu=\rho v \text {, drag force }=3 \pi \mu d V\).
