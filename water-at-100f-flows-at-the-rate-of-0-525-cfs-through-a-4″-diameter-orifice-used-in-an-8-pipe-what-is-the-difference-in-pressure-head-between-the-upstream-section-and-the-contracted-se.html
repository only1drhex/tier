Guest Question.

Water at 100°F flows at the rate of 0.525 cfs through a 4″-diameter orifice used in an 8″ pipe. What is the difference in pressure head between the upstream section and the contracted section (vena contracta section)?

Admin
Solution

In Diagram C of the Appendix, it is observed that c′ varies with Reynolds number. Note that Reynolds number must be calculated for the orifice cross section, not for the contracted section of the jet or for the pipe section. This value is

\(\operatorname{Re}=\frac{V_{o} D_{o}}{v}=\frac{\left(4 Q / \pi D_{o}^{2}\right) D_{o}}{v}=\frac{4 Q}{v \pi D_{o}}=\frac{(4)(0.525)}{(\pi)(0.00000739)(4 / 12)}=271,000\).

\(\text { Diagram } C \text { for } \beta=0.500 \text { gives } c^{\prime}=0.604 \).

Applying the Bernoulli theorem, pipe section to jet section, produces the general equation for incompressible fluids, as follows:

\(\left(\frac{p_{8}}{\gamma}+\frac{V_{8}^{2}}{2 g}+0\right)-\left[\frac{1}{c_{v}^{2}}-1\right] \frac{V_{ jet }^{2}}{2 g}=\left(\frac{p_{ jet }}{\gamma}+\frac{V_{ jet }^{2}}{2 g}+0\right)\).

and \(Q=A_{8} V_{8}=\left(c_{c} A_{4}\right) V_{ jet }\).

\(\text { Substituting for } V_{8} \text { in terms of } V_{ jet } \text { and solving, }\)

\(\frac{V_{\text {jet }}^{2}}{2 g}=c_{v}^{2}\left(\frac{p_{8} / \gamma-p_{\text {jet }} / \gamma}{1-c^{2}\left(A_{4} / A_{8}\right)^{2}}\right) \quad \text { or } \quad V_{\text {jet }}=c_{v} \sqrt{\frac{2 g\left(p_{8} / \gamma-p_{\text {jet }} / \gamma\right)}{1-c^{2}\left(D_{4} / D_{8}\right)^{4}}}\).

Then \(Q=A_{\text {jet }} V_{\text {jet }}=\left(c_{c} A_{4}\right) \times c_{v} \sqrt{\frac{2 g\left(p_{8} / \gamma-p_{\text {jet }} / \gamma\right)}{1-c^{2}\left(D_{4} / D_{8}\right)^{4}}}=c A_{4} \sqrt{\frac{2 g\left(p_{8} / \gamma-p_{\text {jet }} / \gamma\right)}{1-c^{2}\left(D_{4} / D_{8}\right)^{4}}}\).

More conveniently, for an orifice with velocity of approach and a contracted jet, the equation can be written

\(Q=\frac{c^{\prime} A_{4}}{\sqrt{1-\left(D_{4} / D_{8}\right)^{4}}} \sqrt{2 g(\Delta p / \gamma)}\). (1).

or \(Q=K A_{4} \sqrt{2 g(\Delta p / \gamma)}\) (2).

where K is called the flow coefficient. The meter coefficient c’ can be determined experimentally for a given ratio of diameter of orifice to diameter of pipe, or the flow coefficient K may be preferred.
Proceeding with the solution by substituting in the above expression (1),

\(0.525=\frac{0.604 \times \frac{1}{4} \pi(4 / 12)^{2}}{\sqrt{1-(1 / 2)^{4}}} \sqrt{2 g(\Delta p / \gamma)} \text { and } \Delta p / \gamma=\left(p_{8} / \gamma-p_{\text {jet }} / \gamma\right)=1.44 ft \text { water }\).
