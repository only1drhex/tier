Guest Question.

The continuous random variables X and Y have a joint PDF proportional to \(xy(x−y)^{2}\) with 0 ≤ x ≤ 1 and 0 ≤ y ≤ 1. Find the marginal distributions for X and Y and show that they are negatively correlated with correlation coefficient \(-\frac { 2 }{ 3 }\)

Admin
Solution

This PDF is clearly symmetric between x and y. We start by finding its normalisation constant c:

\(\int _{ 0 }^{ 1 }{ \int _{ 0 }^{ 1 }{ c({ x }^{ 3 }y-2{ x }^{ 2 }{ y }^{ 2 }+x{ y }^{ 3 }) } } dxdy=c(\frac { 1 }{ 4 } \frac { 1 }{ 2 } -2\frac { 1 }{ 3 } \frac { 1 }{ 3 } +\frac { 1 }{ 2 } \frac { 1 }{ 4 } )=\frac { c }{ 3 }\)

Thus, we must have that c = 36.
The marginal distribution for x is given by

\(f(x) = 36\int _{ 0 }^{ 1 }{ c({ x }^{ 3 }y-2{ x }^{ 2 }{ y }^{ 2 }+x{ y }^{ 3 }) } dy \)

=\(36 (\frac { 1 }{ 2 } { x }^{ 3 }-\frac { 2 }{ 3 } { x }^{ 2 }+\frac { 1 }{ 4 } x)\)

=\( 18{x}^{3} − 24{x}^{2} + 9x,\)

and the mean of x by

\(μX = \bar {x} =\int _{ 0 }^{ 1 }{(18{ x }^{ 4 }-24{ x }^{ 3 }+9 { x }^{ 2 }) } dx=\frac { 18 }{ 5 } -\frac { 24 }{ 4 } +\frac { 9 }{ 3 }=\frac { 3 }{ 5 }\)

By symmetry, the marginal distribution and the mean for y are \(18{y}^{3} −24{y}^{2} +9y\) and \( \frac { 3 }{ 5 } \), repectively.

To calculate the correlation coefficient we also need the variances of x and y and their covariance. The variances, obviously equal, are given by

\({ \sigma }_{ X }^{ 2 }=\int _{ 0 }^{ 1 }{ { x }^{ 2 }(18{ x }^{ 3 }-24{ x }^{ 2 }+9x) } dx-{ \left( \frac { 3 }{ 5 } \right) }^{ 2 }\)

=\(\frac { 18 }{ 6 } -\frac { 24 }{ 5 } +\frac { 9 }{ 4 } -\frac { 9 }{ 25 } \)

=\(\frac { 900−1440+675−108 }{ 300 } =\frac { 9 }{ 100 } \)

The standard deviations \({σ}_{X}\) and \({σ}_{Y}\) are therefore both equal to 3/10.
The covariance is calculated next; it is given by

\(Cov[X,Y ] = \left< XY \right> − {μ}_{X}{μ}_{Y}\)

=\(36 \int _{ 0 }^{ 1 }{ \int _{ 0 }^{ 1 }{ ({ x }^{ 4 }{ y }^{ 2 }−2{ x }^{ 3 }{ y }^{ 3 }+{ x }^{ 2 }{ y }^{ 4 }) } } dxdy-\frac { 3 }{ 5 } \frac { 3 }{ 5 } \)

=\(\frac { 36 }{ 5×3 } -\frac { 72 }{ 4×4 } +\frac { 36 }{ 3×5 } -\frac { 9 }{ 25 } \)

=\(\frac { 12 }{ 5 } -\frac { 9 }{ 2 } +\frac { 12 }{ 5 } -\frac { 9 }{ 25 } \)

=\(\frac { 120−225+120−18 }{ 50 } =-\frac { 3 }{ 50 } \)

Finally,

\(Corr[X,Y ]=\frac { Cov[X,Y] }{ { \sigma }_{ X }{ \sigma }_{ Y } } =\frac { -\frac { 3 }{ 50 } }{ \frac { 3 }{ 10 } \frac { 3 }{ 10 } } =-\frac { 2 }{ 3 } \)
