Guest Question.

The output voltage waveform of a single-phase inverter is the same as that of Fig. 11.4(a). Design an output filter so that the THD factor, \(THD_{vo}\) ≤ 5%. The inverter input voltage is 220 V dc and the output voltage fundamental component frequency is 50 Hz.

Admin
Solution

As shown in Fig. 11.4(a) the dominant harmonic component is the fifth with an amplitude of:

\(\widehat{V}_{o,5}=0.22V_{in} = 0.22 \times 220 = 48.4 V\)

To achieve a \(THD_{vo} \leq 5\%\), an output filter must be designed so that the dominant harmonic component, which is the fifth, to be reduced to \(30\%\) of the fundamental component (i.e., from \(0.20 V_{in}\) to \(0.03 V_{in}\)). Therefore, using Eq. (\(\frac{\widehat{V}_{o,n}}{\widehat{V}_{if,n}} \approx \frac{1}{1 – n^2 \omega^2_o |L_oC_o|}\)) the following results are obtained:

\(\frac{\widehat{V}_{o,5}}{\widehat{V}_{if,5}} =\frac{0.030V_{in}}{0.20V_{in}} =\frac{1}{1 – (5)^2(2\pi \times 50)^2 |L_oC_o|}\) or

\(0.15 =\frac{1}{1 – 2464900 |L_oC_o|}\) or \(|L_oC_o| = 2.29 \times 10^{-6}\)

Selecting a filter capacitor \(C_o = 30 \mu F\), the output filter inductance is:

\(L_o = \frac{2.29 \times 10^{-6}}{30 \times 10^{-6}} = 76 mH\)

The output filter cut-off or break frequency is:

\(\omega_b =\frac{1}{\sqrt{L_oC_o}} = \frac{1}{\sqrt{2.29 \times 10^{-6}}} = 660.8 \frac{rad}{s}\) or \(f_b = \frac{\omega_b}{2\pi} =105 Hz\)
