Guest Question.

Consider the spring–mass system illustrated in Fig. 7.3 and described mathematically in system (12). Suppose that \(m_{1}=m_{2}=1, k_{1}=3, \text { and } k_{2}=2 . \text { Find } y_{1}(t) \text { and } y_{2}(t)\) if the initial conditions are

\(y_{1}(0)=0, \quad y_{2}(0)=10\)
\(y_{1}^{\prime}(0)=0, \quad y_{2}^{\prime}(0)=0\).

Admin
Solution

System (12) has the form y” (t) = Ay(t), where
\(y (t)=\left[\begin{array}{l}y_{1}(t) \\y_{2}(t)\end{array}\right] \quad \text { and } \quad A=\left[\begin{array}{rr}-5 & 2 \\2 & -2\end{array}\right]\).
By Eq. (13) \(\omega^{2} e^{\omega t} u -e^{\omega t} A u = \theta\), a function x of the form
\(x (t)=e^{\beta t} u\)
will satisfy y” (t) = Ay(t) if \(\beta^{2}\) is an eigenvalue of A.
The eigenvalues of A are \(\lambda_{1}=-1 \text { and } \lambda_{2}=-6\), with corresponding eigenvectors
\(u _{1}=\left[\begin{array}{l}1 \\2\end{array}\right] \quad \text { and } \quad u _{2}=\left[\begin{array}{r}-2 \\1\end{array}\right]\).
Thus four solutions of y” (t) = Ay(t) are
\(x _{1}(t)=e^{i t} u _{1}, \quad x _{2}(t)=e^{-i t} u _{1}\),
\(x _{3}(t)=e^{\sqrt{6} i t} u _{2}, \quad x _{4}(t)=e^{-\sqrt{6} i t} u _{2}\).
The general solution is
\(y (t)=a_{1} x _{1}(t)+a_{2} x _{2}(t)+a_{3} x _{3}(t)+a_{4} x _{4}(t)\). (14)
To satisfy the initial conditions, we need to choose the previous \(a_{i}\) so that
\(y (0)=\left[\begin{array}{r}0 \\10\end{array}\right] \text { and } y ^{\prime}(0)\left[\begin{array}{l}0 \\0\end{array}\right]\).
An evaluation of Eq. (14) shows that
\(y (0)=\left(a_{1}+a_{2}\right) u _{1}+\left(a_{3}+a_{4}\right) u _{2}\)
\(y ^{\prime}(0)=i\left(a_{1}-a_{2}\right) u _{1}+i \sqrt{6}\left(a_{3}-a_{4}\right) u _{2}\).
Since y’ (0) = θ, we see that \(a_{1}=a_{2} \text { and } a_{3}=a_{4}\). With this information in the condition \(y (0)=[0,10]^{T}\), it follows that \(a_{1}=2 \text { and } a_{3}=1\).
Finally, by Eq. (14), we obtain
\(y (t)=2\left[ x _{1}(t)+ x _{2}(t)\right]+\left[ x _{3}(t)+ x _{4}(t)\right]\)
\(=\left[\begin{array}{l}4 \cos t-4 \cos (\sqrt{6} t) \\8 \cos t+2 \cos (\sqrt{6} t)\end{array}\right]\).
