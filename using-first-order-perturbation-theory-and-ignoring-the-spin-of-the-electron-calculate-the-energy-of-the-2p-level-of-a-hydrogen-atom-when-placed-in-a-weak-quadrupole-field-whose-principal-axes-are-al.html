Guest Question.

Using first-order perturbation theory, and ignoring the spin of the electron, calculate the energy of the 2p level of a hydrogen atom when placed in a weak quadrupole field whose principal axes are along the xyz axes: \(\hat{H} _{p} =∑^{\mu =2}_{\mu =-2} Q_{\mu } r^{2} Y_{2\mu } (\Omega )\), where \(Q_{\mu } \) are real numbers, with \(Q_{-1} =Q_{1} =0\) and \(Q_{-2} =Q_{2} \), and \(Y_{2\mu } (\Omega )\) are spherical harmonics.

Admin
Solution

In the absence of the field, the energy levels of the \(|2, 1,m 〉 \) states are threefold degenerate: \(|2,1,-1 〉,|2,1,0 〉, \) and \( |2,1,1 〉 \), and hence correspond to the same energy \(E_{2} =-R/4 \), where R = 13.6 eV is the Rydberg constant.

When the quadrupole field is turned on, and since \(Q_{-2} ,Q_{0} \), and \(Q_{2} \) are small, we can treat the quadrupole interaction \(\hat{H} _{p} =Q_{-2} r^{2} Y_{2-2} (\Omega )+ Q_{0}r^{2} Y_{20} (\Omega )+Q_{2}r^{2} Y_{22} (\Omega ) \) as a perturbation. To calculate the p split, we need to use degenerate perturbation theory, which, in a first step, requires calculating the matrix

\(\left(\begin{matrix} 〈2,1,-1|\hat{H} _{p}|2,1,-1 〉 & 〈2,1,-1|\hat {H} _{p}|2,1,0 〉 & 〈2,1,-1|\hat{H} _{p}|2,1,1 〉 \\ 〈2,1,0|\hat{H} _{p} |2,1,-1 〉 & 〈2,1,0|\hat{H} _{p}|2,1,0 〉 & 〈2,1,0|\hat{H} _{p}|2,1,1 〉 \\ 〈2,1,1|\hat{H} _{p}|2,1,-1 〉 & 〈2,1,1|\hat{H} _{p}|2,1,0 〉 & 〈2,1,1|\hat {H} _{p}|2,1,1 〉 \end{matrix} \right) \), (9.375)

where

\(〈2,1,m^{\prime } |\hat{H} _{p}|2,1,m 〉=〈2,1|r^{2}|2,1 〉〈1,m^{\prime } |Q_{-2} Y_{2-2}+Q_{0} Y_{20}+Q_{2} Y_{22}|1,m 〉 \). (9.376)

The radial part is easy to obtain (Chapter 6):

\(〈n.l|r^{2} |n,l 〉=\int_{0}^{+\infty } r^{4} \left|R_{nl} \right| ^{2} dr=\frac{1}{2} n^{2}\left[5n^{5}+1-3l(l+1) \right] a^{2}_{0} \); (9.377)

hence

\(〈2,1|r^{2}|2,1 〉=30 a^{2}_{0}\). (9.378)

As for the angular part, it can be inferred from the Wigner–Eckart theorem:

\(〈l^{\prime }, m^{\prime }|Y_{2\mu }|1,m 〉=〈l,2;m,\mu |l^{\prime }, m^{\prime } 〉〈l^{\prime }\parallel Y_{2} \parallel 1 〉\); (9.379)

the reduced matrix element \(〈l^{\prime }\parallel Y_{2} \parallel 1 〉\) was calculated in Chapter 7: \(〈l^{\prime }\parallel Y_{2} \parallel 1 〉=\sqrt{5/4\pi } (2l+1)/(2l^{\prime }+1) 〈l,2;0,0 |l^{\prime },0 〉\) and hence

\(〈l^{\prime }, m^{\prime }|Y_{2\mu }|1,m 〉=\sqrt{\frac{5}{4\pi } } \sqrt{\frac{2l+1}{2l^{\prime }+1} } 〈l,2;0,0 |l^{\prime },0 〉〈l,2;m, \mu |l^{\prime },m^{\prime } 〉\). (9.380)

Using the coefficients \(〈l,2;m,0 |l,m 〉 [3m^{2} -l(l+1)]l\sqrt{l(2l-1)(l+1)(2l+3)} \) and \(〈l,2;m,\pm 2,\pm 2|l,m 〉 =\sqrt{\frac {3(l\pm m-1)(l\pm m)(l\mp m+1)(l\mp m+2)}{2l(2l-1)(l+1)(2l+3)} } \), we have

\(〈1,-1|Y_{2-2}|1,1 〉 =〈1,1|Y_{22}|1,-1 〉 =-\sqrt{\frac{3}{10\pi } } \), (9.381)

 

\(〈1,-1|Y_{20}|1,-1 〉 =〈1,1|Y_{20}|1,1 〉 =-\frac{1}{\sqrt{20\pi } } \), (9.382)

 

\(〈1,0|Y_{20}|1,0 〉 =\frac{1}{\sqrt{5\pi } } \). (9.383)

These expressions can also be obtained from the following relations:

\(∫ Y^{*}_{lm} (\Omega )Y_{20} (\Omega )Y_{lm} (\Omega )d \Omega =\sqrt{\frac{5}{4\pi } } \frac{l(l+1)-3m^{2} }{(2l-1)(2l+3)} \), (9.384)

 

\(∫ Y^{*}_{lm+2} (\Omega )Y_{22} (\Omega )Y_{lm} (\Omega )d\Omega =∫ Y^{*}_{lm} (\Omega )Y_{2-2} (\Omega )Y_{lm+2} (\Omega )d\Omega \)

 

\(=\sqrt{\frac{15}{8\pi } }\frac{\sqrt{(l-m-1)(l-m)(l+m+1)(l+m+2)} }{(2l-1)(2l+3)} \). (9.385)

Combining (9.376) to (9.383) we can write the matrix (9.375) as

\(30 a^{2}_{0}\left(\begin{matrix} -\frac{Q_{0}}{\sqrt{20\pi } } & 0 & -Q_{2}\sqrt{\frac{3}{10\pi } } \\ 0 & \frac{Q_{0}}{\sqrt{5\pi } } & 0 \\ -Q_{2}\sqrt{\frac{3}{10\pi } }& 0 & -\frac{Q_{0}}{\sqrt{20\pi } } \end{matrix} \right) \). (9.386)

The diagonalization of this matrix leads to the following eigenvalues:

\(E^{(1)} _{1} =-30\frac{a^{2}_{0}}{\sqrt{10\pi } } \left(\frac {Q_{0}}{\sqrt{2} }+Q_{2}\sqrt{3} \right) \), (9.387)

 

\(E^{(1)} _{2} =30\frac{Q_{0} a^{2}_{0}}{\sqrt{5\pi } } \), (9.388)

 

\(E^{(1)} _{3} =30\frac{a^{2}_{0}}{\sqrt{10\pi } } \left(-\frac {Q_{0}}{\sqrt{2} }+Q_{2}\sqrt{3} \right) \). (9.389)

Thus, to first-order perturbation theory, the energies of the p level are given by

\(E_{21} =-\frac{R}{4} -30\frac{a^{2}_{0}}{\sqrt{10\pi } } \left (\frac{Q_{0}}{\sqrt{2} }+Q_{2}\sqrt{3} \right) \), (9.390)

 

\(E_{22} =-\frac{R}{4} +30\frac{Q_{0} a^{2}_{0}}{\sqrt{5\pi } } \), (9.391)

 

\(E_{23} =-\frac{R}{4}+30\frac{a^{2}_{0}}{\sqrt{10\pi } } \left (-\frac{Q_{0}}{\sqrt{2} }+Q_{2}\sqrt{3} \right) \). (9.392)

So the quadrupole interaction has lifted all the degeneracies of the p level.
