Guest Question.

Let us determine the voltage \(\mathbf{V}_{o}\) in the circuit in Fig. 8.22a. In this example we will use node equations, loop equations, Th√©venin’s theorem, and Norton’s theorem. We will omit the techniques of superposition and source transformation. Why?

Admin
Solution

1. Nodal Analysis To perform a nodal analysis, we label the node voltages and identify the supernode as shown in Fig. 8.22b. The constraint equation for the supernode is

 

\(\mathbf{V}_{3}+12 \underline{/ 0^{\circ}}=\mathbf{V}_{1}\)

 

and the KCL equations for the nodes of the network are

 

\(\begin{array}{r}\frac{\mathbf{V}_{1}-\mathbf{V}_{2}}{-j 1}+\frac{\mathbf{V}_{3}-\mathbf{V}_{2}}{1}-4 \underline{/ 0^{\circ}}+\frac{\mathbf{V}_{3}-\mathbf{V}_{o}}{1}+\frac{\mathbf{V}_{3}}{j 1}=0 \\\\\frac{\mathbf{V}_{2}-\mathbf{V}_{1}}{-j 1}+\frac{\mathbf{V}_{2}-\mathbf{V}_{3}}{1}-2\left(\frac{\mathbf{V}_{3}-\mathbf{V}_{o}}{1}\right)=0 \\\\4 \underline{/ 0^{\circ}}+\frac{\mathbf{V}_{o}-\mathbf{V}_{3}}{1}+\frac{\mathbf{V}_{o}}{1}=0\end{array}\)

 

At this point we can solve the foregoing equations using a matrix analysis or, for example, substitute the first and last equations into the remaining two equations, which yields

 

\(\begin{aligned}3 \mathbf{V}_{o}-(1+j) \mathbf{V}_{2} &=-(4+j 12) \\\\-(4+j 2) \mathbf{V}_{o}+(1+j) \mathbf{V}_{2} &=12+j 16\end{aligned}\)

 

Solving these equations for \(\mathbf{V}_{o}\) yields

 

\(\begin{aligned}\mathbf{V}_{o} &=\frac{-(8+j 4)}{1+j 2} \\\\&=+4 \underline{/ 143.13^{\circ}} \mathrm{V}\end{aligned}\)

 

2. Loop Analysis The mesh currents for the network are defined in Fig. 8.22c. The constraint equations for the circuit are

 

\(\begin{aligned}&\mathbf{I}_{2}=-4 \underline{/ 0^{\circ}} \\\\&\mathbf{I}_{x}=\mathbf{I}_{4}-\mathbf{I}_{2}=\mathbf{I}_{4}+4 \underline{/ 0^{\circ}} \\\\&\mathbf{I}_{3}=2 \mathbf{I}_{x}=2 \mathbf{I}_{4}+8 \underline{/ 0^{\circ}} \end{aligned}\)

 

The KVL equations for mesh 1 and mesh 4 are

 

\(\begin{aligned}-j 1 \mathbf{I}_{1}+1\left(\mathbf{I}_{1}-\mathbf{I}_{3}\right) &=-12 \underline{/ 0^{\circ}} \\\\ 1\left(\mathbf{I}_{4}-\mathbf{I}_{3}\right)+1\left(\mathbf{I}_{4}-\mathbf{I}_{2}\right)+1 \mathbf{I}_{4} &=0\end{aligned}\)

 

Note that if the constraint equations are substituted into the second KVL equation, the only unknown in the equation is \(\mathbf{I}_{4}\). This substitution yields

 

\(\mathbf{I}_{4}=+4 \underline{/143.13^{\circ}} \mathrm{A}\)

 

and hence,

 

\(\mathbf{V}_{o}=+4 \underline{/143.13^{\circ}} \mathrm{V}\)

 

3. Th√©venin’s Theorem In applying Th√©venin’s theorem, we will find the open-circuit voltage and then determine the Th√©venin equivalent impedance using a test source at the opencircuit terminals. We could determine the Th√©venin equivalent impedance by calculating the short-circuit current; however, we will determine this current when we apply Norton’s theorem.

The open-circuit voltage is determined from the network in Fig. 8.23a. Note that \(\mathbf{I}_{x}^{\prime}=4 \underline{/ 0^{\circ}} \mathrm{A}\) and since \(2 \mathbf{I}_{x}^{\prime}\) flows through the inductor, the open-circuit voltage \(\mathbf{V}_{\mathrm{oc}}\) is

 

\(\begin{aligned}\mathbf{V}_{oc} &=-1\left(4 \underline{/ 0^{\circ}}\right)+j 1\left(2 \mathbf{I}_{x}^{\prime}\right) \\\\&=-4+j 8 \mathrm{~V}\end{aligned}\)

 

To determine the Th√©venin equivalent impedance, we turn off the independent sources, apply a test voltage source to the output terminals, and compute the current leaving the test source. As shown in Fig. 8.23b, since \(\mathbf{I}_{x}^{\prime \prime}\) flows in the test source, KCL requires that

 

the current in the inductor be \(\mathrm{I}_{x}^{\prime \prime}\) also. KVL around the mesh containing the test source indicates that

 

\(j 1 \mathbf{I}_{x}^{\prime \prime}-1 \mathbf{I}_{x}^{\prime \prime}-\mathbf{V}_{\text {test }}=0\)

 

Therefore,

 

\(\mathbf{I}_{x}^{\prime \prime}=\frac{-\mathbf{V}_{\text {test }}}{1-j}\)

 

Then

 

\(\begin{aligned}\mathbf{Z}_{\mathrm{T}_{\mathrm{h}}} &=\frac{\mathbf{V}_{\text {test }}}{-\mathbf{I}_{x}^{\prime \prime}} \\\\&=1-j \Omega\end{aligned}\)

 

If the Th√©venin equivalent network is now connected to the load, as shown in Fig. 8.23c, the output voltage \(\mathbf{V}_{o}\) is found to be

 

\(\begin{aligned}\mathbf{V}_{o} &=\frac{-4+8 j}{2-j 1}(1) \\\\&=+4 \underline{ /143.13^{\circ}} \mathrm{V}\end{aligned}\)

 

4. Norton’s Theorem In using Norton’s theorem, we will find the short-circuit current from the network in Fig. 8.24a. Once again, using the supernode, the constraint and KCL equations are

 

\(\begin{array}{r}\mathbf{V}_{3}+12 \underline{/ 0^{\circ}}=\mathbf{V}_{1} \\\\\frac{\mathbf{V}_{2}-\mathbf{V}_{1}}{-j 1}+\frac{\mathbf{V}_{2}-\mathbf{V}_{3}}{1}-2 \mathbf{I}_{x}^{\prime \prime \prime}=0 \\\\\frac{\mathbf{V}_{1}-\mathbf{V}_{2}}{-j 1}+\frac{\mathbf{V}_{3}-\mathbf{V}_{2}}{1}-4 \underline{/ 0^{\circ}}+\frac{\mathbf{V}_{3}}{j 1}+\mathbf{I}_{x}^{\prime \prime \prime}=0 \\\\\mathbf{I}_{x}^{\prime \prime \prime}=\frac{\mathbf{V}_{3}}{1}\end{array}\)

 

Substituting the first and last equations into the remaining equations yields

 

\(\begin{aligned}(1+j) \mathbf{V}_{2}-(3+j) \mathbf{I}_{x}^{\prime \prime} &=j 12 \\\\-(1+j) \mathbf{V}_{2}+(2) \mathbf{I}_{x}^{\prime \prime \prime} &=4-j 12\end{aligned}\)

 

Solving these equations for \(\mathbf{I}_{x}^{\prime \prime \prime}\) yields

 

\(\mathbf{I}_{x}^{\prime \prime \prime}=\frac{-4}{1+j} \mathrm{~A}\)

 

The KCL equation at the right-most node in the network in Fig. 8.24a is

 

\(\mathbf{I}_{x}^{\prime \prime \prime}=4 \underline{/ 0^{\circ}}+\mathbf{I}_{\mathrm{sc}}\)

 

Solving for \(\mathbf{I}_{\mathrm{sc}}\), we obtain

 

\(\mathbf{I}_{\mathrm{se}}=\frac{-(8+j 4)}{1+j} \mathrm{~A}\)

 

The Th√©venin equivalent impedance was found earlier to be

 

\(\mathbf{Z}_{\mathrm{Th}}=1-j \Omega\)

 

Using the Norton equivalent network, the original network is reduced to that shown in Fig. 8.24b. The voltage \(\mathrm{V}_{o}\) is then

 

\(\begin{aligned}\mathbf{V}_{o} &=\frac{-(8+j 4)}{1+j}\left[\frac{(1)(1-j)}{1+1-j}\right] \\\\&=-4\left[\frac{3-j}{3+j}\right] \\\\&=+4 \underline{/+143.13^{\circ}} \mathrm{V}\end{aligned}\)
