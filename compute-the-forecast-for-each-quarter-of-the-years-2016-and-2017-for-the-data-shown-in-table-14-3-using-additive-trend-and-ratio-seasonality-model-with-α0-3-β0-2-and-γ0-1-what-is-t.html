\(S_{t}=\alpha\left(\frac{D_{t}}{I_{t-L}}\right)+(1-\alpha)\left(S_{t-1}+T_{t-1}\right)\) (14.17)

 

\(T_{t}=\beta\left(S_{t}-S_{t-1}\right)+(1-\beta) T_{t-1}\) (14.15)

 

\(I_{t}=\gamma\left(\frac{D_{t}}{S_{t}}\right)+(1-\gamma) I_{t-L}\) (14.16)

 

\(F_{t+1}=\left(S_{t}+T_{t}\right)\left(I_{t-L+1}\right)\) (14.18)
