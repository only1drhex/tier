Guest Question.

Select a W shape of A992 steel for the beam–column of Figure. This member is part of a braced frame and is subjected to the service-load axial force and bending moments shown (the end shears are not shown). Bending is about the strong axis, and \(K_x = K_y = 1.0\). Lateral support is provided only at the ends. Assume that \(B_1 = 1.0.\)

Admin
Solution

The factored axial load is
\(P_{nt} = P_u = 1.2P_D + 1.6P_L = 1.2(54) + 1.6(147) = 300 kips\)
(There is no amplification of axial loads in members braced against sidesway.) The factored moment at each end is
\(M_{ntx} = 1.2M_D + 1.6M_L = 1.2(18) + 1.6(49) = 100 ft-kips\)
Since \(B_1 = 1.0\), the factored load bending moment is
\(M_{ux} = B_{1}M_{ntx} = 1.0(100) = 100 ft-kips\)
The effective length for compression and the unbraced length for bending are the same:
\(L_{c} = L_{b} = 16 ft\)
The bending moment is uniform over the unbraced length, so \(C_{b}\) = 1.0. Try a W10 shape. From Table 6-2, Try a W10 \(\times\) 60, with \(\phi _{c}P_{n} = 530\) kips and \(\phi _{b}M_{nx}\) = 253 ft-kips.
Determine which interaction equation to use:

\(\frac{P_u}{\phi _{c}P_{n}}=\frac {300}{530} = 0.5660 > 0.2 \therefore \)AISC Equation H1-1a controls.

 

\(\frac{P_u}{\phi _{c}P_{n}}+\frac{8}{9}\left (\frac{M_{ux}}{\phi _{b} M_{nx}} + \frac{M_{uy}}{\phi _{b} M_{ny}} \right )= 0.5660 +\frac{8}{9}\left (\frac{100}{253} +0 \right ) = 0.917 < 1.0 (OK)\)

To be sure that we have found the lightest W10, try the next lighter one, a

W10 \(\times\) 54, with \( \phi _{c}P_{n} = 471\) kips and \( \phi _{b} M_{nx} = 224 ft-kips.\)
From AISC Equation H1-1a,
\(\frac{P_u}{\phi _{c}P_{n}}+\frac{8}{9}\left (\frac{M_{ux}}{\phi _{b} M_{nx}} + \frac{M_{uy}}{\phi _{b} M_{ny}} \right )=\frac{300}{471}+\frac{8}{9}\left (\frac{100}{224} +0 \right )= 1.03 > 1.0 (N.G.)\)
Try a W12 shape. Try a W12 \(\times\) 58, with \(\phi _{c}P_{n}\) = 499 kips and \(\phi _{b} M_{nx} \)= 283 ft-kips.
\(\textbf{Determine which interaction equation to use:}\)
\(\frac{P_u}{\phi _{c}P_{n}}=\frac {300}{499} = 0.6012 > 0.2 \therefore \)AISC Equation H1­1a controls.

\(\frac{P_u}{\phi _{c}P_{n}}+\frac{8}{9}\left (\frac{M_{ux}}{\phi _{b} M_{nx}} + \frac{M_{uy}}{\phi _{b} M_{ny}} \right )=0.6012 +\frac{8}{9}\left (\frac{100}{283} +0 \right )= 0.915 < 1.0 (OK)\)

To be sure that we have found the lightest W12, try the next lighter one, a

W12\(\times \)53, with \(\phi_{c}P_{n} = 453\) kips and \(\phi _{b} M_{nx}= 252 ft-kips.\)
From AISC Equation H1-1a,
\(\frac{P_u}{\phi _{c}P_{n}}+\frac{8}{9}\left (\frac{M_{ux}}{\phi _{b} M_{nx}} + \frac{M_{uy}}{\phi _{b} M_{ny}} \right )=\frac{300}{453} +\frac{8}{9}\left (\frac{100}{252} +0 \right )= 1.01 > 1.0 (N.G.)\)
Check other categories of shapes to be sure that we have found the lightest shape.
The lightest W14 that is a possibility is a W14 \(\times\) 53, with \(\phi _{c}P_{n}= 338\) kips and\(\phi _{b} M_{nx} = 254\) ft-kips.
From AISC Equation H1-1a,
\(\frac{P_u}{\phi _{c}P_{n}}+\frac{8}{9}\left (\frac{M_{ux}}{\phi _{b} M_{nx}} + \frac{M_{uy}}{\phi _{b} M_{ny}} \right )=\frac{300}{338} +\frac{8}{9}\left (\frac{100}{254} +0 \right )= 1.24 > 1.0 (N.G.)\)
There are no other possibilities in the deeper W-shape groups.
Use a W12 \(\times\) 58.
The required axial load strength is
\(P_a = P_D + P_L = 54 + 147 = 201 kips\)
The moment at each end is
\(M_{ntx} = M_D + M_L = 18 + 49 = 67 ft-kips\)
Since \(B_1 = 1.0\), the required bending moment strength is
\(M_{ax} = B_{1}M_{ntx} = 1.0(67) = 67 ft-kips\)
The effective length for compression and the unbraced length for bending are the same:
\(L_c = L_b = 16 ft.\)
The bending moment is uniform over the unbraced length, so \(C_b \)= 1.0. Try a W10 shape. From Table 6-2, try a W10 \(\times\) 60, with\( P_{n}/\Omega _{c} = 352 \)kips and \(M_{nx}/\Omega _{b}\) = 169 ft-kips.

Determine which interaction equation to use:
\(\frac{P_{a}}{P_{n}/\Omega _{c}}=\frac{201}{352} = 0.5710 > 0.2 \therefore \)AISC Equation H1-1a controls.

 

\(\frac{P_{a}}{P_{n}/\Omega _{c}}=\frac{8}{9}\left (\frac{M_{ax}}{ M_{nx}/\Omega _{b}} + \frac{M_{ay}}{ M_{ny}/\Omega _{b}} \right )=0.5710 +\frac{8}{9}\left ( \frac{67}{169}+0 \right ) = 0.923 < 1.0 (OK)\)
To be sure that we have found the lightest W10, try the next lighter one, a
W10 \(\times\) 54, with\( P_{n}/\Omega _{c} \)= 314 kips and \(M_{nx}/\Omega _{b}\) = 149 ft-kips.
From AISC Equation H1-1a,
\(\frac{P_{a}}{P_{n}/\Omega _{c}}=\frac{8}{9}\left (\frac{M_{ax}}{ M_{nx}/\Omega _{b}} + \frac{M_{ay}}{ M_{ny}/\Omega _{b}} \right )=\frac{201}{314} +\frac{8}{9}\left ( \frac{67}{169}+0 \right ) =1.04 > 1.0 (N.G.)\)
Try a W12 shape. Try a W12 \(\times\) 58, with \(P_{n}/\Omega _{c}\) = 332 kips and\( M_{nx}/\Omega _{b}\) = 189 ft-kips.
Determine which interaction equation to use:
\(\frac{P_{a}}{P_{n}/\Omega _{c}}=\frac{201}{332}= 0.6054 > 0.2 \therefore\) AISC Equation H1-1a controls.
\(\frac{P_{a}}{P_{n}/\Omega _{c}}=\frac{8}{9}\left (\frac{M_{ax}}{ M_{nx}/\Omega _{b}} + \frac{M_{ay}}{ M_{ny}/\Omega _{b}} \right )= 0.6054+\frac{8}{9}\left ( \frac{67}{189}+0 \right ) = 0.921 < 1.0 (OK)\)
To be sure that we have found the lightest W12, try the next lighter one, a W12\( \times \)53, with \(P_{n}/\Omega _{c} = 301 kips and M_{nx}/\Omega _{b} =168 ft-kips.\)
From AISC Equation H1-1a,
\(\frac{P_{a}}{P_{n}/\Omega _{c}}=\frac{8}{9}\left (\frac{M_{ax}}{ M_{nx}/\Omega _{b}} + \frac{M_{ay}}{ M_{ny}/\Omega _{b}} \right )=\frac{201}{301} +\frac{8}{9}\left ( \frac{67}{168}+0 \right ) = 1.02 > 1.0 (N.G.)\)
Check other categories of shapes to be sure that we have found the lightest shape. The lightest W14 that is a possibility is a W14 \(\times\)53, with \( P_{n}/\Omega _{c}=\) 225 kips and\( M_{nx}/\Omega _{b} =169 ft-kips.\)

From AISC Equation H1-1a,
\(\frac{P_{a}}{P_{n}/\Omega _{c}}=\frac{8}{9}\left (\frac{M_{ax}}{ M_{nx}/\Omega _{b}} + \frac{M_{ay}}{ M_{ny}/\Omega _{b}} \right )=\frac{201}{225} +\frac{8}{9}\left ( \frac{67}{169}+0 \right ) = 1.25 > 1.0 (N.G.)\) There are no other possibilities in the deeper W-shape groups.

Use a W12 \(\times\) 58.
