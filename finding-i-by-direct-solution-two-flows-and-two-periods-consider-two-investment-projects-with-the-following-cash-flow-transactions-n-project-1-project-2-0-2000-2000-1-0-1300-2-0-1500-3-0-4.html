Finding i* by Direct Solution: Two Flows and Two Periods

Consider two investment projects with the following cash flow transactions:

n Project 1 Project 2
0 -$2,000 – $2,000
1 0 1,300
2 0 1,500
3 0
4 3,500
Admin
Solution

Given: Cash flows for two projects.

Find: i* for each project.

Project 1: Solving for i* in \(PW(i^*) = 0\) is identical to solving \(FW(i^*) = 0\), because \(FW\) equals \(PW\) times a constant. We could do either here, but we will set \(FW(i^*) = 0\) to demonstrate the latter. Using the single-payment future-worth relationship, we obtain

\(FW(i^*) = -\$ 2,000(F/P, i^*, 4) + \$ 3,500 = 0\) ,

\(\$ 3,500 = \$ 2,000(F/P, i^*, 4) = \$ 2,000(1 + i^*)^4\) ,

\(1.75 = (1 + i^*)^4\) .

Solving for i* yields

\(i^* =\sqrt[4]{1.75} -1 \)

 

\(= 0.1502\) or \(15.02\%\).

Project 2: We may write the \(NPW\) expression for this project as

\(PW(i) = -\$ 2,000 +\frac{\$1,300}{(1 + i)}+\frac{\$1,500}{(1 + i)^2}=0\).

Let \(X = 1/(1 + i).We may then rewrite \)PW(i) as a function of \(X\) as follows:

\(PW(X) = -\$ 2,000 + \$ 1,300X + \$ 1,500X^2 = 0\).

This is a quadratic equation that has the following solution:\(^4\)

\(X =\frac{-1,300\pm \sqrt{1,300^2 - 4(1,500)(-2,000)} }{2(1,500)}\)

 

\(=\frac{-1,300\pm 3,700}{3,000}\)

 

\(= 0.8\) or \(-1.667\).

Replacing \(X\) values and solving for \(i\) gives us

\(0.8 =\frac{1}{(1+i)} \longrightarrow i = 25\%\) ,

\(-1.667 =\frac{1}{(1+i)} \longrightarrow -160\%\) .

Since an interest rate less than \(-100\%\) has no economic significance, we find that the projectâ€™s i* is \(25\%\).

COMMENTS: In both projects, one sign change occurred in the net cash flow series, so we expected a unique i* . Also, these projects had very simple cash flows. When cash flows are more complicated, generally we use a trial-and-error method or a computer to find i*.

 

\(^4\) The solution of the quadratic equation \(aX^2 + bX + c = 0\) is \(X =\frac{-b \pm \sqrt{b^2-4ac}}{2a}\)
