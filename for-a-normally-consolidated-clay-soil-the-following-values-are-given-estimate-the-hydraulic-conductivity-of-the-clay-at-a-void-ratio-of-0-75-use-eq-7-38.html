Guest Question.

For a normally consolidated clay soil, the following values are given:

Void ratio k (cm/sec)
1.1 \(0.302 \times 10^{-7} \)
0.9 \(0.12 \times 10^{-7} \)

Estimate the hydraulic conductivity of the clay at a void ratio of 0.75. Use Eq. (7.38).

Admin
Solution

From Eq. (7.38),

\(k=C(\frac{e^n}{1+e} )\)

\(\frac{k_1}{k_2}=\frac{(\frac{e^{n}_{1} }{1+e_1} )}{(\frac{e^{n}_{2} }{1+e_2} )}\)

\(\frac{0.302 \times 10^{-7}}{0.12 \times 10^{-7}} =\frac{\frac{(1.1)^n}{1+1.1} }{\frac{(0.9)^n}{1+0.9} } \)

2.517 \(= (\frac{1.9}{2.1} )(\frac{1.1}{0.9} )^n\)

2.782 \(= (1.222 )^n\)

n \(= \frac{\log (2.782)}{\log (1.222)}=\frac{0.444}{0.087} = 5.1 \)

so

\(k=C(\frac{e^{5.1} }{1+e} )\)

To find C,

\(0.302 \times 10^{-7} = C\left[\frac{(1.1)^{5.1} }{1+1.1} \right] = (\frac{1.626}{2.1} )C\)

C \(= \frac{(0.302 \times 10^{-7})(2.1)}{1.626} = 0.39 \times 10^{-7}\)

Hence,

\(k = (0.39 \times 10^{-7} cm/sec )(\frac{e^n}{1+e} )\)

At a void ratio of 0.75,

\(k = (0.39 \times 10^{-7})(\frac{0.75^{5.1} }{1+0.75} ) = 0.514 \times 10^{-8} cm/sec\)
