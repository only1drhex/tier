Guest Question.

A single-line diagram of a four-bus system is shown in Figure 7.20, for which \(Z _{\text {bus }}\) is given below:

\(Z _{ bus }=j\left[\begin{array}{llll}0.25 & 0.2 & 0.16 & 0.14 \\0.2 & 0.23 & 0.15 & 0.151 \\0.16 & 0.15 & 0.196 & 0.1 \\0.14 & 0.151 & 0.1 & 0.195\end{array}\right] \text { per unit }\)

Let a three-phase fault occur at bus 2 of the network.
(a) Calculate the initial symmetrical rms current in the fault.
(b) Determine the voltages during the fault at buses 1, 3, and 4.
(c) Compute the fault currents contributed to bus 2 by the adjacent unfaulted buses 1, 3, and 4.
(d) Find the current flow in the line from bus 3 to bus 1. Assume the prefault voltage \(V_{f}\) at bus 2 to be \(1 \angle 0^{\circ} \text { p.u. }\), and neglect all prefault currents.

Admin
Solution

(a) \(\bar{I}_{f}^{\prime \prime}=\frac{1.0}{\bar{Z}_{22}}=\frac{1.0}{j 0.23}=-j 4.348 pu \leftarrow \text { Due to the fault }\)

Note: Because load currents are neglected, the prefault voltage at each bus is \(1.0 \angle 0^{\circ} pu\) , the same as \(\bar{V}_{f}\) at bus 2.

 

(b) Voltages during the fault are calculated below:

\(\left[\begin{array}{c}\bar{V}_{1} \\\bar{V}_{2} \\\bar{V}_{3}\\\bar{V}_{4}\end{array}\right]=\left[\begin{array}{c}1-\frac{j 0.2}{j 0.23} \\0 \\1-\frac{j 0.15}{j 0.23} \\1-\frac{j 0.151}{j 0.23}\end{array}\right]=\left[\begin{array}{c}0.134 \\0 \\0.3478 \\0.3435\end{array}\right] pu \leftarrow\)

 

(c) Current flow in line 3-1 is

\(\bar{I}_{31}=\frac{\bar{V}_{3}-\bar{V}_{1}}{\bar{Z}_{3-1}}=\frac{0.3478-0.1304}{j 0.25}=-j 0.8696 pu \leftarrow\)

 

(d) Fault currents contributed to bus 2 by the adjacent unfaulted buses are calculated below:

From Bus 1: \(\frac{\bar{V}_{1}}{\bar{Z}_{2-1}}=\frac{0.1304}{j 0.125}=-j 1.0432 \leftarrow\)

From Bus 3: \(\frac{\bar{V}_{3}}{\bar{Z}_{2-3}}=\frac{0.3478}{j 0.25}=-j 1.3912 \leftarrow\)

From Bus 4: \(\frac{\bar{V}_{4}}{\bar{Z}_{2-4}}=\frac{0.3435}{j 0.2}=-j 1.7175 \leftarrow\)

Which is approximately same as \(\bar{I}_{f}^{\prime \prime}\).
