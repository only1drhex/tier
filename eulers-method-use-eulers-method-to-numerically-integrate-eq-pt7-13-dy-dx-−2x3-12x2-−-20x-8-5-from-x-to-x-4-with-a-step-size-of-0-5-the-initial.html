Euler’s Method

Use Euler’s method to numerically integrate Eq. (PT7.13):

\(\frac{dy}{dx} = −2x^3 + 12x^2 − 20x + 8.5\)

from x = to x = 4 with a step size of 0.5. The initial condition at x = 0 is y = 1. Recall that the exact solution is given by Eq. (PT7.16):

\(y = −0.5x^4 + 4x^3 − 10x^2 + 8.5x + 1\)
Admin
Solution

Equation (25.2) can be used to implement Euler’s method:

\(y(0.5) = y(0) + f(0, 1)0.5\)

 

\( y_{i+1}=y_i + f(x_i, y_i)h\) (25.2)

where y(0) = 1 and the slope estimate at x = 0 is

\(f(0, 1) = −2(0)^3 + 12(0)^2 − 20(0) + 8.5 = 8.5\)

Therefore,

\(y(0.5) = 1.0 + 8.5(0.5) = 5.25\)

The true solution at x = 0.5 is

\(y = −0.5(0.5)^4 + 4(0.5)^3 − 10(0.5)^2 + 8.5(0.5) + 1 = 3.21875\)

Thus, the error is

\(E_t = true − approximate = 3.21875 − 5.25 = −2.03125\)

or, expressed as percent relative error, \(\varepsilon _{t}= −63.1\)%. For the second step,

\(y(1) = y(0.5) + f(0.5, 5.25)0.5\)

\(= 5.25 + [−2(0.5)^3 + 12(0.5)^2 − 20(0.5) + 8.5]0.5\)

\(= 5.875\)

The true solution at x = 1.0 is 3.0, and therefore, the percent relative error is −95.8%.
The computation is repeated, and the results are compiled in Table 25.1 and Fig. 25.3.
