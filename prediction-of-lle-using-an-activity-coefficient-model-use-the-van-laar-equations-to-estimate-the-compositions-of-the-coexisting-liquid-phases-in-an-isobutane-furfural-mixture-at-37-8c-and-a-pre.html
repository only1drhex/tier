Guest Question.

Prediction of LLE Using an Activity Coefficient Model

Use the van Laar equations to estimate the compositions of the coexisting liquid phases in an isobutane-furfural mixture at 37.8°C and a pressure of 5 bar. (You may assume that the van Laar constants for this system given in Table 9.5-1 are applicable at this pressure.)

Admin
Solution

The compositions of the coexisting phases are the solution of the following set of algebraic equations:

 

\(x_{1}^{ I } \gamma_{1}^{ I }=x_{1}^{ I } \exp \left\{\frac{\alpha}{\left[1+\frac{\alpha x_{1}^{ I }}{\beta\left(1-x_{1}^{ I }\right)}\right]^{2}}\right\}=x_{1}^{ II } \exp \left\{\frac{\alpha}{\left[1+\frac{\alpha x_{1}^{ II }}{\beta\left(1-x_{1}^{ II }\right)}\right]^{2}}\right\}=x_{1}^{ II } \gamma_{1}^{ II }\) (a)

 

\(x_{2}^{ I } \gamma_{2}^{ I }=x_{2}^{ I } \exp \left\{\frac{\beta}{\left[1+\frac{\beta x_{2}^{ I }}{\alpha\left(1-x_{2}^{ I }\right)}\right]^{2}}\right\}=x_{2}^{ II } \exp \left\{\frac{\beta}{\left[1+\frac{\beta x_{2}^{ II }}{\alpha\left(1-x_{2}^{ II }\right)}\right]^{2}}\right\}=x_{2}^{ II } \gamma_{2}^{ II }\) (b)

 

\(x_{1}^{ I }+x_{2}^{ I }=1\) (c)

 

\(x_{1}^{ II }+x_{2}^{ II }=1\) (d)

 

Here isobutane will be taken as species 1 and furfural as species 2; thus, from Table 9.5-1, α = 2.62 and β = 3.02. These equations may be solved directly using an equation-solving program such as MATHCAD. Alternatively, the following graphical procedure can be used in solving the equations:

 

i. A guess is made for the value of \(x_{1}^{ I }\).

ii. Equation a is then used to compute \(x_{1}^{ II }, \text { and Eq. c to compute } x_{2}^{ I }\).

iii. \(x_{2}^{ II }\) is then be computed using Eq. b.

iv. Finally, the check whether Eq. d is satisfied with the computed values of \(x_{1}^{ II } \text { and } x_{2}^{ II }\). If it is, we have found the solution to the equations. If Eq. d is not satisfied, a new value of \(x_{1}^{ I }\) is chosen, and the calculation is repeated.

The difficult part of this computation is the solution of Eqs. a and b for \(x_{1}^{ II } \text { and } x_{2}^{ II }\). One way to solve these equations, if one does not wish to use an equation-solving program, is as follows. First, the values of the product \(x_{1} \gamma_{1}\) are computed for various values of \(x_{1}\) (see the table that follows). From a graph of \(x_{1} \gamma_{1} \text { versus } x_{1}\) (see Fig. 1), we can easily obtain the value of (see Fig. 1), we can easily obtain the value of \(x_{1}\gamma_{1}\) forany choice of \(x_{1}\) and then locate other values of \(x_{1}\) that have the same value of \(x_{1} \gamma_{1}\). Thus, given the trial value of \(x_{1}^{ I }, x_{1}^{ II }\) is quickly found. In a similar fashion, computing and plotting \(x_{2} / 2\) as a function of \(x_{2}\left(\text { or } x_{1}=1-x_{2}\right)\) provides an easy method of finding \(x_{2}^{ II } \text { given } x_{2}^{ I }=1-x_{1}^{ I }\). We then need to check whether Eq. d is satisfied. If not, the calculation is repeated with another guess for \(x_{1}^{ I }\).

Using MATHCAD, we obtain the following compositions for the coexisting liquid phases:

 

\(x_{1}^{ I }=0.1128 \quad x_{1}^{ II }=0.9284\)

 

and

 

\(x_{2}^{ I }=0.8872 \quad x_{2}^{ II }=0.0716\)

 

At these compositions the activity coefficients are

 

\(\begin{array}{ll}\gamma_{1}^{ I }=8.375 & \gamma_{1}^{ II }=1.018 \\\gamma_{2}^{ I }=1.030 & \gamma_{2}^{ II }=12.77\end{array}\)

 

and

 

\(\begin{array}{l}x_{1}^{ I } \gamma_{1}^{ I }=0.1128 \times 8.375=0.945=x_{1}^{ II } \gamma_{1}^{ II }=0.9284 \times 1.018 \\x_{2}^{ I } \gamma_{2}^{ I }=0.8872 \times 1.030=0.914=x_{2}^{ II } \gamma_{2}^{ II }=0.0716 \times 12.77\end{array}\)

 

There are no experimental liquid-liquid phase equilibrium data for the isobutane-furfural system with which we can compare our predictions. However, binary mixtures of furfural and, separately, 2,2-dimethyl pentane, 2-methyl pentane, and hexane, which are similar to the binary mixture here, all exhibit liquid-liquid phase separation,\(^{6}\) with one phase containing between 89 and 90 mol % furfural \(\left(x_{2}^{ I }\right)\) and the other between 6 and 7 mol % furfural \(\left(x_{2}^{ II }\right)\).
