Guest Question.

(a) Show how \(\hat{J}_{x} \) and \(\hat{J}_{y} \) transform under a rotation of (finite) angle α about the z-axis.

Using these results, determine how the angular momentum operator \(\hat{\vec{J}}\) transform under the rotation.

(b) Show how a vector operator \(\hat{\vec{A}}\) transforms under a rotation of angle α about the y-axis.

(c) Show that \(e^{i\pi \hat{J}_{z} /\hbar } e^{i\alpha \hat{J}_{y} /\hbar } e^{-i\pi \hat{J}_{z} /\hbar}= e^{-i\alpha \hat{J}_{y} /\hbar }\).

Admin
Solution

(a) The operator corresponding to a rotation of angle α about the z-axis is given by \(\hat{R}_{z}(\alpha )=e^{-i\alpha \hat{J}_{z} /\hbar }\). Under this rotation, an operator \(\hat{B} \) transforms like \(\hat{B}^{\prime } =\hat{R}^{\dagger }_{z} \hat {B} \hat{R}_{z}=e^{i\alpha \hat{J}_{z} /\hbar }\hat{B} e^{-i\alpha \hat{J}_{z} /\hbar}\).

Using the relation

\(e^{\hat{A}}\hat{B} e^{-\hat{A}} =\hat{B}+\left[\hat {A},\hat {B}\right] +\frac{1}{2!} \left[\hat{A},\left[\hat{A},\hat{B}\right] \right] +\frac{1}{3!}\left[\hat{A},\left[\hat{A},\left[\hat{A},\hat{B}\right] \right] \right] +… \), (7.338)

along with the commutation relations \(\left[\hat{J}_{z}, \hat {J}_{y}\right] =-i\hbar \hat{J}_{x} \) and \(\left[\hat {J}_{z},\hat{J}_{x}\right] =i\hbar \hat{J}_{y} \), we have

\(e^{i\alpha \hat{J}_{z} /\hbar }\hat{J}_{x} e^{-i\alpha \hat {J}_{z} /\hbar}=\hat{J}_{x} +\frac{i\alpha }{\hbar } \left[\hat {J}_{z},\hat{J}_{x}\right] -\frac{\alpha ^{2} }{2!\hbar ^{2} } \left [\hat{J}_{z},\left[\hat{J}_{z},\hat{J}_{x}\right] \right] \)

 

\(-\frac{i\alpha ^{3} }{3!\hbar ^{3} }\left[\hat{J}_{z},\left[\hat {J}_{z},\left[\hat{J}_{z},\hat{J}_{x}\right] \right] \right] +… \)

 

\(=\hat{J}_{x} -\alpha \hat{J}_{y} -\frac{\alpha ^{2} }{2!}\hat {J}_{x}+\frac{\alpha ^{3} }{3!}\hat{J}_{y}+\frac{\alpha ^{4} }{4!} \hat{J}_{x}-\frac{\alpha ^{5} }{5!}\hat{J}_{y}+… \)

 

\(=\hat{J}_{x}\left(1-\frac{\alpha ^{2} }{2!}+\frac{\alpha ^{4} }{4!}+…\right) -\hat{J}_{y}\left(\alpha -\frac{\alpha ^{3} }{3!}+\frac {\alpha ^{5} }{5!}-…\right) \)

 

\(=\hat{J}_{x}\cos \alpha -\hat{J}_{y}\sin \alpha \). (7.339)

Similarly, we can show that

\(e^{i\alpha \hat{J}_{z} /\hbar }\hat{J}_{y}e^{-i\alpha \hat {J}_{z} /\hbar }=\hat{J}_{y}\cos \alpha -\hat{J}_{x}\sin \alpha \). (7.340)

As \(\hat{J}_{z}\) is invariant under an arbitrary rotation about the z-axis \((e^{i\alpha \hat{J}_{z} /\hbar }\hat {J}_{z} e^{-i\alpha \hat{J}_{z} /\hbar }=\hat{J}_{z})\), we can condense equations (7.339) and (7.340) into a single matrix relation:

\(e^{i\alpha \hat{J}_{z} /\hbar }\hat{\vec{J}}e^{-i\alpha \hat {J}_{z} /\hbar }=\left(\begin{matrix} \cos \alpha & -\sin \alpha & 0 \\ \sin \alpha & \cos \alpha & 0 \\ 0 & 0 & 1 \end{matrix} \right) \left (\begin{matrix} \hat{J}_{x} \\ \hat{J}_{y} \\ \hat{J}_{z} \end {matrix} \right) \). (7.341)

(b) Using the commutation relations \(\left[\hat{J}_{y},\hat {A}_{x}\right] =-i\hbar \hat{A}_{z}\) and \( \left[\hat {J}_{y},\hat{A}_{z}\right] =i\hbar \hat{A}_{x}\) (see (7.282) to (7.284))

\(\left[\hat{A}_{x},\hat{J}_{x}\right] =\left[\hat{A}_{y},\hat {J}_{y}\right] =\left[\hat{A}_{z},\hat{J}_{z}\right] =0 \), (7.282)

 

\(\left[\hat{A}_{x},\hat{J}_{y}\right] =i\hbar \hat{A}_{z}, \left[\hat{A}_{y},\hat{J}_{z}\right] =i\hbar \hat{A}_{x}, \left [\hat{A}_{z},\hat{J}_{x}\right] =i\hbar \hat{A}_{y} \), (7.283)

 

\(\left[\hat{A}_{x},\hat{J}_{z}\right] =-i\hbar \hat{A}_{y}, \left[\hat{A}_{y},\hat{J}_{x}\right] =-i\hbar \hat{A}_{z}, \left [\hat{A}_{z},\hat{J}_{y}\right] =-i\hbar \hat{A}_{x} \). (7.284)

along with (7.338), we have

\(e^{i\alpha \hat{J}_{y} /\hbar }\hat{A}_{x}e^{-i\alpha \hat {J}_{y} /\hbar }=\hat{A}_{x}+\frac{i\alpha }{\hbar } \left[\hat {J}_{y},\hat{A}_{x}\right] -\frac{\alpha ^{2} }{2!\hbar ^{2} } \left [\hat{J}_{y},\left[\hat{J}_{y},\hat{A}_{x}\right] \right] \)

 

\(-\frac{i\alpha ^{3} }{3!\hbar ^{3} }\left[\hat{J}_{y},\left[\hat {J}_{y},\left[\hat{J}_{y},\hat{A}_{x}\right] \right] \right] +… \)

 

\(=\hat{A}_{x} +\alpha \hat{A}_{z} -\frac{\alpha ^{2} }{2!}\hat {A}_{x}-\frac{\alpha ^{3} }{3!}\hat{A}_{z}+\frac{\alpha ^{4} }{4!} \hat{A}_{x}+\frac{\alpha ^{5} }{5!}\hat{J}_{z}+… \)

 

\(=\hat{A}_{x}\left(1-\frac{\alpha ^{2} }{2!}+\frac{\alpha ^{4} }{4!}+…\right) +\hat{A}_{z}\left(\alpha -\frac{\alpha ^{3} }{3!}+\frac {\alpha ^{5} }{5!}+…\right) \)

 

\(=\hat{A}_{x}\cos \alpha +\hat{A}_{z}\sin \alpha \). (7.342)

Similarly, we can show that

\(\hat{A}^{\prime }_{z} =e^{i\alpha \hat{J}_{y} /\hbar }\hat {A}_{z}e^{-i\alpha \hat{J}_{y} /\hbar }=-\hat{A}_{x}\sin \alpha + \hat{A}_{z}\cos \alpha \). (7.343)

Also, since \(\hat{A}_{y} \) is invariant under an arbitrary rotation about the y-axis, we may combine equations (7.342) and (7.343) to find the vector operator \(\hat{\vec{A}^{\prime } }\) obtained by rotating \(\hat{\vec{A}} \) through an angle α about the y-axis:

\(\hat{\vec{A}^{\prime } }=e^{i\alpha \hat{J}_{y} /\hbar }\hat {\vec{A}} e^{-i\alpha \hat{J}_{y} /\hbar }=\left(\begin{matrix} \cos \alpha & 0 & \sin \alpha \\ 0 & 1 & 0 \\ -\sin \alpha & 0 & \cos \alpha \end{matrix} \right) \left(\begin{matrix} \hat{A}_{x} \\ \hat{A}_{y} \\ \hat{A}_{z} \end{matrix} \right) \). (7.344)

(c) Expanding \(e^{i\alpha \hat{J}_{y} /\hbar }\) and then using (7.340), we obtain

\(e^{i\pi \hat{J}_{z} /\hbar } e^{i\alpha \hat{J}_{y} /\hbar } e^{-i\pi \hat{J}_{z} /\hbar}=\sum\limits_{n=0}^{\infty }{\frac{(i \alpha /\hbar )^{n} }{n!} e^{i\pi \hat{J}_{z} /\hbar }\left(\hat {J}_{y}\right) ^{n} e^{-i\pi \hat{J}_{z} /\hbar}} \)

 

\(=\sum\limits_{n=0}^{\infty }\frac{(i\alpha /\hbar )^{n} }{n!} \left(\hat{J}_{y}\cos \pi +\hat{J}_{x}\sin \pi \right) ^{n} = \sum \limits_{n=0}^{\infty }\frac{(i\alpha /\hbar )^{n} }{n!}\left(\hat {J}_{y}\right) ^{n} \)

 

\(=e^{-i\alpha \hat{J}_{y} /\hbar } \). (7.345)
