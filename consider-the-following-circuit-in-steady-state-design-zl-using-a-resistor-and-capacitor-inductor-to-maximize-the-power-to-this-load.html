Guest Question.

Consider the following circuit in steady state.

Design \(Z_{l}\) using a resistor and capacitor/inductor to maximize the power to this load.

Admin
Solution

In the phasor domain, the circuit is as follows.

Using mesh analysis and noting that \(\underline{i}_{x}=\underline{i}_{a}, \mathrm{KVL}\) in mesh \(a\) can be written as

  • \(\operatorname{KVL}(a):-100-j 100+j 200 \underline{i}_{a}+100 \underline{i}_{x}+100 \underline{i}_{a}=0 \longrightarrow \underline{i}_{a}=1 / 2 \mathrm{~A}\).

We also note that \(\underline{i}_{a}\) does not depend on whether an open or short circuit is used for \(Z_{l}\). For the open-circuit case, KVL in mesh \(b\) can be derived as

  • \(\mathrm{KVL}(b):-100 \underline{i}_{x}+j 200 \underline{i}_{b}-j 100 \underline{i}_{b}+100 \underline{i}_{b}=0\),

leading to

\(\underline{i}_{b}=\frac{\underline{i}_{a}}{1+j}=\frac{1-j}{4} \mathrm{~A} . \)

Therefore, the open-circuit voltage can be found to be

\(\underline{v}_{\mathrm{oc}}=-j 100 \times \underline{i}_{b}=(-25-j 25) \mathrm{V} .\)

For the short-circuit case, we have

  • \(\operatorname{KVL}(b):-100 \underline{i}_{x}+j 200 \underline{i}_{b}+100 \underline{i}_{b}=0\),

where the capacitor impedance is omitted due to the short circuit. Then the value of \(\underline{i}_{b}\) can be found to be

\(\underline{i}_{b}=\frac{\underline{i}_{a}}{1+j 2}=\frac{1-j 2}{10} \mathrm{~A} .\)

This value corresponds to the short-circuit current, that is,

\(\underline{i}_{sc}=\underline{i}_{b}=\frac{1-j 2}{10} \mathrm{~A} .\)

Therefore, the Th√©venin impedance is

\(Z_{\mathrm{th}}=\frac{-25-j 25}{(1-j 2) / 10}=-250 \frac{1+j}{1-j 2}=-50(1+j)(1+j 2)=50(1-j 3) \Omega .\)

Hence, in order to maximize the transferred power, one needs to select

\(Z_{l}=Z_{\text {th }}^{*}=50(1+j 3) \Omega .\)

Such a load can be designed by connecting a \(50 \Omega\) resistor in series to a \(150 / \omega=0.15 \mathrm{H}\) inductor.
