Guest Question.

Calculate the density of \(Cl_2\) at STP.

Admin
Solution

READ To find the density, we need the molar mass of \(Cl_2\). Using the periodic table, we find it to be 70.90 g/mol.

PLAN The unit for density for gases is g/L.

Solution map: g/mol â†’ g/L

SETUP 1 mol = 22.4 L, so the conversion factor is 1 mol /22.4 L.

CALCULATE \((\frac{70.90 g}{1 \cancel{mol}} ) (\frac{1 \cancel{mol}}{22.4 L} ) \) = 3.165 g/L

CHECK Compare the answer 3.179/L to \(Cl_2\) density in Table 12.4.
