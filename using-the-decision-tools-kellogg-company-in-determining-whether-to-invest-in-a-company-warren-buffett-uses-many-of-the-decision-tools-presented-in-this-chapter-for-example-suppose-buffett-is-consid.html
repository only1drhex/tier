USING THE DECISION TOOLS Kellogg Company

In determining whether to invest in a company, Warren Buffett uses many of the decision tools presented in this chapter. For example, suppose Buffett is considering investing in Kellogg Company. The condensed financial statements of Kellogg Company for two recent years are presented here.

Solution

1. Liquidity
a. Current ratio:

Current year: \(\frac{\$ 3,267}{\$ 3,835}=0.85: 1 \) Prior year: \(\quad 0.75: 1\)

b. Inventory turnover:

\(\text { Current year: } \frac{\$ 8,689}{(\$ 1,248+\$ 1,365) / 2}=6.7 \text { times } \quad\qquad \text { Prior year: } \quad 6.7 \text { times }\)

We see that the current ratio increased. However, a current ratio below \(1: 1\) could be problematic. The inventory turnover remained constant. Combined these ratios suggest the company’s liquidity was unchanged or slightly improved.

 

2. Solvency
a. Debt to assets ratio:

\(\text { Current year: } \frac{\$ 3,835+\$ 8,032}{\$ 15,474}=77 \% \quad\qquad \text { Prior year: } 84 \%\)

b. Times interest earned:

\(\text { Current year: } \frac{\$ 1,807+\$ 792+\$ 235}{\$ 235}=12.1 \text { times } \quad \qquad\text { Prior year: } 6.1 \text { times }\)

Kellogg’s debt to assets ratio decreased, suggesting its solvency improved. Also, its times interest earned increased, suggesting its solvency improved.

 

3. Profitability
a. Return on assets:

\(\text { Current year: } \frac{\$ 1,807}{(\$ 15,474+\$ 15,169) / 2}=11.8 \% \quad\qquad \text { Prior year: } \quad 7.1 \%\)

b. Profit margin:

\(\text { Current year: } \frac{\$ 1,807}{\$ 14,792}=12.2 \% \qquad\quad \text { Prior year: } \quad 6.8 \%\)

c. Return on common stockholders’ equity:

\(\text { Current year: } \frac{\$ 1,807}{(\$ 3,607+\$ 2,465) / 2}=60 \% \quad\qquad \text { Prior year: } \quad 45 \%\)

 

Kellogg’s return on assets, profit margin, and return on stockholders’ equity all increased. This suggests a broad improvement in profitability.
