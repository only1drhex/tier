Guest Question.

Let us determine the current \(\mathbf{I}_{o}\) in the network in Fig. 8.17a using nodal analysis, loop analysis, superposition, source exchange, Thévenin’s theorem, and Norton’s theorem.

Admin
Solution

1. Nodal Analysis We begin with a nodal analysis of the network. The KCL equation for the supernode that includes the voltage source is

 

\(\frac{\mathbf{V}_{1}}{1+j}-2 \underline {/ 0^{\circ}}+\frac{\mathbf{V}_{2}}{1}+\frac{\mathbf{V}_{2}}{1-j}=0\)

 

and the associated \(\mathrm{KVL}\) constraint equation is

 

\(\mathbf{V}_{1}+6 \underline {/ 0^{\circ}}=\mathbf{V}_{2}\)

 

Solving for \(\mathbf{V}_{1}\) in the second equation and using this value in the first equation yields

 

\(\frac{\mathbf{V}_{2}-6 \underline {/ 0^{\circ}}}{1+j}-2 \underline {/ 0^{\circ}}+\mathbf{V}_{2}+\frac{\mathbf{V}_{2}}{1-j}=0\)

 

or

 

\(\mathbf{V}_{2}\left[\frac{1}{1+j}+1+\frac{1}{1-j}\right]=\frac{6+2+2 j}{1+j}\)

Solving for \(\mathbf{V}_{2}\), we obtain

 

\(\mathbf{V}_{2}=\left(\frac{4+j}{1+j}\right) \mathrm{V}\)

 

Therefore,

 

\(\mathbf{I}_{o}=\frac{4+j}{1+j}=\left(\frac{5}{2}-\frac{3}{2} j\right) \mathrm{A}\)

 

2. Loop Analysis The network in Fig. 8.17b is used to perform a loop analysis. Note that one loop current is selected that passes through the independent current source. The three loop equations are

 

\(\begin{aligned}\mathbf{I}_{1} &=-2 \underline{/ 0^{\circ}} \\\\1\left(\mathbf{I}_{1}+\mathbf{I}_{2}\right)+j 1\left(\mathbf{I}_{1}+\mathbf{I}_{2}\right)-6 \underline{/ 0^{\circ}}+1\left(\mathbf{I}_{2}+\mathbf{I}_{3}\right)-j 1\left(\mathbf{I}_{2}+\mathbf{I}_{3}\right) &=0 \\\\1 \mathbf{I}_{3}+1\left(\mathbf{I}_{2}+\mathbf{I}_{3}\right)-j 1\left(\mathbf{I}_{2}+\mathbf{I}_{3}\right) &=0\end{aligned}\)

 

Combining the first two equations yields

 

\(\mathbf{I}_{2}(2)+\mathbf{I}_{3}(1-j)=8+2 j\)

 

The third loop equation can be simplified to the form

 

\(\mathbf{I}_{2}(1-j)+\mathbf{I}_{3}(2-j)=0\)

 

Solving this last equation for \(\mathbf{I}_{2}\) and substituting the value in to the previous equation yields

 

\(\mathbf{I}_{3}\left[\frac{-4+2 j}{1-j}+1-j\right]=8+2 j\)

 

or

 

\(\mathbf{I}_{3}=\frac{-10+6 j}{4}\)

 

and finally

 

\(\mathbf{I}_{o}=-\mathbf{I}_{3}=\left(\frac{5}{2}-\frac{3}{2} j\right) \mathrm{A}\)

 

3. Superposition In using superposition, we apply one independent source at a time. The network in which the current source acts alone is shown in Fig. 8.18a. By combining the two parallel impedances on each end of the network, we obtain the circuit in Fig. 8.18b, where

 

\(\mathbf{Z}^{\prime}=\frac{(1+j)(1-j)}{(1+j)+(1-j)}=1 \Omega\)

 

Therefore, using current division

 

\(\mathbf{I}_{o}^{\prime}=1 \angle 0^{\circ} \mathrm{A}\)

 

The circuit in which the voltage source acts alone is shown in Fig. 8.18c. The voltage \(\mathbf{V}_{1}^{\prime \prime}\) obtained using voltage division is

 

\(\begin{aligned}\mathbf{V}_{1}^{\prime \prime} &=\frac{\left(6 \underline{/0^{\circ}}\right)\left[\frac{1(1-j)}{1+1-j}\right]}{1+j+\left[\frac{1(1-j)}{1+1-j}\right]} \\\\&=\frac{6(1-j)}{4} \mathrm{~V} \end{aligned}\)

 

and hence,

 

 

\(\mathbf{I}_{0}^{\prime \prime}=\frac{6}{4}(1-j) \mathrm{A}\)

 

 

Then

 

\(\mathbf{I}_{o}=\mathbf{I}_{o}^{\prime}+\mathbf{I}_{o}^{\prime \prime}=1+\frac{6}{4}(1-j)=\left(\frac{5}{2}-\frac{3}{2} j\right) \mathrm{A}\)

 

4. Source Exchange As a first step in the source exchange approach, we exchange the current source and parallel impedance for a voltage source in series with the impedance, as shown in Fig. 8.19a.

 

Adding the two voltage sources and transforming them and the series impedance into a current source in parallel with that impedance are shown in Fig. 8.19b. Combining the two impedances that are in parallel with the 1-Ω resistor produces the network in Fig. 8.19c, where

 

\(\mathbf{Z}=\frac{(1+j)(1-j)}{1+j+1-j}=1 \Omega\)

 

Therefore, using current division,

 

\(\begin{aligned}\mathbf{I}_{o} &=\left(\frac{8+2 j}{1+j}\right)\left(\frac{1}{2}\right)=\frac{4+j}{1+j} \\\\&=\left(\frac{5}{2}-\frac{3}{2} j\right) \mathrm{A}\end{aligned}\)

 

5. Thévenin Analysis In applying Thévenin’s theorem to the circuit in Fig. 8.17a, we first find the open-circuit voltage, \(\mathbf{V}_{\text {oc }}\), as shown in Fig. 8.20a. To simplify the analysis, we perform a source exchange on the left end of the network, which results in the circuit in Fig. 8.20b. Now using voltage division,

 

\(\mathbf{V}_{\mathrm{oc}}=[6+2(1+j)]\left[\frac{1-j}{1-j+1+j}\right]\)

 

or

 

\(\mathbf{V}_{\mathrm{oc}}=(5-3 j) \mathrm{V}\)

 

 

The Thévenin equivalent impedance, \(\mathbf{Z}_{\mathrm{Th}}\), obtained at the open-circuit terminals when the current source is replaced with an open circuit and the voltage source is replaced with a short circuit is shown in Fig. 8.20c and calculated to be

 

\(\mathbf{Z}_{\mathrm{Th}}=\frac{(1+j)(1-j)}{1+j+1-j}=1 \Omega\)

 

 

Connecting the Thévenin equivalent circuit to the 1-Ω resistor containing \(\mathbf{I}_{o}\) in the original network yields the circuit in Fig. 8.20d. The current \(\mathbf{I}_{o}\) is then

 

\(\mathbf{I}_{o}=\left(\frac{5}{2}-\frac{3}{2} j\right) \mathrm{A}\)

 

6. Norton Analysis Finally, in applying Norton’s theorem to the circuit in Fig. 8.17a, we calculate the short-circuit current, \(\mathbf{I}_{\mathrm{se}}\), using the network in Fig. 8.21a. Note that because of the short circuit, the voltage source is directly across the impedance in the left-most branch. Therefore,

 

\(\mathbf{I}_{1}=\frac{6 \underline{/0^{\circ}}}{1+j}\)

 

Then using KCL,

 

\(\begin{aligned}\mathbf{I}_{\mathrm{sc}} &=\mathbf{I}_{1}+2 \underline{/0^{\circ}}=2+\frac{6}{1+j} \\\\&=\left(\frac{8+2 j}{1+j}\right) \mathrm{A}\end{aligned}\)

 

 

The Thévenin equivalent impedance, \(\mathbf{Z}_{\mathrm{T}}\), is known to be 1 Ω and, therefore, connecting the Norton equivalent to the 1-Ω resistor containing \(\mathbf{I}_{o}\) yields the network in Fig. 8.21b. Using current division, we find that

 

\(\begin{aligned}\mathbf{I}_{o} &=\frac{1}{2}\left(\frac{8+2 j}{1+j}\right) \\\\&=\left(\frac{5}{2}-\frac{3}{2} j\right) \mathrm{A}\end{aligned}\)

In source exchange, a voltage source in series with an impedance can be exchanged for a current source in parallel with the impedance, and vice versa. Repeated application systematically reduces the number of circuit elements.
