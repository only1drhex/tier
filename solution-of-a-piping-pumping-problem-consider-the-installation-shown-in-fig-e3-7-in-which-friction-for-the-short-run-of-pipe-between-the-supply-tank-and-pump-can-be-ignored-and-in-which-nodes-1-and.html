Solution of a Piping/Pumping Problem

Consider the installation shown in Fig. E3.7, in which friction for the short run of pipe between the supply tank and pump can be ignored, and in which nodes 1 and 2 are essentially at the same elevation. The equation for the pressure increase (psi) across the centrifugal pump is (see Eqn. (4.4)):

\(\Delta h=a-b Q^{n}\) (4.4)

\(\Delta p=p_{2}-p_{1}=a-b Q_{\mathrm{A}}^{2},\) (E3.7.1)

in which \(\mathrm{a}=72 \mathrm{psi}\) and \(\mathrm{b}=0.0042 \mathrm{psi} /(\mathrm{gpm})^{2}\). The friction factors in the three pipes are \(f_{\mathrm{FA}}=0.00523, f_{\mathrm{FB}}=0.00584\), and \(f_{\mathrm{FC}}=0.00556\), and the density of the water being pumped is \(\rho=62.4 \mathrm{lb}_{\mathrm{m}} / \mathrm{ft}^{3}\).

Other parameters are given in Tables E3.7.1 and E3.7.2. Assume that the pipe lengths have already included the equivalent lengths of all fittings and valves.

Solution

By dividing Eqn. (3.92) by \(g\), inserting the value for \(\pi\), and substituting appropriate conversion factors, the energy balance for a pipe element becomes:

\(g \Delta z+\frac{\Delta p}{\rho}+\frac{32 f_{\mathrm{F}} Q^{2} L}{\pi^{2} D^{5}}=0\) (3.92)

\(\Delta z+4.636 \times 10^{3} \frac{\Delta p}{\rho g}+4.01 \frac{f_{\mathrm{F}} Q^{2} L}{g D^{5}}=0,\) (E3.7.2)

in which the variables have the following units: \(z, \mathrm{ft} ; p, \mathrm{psi} ; \rho, \mathrm{lb}_{\mathrm{m}} / \mathrm{ft}^{3} ; Q, \mathrm{gpm}\); \(L, \mathrm{ft} ; g, \mathrm{ft} / \mathrm{s}^{2} ;\) and \(D\), in. The energy balance is now applied three times:

Between points 2 and 3

\(f_{1}=z_{3}-z_{2}+4.636 \times 10^{3} \frac{p_{3}-p_{2}}{\rho g}+4.01 \frac{f_{\mathrm{FA}} Q_{\mathrm{A}}^{2} L_{\mathrm{A}}}{g D_{\mathrm{A}}^{5}}=0.\) (E3.7.3)

Between points 3 and 4

\(f_{2}=z_{4}-z_{3}+4.636 \times 10^{3} \frac{p_{4}-p_{3}}{\rho g}+4.01 \frac{f_{\mathrm{FB}} Q_{\mathrm{B}}^{2} L_{\mathrm{B}}}{g D_{\mathrm{B}}^{5}}=0.\) (E3.7.4)

Between points 3 and 5

\(f_{3}=z_{5}-z_{3}+4.636 \times 10^{3} \frac{p_{5}-p_{3}}{\rho g}+4.01 \frac{f_{\mathrm{FC}} Q_{\mathrm{C}}^{2} L_{\mathrm{C}}}{g D_{\mathrm{C}}^{5}}=0.\) (E3.7.5)

Considerations of continuity at the branch point 3 and the performance of the pump give:

Continuity at point 3

\(f_{4}=Q_{\mathrm{A}}-Q_{\mathrm{B}}-Q_{\mathrm{C}}=0,\) (E3.7.6)

Between points 1 and 2

\(f_{5}=p_{2}-\left(p_{1}+a-b Q_{\mathrm{A}}^{2}\right)=0 .\) (E3.7.7)

These last relationships represent five simultaneous nonlinear equations. The values of the five unknowns \(p_{2}, p_{3}, Q_{\mathrm{A}}, Q_{\mathrm{B}}\), and \(Q_{\mathrm{C}}\) are found by using the Excel Solver, for which the results are shown in Table E3.7.3. The strategy is governed by noting that although each of the \(f_{i}, i=1, \ldots, 5\) should equal zero at convergence, the Solver can only minimize a single cell. Therefore, the cell identified as “Sum” is the sum of the absolute values, \(\sum_{i=1}^{5}\left|f_{i}\right|\), and if this is brought close to zero, then all the individual \(f_{i}\) values must also be essentially zero.

The same final values for the two unknown pressures and the three flow rates are obtained for different starting estimates of these five variables. Observe that each of the \(f_{i}\) values is essentially zero, and that continuity is observed: \(Q_{\mathrm{A}}=Q_{\mathrm{B}}+\) \(Q_{\mathrm{C}}\). Solutions for the same basic configuration of tanks, pump, and pipes, but with, for example, different elevations and pump characteristics, could easily be obtained by changing the appropriate input cells and repeating the solution. However, if a different system were to be investigated, the structure of the spreadsheet would have to be changed.

For simplicity in this example, constant yet realistic values were specified for the friction factors in the pipes A, B, and C. A simple extension would be to incorporate extra cells to allow for variability with the Reynolds number and roughness ratio, as in Eqn. (3.50).

\(\operatorname{Re} \leq 2,000: \quad f_{\mathrm{F}}=\frac{16}{\operatorname{Re}}\),

\(\operatorname{Re}>4,000: \quad f_{\mathrm{F}}=\left\{-1.737 \ln \left[0.269 \frac{\varepsilon}{D}-\frac{2.185}{\operatorname{Re}} \ln \left(0.269 \frac{\varepsilon}{D}+\frac{14.5}{\operatorname{Re}}\right)\right]\right\}^{-2} .\) (3.50)
