Guest Question.

Use the Adams-Bashforth-Moulton method to solve \(y^{\prime}=y^{2}+1 ; y(0)=0\), on the interval [0, 1] with h = 0.1.

Admin
Solution

Here \(f(x, y)=y^{2}+1, x_{0}=0, \text { and } y_{0}=0 \text {. }\) Using Table 19-5, we find the three additional starting values to be \(y_{1}=0.1003346, y_{2}=0.2027099, \text { and } y_{3}=0.3093360\). Thus,

\(\begin{aligned}&y_{0}^{\prime}=\left(y_{0}\right)^{2}+1=(0)^{2}+1=1 \\&y_{1}^{\prime}=\left(y_{1}\right)^{2}+1=(0.1003346)^{2}+1=1.0100670 \\&y_{2}^{\prime}=\left(y_{2}\right)^{2}+1=(0.2027099)^{2}+1=1.0410913 \\&y_{3}^{\prime}=\left(y_{3}\right)^{2}+1=(0.3093360)^{2}+1=1.0956888\end{aligned}\)

Then, using Eqs. (19.6), beginning with n = 3, and Eq. (19.3), we compute

\(\begin{aligned}n=3: \quad x_{4} &=0.4 \\p y_{4} &=y_{3}+(h / 24)\left(55 y_{3}^{\prime}-59 y_{2}^{\prime}+37 y_{1}^{\prime}-9 y_{0}^{\prime}\right) \\&=0.3093360+(0.1 / 24)[55(1.0956888)-59(1.0410913)+37(1.0100670)-9(1)] \\&=0.4227151 \\p y_{4}^{\prime} &=\left(p y_{4}\right)^{2}+1=(0.4227151)^{2}+1=1.1786881 \\y_{4} &=y_{3}+(h / 24)\left(9 p y_{4}^{\prime}+19y_{3}^{\prime}-5 y_{2}^{\prime}+y_{1}^{\prime}\right) \\&=0.3093360+(0.1 / 24)[9(1.1786881)+19(1.0956888)-5(1.0410913)+1.0100670] \\&=0.4227981 \\y_{4}^{\prime} &=\left(y_{4}\right)^{2}+1=(0.4227981)^{2}+1=1.1787582\end{aligned}\)

 

\(\begin{aligned}n=4: \quad x_{5} &=0.5 \\p y_{5} &=y_{4}+(h / 24)\left(55 y_{4}^{\prime}-59 y_{3}^{\prime}+37 y_{2}^{\prime}-9 y_{1}^{\prime}\right) \\&=0.4227981+(0.1 / 24)[55(1.1787582)-59(1.0956888)+37(1.0410913)-9(1.0100670)] \\&=0.5461974 \\p y_{5}^{\prime} &=\left(p y_{5}\right)^{2}+1=(0.5461974)^{2}+1=1.2983316 \\y_{5} &=y_{4}+(h / 24)\left(9 p y_{5}^{\prime}+19 y_{4}^{\prime}-5 y_{3}^{\prime}+y_{2}^{\prime}\right) \\&=0.4227981+(0.1 / 24)[9(1.2983316)+19(1.1787582)-5(1.0956888)+1.0410913] \\&=0.5463149 \\y_{5}^{\prime} &=\left(y_{5}\right)^{2}+1=(0.5463149)^{2}+1=1.2984600\end{aligned}\)

 

\(\begin{aligned}n=5: \quad x_{6} &=0.6 \\p y_{6} &=y_{5}+(h / 24)\left(55 y_{5}^{\prime}-59 y_{4}^{\prime}+37 y_{3}^{\prime}-9 y_{2}^{\prime}\right) \\&=0.5463149+(0.1 / 24)[55(1.2984600)-59(1.1787582)+37(1.0956888)-9(1.0410913)] \\&=0.6839784 \\p y_{6}^{\prime} &=\left(p y_{6}\right)^{2}+1=(0.6839784)^{2}+1=1.4678265 \\y_{6} &=y_{5}+(h / 24)\left(9 p y_{6}^{\prime}+19 y_{5}^{\prime}-5 y_{4}^{\prime}+y_{3}^{\prime}\right) \\&=0.5463149+(0.1 / 24)[9(1.4678265)+19(1.2984600)-5(1.1787582)+1.0956888] \\&=0.6841611 \\y_{6}^{\prime} &=\left(y_{6}\right)^{2}+1=(0.6841611)^{2}+1=1.4680764\end{aligned}\)

 

Continuing in this manner, we generate Table 19-7.
