Guest Question.

Gases like argon and neon have constant specific heats. Develop an expression for the ideal gas contribution to Helmholtz function in Eq.12.92 for these cases.

Admin
Solution

The ideal gas Helmholtz function is from its definition, see Eq.12.92,

\(a ^{*}= u ^{*}- Ts ^{*}= h ^{*}- RT – T s ^{*}\)

We have

\(\begin{aligned}h ^{*} &= h _{ o }^{*}+\int C _{ Po } d T = h _{ o }^{*}+ C _{ Po }\left( T – T _{ o }\right) \\s ^{*} &= s _{ o }^{*}+\int \frac{ C _{ Po }}{ T } d T – R \ln \left(\frac{\rho T }{\rho_{ o } T _{ o }}\right) \\&= s _{ o }^{*}+ C _{ Po } \ln \left( T / T _{o}\right)- R \ln \left(\frac{\rho T }{\rho_{ o } T _{ o }}\right)\end{aligned}\)

So now we get

\(\begin{aligned}a^{*} &=h_{o}^{*}+C_{P o}\left(T-T_{o}\right)-R T-T s^{*} \\&=h_{o}^{*}-T s_{o}^{*}+C_{P_{o}}\left(T-T_{o}\right)-C_{P o} T \ln \left(\frac{T}{T_{o}}\right)-R T+R T \ln \left(\frac{\rho T}{\rho_{o} T_{o}}\right) \\&=C_{o}+C_{1} T-C_{2} T \ln \left(\frac{T}{T_{o}}\right)+R T \ln \left(\frac{\rho}{\rho_{o}}\right)\end{aligned}\)

where

\(C _{ o }= h _{ o }^{*}- C _{ Po } T _{o} ; \quad C _{1}= C _{ Po }- R – s _{o}^{*} ; \quad C _{2}= C _{ Po }- R\)

………………………………………

Eq.12.92 : \(a^{*}=h^{*}-R T-T s^{*}\)
