Turbulent Flow in an Obstructed U-Duct (COMSOL)

Problem Statement

Consider the duct in the \(x / y\)-plane shown in Fig. E9.5.1, in which the dimensions are in meters, and which extends indefinitely normal to the plane of the diagram. Two reacting gases, each with density \(\rho=1.18 \mathrm{~kg} / \mathrm{m}^{3}\) and kinematic viscosity \(\nu=1.83 \times 10^{-5} \mathrm{~m}^{2} / \mathrm{s}\), enter at the top left with velocities \(V_{1}=30 \mathrm{~m} / \mathrm{s}\) and \(V_{2}=10 \mathrm{~m} / \mathrm{s}\), and leave at the bottom left at zero pressure.

(a) Calculate the Reynolds number (based on the channel width of \(0.4 \mathrm{~m}\) after the gases have mixed) and verify that the flow is likely to be turbulent \((\operatorname{Re}>5,000,\), say).

(b) If the flow is indeed turbulent, use COMSOL to solve the problem using the \(k-\varepsilon\) method.

(c) Construct the following plots and comment on them:

  1. Velocity magnitude.
  2. Turbulent kinematic viscosity.
  3. Pressure.
  4. Logarithm of the turbulent kinetic energy.
  5. Logarithm of the turbulent dissipation rate.
  6. Arrows that show the velocity vectors.
  7. Streamlines.
Admin
Solution

Chapter 14 has more details about running COMSOL Multiphysics. All mouse clicks are left-clicks (the same as Select) unless denoted as right-clicks (R).

Check the Reynolds Number

Note that the width of each inlet is \(0.1 \mathrm{~m}\) and the width after mixing is \(W=\) \(0.4 \mathrm{~m}\). The mean velocity after mixing is therefore \(\bar{V}=(30 \times 0.1+10 \times 0.1) / 0.4=\) \(10 \mathrm{~m} / \mathrm{s}\). Thus, the Reynolds number is \(\bar{V} W / \nu=10 \times 0.4 / 1.83 \times 10^{-5}=218,530\), highly turbulent.

Select the Physics

  1. Open COMSOL and L-click Model Wizard, 2D, Fluid Flow (the little rotating triangle is called a “glyph”), Single Phase, Turbulent Flow, Turbulent Flow \(k-\varepsilon\), Add. L-click Study, Stationary, Done.

Define the Parameters

  1. R-click Global Definitions and select Parameters. In the Settings window to the right of the Model Builder enter the parameter rho1 in the Name column and \(1.18\left[\mathrm{~kg} / \mathrm{m}^{3}\right]\) (as kg/ m^3 ) in the Expression column. Similarly, enter the kinematic viscosity nu 1 as \(1.83 \mathrm{e}-5\left[\mathrm{~m}^{2} / \mathrm{s}\right]\) and the two inlet velocities, \(V 1\) and \(V 2\), as \(30[\mathrm{~m} / \mathrm{s}]\) and \(10[\mathrm{~m} / \mathrm{s}]\), respectively. For later convenience, define the dynamic viscosity \(m u 1\) using the expression \(n u 1 * r h o 1\).

Establish the Geometry

  1. Create the geometry using simple rectangles and Boolean operations. Rightclick Geometry 1 and select Rectangle to create the rectangle \(r 1\). Enter the values for Width and Height as \(4[\mathrm{~m}]\) and \(1[\mathrm{~m}]\). Note the default position is based on the lower left corner being located at \((0,0)\). The units may be omitted because the default is \([\mathrm{m}]\). L-click Build Selected.
  2. Similarly, create the following four rectangles with width, height, and base combinations: \(r 2\), Size \((3.5,0.2)\), Base \((0.0,0.4) ; r 3\), Size \((0.5,0.2)\), Base \((0.0,0.7) ;\) \(r 4\), Size \((0.5,0.2)\), Base \((1.5,0.2) ;\) and \(r 5\), Size \((0.5,0.2)\), Base \((2.5,0.0) . \mathrm{L}-\mathrm{click}\) Build All.
  3. Form the final geometry. R-click Geometry and L-click Boolean and Partitions, Difference. Select the largest rectangle, \(r 1\), by L-clicking it and note that it appears under Objects to add. Activate the Objects to subtract by L-clicking the On/Off toggle and select the remaining embedded rectangles, \(r 2\) through \(r 5\), to include them under Objects to subtract. L-click Build Selected.

Define the Fluid Properties

  1. R-click the Materials node within Component 1 and select Blank Material. Enter rho1 and \(m u 1\) as the values for Density and Dynamic viscosity.

Define the Boundary Conditions

  1. Select the Wall 1 node in the Turbulent Flow \(k-\varepsilon\) tree and inspect and note that the default wall boundary condition is Wall functions. R-click Turbulent Flow \(k-\varepsilon\) and select Inlet to define the first inlet boundary condition, Inlet 1. L-click to select the highest left edge \((\# 7)\), which is Inlet 1. Change the value for \(U_{0}\) to \(V 1\). Similarly, add the second inlet boundary condition, Inlet 2, by selecting the next lower edge \((\# 4)\) and setting \(U_{0}\) to \(V 2\).
  2. Define the zero-pressure outlet condition. R-click Turbulent Flow \(k-\varepsilon\) and select Outlet to define an outlet boundary condition. L-click to select the lowest left edge \((\# 1)\), which is the outlet. Note the default is Pressure outlet and the reference pressure is zero. Additionally, ensure the Suppress backflow is deselected and Normal flow is selected. Click Turbulent Flow \(k-\varepsilon\) and then the Equations glyph under Settings to see the equations being solved.

Create the Mesh and Solve the Problem

  1. L-click the Mesh node and set the Element size to Fine, which is a reasonable starting size for the mesh. L-click Build All and use the graphics function to zoom and inspect the default boundary layer mesh. The image may be moved around by “grabbing” it while holding the Control key down on a Mac (or R-click for Windows and Linux). After a R-click on Mesh 1, note under Statistics that there are 36,886 elements – and 137,685 degrees of freedom (unknowns) to be solved.
  2. Following the recommendations in Section 14.7, modify the boundarylayer mesh. Within the Mesh node, under Settings, change the Sequence type to User-controlled mesh, allowing custom setting for the boundary-layer mesh. Note that the original mesh setting in the previous step is propagated to the present user settings. Expand the Boundary Layers node by L-clicking on the glyph to its left and L-click Boundary Layer Properties. Change the Number of boundary layers, Boundary layer stretching factor, and Thickness adjustment factor to 8 , 1.1, and 1, respectively, as shown in Fig. E9.5.2. This will produce a boundary mesh with reasonable wall lift-off in viscous units. L-click Build Selected.
  3. Save the model file periodically.
  4. R-click the Study node and select Compute. This will take more time to solve than other examples. Above the Graphics window, L-click the Convergence Plot tab to watch the solution convergence (466 s for the author’s computer), resulting in a surface plot of the velocity magnitude.
  5. Inspect the default results for this physics. Particularly note the default Wall Resolution plots and refer to the explanation in Section 14.7.

Construct the Necessary Plots

  1. Make a surface plot to inspect solution variables. R-click Results and L-click 2D Plot Group. R-click the new 2D Plot Group 4 and L-click Surface. In the Settings window, change the Expression being plotted by L-clicking the Expression Builder (the little triangular glyph) at the right of Expression and Leftclick Component 1, Turbulent Flow \(k-\varepsilon\), Turbulence variables, spf.nuT – Turbulent kinematic viscosity. L-click Plot.
  2. R-click Results and L-click 2D Plot Group. R-click the new 2D Plot Group 5 and L-click Surface. In the Settings window, replace the Expression with the variable \(p\), which is pressure. L-click Plot.
  3. Repeat Step 15 using \(2 \mathrm{D}\) Plot Group 6 and \(\log (k)\), the logarithm of the turbulent kinetic energy (which gives a better overview than \(k\) itself). L-click Plot (not shown here).
  4. Repeat Step 15 using \(2 \mathrm{D}\) Plot Group 7 and \(\log (e p)\), the logarithm of the turbulent dissipation rate (which gives a better overview than ep itself). L-Plot.
  5. Create an arrow plot to visualize flow direction. R-click Results and L-click \(2 \mathrm{D}\) Plot Group 8. R-click the newly created plot group and L-click Arrow Surface. L-click Plot to update the vector scene. Note that the default arrow resolution is too low to visualize the flow. In the Settings Panel, modify the number of \(x\) and \(y\) grid points to be 40 . L-click Plot to update the scene.
  6. Display the streamlines. R-click Results and L-click 2D Plot Group \(9 .\) R-click the new 2D Plot Group and L-click Streamline. Several recirculation zones in this example present an issue for defining the streamline starting positions. So, change the Positioning option in the Streamline Positioning section of the Settings panel to Uniform density. Change the Separating distance to \(0.01 .\) L-click Plot.
