Guest Question.

The position of an object moving along the x axis varies with time as in Figure 2.7a. Graph the velocity versus time and the acceleration versus time for the object.

Admin
Solution

The velocity at any instant is the slope of the tangent to the x -t graph at that instant. Between t = 0 and \( t =t_A \), the slope of the x -t graph increases uniformly, and so the velocity increases linearly, as shown in Figure 2.7b.
Between \( t_{ A } \text { and } t_{ B } \), the slope of the x -t graph is constant, and so the velocity remains constant. At \( t_D \), the slope of the x -t graph is zero, so the velocity is zero at that instant.
Between \( t_{ D } \text { and } t_{ E } \), the slope of the x -t graph and thus the velocity are negative and decrease uniformly in this interval.
In the interval \( t_{ E } \text { to } t_{ F } \), the slope of the x-t graph is still negative, and at \( t_F \) it goes to zero. Finally, after \( t_F \), the slope of the x -t graph is zero, meaning that the object is at rest for \( t>t_{ F } \) .

The acceleration at any instant is the slope of the tangent to the \(v_x -t\) graph at that instant. The graph of acceleration versus time for this object is shown in Figure 2.7c. The acceleration is constant and positive between 0 and \( t_A \), where the slope of the \(v_x -t\) graph is positive. It is zero between \( t_{ A } \text { and } t_{ B } \) and for \( t>t_{ F } \) because the slope of the \(v_x -t\) graph is zero at these times. It is negative between \( t_{ B } \text { and } t_{ E } \) because the slope of the \(v_x -t\) graph is negative during this interval.

Note that the sudden changes in acceleration shown in Figure 2.7c are unphysical. Such instantaneous changes cannot occur in reality.
