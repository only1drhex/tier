Guest Question.

Choked Fanno Flow

GIVEN Standard atmospheric air [\(T_{0}=288 K , p_{0}=101\) kPa(abs) ] is drawn steadily through a frictionless, adiabatic converging nozzle into an adiabatic, constant area duct as shown in Fig. E11.12a. The duct is 2 m long and has an inside diameter of 0.1 m. The average friction friction factor for the duct is estimated as being equal to 0.02.

FIND What is the maximum mass flowrate through the duct? For this maximum flowrate, determine the values of static temperature, static pressure, stagnation temperature, stagnation pressure, and velocity at the inlet [section (1)] and exit [section (2)] of the constant area duct. Sketch a temperature–entropy diagram for this flow.

Admin
Solution

We consider the flow through the converging nozzle to be isentropic and the flow through the constant area duct to be Fanno flow. A decrease in the pressure at the exit of the constant area duct (back pressure) causes the mass flowrate through the nozzle and the duct to increase. The flow throughout is subsonic. The maximum flowrate will occur when the back pressure is lowered to the extent that the constant area duct chokes and the Mach number at the duct exit is equal to 1. Any further decrease of back pressure will not affect the flowrate through the nozzle–duct combination.

For the maximum flowrate condition, the constant area duct must be choked, and

 

\(\frac{f\left(\ell^{*}-\ell_{1}\right)}{D}=\frac{f\left(\ell_{2}-\ell_{1}\right)}{D}=\frac{(0.02)(2 m )}{(0.1 m )}=0.4\) (1)

 

With \(k=1.4 \text { for air and the above calculated value of } f\left(\ell^{*}-\ell_{1}\right) /\) D = 0.4, we could use Eq. 11.98 to determine a value of Mach number at the entrance of the duct [section (1)]. With \(k=1.4 \text { and }\) \(Ma _{1}\) known, we could then rely on Eqs. 11.101, 11.103, 11.107, and 11.109 to obtain values of \(T_{1} / T^{*}, V_{1} / V^{*}, p_{1} / p^{*}, \text { and } p_{0,1} / p_{0}^{*}\).

 

\(\frac{1}{k} \frac{\left(1- Ma ^{2}\right)}{ Ma ^{2}}+\frac{k+1}{2 k} \ln \left\{\frac{[(k+1) / 2] Ma ^{2}}{1+[(k-1) / 2] Ma ^{2}}\right\}=\frac{f\left(\ell^{*}-\ell\right)}{D}\) (11.89)

 

\(\frac{T}{T^{*}}=\frac{(k+1) / 2}{1+[(k-1) / 2] Ma ^{2}}\) (11.101)

 

\(\frac{V}{V^{*}}=\left\{\frac{[(k+1) / 2] Ma ^{2}}{1+[(k-1) / 2] Ma ^{2}}\right\}^{1 / 2}\) (11.103)

 

\(\frac{p}{p^{*}}=\frac{1}{ Ma }\left\{\frac{(k+1) / 2}{1+[(k-1) / 2] Ma ^{2}}\right\}^{1 / 2}\) (11.107)

 

\(\frac{p_{0}}{p_{0}^{*}}=\frac{1}{ Ma }\left[\left(\frac{2}{k+1}\right)\left(1+\frac{k-1}{2} Ma ^{2}\right)\right]^{[(k+1) / 2(k-1)]}\) (11.109)

 

Alternatively, for air \((k=1.4) \text {, we can use Fig. D. } 2 \text { with } f\left(\ell^{*}-\ell_{1}\right) /\) \(D=0.4 \text { and read off values of } Ma _{1}, T_{1} / T^{*}, V_{1} / V^{*}, p_{1} / p^{*} \text {, and }\) \(p_{0,1} / p_{0}^{*}\).

The pipe entrance Mach number, \(Ma _{1} \text {, }\), also represents the Mach number at the throat 1and exit2 of the isentropic, converging nozzle. Thus, the isentropic flow equations of Section 11.4 or Fig. D.1 can be used with \(Ma _{1} \text {, }\). We use Fig. D.1 in this example.

With \(Ma _{1} \text {, }\) known, we can enter Fig. D.1 and get values of \(T_{1} / T_{0}, p_{1} / p_{0}, \text { and } \rho_{1} / \rho_{0}\). Through the isentropic nozzle, the values \(\text { of } T_{0}, p_{0} \text {, and } \rho_{0} \text { are each constant, and thus } T_{1}, p_{1} \text {, and } \rho_{1} \text { can be }\) readily obtained.

Since \(T_{0}\) also remains constant through the constant area duct (see Eq. 11.75), we can use Eq. 11.63 to get \(T^{*}\). Thus,

 

\(\frac{T^{2}}{T_{0}}=\frac{2}{k+1}\) (11.63)

 

\(T+\frac{(\rho V)^{2} T^{2}}{2 c_{p}\left(p^{2} / R^{2}\right)}=T_{0}=\text { constant }\) (11.75)

 

\(\frac{T^{*}}{T_{0}}=\frac{2}{k+1}=\frac{2}{1.4+1}=0.8333\) (2)

 

\(\text { Since } T_{0}=288 K \text {, we get from Eq. } 2 \text {, }\)

 

\(T^{*}=(0.8333)(288 K )=240 K =T_{2}\) (3)

 

With \(T^{*}\) known, we can calculate \(V^{*}\) from Eq. 11.36 as

 

\(c=\sqrt{R T k}\) (11.36)

 

\(\begin{aligned}V^{*} &=\sqrt{R T^{*} k} \\&=\sqrt{[(286.9 J ) /( kg \cdot K )](240 K )(1.4)} \\&=310( J / kg )^{1 / 2}\end{aligned}\)

 

\(\text { Thus, since } 1 J / kg =1 N \cdot m / kg =1\left( kg \cdot m / s ^{2}\right) \cdot m / kg =( m / s )^{2} \text {, }\) we obtain

 

\(V^{*}=310 m / s =V_{2}\) (4)

 

\(\text { Now } V_{1} \text { can be obtained from } V^{*} \text { and } V_{1} / V^{*} \text {. Having } A_{1}, \rho_{1} \text {, and }\) \(V_{1} \text { we can get the mass flowrate from }\)

 

\(\dot{m}=\rho_{1} A_{1} V_{1}\) (5)

 

Values of the other variables asked for can be obtained from the ratios mentioned.

Entering Fig. D.2 with \(f(\ell *-\ell) / D=0.4 \text { we read }\)

 

\(Ma _{1}=0.63\) (7)

 

\(\frac{T_{1}}{T^{*}}=1.1\) (8)

 

\(\frac{V_{1}}{V^{*}}=0.66\) (9)

 

\(\frac{p_{1}}{p^{*}}=1.7\) (10)

 

\(\frac{p_{0,1}}{p_{0}^{*}}=1.16\) (11)

 

\(\text { Entering Fig. D.1 with } Ma _{1}=0.63 \text { we read }\)

 

\(\frac{T_{1}}{T_{0}}=0.93\) (12)

 

\(\frac{p_{0}}{p_{0,1}}=0.76\) (13)

 

\(\frac{\rho_{1}}{\rho_{0,1}}=0.83\) (14)

 

Thus, from Eqs. 4 and 9 we obtain

 

\(V_{1}=(0.66)(310 m / s )=205 m / s\)

 

From Eq. 14 we get

 

\(\rho_{1}=0.83 \rho_{0,1}=(0.83)\left(1.23 kg / m ^{3}\right)=1.02 kg / m ^{3}\)

 

and from Eq. 5 we conclude that

 

\(\begin{aligned}\dot{m} &=\left(1.02 kg / m ^{3}\right)\left[\frac{\pi(0.1 m )^{2}}{4}\right](206 m / s ) \\&=1.65 kg / s\end{aligned}\)

 

From Eq. 12, it follows that

 

\(T_{1}=(0.93)(288 K )=268 K\)

 

Equation 13 yields

 

\(p_{1}=(0.76)[101 kPa ( abs )]=77 kPa ( abs )\)

 

The stagnation temperature, \(T_{0}\), remains constant through this adiabatic flow at a value of

 

\(T_{0,1}=T_{0,2}=288 K\)

 

The stagnation pressure, \(p_{0}\), at the entrance of the constant area duct is the same as the constant value of stagnation pressure through the isentropic nozzle. Thus

 

\(p_{0,1}=101 kPa ( abs )\)

 

To obtain the duct exit pressure \(\left(p_{2}=p^{*}\right)\) we can use Eqs. 10 and 13. Thus,

 

\(\begin{aligned}p_{2} &=\left(\frac{p^{*}}{p_{1}}\right)\left(\frac{p_{1}}{p_{0,1}}\right)\left(p_{0,1}\right)=\left(\frac{1}{1.7}\right)(0.76)[101 kPa ( abs )] \\&=45 kPa ( abs )\end{aligned}\)

 

For the duct exit stagnation pressure \(\left(p_{0,2}=p_{0}^{*}\right)\) we can use Eq. 11 as

 

\(\begin{aligned}p_{0,2} &=\left(\frac{p_{0}^{*}}{p_{0,1}}\right)\left(p_{0,1}\right)=\left(\frac{1}{1.16}\right)[101 kPa ( abs )] \\&=87.1 kPa ( abs )\end{aligned}\)

 

The stagnation pressure, \(p_{0}\), decreases in a Fanno flow because of friction.

COMMENT Use of graphs such as Figs. D.1 and D.2 illustrates the solution of a problem involving Fanno flow. The T–s diagram for this flow is shown in Fig. E.11.12b, where the entropy difference, \(S_{2}-S_{1}\), is obtained from Eq. 11.22.

 

\(s_{2}-s_{1}=c_{p} \ln \frac{T_{2}}{T_{1}}-R \ln \frac{p_{2}}{p_{1}}\) (11.22)
