Guest Question.

Assume the compressor and the gas turbine in both have an efficiency of 80 percent. Using the Brayton cycle determine the back work ratio and the thermal efficiency.

Admin
Solution
\( \text { We can calculate the quantities asked for if we determine } W_{\text {comp }}, W_{\text {turb }} \text {, and } q_{\text {in. }} \text {. The compressor work is } \)

\( w_{\text {comp }}=\frac{w_{\text {comp }, s}}{\eta_{\text {comp }}}=\frac{C_{p}}{\eta_{\text {comp }}}\left(T_{2^{\prime}}-T_{1}\right) \).

\( \text { where } w_{\text {comp. } s} \text { is the isentropic work, } T_{2} \text { is the temperature of state } \) 2′ assuming an isentropic process, and state 2 is the actual
state. We then have, using \( T_{2^{\prime}}=T_{2} \) from ,

\( w_{\text {comp }}=\left(\frac{1.00}{0.8}\right)(472-298)=217.5 kJ / kg \).

\( \text { Likewise, there results } w_{\text {turb }}=\eta_{\text {turb }} w_{\text {turb }, s}=\eta_{\text {turb }} C_{p}\left(T_{3}-T_{4}\right)=(0.8)(1.00)(1123-709.1)=331.1 kJ / kg \text {, where } T_{4^{\prime}}=T_{4} \text { is } \) calculated in . State 4′ is the isentropic state and state 4 is the actual state. The back work ratio is then

\( \frac{w_{\text {comp }}}{w_{\text {turb }}}=\frac{217.5}{331.1}=0.657 \quad \text { or } \quad 65.7 \% \).

The heat transfer input necessary in this cycle is \( q_{\text {in }}=h_{3}-h_{2}=C_{p}\left(T_{3}-T_{2}\right), \text { where } T_{2} \) is the actual temperature of the air leaving the compressor. It is found by returning to the compressor:

\( w_{\text {comp }}=C_{p}\left(T_{2}-T_{1}\right) \quad 217.5=(1.00)\left(T_{2}-298\right) \quad \therefore T_{2}=515.5 K \).

\( \text { Thus, } q_{\text {in }}=(1.00)(1123-515.5)=607.5 kJ / kg \text {. The thermal efficiency of the cycle can then be written as } \)

\( \eta=\frac{w_{\text {net }}}{q_{\text {in }}}=\frac{w_{\text {turb }}-w_{\text {comp }}}{q_{\text {in }}}=\frac{331.1-217.5}{607.5}=0.187 \quad \text { or } \quad 18.7 \% \).
