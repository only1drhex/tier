Guest Question.

The single-phase regenerative PWM rectifier of Fig. 9.14(a) has the following specifications:

Input voltage = 240 V, 50 Hz, output voltage = 500 V dc.

Active output power = 10 kW, applies SPWM modulation.

A control loop is applied so that the power factor to be considered unity. The full-bridge semiconductor switches are driven by high frequency SPWM signals, and the switching losses are considered to be negligible. If the amplitude modulation factor \(m_a = 0.9\), calculate the following:

a) The rms input current.

b) The rms of the reflected voltage fundamental component and its phase displacement with respect to the input voltage source.

c) The value of the input inductor assuming that its internal resistance is zero.

d) The value of the output capacitor so that the peak-to-peak ripple of the output voltage second harmonic component not to exceed 20 V.

Admin
Solution

a) Since the rectifier is considered to be ideal and since the input current is sinusoidal and in phase with the input ac voltage, the following results are obtained:

\(P_i =\widetilde{V}_i \widetilde{I}_{i,1} \cos \varphi_1 =\widetilde{V}_i \widetilde{I}_{i} \cos \varphi 0Â° =\widetilde{V}_i \widetilde{I}_{i} = P_o\) and therefore \(\widetilde{I}_{i,1} = \widetilde{I}_{i} = \frac{P_i}{\widetilde{V}_i} = \frac{10000}{240}\)

 

\(= 41.7 A\)

b) Since the rectifier operates with SPWM control technique the reflected voltage fundamental component is given by:

\(\widetilde{V}_{ab,1} = \frac{\overline{V}_om_a}{\sqrt{2}} = \frac{(500)(0.9)}{\sqrt{2}} = 318.2 V\)

c) Using Fig. 9.16(c) the rms inductor voltage is given by:

\(\widetilde{V}_{L,1} = \sqrt{\widetilde{V}^2_{ab,1} – \widetilde{V}^2_i} = \sqrt{318.2^2 – 240^2} = 208.9 V\)

According to the phasor diagram of Fig. 9.16(c):

\(\delta = \tan^{-1} (\frac{\widetilde{V}_{L,1}}{\widetilde{V}_i}) = \tan^{-1} (\frac{208.9}{240}) = 41.036Â°\)

Since the input inductor reactance is \(X_L = \frac{\widetilde{V}_{L,1}}{\widetilde{I}_{i,1}} = \frac{208.9}{41.7} = 5 \Omega\), then the inductor value is:

\(L_i = \frac{X_L}{\omega} = \frac{X_L}{2\pi f} = \frac{5}{2\pi \times 50} =15.9 mH\)

d) The instantaneous input active power of the rectifier for sinusoidal input current (i.e., for high switching frequency) is given by:

\(p_i = v_i(t) i_i(t) =\widehat{V}_i \sin \omega t \cdot \widehat{I}_i \sin \omega t =\widehat{V}_i \widehat{I}_i \sin^2 \omega t = \frac{\widehat{V}_i \widehat{I}_i}{2} (1-\cos 2\omega t)\)

 

\(= \widetilde{V}_i \widetilde{I}_i – \widetilde{V}_i \widetilde{I}_i \cos 2\omega t = \widetilde{V}_i \widetilde{I}_i (1-\cos 2\omega t)\)

The instantaneous power before the output filter is: \(p_{before-filter} = \overline{V}_o i_f\)
Therefore:

\(p_i = \widetilde{V}_i \widetilde{I}_i (1-\cos 2\omega t) =p_{before-filter} = \overline{V}_o i_f\) or \(i_f = \overline{I}_o + i_C =\frac{\widetilde{V}_i \widetilde{I}_i}{\overline{V}_o}(1-\cos 2\omega t)\)

Therefore, under ideal operating conditions:

\(i_C =\frac{\widetilde{V}_i \widetilde{I}_i}{\overline{V}_o}(1-\cos 2\omega t) – \overline{I}_o =\frac{\widetilde{V}_i \widetilde{I}_i}{\overline{V}_o} – \frac{\widetilde{V}_i \widetilde{I}_i}{\overline{V}_o} \cos 2\omega t – \overline{I}_o = \overline{I}_o – \overline{I}_o \cos 2\omega t – \overline{I}_o\)

 

\(= \overline{I}_o \cos 2\omega t\)

where \(\overline{I}_o =\frac{P_o}{\overline{V}_o} = \frac{10000}{500} = 20 A\).

The output capacitor reactance at the frequency of 100 Hz is:

\(X_{C,2} = \frac{1}{2 \omega C_o} = \frac{1}{2(2\pi f)C_o}= \frac{1}{2(2\pi \times 50)C_o} = \frac{\widehat{V}_{o,2}}{\widehat{I}_{C,2}} =\frac{\frac{20}{2}}{20} = 0.5 \Omega\)

Therefore, the required capacitance value is \(C_o = 3185 \mu F\).
