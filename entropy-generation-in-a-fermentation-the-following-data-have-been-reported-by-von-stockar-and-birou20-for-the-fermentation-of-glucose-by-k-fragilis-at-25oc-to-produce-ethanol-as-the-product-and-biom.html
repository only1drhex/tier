Guest Question.

Entropy Generation in a Fermentation

The following data have been reported by von Stockar and \(\text { Birou }^{20}\) for the fermentation of glucose by K. fragilis at 25ºC to produce ethanol as the product and biomass of elemental composition \(CH _{1.75} O _{0.52} N _{0.15}\):

\(Y_{ B / S }=0.572 \quad Y_{ CO _{2} / s }=0.398 \quad \text { and } \quad Y_{ Q / s }=-180.9\) \(\frac{ kJ }{\text { C-mole }}\)

Compute the following per C-mole of substrate consumed:

a. The amount of product formed \(\left(Y_{ P / S }\right)\)

b. The amount of ammonia consumed \(\left(Y_{ N / S }\right)\)

c. The amount of oxygen consumed \(\left(Y_{ O _{2} / S }\right)\)

d. The amount of heat released \(\left(Y_{ Q / S }\right)\)

e. The amount of entropy generated \(\left(S_{\text {gen }}\right)\)

Admin
Solution

To begin, we compute the generalized degree of reduction of the biomass produced:

 

\(\xi_{ B }=4 \times C + H -2 \times O =4+1.75-2 \times 0.52=4.71 \)

 

Therefore, for the biomass,

 

\(\triangle_{ c } G_{ B }=112 \times \xi_{ B }=527.5 \frac{ kJ }{ C – mole } \quad \text { and } \quad \Delta_{ c } H_{ B }=110.9 \times \xi_{ B }=522.3 \frac{ kJ }{ C – mole }\)

 

The heats and Gibbs energies of all other components are available in Table 15.7-2.

a. By the carbon balance,

 

\(1=Y_{ B / s }+Y_{ P / s }+Y_{ CO _{2} / s }\)

 

so that

 

\(Y_{ P / s }=1-Y_{ B / s }-Y_{ CO _{2} / s }=1-0.572-0.398=0.030<br /> \)

 

b. By the nitrogen balance,

 

\(Y_{ N / s }=0.15 \times Y_{ N / s }=0.15 \times 0.572=0.086\)

 

c. From Eq. 15.7-12, we have

 

\(Y_{ O / S }=\frac{\xi_{ S }+\xi_{ N } Y_{ N / s }-\xi_{ B } Y_{ B / S }-\xi_{ P } Y_{ P / S }}{4}\) (15.7-12)

 

\(\begin{aligned}Y_{ O _{2} / s } &=\frac{\xi_{ S }+\xi_{ N } \cdot Y_{ N / s }-\xi_{ B } \cdot Y_{ B / s }-\xi_{ P } \cdot Y_{ P / S }}{4} \\&=\frac{4+0.086 \times 3-4.71 \times 0.572-0.030 \times 6}{4}=0.346\end{aligned}\)

 

d. From the energy balance,

 

\(\begin{aligned}Y_{ Q / s } &=Y_{ B / S } \cdot \triangle_{ c } H_{ B }+Y_{ P / s } \cdot \triangle_{ c } H_{ P }-\triangle_{ c } H_{ S }-Y_{ N / s } \cdot \triangle_{ c } H_{ N } \\&=0.572 \cdot 522.3+0.030 \cdot 684.5-467.8-0.086 \cdot 348.1=-178.5\end{aligned}\) \(\frac{ kJ }{\text { C-mole }}\)

 

e. From the entropy balance, we have

 

\(\begin{aligned}\underline{S}_{\text {gen }} &=\frac{1}{T}\left(\triangle_{ c } G_{ S }+Y_{ N / S } \cdot \triangle_{ c } G_{ N }-Y_{ B / S } \cdot \triangle_{ c } G_{ B }-Y_{ P / S } \cdot \triangle_{ c } G_{ P }\right) \\&=\frac{(478.7+0.086 \cdot 391.9-0.572 \cdot 527.5-0.030 \cdot 684.5)}{298.15}=0.640\end{aligned}\) \(\frac{ kJ }{\text { C-mol K }}\)

 

Finally, it is of interest to compare the heat release calculated in part (d) above, \(Y_{ Q / s }\) = −178.5 kJ/ C-mole, with the more approximate result from Eq. 15.9-22:

 

\(Y_{ Q / S }=-443.6 \times Y_{ O / S }=-443.6 \times 0.346=-153.5\) \(\frac{ kJ }{ C – mole }\)

 

Though the approximate result is not very accurate, it is not unreasonable given the simplicity of the calculation.
