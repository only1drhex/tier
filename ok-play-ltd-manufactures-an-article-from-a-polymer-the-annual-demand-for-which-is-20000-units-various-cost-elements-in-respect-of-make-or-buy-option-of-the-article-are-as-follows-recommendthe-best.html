Guest Question.

OK Play Ltd. manufactures an article from a polymer, the annual demand for which is 20,000 units.Various cost elements in respect of make or buy option of the article are as follows. Recommendthe best option.

Buy Make
Unit Cost of Item \(\text { (₹) }\) 85 52
Procurement cost/order \(\text { (₹) }\) 750
Set-up cost/set-up \(\text { (₹) }\) 1,000
Carrying cost/per item/per year \(\text { (₹) }\) 25 12
Annual rate of Production 35000 units
Admin
Solution

Make option:

 

\(\begin{array}{l}C_{s}=₹ 1,000 / \text { set-up } \\C_{c}=₹ 12 / \text { unit/year } \\D=20,000 \text { units/year } \\r=35,000 \text { units/year } \\P=₹ 52 / \text { unit }\end{array}\)

 

\(\begin{array}{l}Q_{P}=\sqrt{\frac{2 C_{s} D}{C_{c}(1-D / r)}} \\Q_{P}=\sqrt{\frac{2 \times 1,000 \times 20,000}{12(1-20,000 / 35,000)}}=2,788.867 \approx 2,789 \text { units (approximately) }\end{array}\)

 

\(\begin{aligned}T C_{P} &=D \times P+\frac{D C_{o}}{Q_{P}}+C_{c}(r-D) \frac{Q_{P}}{2 \times r} \\&=20,000 \times 52+\frac{20,000 \times 12}{2,789}+12 \times(35,000-20,000) \times \frac{2,789}{2 \times 35,000} \\&=₹ 10,47,257.77\end{aligned}\)

 

Buy option:

 

\(\begin{array}{l}C_{o}=\text { ₹750/order } \\C_{c}=\text { ₹25/unit/year } \\D=20,000 \text { units/year } \\P=\text { ₹85/unit }\end{array}\)

 

\(Q_{o}=\sqrt{\frac{2 C_{o} D}{C_{c}}}\)

 

\(Q_{o}=\sqrt{\frac{2 \times 750 \times 20,000}{25}}=1,095.45 \approx 1,096 \text { units (approximately) }\)

 

\(\begin{aligned}T C_{o} &=D \times P+\frac{D C_{o}}{Q_{o}}+\frac{Q_{o} \times C_{c}}{2} \\&=20,000 \times 85+\frac{20,000 \times 750}{1,096}+\frac{1,096 \times 25}{2} \\&=₹ 17,27,386.131\end{aligned}\)

 

From the above calculations, it can be seen that the cost of making is less than the cost of buying. Therefore, the company should opt for making the item in-house rather than buying it from an outside vendor.
