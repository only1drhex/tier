Guest Question.

The Trimerization of a Pancreatic Hormone Glucagon is a 29-amino peptide hormone of the pancreas. At pH = 10.6 it is known that glucagon can associate or trimerize, and the following standard-state (ideal 1-molal solution at 25ºC) information is available:

\(G \stackrel{K}{\rightleftharpoons} \frac{1}{3} G_{3}\) \(\Delta_{ rxn } G^{\circ}=-30.6 \frac{ kJ }{ mol } \quad \Delta_{ rxn } H^{\circ}=-131 \frac{ kJ }{ mol } \quad \text { and } \quad \Delta_{ rxn } C_{ P }^{\circ}=-1.8 \frac{ kJ }{ mol K }\)

If the initial concentration of glucagon is 0.01 M (which is sufficiently low for the solution to be considered ideal), what is the fraction of glucagon trimerized over the temperature range from 0 to 50ºC?

Admin
Solution

The equilibrium relation is

 

\(K_{a}=e^{-\left(\Delta_{ rxn } G^{ O } / R T\right)}=\frac{\left(a_{ G _{3}}\right)^{1 / 3}}{a_{ G }} \cong \frac{\left(M_{ G _{3}}\right)^{1 / 3} \cdot(1 M )^{2 / 3}}{M_{ G }}\)

 

and the mass balances are

 

\(M_{ G }=M_{ G , o }-\alpha=0.01-\alpha \quad \text { and } \quad M_{ G _{3}}=\frac{\alpha}{3}\)

 

so that

 

\(K_{a}=\frac{\left(\frac{\alpha}{3}\right)^{1 / 3} \cdot(1 M )^{2 / 3}}{0.01-\alpha}\)

 

To calculate the equilibrium constant at any temperature, we use

 

\(\ln \frac{K_{a}(T)}{K_{a}(T=298.15 K )}=\int_{298.15 K }^{T} \frac{\Delta_{ rxn } H^{\circ}(T)}{R T^{2}} d T\)

 

with

 

\(\begin{aligned}\Delta_{ rxn } H^{ o }(T) &=\Delta_{ rxn } H^{\circ}(T=298.15 K )+\int_{298.15 K }^{T} \Delta_{ rxn } C_{ P }^{\circ} d T \\&=\Delta_{ rxn } H^{\circ}(T=298.15 K )+\triangle_{ rxn } C_{ P }^{\circ} \cdot(T-298.15 K )\end{aligned}\)

 

so that

 

\(\begin{aligned}\ln \frac{K_{a}(T)}{K_{a}(T=298.15 K )}=& \int_{298.15 K }^{T} \frac{\Delta_{ rxn } H^{\circ}(T=298.15 K )}{R T^{2}} d T \\&+\int_{298.15 K }^{T} \frac{\Delta_{ rxn } C_{ P }^{\circ} \cdot(T-298.15 K )}{R T^{2}} d T \\=&-\Delta_{ rxn } H^{ o }(T=298.15 K ) \cdot\left(\frac{1}{T}-\frac{1}{298.15 K }\right) \\&+\frac{\Delta_{ rxn } C_{ P }^{ o }}{R}\left[\ln \frac{T}{298.15 K }+(298.15 K ) \cdot\left(\frac{1}{T}-\frac{1}{298.15 K }\right)\right]\end{aligned}\)

 

We obtain the following values:

 

\(T\left({ }^{\circ} C \right)\) Ka(T) α(M) Percent Trimerized
0 \(1.237 \times 10^{7}\) 0.01000. 100.0
25 \(2.297 \times 10^{5}\) 0.009999 99.99
50 \(1.944 \times 10^{3}\) 0.009923 99.23

 

Therefore, over the temperature range of 0 to 50ºC glucagon is almost completely trimerized.
