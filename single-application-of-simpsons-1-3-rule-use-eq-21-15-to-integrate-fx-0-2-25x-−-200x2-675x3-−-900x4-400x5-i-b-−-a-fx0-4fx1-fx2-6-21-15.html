Guest Question.

Single Application of Simpson’s 1/3 Rule

Use Eq. (21.15) to integrate

\(I\cong \underbrace{(ba)}_{Width} \underbrace{\frac{ f(x_0) + 4f(x_1) + f(x_2)}{6} }_{Average height} \) (21.15)

\(f(x) = 0.2 + 25x − 200x^2 + 675x^3 − 900x^4 + 400x^5\)

from a = 0 to b = 0.8. Recall that the exact value is 1.640533.

Admin
Solution

\(\begin{matrix} f(0) = 0.2 &&f(0.4) = 2.456&& f(0.8) = 0.232 \end{matrix} \)

Therefore, Eq. (21.15) can be used to compute

\(I\cong \underbrace{(ba)}_{Width} \underbrace{\frac{ f(x_0) + 4f(x_1) + f(x_2)}{6} }_{Average height} \) (21.15)

\( I\cong 0.8\frac{0.2 + 4(2.456) + 0.232}{6}= 1.367467 \)

which represents an exact error of

\(E_{t} = 1.640533 − 1.367467 = 0.2730667\) \( \varepsilon _{t}= 16.6 \)%

which is approximately five times more accurate than for a single application of the trapezoidal rule ( ).
The estimated error is [Eq. (21.16)]

\( E_{t}=-\frac{(ba)^{5}}{2880}f^{(4)}(\xi ) \) (21.16)

\( E_{a}=-\frac{(0.8)^5}{2880}(−2400) = 0.2730667 \)

where −2400 is the average fourth derivative for the interval, as obtained using Eq. (PT6.4). As was the case in , the error is approximate \( (E_a)\) because the average fourth derivative is not an exact estimate of \( f^{(4)} (\xi )\). However, because this case deals with a fifthorder polynomial, the result matches.

\(Mean=\frac{\int_{a}^{b}{f(x)dx} }{ba} \) (PT6.4)
