Guest Question.

Using waiting line analysis to size accumulation conveyors at workstations

Recall in it was given that \(c=1, \lambda=1\), and \(\mu=4 / 3\). It was desired to determine the length of an accumulation line \((N)\) such that \(P_{N}=0.02\).

Admin
Solution

From Table 10.36, it is seen that

\(P_{0}=\frac{1-\rho}{1-\rho^{N+1}}\)
Table 10.36 Summary of Operating Characteristics for the \((M|M|I)\) Queue
\((M|M| 1):(G D|N| \infty)^{a}\) \((M|M| 1):(G D|\infty| \infty)^{b}\)
\(L_{q}\) \(\frac{\rho^{2}\left[1-\rho^{N}-N \rho^{N-1}(1-\rho)\right]}{(1-\rho)\left(1-\rho^{N+1}\right)}\) \(\frac{\rho^{2}}{1-\rho} \)
L \(\frac{\rho\left[1-\rho^{N}-N \rho^{N}(1-\rho)\right]}{(1-\rho)\left(1-\rho^{N+1}\right)}\) \(\frac{\rho}{1-\rho}\)
\(W_{q}\) \(\frac{\rho\left[1-\rho^{N}-N \rho^{N-1}(1-\rho)\right]}{\mu(1-\rho)\left(1-\rho^{N}\right)}\) \(\frac{\rho}{\mu(1-\rho)}\)
W \(\frac{\left[1-\rho^{N}-N \rho^{N}(1-\rho)\right]}{\mu(1-\rho)\left(1-\rho^{N}\right)}\) \(\frac{1}{\mu(1-\rho)}\)
U \(\frac{\rho\left(1-\rho^{N}\right)}{1-\rho^{N+1}}\) \(\rho\)
\(\hat{\lambda}\) \(\frac{\lambda\left(1-\rho^{N}\right)}{1-\rho^{N+1}}\) \(\lambda\)
\(P_{0}\) \(\frac{1-\rho}{1-\rho^{N+1}}\) \(1-\rho\)
\(a_{\rho} \neq 1\)
\(b_{\rho}<1\)

where \(\rho=\lambda / \mu\), or \(0.75 .\) From Equation \(10.152\), it is seen that the value of \(N\) is to be determined such that

\(P_{n}= \begin{cases}\frac{(c \rho)^{n}}{n !} P_{0} & n=0,1, \ldots, c-1 \\ \frac{c^{c} \rho^{n}}{c !} P_{0} & n=c, \ldots, N \\ 0 & n>N\end{cases}\) (10.152)

\(P_{N}=\frac{\rho^{N}(1-\rho)}{1-\rho^{N+1}} \leq 0.02\)

Or

\(\begin{aligned}&\rho^{N}-\rho^{N+1}=0.02-0.02 \rho^{N+1} \\&\rho^{N}(1-0.98 \rho) \leq 0.02 \\&\rho^{N} \leq \frac{0.02}{1-0.98 \rho} \\&(0.75)^{N} \leq \frac{0.02}{1-0.98(0.75)}\end{aligned}\)

Taking the logarithm of both sides gives

\(\begin{aligned}&N \log (0.75) \leq \log (0.07547) \\&N \geq 8.982\end{aligned}\)

Thus, \(N=9\), and the accumulation line will have space to accommodate eight parts. The operating characteristics for the system can be obtained using the results given in Table 10.36. Namely, for \(N=9, \lambda=1, \mu=4 / 3, c=1\), and \(\rho=\lambda / \mathrm{c} \mu=0.75\),

\(\begin{aligned}&L=\frac{\rho\left[1-\rho^{N} N \rho^{N}(1-\rho)\right]}{(1-\rho)\left(1-\rho^{N-1}\right)}=2.403 \\&L_{q}=\frac{\rho^{2}\left[1-\rho^{N} N \rho^{N-1}(1-\rho)\right]}{(1-\rho)\left(1-\rho^{N-1}\right)}=1.668 \\&W=\frac{1-\rho^{N} N \rho^{N}(1-\rho)}{\mu(1-\rho)\left(1-\rho^{N}\right)}=2.452 \text { minutes } \\&W_{q}=\frac{\left[1-\rho^{N} N \rho^{N-1}(1-\rho)\right]}{\mu(1-\rho)\left(1-\rho^{N}\right)}=1.702 \text { minutes } \\&U=\frac{\rho\left(1-\rho^{N}\right)}{1-\rho^{N+1}}=0.735 \quad \text { or } \quad 73.5 \%\end{aligned}\)

For a \((M|M| 1):(\mathrm{GD}|N| \infty)\) queue, Figure \(10.62\) provides the probability of the system not being full. If \(\rho=0.75\), and it is desired to have no more than \(2 \%\) of customers turned away, then \(1-P_{N}=0.98\), and \(N=9 .\) Similarly, for no more than \(8 \%\) of customers turned away, \(N=5\left(P_{N}=0.0722\right.\) for \(N=5\) when \(\left.\rho=0.75\right)\).

Values of \(L\) and \(L_{q}\) can be obtained using Figures \(10.63\) and \(10.64\) for the \((M|M| c):(\mathrm{GD}|\infty| \infty)\) case. Notice, when \(N=\infty\), then \(U=\rho .\) Figures \(10.63\) and \(10.64\) indicate an important concept when designing material handling systems. Namely, as equipment utilization \((\rho)\) increases, the waiting lines for material to be moved increase exponentially. It is a common tendency for managers to evaluate material handling effectiveness on the basis of equipment utilization; in many cases, that is counterproductive for the overall system.
