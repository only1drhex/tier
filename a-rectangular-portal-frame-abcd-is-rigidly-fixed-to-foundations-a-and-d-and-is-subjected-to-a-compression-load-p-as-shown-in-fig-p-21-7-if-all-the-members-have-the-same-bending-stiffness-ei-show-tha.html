Guest Question.

A rectangular portal frame ABCD is rigidly fixed to foundations A and D and is subjected to a compression load P as shown in Fig. P.21.7. If all the members have the same bending stiffness EI show that the buckling loads for modes which are symmetrical about the vertical centre line are given by the transcendental equation

\(\frac{\mu a}{2}=-\frac{1}{2}\left(\frac{a}{b}\right) \tan \left(\frac{\mu a}{2}\right)\)

where \(\mu^{2}=P / E I\).

Admin
Solution

The deflected shape of each of the members AB and BC is shown in Fig. S.21.7.

For the member AB and from Eq. (13.3)

\(\frac{ d ^{2} v}{ d x^{2}}=\frac{M}{E I}\) (13.3)

\(E I \frac{ d ^{2} v_{1}}{ d x_{1}^{2}}=-M_{ B }\)

Then

\(E I \frac{ d v_{1}}{ d x_{1}}=-M_{ B } x_{1}+C_{1}\)

When \(x_{1}=b, d v_{1} / d x_{1}=0\) so that \(C_{1}=M_{ B } b\) and

\(E I \frac{ d v_{1}}{ d x_{1}}=-M_{ B }\left(x_{1}-b\right)\) (i)

At B when \(x_{1}=0\) Eq. (i) gives

\(\frac{ d v_{1}}{ d x_{1}}=\frac{M_{ B } b}{E I}\) (ii)

In the member BC Eq. (13.3) gives

\(E I \frac{ d ^{2} v}{ d x^{2}}=-P v+M_{ B }\)

or

\(\frac{ d ^{2} v}{ d x^{2}}+\mu^{2} v=\frac{M_{ B }}{E I}\) (iii)

where \(\mu^{2}=P / E I\). The solution of Eq. (iii) is

\(v=C_{2} \cos \mu x+C_{3} \sin \mu x+\frac{M_{ B }}{P}\) (iv)

When x = 0, υ = 0 so that \(C_{2}=-M_{ B } / P\). Also, when x = a/2, dv/dx = 0 which gives

\(C_{3}=C_{2} \tan \frac{\mu a}{2}=-\frac{M_{ B }}{P} \tan \frac{\mu a}{2}\)

Eq. (iv) then becomes

\(v=-\frac{M_{ B }}{P}\left(\cos \mu x+\tan \frac{\mu a}{2} \sin \mu x-1\right)\)

Then

\(\frac{ d v}{ d x}=-\frac{M_{ B }}{P}\left(-\mu \sin \mu x+\mu \tan \frac{\mu a}{2} \cos \mu x\right)\)

At B when x = 0,

\(\frac{ d v}{ d x}=-\frac{M_{ B }}{P} \mu \tan \frac{\mu a}{2}\) (v)

Since \(d v_{1} / d x_{1}= d v / d x\) at B then, from Eqs (ii) and (v)

\(\frac{b}{E I}=-\frac{\mu}{P} \tan \frac{\mu a}{2}\)

Rearranging

\(\frac{\mu a}{2}=\frac{-1}{2}\left(\frac{a}{b}\right) \tan \frac{\mu a}{2}\)
