Guest Question.

The Shooting Method

Use the shooting method to solve Eq. (27.1)

\(\frac{d^{2} T}{d x^{2}}+h^{\prime}\left(T_{a}-T\right)=0\) (27.1)

for a 10-m rod with \(h^{\prime}=0.01 m ^{-2}, T_{a}=20,\) and the boundary conditions

T(0) = 40 T(10) = 200

Admin
Solution

Using the same approach as was employed to transform Eq. (PT7.2) into
Eqs. (PT7.3) through (PT7.6),

\(m \frac{d^{2} x}{d t^{2}}+c \frac{d x}{d t}+k x=\mid 0\) (PT7.2)

 

\(y=\frac{d x}{d t}\) (PT7.3)

\(\frac{dy}{dt}=\frac{d^2x}{dt^2}\) (PT7.4)

\(m\frac{dy}{dt}+cy+kx=0\) (PT7.5)

\(\frac{d y}{d t}=-\frac{c y+k x}{m}\) (PT7.6)

he second-order equation can be expressed as two first order ODEs:

\(\frac{d T}{d x}=z\) (E27.1.1)

 

\(\frac{d z}{d x}=h^{\prime}\left(T-T_{a}\right)\) (E27.1.2)

To solve these equations, we require an initial value for z. For the shooting method, we guess a valueâ€”say, z(0) = 10. The solution is then obtained by integrating Eq. (E27.1.1) and (E27.1.2) simultaneously. For example, using a fourth-order RK method with a step size of 2, we obtain a value at the end of the interval of T(10) = 168.3797 (Fig. 27.3a), which differs from the boundary condition of T(10) = 200. Therefore, we make another guess, z(0) = 20, and perform the computation again. This time, the result of T(10) = 285.8980 is
obtained (Fig. 27.3b).

Now, because the original ODE is linear, the values

z(0) = 10 T(10) = 168.3797

and

z(0) = 20 T(10) = 285.8980

are linearly related. As such, they can be used to compute the value of z(0) that yields T(10) = 200. A linear interpolation formula [recall Eq. (18.2)]

\(f_{1}(x)=f\left(x_{0}\right)+\frac{f\left(x_{1}\right)-f\left(x_{0}\right)}{x_{1}-x_{0}}\left(x-x_{0}\right)\) (18.2)

can be employed for this purpose:

\(z(0)=10+\frac{20-10}{285.8980-168.3797}(200-168.3797)=12.6907\)

This value can then be used to determine the correct solution, as depicted in Fig. 27.3c.
