Guest Question.

Outside cool air at 15°C and 40% relative humidity (airstream 1) is mixed with inside air taken near the ceiling at 32°C and 70% relative humidity (airstream 2). Determine the relative humidity and temperature of the resultant airstream 3 if the outside flow rate is 40 m3/ min and the inside flow rate is 20 m³/ min.

Admin
Solution

An energy and mass balance of the mixing of airstream 1 with airstream 2 to produce airstream 3 would reveal the following facts relative to the psychrometric chart:
State 3 lies on a straight line connecting state 1 and state 2.

\( \text { The ratio of the distance } 2-3 \text { to the distance } 3-1 \text { is equal to } \dot{m}_{a 1} / \dot{m}_{a 2} \text {. } \) \( \text { State } 1 \text { and state } 2 \text { can be located on the psychrometric chart. We must determine } \dot{m}_{a 1} \text { and } \dot{m}_{a 2}: \)

\( P_{a 1}=P-P_{v 1}=100-1.7=98.3 kPa \quad P_{a 2}=P-P_{v 2}=100-4.8=95.2 kPa \).

\( \therefore \rho_{a 1}=\frac{98.3}{(0.287)(288)}=1.19 kg _{a} / m ^{3} \quad \rho_{a 2}=\frac{95.2}{(0.287)(305)}=1.09 kg _{a} / m ^{3} \).

\( \therefore \dot{m}_{a 1}=(40)(1.19)=47.6 kg _{a} / min \quad \dot{m}_{a 2}=(20)(1.09)=21.8 kg _{a} / min \).

\( \text { State } 3 \text { is located by the ratio } d_{2-3} / d_{3-1}=\dot{m}_{a 1} / \dot{m}_{a 2}=47.6 / 21.8=2.18, \text { where } d_{2-3} \text { is the distance from state } 2 \text { to state } 3 . \text { State } 3 \) \( \text { is positioned on the psychrometric chart, and we find } \phi_{3}=63 \% \text { and } T_{3}=20.2^{\circ} C \text {. } \)
