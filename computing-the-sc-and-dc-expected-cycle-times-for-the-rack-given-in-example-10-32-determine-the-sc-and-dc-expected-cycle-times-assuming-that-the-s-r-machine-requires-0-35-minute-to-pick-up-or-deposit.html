Guest Question.

Computing the \(S C\) and \(D C\) expected cycle times

For the rack given in , determine the \(S C\) and \(D C\) expected cycle times assuming that the \(\mathrm{S} / \mathrm{R}\) machine requires \(0.35\) minute to pick up or deposit a load, travels horizontally at an average speed of \(350 \mathrm{fpm}\), and travels vertically at an average speed of \(60 \mathrm{fpm}\).

The dimensions (inches) of the storage rack are \(56 m\) (length) by \(58 n\) (height), where \(m\) is the number of horizontal addresses and \(n\) is the number of vertical addresses.

Admin
Solution

For the example, \(m=80\) and \(n=12\). Therefore, the rack dimensions are

\(L=56(80) / 12=373.33^{\prime} \quad \text { and } \quad H=58(12) / 12=58^{\prime}\)

(Note: It is assumed the \(\mathrm{S} / \mathrm{R}\) machine travels a maximum of \(373.33^{\prime}\) horizontally and \(58^{\prime}\) vertically. Actually, the shuttle mechanism must be lifted only to the load support position of the twelfth storage level, rather than to the top of the storage rack. Also, the horizontal travel distance is underestimated from \(2^{\prime}\) to \(10^{\prime}\), depending on the exact location of the I/O point.)

The maximum horizontal and vertical travel times are determined as follows:

\(\begin{aligned}t_{b} &=L / b_{v}=373.33 / 350 \mathrm{fpm}=1.067 \text { minutes } \\t_{v} &=H / v_{v}=58 / 60 \mathrm{fpm}=0.967 \text { minute }\end{aligned}\)

Thus,

\(T=\max (1.067,0.967)=1.067 \text { minutes }\)

and

\(Q=\frac{0.967}{1.067}=0.906\)

Therefore, the SC and DC cycle times are

\(\begin{aligned} T_{S C} &=T\left(1+\frac{Q^{3}}{3}\right)+2 T_{P / D} \\ &=1.067\left[1+\frac{(0.906)^{3}}{3}\right]+2(0.35) \\ &=2.06 \text { minutes per single command cycle } \\ T_{D C} &=\frac{T}{30}\left(40+15 Q^{2}-Q^{3}\right)+4 T_{P / D} \\ &=\frac{1.067}{30}\left[40+15(0.906)^{2}-(0.906)^{3}\right]+4(0.35) \\ &=3.23 \text { minutes per DC cycle } \end{aligned}\)

Hence, the average time per operation is \(2.06\) minutes with an SC cycle and \(1.615\) minutes with a DC cycle (since two operations are performed per DC cycle).

Since the expected \(S C\) and \(D C\) travel times were derived without considering the impact of \(\mathrm{S} / \mathrm{R}\) machine acceleration and deceleration, \(b_{v}\) and \(v_{v}\) in the above example reflect the average travel speed of the \(\mathrm{S} / \mathrm{R}\) machine. Alternatively, one can set \(b_{v}\) and \(v_{v}\) equal to the full travel speed of the \(\mathrm{S} / \mathrm{R}\) machine and then adjust the expected \(\mathrm{SC}\) and DC travel times to account for acceleration and deceleration. For expected S/R machine cycle times including acceleration and deceleration, the interested reader may refer to [17].
