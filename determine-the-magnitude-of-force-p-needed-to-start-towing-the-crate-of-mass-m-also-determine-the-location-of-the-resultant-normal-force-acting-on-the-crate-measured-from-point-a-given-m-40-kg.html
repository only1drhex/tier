Guest Question.

Determine the magnitude of force \(P\) needed to start towing the crate of mass \(M\). Also determine the location of the resultant normal force acting on the crate, measured from point \(A\).

Given:

\(M\) = \(40\) \(kg\quad \quad c\) = \(200\) \(mm\)

\({ \mu }_{ s }\) = \(0.3\quad \quad \quad d\) = \(3\)

\(a\) = \(400\) \(mm\quad \quad e\) = \(4\)

\(b\) = \(800\) \(mm\)

Admin
Solution

Initial guesses: \(\quad { N }_{ C }=200N\quad \quad P=50N\)

Given:

\(\sum { { F }_{ x } } =0;\quad \left( \frac { d }{ \sqrt { { d }^{ 2 }+{ e }^{ 2 } } } \right) P-{ \mu }_{ s }{ N }_{ C }=0\)

 

\(\sum { { F }_{ Y } } =0;\quad { N }_{ C }-Mg+\frac { eP }{ \sqrt { { d }^{ 2 }+{ e }^{ 2 } } } =0\)

 

\(\quad \quad \quad \quad \quad \quad \left( \begin{matrix} { N }_{ C } \\ P \end{matrix} \right) =Find\left( { N }_{ C },P \right)\)

 

\(\quad \quad \quad \quad \quad \quad { N }_{ C }=280.2N\quad \quad \quad P=140N\)

 

\(\sum { { M }_{ O } } =0;\quad -{ \mu }_{ s }{ N }_{ C }\left( \frac { a }{ 2 } \right) -{ N }_{ 1 }x+\left( \frac { eP }{ \sqrt { { d }^{ 2 }+{ e }^{ 2 } } } \right) \left( \frac { b }{ 2 } \right) =0\)

 

\(\quad \quad \quad \quad \quad \quad x=\frac { -1 }{ 2 } \frac { { \mu }_{ s }{ N }_{ C }a\sqrt { { d }^{ 2 }+{ e }^{ 2 } } -ePb }{ { N }_{ C }\sqrt { { d }^{ 2 }+{ e }^{ 2 } } } x=123.51mm\)

 

Thus, the distance from \(A\) is \(\quad A=x+\frac { b }{ 2 } \quad \quad A=523.51mm\)
