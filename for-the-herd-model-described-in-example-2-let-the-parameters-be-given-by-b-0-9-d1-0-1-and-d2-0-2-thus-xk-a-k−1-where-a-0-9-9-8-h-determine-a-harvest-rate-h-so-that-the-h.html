Guest Question.

For the herd model described in , let the parameters be given by b = 0.9, \(d_{1}=0.1, \text { and } d_{2}=0.2 . \text { Thus } x _{k}=A x _{k-1}\), where

\(A=\left[\begin{array}{cc}0 & .9 \\.9 & .8-h\end{array}\right]\).

Determine a harvest rate h so that the herd neither dies out nor grows without bound.

Admin
Solution

For any given harvest rate h, the matrix A will have eigenvalues \(\lambda_{1} \text { and } \lambda_{2}\), where \( \left|\lambda_{1}\right| \geq\left|\lambda_{2}\right| . \text { If }\left|\lambda_{1}\right|<1, \text { then }\left\{ x _{k}\right\} \rightarrow \theta\), and the herd is dying out. If \(\left|\lambda_{1}\right|>1\), then \(\left\{\left\| x _{k}\right\|\right\} \rightarrow \infty\), which indicates that the herd is increasing without bound.

Therefore, we want to select h so that \(\lambda_{1}=1\). For any given h, \(\lambda_{1} \text { and } \lambda_{2}\) are roots of the characteristic equation p(t) = 0, where

\(p(t)=\operatorname{det}(A-t I)=t^{2}-(.8-h) t-.81\).

To have \(\lambda_{1}=1\) we need p(1) = 0, or

\((1)^{2}-(.8-h)(1)-.81=0\),

or
h − .61 = 0.

Thus a harvest rate of h = 0.61 will lead to \(\lambda_{1}=1 \text { and } \lambda_{2}=-0.81\).
