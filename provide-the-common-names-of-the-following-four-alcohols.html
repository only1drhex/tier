Guest Question.

Provide the common names of the following four alcohols.

Admin
Solution

Common names are derived from the alkyl group that corresponds to the parent alkane. The name of the alkyl group is followed by the word alcohol. First, identify the carbinol carbon and the alkyl groups of which they are a part.

In the structures above, the carbinol carbon is highlighted in red. You may wish to review the names of the continuous-and branched-chain alkyl groups in Tables 10.5 and 10.6.

TABLE 10.5 Names and Formulas of the First Five Continuous-Chain Alkyl Groups
Alkyl Group Structure Name
\(—CH_{3}\) Methyl
\(—CH_{2}CH_{3}\) Ethyl
\(—CH_{2}CH_{2}CH_{3}\) Propyl
\(—CH_{2}CH_{2}CH_{2}CH_{3}\) Propyl
\(—CH_{2}CH_{2}CH_{2}CH_{2}CH_{3}\) Pentyl
TABLE 10.6 Structures and Names of Some Branched-Chain Alkyl Groups
Structure Classification Common Name IUPAC Name
2° Isopropyl\(^{*}\) 1-Methylethyl
1° Isobutyl\(^{*}\) 1-Methylethyl
2° sec-Butyl\(^{†}\) 1-Methylpropyl
3° t-Butyl or tert-Butyl\(^{‡}\) 1,1-Dimethylethyl

a. In structure a, the carbinol carbon is part of a four-carbon continuous chain alkyl group. Referring to Table 10.5, we see that this is the butyl group. The common name for this alcohol is butyl alcohol.

b. In structure b, the carbinol carbon is bonded to two carbon atoms. From Table 10.6, we see that this is the sec-butyl group. The common name of this structure is sec-butyl alcohol.

c. In structure c, the alkyl group has two methyl groups. From Table 10.6, we see that this is an iso-alkyl group. In this case, it is the isobutyl group and the alcohol is isobutyl alcohol.

d. In structure d, the carbinol carbon is attached to three carbon atoms. From Table 10.6, we see that this is the tertor t-butyl group. This alcohol is tert-butyl alcohol.
