Guest Question.

A single-phase full-bridge inverter with square-wave output voltage has the following specifications:

Each output voltage pulse has a width of \(\delta\) =180°;

dc input voltage: V\(_{in}\) =100 Vdc;

Load resistance: R = 10 Ω;

Load inductance: L = 25 mH;

Output voltage fundamental frequency: f = 50 Hz.

Calculate the following:

a) The output current function.

b) The active power consumed by the load.

c) The average input current.

Admin
Solution

From the specifications of the inverter, the frequency of the output voltage and the time constant of the load are:

\(T=\frac{1}{f}=\frac{1}{50}=0.02 s\) \(\frac{1}{\tau}=\frac{R}{L} =\frac{10}{25 \times 10^{-3}} = 400 s^{-1}\)

a) During the first half-cycle of the output voltage, switches \(S_1\) and \(S_4\) conduct and, consequently, the equivalent circuit of the inverter is the following:

The inverter output current during the first half-cycle is:

\(i_o(t)\) = steady-state component + transient component

\(=i_F + i_N =\frac{V_{in}}{R} +Ae^{-\frac{R}{L}t}\) for \(0 ≤ t ≤ \frac{T}{2}\) (1)

During the second half-cycle switches \(S_2\) and \(S_3\) conduct (after switches \(S_1, S_4\) have been turned off before) and, consequently, the inverter equivalent circuit is:

The inverter output current during the second half-cycle is:

\(i_o(t)\) = steady-state component + transient component

\(= i_F + i_N =\frac{-V_{in}}{R} +Be^{-\frac{R}{L}t}\) for \(\frac{T}{2} ≤ t ≤ T\) (2)

The following figure shows the inverter output voltage and current waveforms.

In steady-state the output current is periodical and symmetrical around zero value as shown in the figure shown above. Assuming that the output current varies between a minimum value, \(I_{min}\), and a maximum value, \(I_{max}\), the values of constants A and B of Eqs. (1) and (2) can be calculated as follows:

\(i_o(0) =\frac{V_{in}}{R}+ Ae^0 = I_{min}\) or \(A = I_{min} -\frac{V_{in}}{R}\)

and

\(i_o (\frac{T}{2}) =-\frac{V_{in}}{R}+ Be^0 = I_{max}\) or \(B = I_{max} +\frac{V_{in}}{R}\)

Therefore, the output current during the two half-cycles is given by:

\(i_o =\frac{V_{in}}{R} + (I_{min} -\frac{V_{in}}{R}) e^{-\frac{R}{L}t}\) for \(0 ≤ t ≤ \frac{T}{2}\)

and

\(i_o =-\frac{V_{in}}{R} + (I_{max} +\frac{V_{in}}{R}) e^{-\frac{R}{L}t}\) for \(\frac{T}{2} ≤ t ≤ T\) (3)

Using Eq. (3) the inverter minimum and maximum output current values can be calculated as follows:

\(i_o (\frac{T}{2})=I_{max} =\frac{V_{in}}{R} + (I_{min} -\frac{V_{in}}{R}) e^{-\frac{R}{L}(\frac{T}{2})}\) (4)

Due to the symmetry of \(i_o, I_{min} =-I_{max}\). Therefore, from Eq. (4) the values of \(I_{max}\) and \(I_{min}\) are obtained:

\(I_{max} = -I_{min} = \frac{V_{in}}{R}[\frac{1-e^{-\frac{R}{L}(\frac{T}{2})}}{1+e^{-\frac{R}{L}(\frac{T}{2})}}]\) (5)

Substituting the values of T and R/L into Eq. (5) the upper and lower limits of the output current are found:

\(I_{max} = -I_{min} = \frac{100}{10}[\frac{1-e^{-400(0.02/2)}}{1+e^{-400(0.02/2)}}] =9.64 A\)

b) The active power consumed by the load is given by:

\(P_o\) = output active power =\(\widetilde{I}^2_o R\) (6)

where \(\widetilde{I}_o\) = rms output current = \([\frac{1}{T}\int_{0}^{t}{i^2_o (t)dt}]^{1/2}\)

Since the output current has half-wave symmetry, the rms value can be calculated as follows:

\(\widetilde{I}_o = \sqrt{\frac{1}{T/2}\int_{0}^{T/2}{\left[\frac{V_{in}}{R}+\left(I_{min}-\frac{V_{in}}{R} \right)e^{-\frac{R}{L}t } \right]^{2} dt } }\)

 

\(=\sqrt{\frac{2}{0.02}\int_{0}^{\frac{0.02}{2}}{\left[\frac{100}{10}+\left(-9.64-\frac{100}{10} \right)e^{-400t } \right]^{2} dt } }\)

 

\(=\sqrt{100\int_{0}^{0.01}{\left[10+19.64e^{-400t } \right]^{2} dt } }\)

 

\(\approx 7.2 A\)

The rms output current can also be calculated using the following equation:

\(\widetilde{I}_o = \sqrt{\widetilde{I}^2_{o,1}+\widetilde{I}^2_{o,3}+\widetilde{I}^2_{o,5} + . . .} =\sqrt{(\frac{\widetilde{V}_{o,1} }{\left|z_{o,1}\right| } )^2 + (\frac{\widetilde{V}_{o,3} }{\left|z_{o,3}\right| } )^2 + (\frac{\widetilde{V}_{o,5} }{\left|z_{o,5}\right| } )^2 . . .}\)

 

\(\approx \sqrt{(\frac{\widetilde{V}_{o,1} }{\left|z_{o,1}\right| } )^2 + (\frac{\widetilde{V}_{o,3} }{\left|z_{o,3}\right| } )^2 + (\frac{\widetilde{V}_{o,5} }{\left|z_{o,5}\right| } )^2 } = \sqrt{\left(\frac{\frac{4(100)}{\sqrt{2}\pi} }{\sqrt{10^2+(2\pi\cdot 50\cdot 25\cdot 10^{-3})^2} } \right)^2 }\)

 

\(+\sqrt{\left(\frac{\frac{4(100)}{3\sqrt{2}\pi} }{\sqrt{10^2+(6\pi\cdot 50\cdot 25\cdot 10^{-3})^2} } \right)^2 +\left(\frac{\frac{4(100)}{5\sqrt{2}\pi} }{\sqrt{10^2(10\pi\cdot 50\cdot 25\cdot 10^{-3})^2} } \right)^2 } = 7.2 A\)

Therefore, \(P_o = (7.2)^2 \times 10 = 518.4 W\).

c) The average input current is given by:

\(\overline{I}_i =\frac{\overline{P}_i}{V_{in}}=\frac{dc input power}{dc input voltage}\) (7)

Assuming that the inverter is ideal (i.e., exhibits no power losses), then the output active power will be equal to the input active power and, consequently,

\(\overline{I}_i =\frac{\overline{P}_i}{V_{in}} =\frac{P_o}{V_{in}} =\frac{518.4}{100}=5.184 A\)
