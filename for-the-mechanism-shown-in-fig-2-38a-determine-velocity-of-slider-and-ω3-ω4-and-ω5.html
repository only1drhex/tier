Guest Question.

For the mechanism shown in Fig.2.38(a), determine velocity of slider and \( \omega_{3}, \omega_{4}, \text { and } \omega_{5} \).

Admin
Solution

Procedure:
1. Draw the configuration diagram shown in Fig.2.38(a) to a scale of 1 cm = 20 mm.
2. Angular speed of link 2,

\( \omega_{2}=150 rad / s \).

\( v_{b}=\omega_{2} \times O_{2} B=150 \times 30=4500 mm / s \).

3. Draw the velocity diagram shown in Fig.2.38(b) to a scale of 1 cm = 1000 mm/s.

\( \text { (i) Draw } v_{b}=o_{2} b \perp O_{2} B, o_{2} b=4.5 cm \).

\( \text { (ii) Draw } b c \perp B C \text { and } o_{4} c \perp O_{4} C \text { to intersect at point } c \).

\( \text { (iii) Draw } c d \perp C D \text { and } o_{4} d \perp O_{4} D \text { to intersect at } d \).

\( \text { (iv) Draw } d e \perp D E \text { and } o_{2} e \| y y \text { to intersect at } e \).

\( \text { (v) } \omega_{3}=\frac{v_{c b}}{B C}=\frac{b c}{B C}=\frac{5.8 \times 1000}{62.5}=92.8 rad / s , ccw \).

\( \omega_{4}=\frac{v_{c}}{O_{4} C}=\frac{o_{4} c}{O_{4} C}=\frac{5.5 \times 1000}{65}=84.6 rad / s , ccw \).

\( \omega_{5}=\frac{v_{d e}}{D E}=\frac{e d}{D E}=\frac{2.8 \times 1000}{75}=37.3 rad / s , ccw \).

\( \text { Velocity of slider, } v_{e}=o_{A} e=7.2 \times 1000=7200 mm / s \text { or } 7.2 m / s \text {. } \)
