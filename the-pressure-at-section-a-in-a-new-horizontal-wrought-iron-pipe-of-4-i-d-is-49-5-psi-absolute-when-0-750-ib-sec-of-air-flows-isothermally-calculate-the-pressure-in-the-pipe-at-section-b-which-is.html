Guest Question.

The pressure at section A in a new horizontal wrought iron pipe of 4″ I.D. is 49.5 psi absolute when 0.750 Ib/sec of air flows isothermally. Calculate the pressure in the pipe at section B, which is 1800 ft from A. (Absolute viscosity \(=3.90 \times 10^{-7} lb - sec / ft ^{2}\) and t = 90°F.) Use ε = 0.0003 ft.

Admin
Solution

Air has a variable density as pressure conditions change with flow.
The Bernoulli theorem for compressible fluids was applied in Chapter 7 to conditions involving no loss of head (ideal flow). The basic energy expression with loss of head included for a length of pipe dL and where \(z_{1}=z_{2}\) would be

\(\frac{d p}{\gamma}+\frac{V d V}{g}+f\left(\frac{d L}{d}\right)\left(\frac{V^{2}}{2 g}\right)=0\).

\(\text { Dividing by } \frac{V^{2}}{2 g} \),

\(\frac{2 g}{V^{2}}\left(\frac{d p}{\gamma}\right)+\frac{2 d V}{V}+\frac{f}{d} d L=0\).

For steady flow, the number of Ib/sec flowing is constant; then \(W=\gamma Q=\gamma A V, \text { and } W / \gamma A\) can be substituted for V in the pressure head term, giving

\(\frac{2 g \gamma^{2} A^{2}}{W^{2} \gamma} d p+\frac{2 d V}{V}+\frac{f}{d} d L=0\).

\(\text { For isothermal conditions, } p_{1} / \gamma_{1}=p_{2} / \gamma_{2}=R T \text { or } \gamma=p / R T \text {. Substituting for } \gamma \),

\(\frac{2 g A^{2}}{W^{2} R T} \int_{p_{1}}^{p_{2}} p d p+2 \int_{V_{1}}^{V_{2}} \frac{d V}{V}+\frac{f}{d} \int_{0}^{L} d L=0\).

f being considered a constant, as explained below. Integrating and substituting limits,

\(\frac{g A^{2}}{W^{2} R T}\left(p_{2}^{2}-p_{1}^{2}\right)+2\left(\ln V_{2}-\ln V_{1}\right)+f(L / d)=0\) (A).

\(\text { For comparison with the familiar form of the equation }\left(z_{1}=z_{2}\right) \text {, we obtain }\)

\(\left(K p_{1}^{2}+2 \ln V_{1}\right)-f(L / d)=\left(K p_{2}^{2}+2 \ln V_{2}\right)\). (B)

\(\text { where } K=\frac{g A^{2}}{W^{2} R T} \text {. Rearranging }(A) \text { for ready solution, }\)

\(p_{1}^{2}-p_{2}^{2}=\frac{W^{2} R T}{g A^{2}}\left[2 \ln \frac{V_{2}}{V_{1}}+f \frac{L}{d}\right]\). (C)

\(\text { Now } W^{2} / A^{2}=\gamma_{1}^{2} A_{1}^{2} V_{1}^{2} / A_{1}^{2}=\gamma_{1}^{2} V_{1}^{2} \text { and } R T=p_{1} / \gamma_{1} \text {; hence }\)

\(\frac{W^{2} R T}{g A^{2}}=\frac{\gamma_{1} V_{1}^{2} p_{1}}{g}\) (D).

\(\text { Then }(C) \text { becomes }\left(p_{1}-p_{2}\right)\left(p_{1}+p_{2}\right)=\frac{\gamma_{1} p_{1} V_{1}^{2}}{g}\left[2 \ln \frac{V_{2}}{V_{1}}+f \frac{L}{d}\right] \text { and }\).

\(\frac{p_{1}-p_{2}}{\gamma_{1}}=\frac{2\left[2 \ln \frac{V_{2}}{V_{1}}+f \frac{L}{d}\right] \frac{V_{1}^{2}}{2 g}}{1+p_{2} / p_{1}}=\text { lost head }\) (E).

Limiting pressures and velocities will be discussed in Chapter 13.
Before solving this expression, investigation of friction factor f is important, as velocity V is not constant for gases where changes in density may occur.

\(\operatorname{Re}=\frac{V d}{\mu / \rho}=\frac{V d \rho}{\mu}=\frac{W d \rho}{\gamma A \mu}\)

\(\text { Since } g=\frac{\gamma}{\rho}, \operatorname{Re}=\frac{W d}{A g \mu}\) (F).

It should be observed that the Reynolds number is constant for steady flow because fji does not vary if there is no temperature change. Hence friction factor / is constant for the problem even though the velocity will increase as the pressure decreases. Solving equation (F), using the absolute viscosity given,

\(\operatorname{Re}=\frac{0.750 \times 4 / 12 \times 10^{9}}{(\pi / 4)(4 / 12)^{2} \times 32.2 \times 390}=228,000 \text {. From Diagram A-1, } f=0.0205 \text { for } \epsilon / d=0.0009 \).

\(\text { Using equation }(C) \text { and neglecting } 2 \ln V_{2} / V_{1} \text {, which is very small compared to the } f(L / d) \text { term, }\)

\((49.5 \times 144)^{2}-p_{2}^{2}=\frac{(0.750)^{2} \times(53.3)(90+460)}{(32.2)\left[(\pi / 4)(4 / 12)^{2}\right]^{2}}\left[\text { negl. }+(0.0205) \frac{1800}{4 / 12}\right]\).

\(\text { from which } p_{2}=6585 psf =45.7 \text { psi absolute. }\)

AtB:

\(\gamma_{2}=\frac{6585}{(53.3)(90+460)}=0.225 lb / ft ^{3}, \quad V_{2}=\frac{W}{\gamma_{2} A}=\frac{0.750}{0.225 \times 0.0873}=38.2 ft / sec\).

At A:

\(\gamma_{1}=\frac{49.5 \times 144}{(53.3)(90+460)}=0.243 lb / ft ^{3}, \quad V_{1}=\frac{0.750}{0.243 \times 0.0873}=35.4 ft / sec\).

\(\text { Hence } 2 \ln V_{2} / V_{1}=2 \ln (38.2 / 35.4)=0.152 \text {, which is negligible in terms of the } f(L / d) \text { term of }\)\(\text { 111. Therefore the pressure at section } B \text { is } p_{2}=45.7 psi \).

Had the air been treated as incompressible, then

\(\frac{p_{1}-p_{2}}{\gamma_{1}}=f \frac{L}{d}\left(\frac{V^{2}}{2 g}\right)=0.0205 \times \frac{1800}{4 / 12} \times \frac{(35.4)^{2}}{2 g}=2154 ft\).

\(\Delta p=\gamma_{1} h=0.243 \times 2154=523 psf =3.63 psi\).

\(\text { and } p_{2}^{\prime}=49.5-3.6=45.9 psi \text {, an unusually close agreement}\).
