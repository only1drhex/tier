Guest Question.

Estimate the position of the feed point for the separation considered in , for a reflux ratio of 3.

Admin
Solution

Use the Kirkbride equation, equation 11.62. Product distributions taken from,

 

\(\log \left[\frac{N_{r}}{N_{s}}\right]=0.206 \log \left[\left(\frac{B}{D}\right)\left(\frac{x_{f, HK }}{x_{f, LK }}\right)\left(\frac{x_{b, LK }}{x_{d, HK }}\right)^{2}\right]\) (11.62)

 

\(x_{b, LK }=\frac{1}{54.7}=0.018\)

 

\(x_{d, HK }=\frac{1}{45.3}=0.022\)

 

\(\log \left(\frac{N_{r}}{N_{s}}\right)=0.206 \log \left[\frac{54.7}{45.3}\left(\frac{0.20}{0.25}\right)\left(\frac{0.018}{0.022}\right)^{2}\right]\)

 

\(\log \left(\frac{N_{r}}{N_{s}}\right)=0.206 \log (0.65)\)

 

\(\frac{N_{r}}{N_{s}}=0.91\)

 

for R = 3, N = 12

number of stages, excluding the reboiler = 11

 

\(N_{r}+N_{s}=11\)

 

\(N_{s}=11-N_{r}=11-0.91 N_{s}\)

 

\(N_{s}=\frac{11}{1.91}=5.76, \text { say } \underline{\underline{6}}\)

 

Checks with the method used in , where the reflux ratio was 2.5.
