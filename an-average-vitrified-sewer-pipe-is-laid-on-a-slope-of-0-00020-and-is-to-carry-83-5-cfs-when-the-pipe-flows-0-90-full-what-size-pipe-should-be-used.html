Guest Question.

An average vitrified sewer pipe is laid on a slope of 0.00020 and is to carry 83.5 cfs when the pipe flows 0.90 full. What size pipe should be used?

Admin
Solution

From , n = 0.015.
Calculate the hydraulic radius R. (Refer to Fig. 10-7.)

\(R=\frac{A}{p}=\frac{\text { circle }-(\text { sector } A O C E-\text { triangle } A O C D)}{\operatorname{arc} A B C}\).

\(\text { Angle } \theta=\cos ^{-1}(0.40 d / 0.50 d)=\cos ^{-1} 0.800, \theta=36^{\circ} 52^{\prime}\).

\(\text { Area of sector } A O C E=\left[(2)\left(36^{\circ} 52^{\prime}\right) / 360^{\circ}\right]\left(\frac{1}{4} \pi d^{2}\right)=0.1609 d^{2}\).

\(\text { Length of arc } A B C=\pi d-\left[(2)\left(36^{\circ} 52^{\prime}\right) / 360^{\circ}\right](\pi d)=2.498 d \).

\(\text { Area of triangle } A O C D=(2)\left(\frac{1}{2}\right)(0.40 d)\left(0.40 d \tan 36^{\circ} 52^{\prime}\right)=0.1200 d^{2} \).

\(R=\frac{\frac{1}{4} \pi d^{2}-\left(0.1609 d^{2}-0.1200 d^{2}\right)}{2.498 d}=\frac{0.7445 d^{2}}{2.498 d}=0.298 d\).

(a) Using Kutter’s C (assumed as 100 for first calculation),

\(Q=C A \sqrt{R S}, \quad 83.5=(100)\left(0.7445 d^{2}\right) \sqrt{0.298 d(0.00020)}, \quad d^{5 / 2}=145.3, \quad d=7.33 ft\).

Checking on C, R = 0.298 Ã— 7.33 = 2.18 ft. gives C = 112. Recalculating,

\(d^{5 / 2}=(145.3)(100 / 112)=129.7 \quad \text { or } \quad d=7.00 ft \quad(C \text { checks satisfactorily. })\)

(b) Using Manning’s n (and above information),

\(Q=\frac{1.486}{n} A R^{2 / 3} S^{1 / 2}\).

\(83.5=\left(\frac{1.486}{0.015}\right)\left(0.7445 d^{2}\right)(0.298 d)^{2 / 3}(0.00020)^{1 / 2}, \quad d^{8 / 3}=179.4, \quad d=7.00 ft\).
