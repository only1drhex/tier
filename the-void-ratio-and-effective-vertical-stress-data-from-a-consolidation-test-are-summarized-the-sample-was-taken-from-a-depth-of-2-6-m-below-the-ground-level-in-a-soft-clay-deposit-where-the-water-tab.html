Solution
\(e_{0}=2.20 \text { and } G_{s}=2.70 \rightarrow \gamma_{\text {sat }}=15.0 kN / m ^{3}\)

 

\(\text { At } 2.6 m \text { depth, } \sigma_{v 0}^{\prime}=2.6 \times(15.0-9.81)=13.5 kPa\)

 

The e _ log \(\sigma_{v}^{\prime}\) plot is shown on page 166, along with Casagrande’s construction to determine \(\sigma_{p}^{\prime}\), which is about 43 kPa.

 

\(\therefore \text { The clay is overconsolidated with an OCR of } \frac{43}{13.5}=3.2\)
