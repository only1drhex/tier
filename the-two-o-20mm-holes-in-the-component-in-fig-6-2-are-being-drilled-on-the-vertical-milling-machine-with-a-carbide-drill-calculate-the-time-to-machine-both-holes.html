Guest Question.

The two Ø 20mm holes in the component in Fig. 6.2 are being drilled on the vertical milling machine with a carbide drill. Calculate the time to machine both holes.

Admin
Solution

\(V_{c} = 80 m min ^{-1}\) \(N = \frac{1000V_{c}}{\pi D }\)

D = 20 mm

N=? \(N = \frac{1000 × 80}{\pi ×20 } = 1273.24 rpm \)

A=? A = D/2

D = 20mm A = 20/2 = 10 mm

T=? \(T= \frac{L + 2A }{f_{r}N} \)

L = 22mm

A = 10mm \(T= \frac{22 + (2 × 10)}{0.32 × 1273.24} = 0.103 min\)

\(f_{r} = 0.32 mm rev^{- 1}\) (from Table 6.5)

N = 1273.24rev/min

Total time = 2 X 0.103 = 0.206 min
