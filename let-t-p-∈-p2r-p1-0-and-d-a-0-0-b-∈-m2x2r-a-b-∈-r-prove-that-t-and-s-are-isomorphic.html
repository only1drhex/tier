Guest Question.

Let \(\mathbb{T} = \left\{\mathbf{p} \in P_2(\mathbb{R} ) \ \ | \ \ \mathbf{p}(1) = 0 \right\} \) and \(\mathbb{D} = \left\{ \left [ \begin{matrix} a & 0 \\ 0 & b \end{matrix} \right ] \in M _2\times 2 (\mathbb{R} ) \ \ | \ \ a , b \in \mathbb{R} \right\}\) . Prove that \(\mathbb{T}\) and \(\mathbb{S}\) are isomorphic.

Admin
Solution

By the Factor Theorem, every vector \(\mathbf{p} \in \mathbb{T}\) has the form \((1 − x)(a + bx)\). Consequently, a basis for \(\mathbb{T}\) is \(\mathfrak{B} = \left\{1 − x, x − x^2\right\} \). We see that a basis for \(\mathbb{D}\) is \(C = \left\{\left [ \begin{matrix} 1 & 0 \\ 0 & 0 \end{matrix} \right ] , \left [ \begin{matrix} 0 & 0 \\ 0 & 1 \end{matrix} \right ] \right\} \). To define an isomorphism, we can map the vectors in \(\mathfrak{B}\) to the vectors in C. In particular, we define \(L : \mathbb{T} \rightarrow \mathbb{S}\) by

\(L \left(a(1 − x) + b(x − x^2)\right) = a \left [ \begin{matrix} 1 & 0 \\ 0 & 0 \end{matrix} \right ] + b \left [ \begin{matrix} 0 & 0 \\ 0 & 1 \end{matrix} \right ] \)

Linear: Let \( \mathbf{p}(x) = a_1(1 − x) + b_1(x − x^2), \mathbf{q}(x) = a_2(1 − x) + b_2(x − x^2) \in \mathbb{T}\). For any \(s, t \in \mathbb{R}\) we get

\(L(\mathbf{p} s + t \mathbf{q}) = L \left((sa_1 + ta_2)(1 − x) + (sb_1 + tb_2)(x − x^2)\right)\)

\( = (sa_1 + ta_2) \left [ \begin{matrix} 1 & 0 \\ 0 & 0 \end{matrix} \right ]+ (sb_1 + tb_2) \left [ \begin{matrix} 0 & 0 \\ 0 & 1 \end{matrix} \right ]\)

\( = s \left( a_1\left [ \begin{matrix} 1 & 0 \\ 0 & 0 \end{matrix} \right ] + b_1 \left [ \begin{matrix} 0 & 0 \\ 0 & 1 \end{matrix} \right ]\right) + t \left( a_2\left [ \begin{matrix} 1 & 0 \\ 0 & 0 \end{matrix} \right ] + b_2 \left [ \begin{matrix} 0 & 0 \\ 0 & 1 \end{matrix} \right ]\right) \)

= \( sL(\mathbf{p}) + tL(\mathbf{q})\)

Thus, L is linear.
One-to-one: Let \( a(1 − x) + b(x − x^2) \in ker(L)\). Then,

\( \left [ \begin{matrix} 0 & 0 \\ 0 & 0 \end{matrix} \right ] = L \left((1 − x) + b(x − x^2)\right) = a \left [ \begin{matrix} 1 & 0 \\ 0 & 0 \end{matrix} \right ] + b \left [ \begin{matrix} 0 & 0 \\ 0 & 1 \end{matrix} \right ] = \left [ \begin{matrix} a & 0 \\ 0 & b \end{matrix} \right ]\)

Hence, \( a = b = 0\) , so \( Null(L) = \left\{0\right\}\). Therefore, by Theorem 4.7.1 (A linear mapping \(L : \mathbb{U} \rightarrow \mathbb{V}\) is one-to-one if and only if \(Null(L) = \left\{0\right\} \).), L is one-to-one.
Onto: Let \( \left [ \begin{matrix} a & 0 \\ 0 & b \end{matrix} \right ] \in \mathbb{S}\). We have

\( L \left((1 − x) + b(x − x^2)\right) = a \left [ \begin{matrix} 1 & 0 \\ 0 & 0 \end{matrix} \right ] + b \left [ \begin{matrix} 0 & 0 \\ 0 & 1 \end{matrix} \right ] = \left [ \begin{matrix} a & 0 \\ 0 & b \end{matrix} \right ]\)

So, L is onto.
Thus, L is an isomorphism, and hence \( \mathbb{T}\) and \(\mathbb{S}\) are isomorphic.
