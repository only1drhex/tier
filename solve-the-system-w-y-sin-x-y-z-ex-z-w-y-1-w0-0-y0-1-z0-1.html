Guest Question.

Solve the system

\(\begin{aligned}&w^{\prime}+y=\sin x \\&y^{\prime}-z=e^{x} \\&z^{\prime}+w+y=1; \\&w(0)=0, \quad y(0)=1, \quad z(0)=1\end{aligned}\)

Admin
Solution

Denote \(\mathscr{L} \{w(x)\}, \mathscr{L} \{y(x)\} \text {, and } \mathscr{L} \{z(x)\} \text { by } W(s), Y(s) \text {, and } Z(s) \text {, }\), respectively. Then, taking Laplace transforms of all three differential equations, we have

\(\begin{gathered}{[s W(s)-0]+Y(s)=\frac{1}{s^{2}+1}} \\{[s Y(s)-1]-Z(s)=\frac{1}{s-1}} \\{[s Z(s)-1]+W(s)+Y(s)=\frac{1}{s}}\end{gathered}\) or \(\begin{aligned}&s W(s)+Y(s)=\frac{1}{s^{2}+1} \\&s Y(s)-Z(s)=\frac{s}{s-1} \\&W(s)+Y(s)+s Z(s)=\frac{s+1}{s}\end{aligned}\)

The solution to this last system of simultaneous linear equations is

\(W(s)=\frac{-1}{s(s-1)} \quad Y(s)=\frac{s^{2}+s}{(s-1)\left(s^{2}+1\right)} \quad Z(s)=\frac{s}{s^{2}+1}\)

Using the method of partial fractions and then taking inverse transforms, we obtain

\(\begin{aligned}&w(x)= \mathscr{L} ^{-1}\{W(s)\}= \mathscr{L} ^{-1}\left\{\frac{1}{s}-\frac{1}{s-1}\right\}=1-e^{x} \\&y(x)= \mathscr{L} ^{-1}\{Y(s)\}= \mathscr{L} ^{-1}\left\{\frac{1}{s-1}+\frac{1}{s^{2}+1}\right\}=e^{x}+\sin x \\&z(x)= \mathscr{L} ^{-1}\{Z(s)\}= \mathscr{L} ^{-1}\left\{\frac{s}{s^{2}+1}\right\}=\cos x\end{aligned}\)
