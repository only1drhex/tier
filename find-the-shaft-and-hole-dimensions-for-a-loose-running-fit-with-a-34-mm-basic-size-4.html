Guest Question.

Find the shaft and hole dimensions for a loose running fit with a 34-mm basic size.

Admin
Solution

From Table 7–9, the ISO symbol is 34H11∕c11. From , we find that tolerance grade IT11 is 0.160 mm. The symbol 34H11∕c11 therefore says that ΔD = Δd = 0.160 mm. Using Equation (7–36) for the hole, we get

\( D_{max} = D + ΔD = 34 + 0.160 = 34.160\) mm

\( D_{min} = D = 34.000 \)mm

The shaft is designated as a 34c11 shaft. From , the fundamental deviation is \(δ_{F} = −0.120 mm\). Using Equation (7–37), we get for the shaft dimensions

\(d_{max} = d + δ_{F} = 34 + (−0.120) = 33.880 mm\)

 

\( d_{min} = d + δ_{F} − Δd = 34 + (−0.120) − 0.160 = 33.720 mm\)
