Guest Question.

Determine the normal and shear stress on the plane AB shown in Fig. P.14.4 when
i. \(\alpha=60^{\circ}, \sigma_{x}=54 N / mm ^{2}, \sigma_{y}=30 N / mm ^{2}, \tau_{x y}=5 N / mm ^{2}\);
ii. \(\alpha=120^{\circ}, \sigma_{x}=-60 N / mm ^{2}, \sigma_{y}=-36 N / mm ^{2}, \tau_{x y}=5 N / mm ^{2}\).
Ans. (i) \(\sigma_{ n }=52.3 N / mm ^{2}, \tau=7.9 N / mm ^{2}\);
(ii) \(\sigma_{ n }=-58.3 N / mm ^{2}, \tau=7.9 N / mm ^{2}\).

Admin
Solution

The stress system acting on a triangular element formed by the plane AB and the vertical and horizontal sides of the rectangular element is shown in Fig. S.14.4.

Resolving forces perpendicular to AB

\(\sigma_{ n } AB -\sigma_{x} AC \sin \alpha-\sigma_{y} BC \cos \alpha-\tau_{x y} AC \cos \alpha-\tau_{x y} BC \sin \alpha=0\)

i.e.

\(\sigma_{ n }=\sigma_{x} \sin ^{2} \alpha+\sigma_{y} \cos ^{2} \alpha+\tau_{x y} \sin 2 \alpha\) (i)

Resolving forces parallel to AB

\(\tau AB +\sigma_{x} AC \cos \alpha-\sigma_{y} BC \sin \alpha+\tau_{x y} BC \cos \alpha-\tau_{x y} AC \sin \alpha=0\)

i.e.

\(\tau=-\sigma_{x} \sin \alpha \cos \alpha+\sigma_{y} \cos \alpha \sin \alpha-\tau_{x y}\left(\cos ^{2} \alpha-\sin ^{2} \alpha\right)\)

or

\(\tau=-\left(\frac{\sigma_{x}-\sigma_{y}}{2}\right) \sin 2 \alpha-\tau_{x y} \cos 2 \alpha\) (ii)

i. Substituting the given values in Eqs (i) and (ii) in turn gives

\(\sigma_{ n }=52.3 N / mm ^{2}, \tau=7.9 N / mm ^{2}\)

ii. Similarly for this case

\(\sigma_{ n }=-58.3 N / mm ^{2}, \tau=7.9 N / mm ^{2}\).
