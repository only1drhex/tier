Guest Question.

For the single-phase ac voltage controller of Fig. 8.1(a), the following specifications are given: \(v_i = 120 \sqrt{2} \sin \omega t\) and \(R = 15 \Omega\). Calculate:

a) The firing angle so that the output power is 500 W.

b) The rms output current.

c) The average and rms thyristor current values.

d) The power factor and the total harmonic distortion (THD) percentage of the output current.

Admin
Solution

a) Since the load is resistive the load angle \(\varphi = 0°\) and, consequently, the firing angle a can take values from 0° to 180°.

The active power dissipated in the load resistance is given by:

\(P_o = \widetilde{I}^2 R = \frac{\widetilde{V}^2_o}{R}\)

Therefore using the above equation for the given output active power the required rms output voltage should be:

\(\widetilde{V}_o = \sqrt{P_o \times R} = \sqrt{(500)(15)}= 86.6 V\)

Using Eq. (\(\widetilde{V}_o = [\frac{1}{\pi}[\int_{\alpha}^{\pi}{2 \widetilde{V}^2_i \sin^2(\omega t)d(\omega t)} + \int_{\pi +\alpha}^{2\pi}{2 \widetilde{V}^2_i \sin^2(\omega t)d(\omega t)} ]]^{1/2}\)

 

\(=\widetilde{V}_i [\frac{1}{\pi}(\pi – \alpha + \frac{\sin 2 \alpha }{2})]^{\frac{1}{2}}\) ) the firing angle that will generate the above rms output voltage is given by:

\((86.6)^2 = 120^2 \frac{1}{\pi} (\pi – \alpha + \frac{\sin 2\alpha}{2})\)

giving \(\alpha = 1.54 rad\) or \(\alpha = 88.1°\)

The waveforms of this example are shown in the following figure:

b) The rms output current is \(\widetilde{I}_o = \widetilde{I} = \frac{\widetilde{V}_o}{R} = \frac{86.6}{15} = 5.77 A\).

c) From the above waveforms, the following results are obtained:

\(\overline{I}_{Q_1} = \overline{I}_{Q_2} = \frac{1}{2\pi}\int_{\alpha}^{\pi}{\frac{\sqrt{2}\widetilde{V}_i }{R} \sin (\omega t) d(\omega t) = \frac{\widetilde{V}_i}{R}(1 + \cos \alpha)} =\frac{120\sqrt{2}}{15} (1 + \cos 88.1°)\)

 

\(= 1.86 A\)

Since the current pulses are almost half of a pure sinusoidal wave the rms thyristor current will be:

\(\widetilde{I}_{Q_1} =\widetilde{I}_{Q_2} = \frac{\widetilde{I}_{o}}{\sqrt{2}} =\frac{5.77}{\sqrt{2}} = 4.08 A\)

d) Assuming that the ac voltage controller is ideal (i.e., no power losses):

\(\lambda\) = input power factor =\(\frac{P_i}{S_i} = \frac{P_o}{S_i} = \frac{500}{\widetilde{V}_i \widetilde{I}} = \frac{500}{120 \times 5.77} = 0.72\)

Using Eqs. (\(\widehat{I}_{1,1} = \frac{\sqrt{2}\widetilde{V}_i}{\pi R}[\pi – \alpha +\frac{1}{2}\sin 2\alpha] \)), (\(\widehat{I}_{2,1} = – \frac{\sqrt{2}\widetilde{V}_i}{\pi R} \sin^2 \alpha\)), and (\(\widetilde{I}_1 = \frac{1}{\sqrt{2}}[\widehat{I}^2_{1,1} + \widehat{I}^2_{2,1}]^{\frac{1}{2}}\)), the following results are obtained:

\(\widetilde{I}_1 = \frac{1}{\sqrt{2}}[\widehat{I}^2_{1,1} + \widehat{I}^2_{2,1}]^{\frac{1}{2}}\)

where

\(\widehat{I}_{1,1} = \frac{120 \sqrt{2}}{15 \pi}(\pi – 1.54 + \frac{1}{2}\sin (2 – 88.1°)) = 5.76 A\)

 

\(\widehat{I}_{2,1} =\frac{120 \sqrt{2}}{15 \pi} \sin^2 88.1° = \frac{120 \sqrt{2}}{15 \pi} (\frac{1 – \cos 2 \times 88.1°}{2}) = 3.6 A\)

and, therefore, the rms value of the circuit current fundamental component is:

\(\widetilde{I}_1 = \frac{\sqrt{5.76^2 + 3.6^2}}{\sqrt{2}} = 4.8 A\)

Finally, using the above values the circuit current THD is:

\(THD_i \% = \frac{\sqrt{\widetilde{I}^2 – \widetilde{I}^2_1 }}{\widetilde{I}_1} = \frac{\sqrt{5.77^2 – 4.8^2}}{4.8} = 0.667 = 66.7\%\)
