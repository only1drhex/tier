Energy—Effect of Nonuniform Velocity Profile

GIVEN The small fan shown in Fig. E5.26 moves air at a mass flowrate of 0.1 kg/min. Upstream of the fan, the pipe diameter is 60 mm, the flow is laminar, the velocity distribution is parabolic, and the kinetic energy coefficient, \(\alpha_{1}\), is equal to 2.0.

Downstream of the fan, the pipe diameter is 30 mm, the flow is turbulent, the velocity profile is quite uniform, and the kinetic, \(\alpha_{2}\), is equal to 1.08. The rise in static pressure across the fan is 0.1 kPa and the fan motor draws 0.14 W.

FIND Compare the value of loss calculated: (a) assuming uniform velocity distributions, (b) considering actual velocity distributions.

Admin
Solution

Application of Eq. 5.87 to the contents of the control volume shown in Fig. E5.26 leads to

 

\(\frac{p_{\text {out }}}{\rho}+\frac{\alpha_{\text {out }} \bar{V}_{\text {out }}^{2}}{2}+g z_{\text {out }}=\frac{p_{\text {in }}}{\rho}+\frac{\alpha_{\text {in }} \bar{V}_{\text {in }}^{2}}{2}+g z_{\text {in }}+w\underset{\substack{\text { shaft } \\ \text { net in }}}{ }-\text { loss }\) (5.87)

 

(1)

 

or solving Eq. 1 for loss we get

 

\(\text { loss }=w_{\substack{\text { shaft } \\ \text { net in }}}-\left(\frac{p_{2}-p_{1}}{\rho}\right)+\frac{\alpha_{1} \bar{V}_{1}^{2}}{2}-\frac{\alpha_{2} \bar{V}_{2}^{2}}{2}\) (2)

 

To proceed further, we need values of \(w_{\text {shaft net in }} \bar{V}_{1}, \text { and } \bar{V}_{2}\). These quantities can be obtained as follows. For shaft work

 

\(\underset{\text { net in }}{w_{\text {shaft }}}=\frac{\text { power to fan motor }}{m}\)

 

or

 

\(\begin{array}{l}w_{\begin{array}{c}\text { shaft } \\\text { net in }\end{array}}=\frac{(0.14 W )[(1 N \cdot m / s ) / W ]}{0.1 kg / min }(60 s / min )\\=84.0 N \cdot m / kg\end{array}\) (3)

 

For the average velocity at section (1), \(\bar{V}_{1}\), rom Eq. 5.11 we obtain

 

\(\dot{m}=\rho A \bar{V}\) (5.11)

 

\(\begin{aligned}\bar{V}_{1} &=\frac{\dot{m}}{\rho A_{1}} \\&=\frac{\dot{m}}{\rho\left(\pi D_{1}^{2} / 4\right)} \\&=\frac{(0.1 kg / min )(1 min / 60 s )(1000 mm / m )^{2}}{\left(1.23 kg / m ^{3}\right)\left[\pi(60 mm )^{2} / 4\right]} \\&=0.479 m / s\end{aligned}\) (4)

 

For the average velocity at section (2), \(\bar{V}_{2}\),

 

\(\begin{aligned}\bar{V}_{2} &=\frac{(0.1 kg / min )(1 min / 60 s )(1000 mm / m )^{2}}{\left(1.23 kg / m ^{3}\right)\left[\pi(30 mm )^{2} / 4\right]} \\&=1.92 m / s\end{aligned}\) (5)

 

(a) For the assumed uniform velocity profiles (\(\alpha_{1}=\alpha_{2}=1.0\)), Eq. 2 yields

 

\(\text { loss }=w_{\substack{\text { shaft } \\ \text { net in }}}-\left(\frac{p_{2}-p_{1}}{\rho}\right)+\frac{\bar{V}_{1}^{2}}{2}-\frac{\bar{V}_{2}^{2}}{2}\) (6)

 

Using Eqs. 3, 4, and 5 and the pressure rise given in the problem statement, Eq. 6 gives

 

\(\begin{aligned}\text { loss }=& 84.0 \frac{ N \cdot m }{ kg }-\frac{(0.1 kPa )(1000 Pa / kPa )\left(1 N / m ^{2} / Pa \right)}{1.23 kg / m ^{3}} \\&+\frac{(0.479 m / s )^{2}}{2\left[1( kg \cdot m ) /\left( N \cdot s ^{2}\right)\right]}-\frac{(1.92 m / s )^{2}}{2\left[1( kg \cdot m ) /\left( N \cdot s ^{2}\right)\right]}\end{aligned}\)

 

or

 

\(\begin{aligned}\text { loss }=& 84.0 N \cdot m / kg -81.3 N \cdot m / kg \\&+0.115 N \cdot m / kg -1.84 N \cdot m / kg \\=& 0.975 N \cdot m / kg\end{aligned}\) (Ans)

 

(b) For the actual velocity profiles \(\left(\alpha_{1}=2, \alpha_{2}=1.08\right)\), Eq. 1 gives

 

\(\text { loss }=w_{\substack{\text { shaft } \\ \text { net in }}}-\left(\frac{p_{2}-p_{1}}{\rho}\right)+\alpha_{1} \frac{\bar{V}_{1}^{2}}{2}-\alpha_{2} \frac{\bar{V}_{2}^{2}}{2}\) (7)

 

If we use Eqs. 3, 4, and 5 and the given pressure rise, Eq. 7 yields

 

\(\begin{aligned}\text { loss }=& 84 N \cdot m / kg -\frac{(0.1 kPa )(1000 Pa / kPa )\left(1 N / m ^{2} / Pa \right)}{1.23 kg / m ^{3}} \\&+\frac{2(0.479 m / s )^{2}}{2\left[1( kg \cdot m ) /\left( N \cdot s ^{2}\right)\right]}-\frac{1.08(1.92 m / s )^{2}}{2\left[1( kg \cdot m ) /\left( N \cdot s ^{2}\right)\right]}\end{aligned}\)

 

or

 

\(\begin{aligned}\text { loss }=& 84.0 N \cdot m / kg -81.3 N \cdot m / kg \\&+0.230 N \cdot m / kg -1.99 N \cdot m / kg \\=& 0.940 N \cdot m / kg\end{aligned}\) (Ans)

 

COMMENT The difference in loss calculated assuming uniform velocity profiles and actual velocity profiles is not large compared to \(W_{\text {shaft net in }}\) for this fluid flow situation.
