Guest Question.

Estimate the surface tension of pure methanol at \(20^{\circ} C \text {, density } 791.7 kg / m ^{3}\), molecular weight 32.04.

Admin
Solution

Calculation of parachor, \(CH _{3} OH\), Table 8.7.

 

Table 8.7. Contribution to Sugdens’s parachor for organic compounds (Sugden, 1924)
Atom, group or bond Contribution Atom, group or bond Contribution
C 4.8 Si 25.0
H 17.1 AI 38.6
H in (OH) 11.3 Sn 57.9
O 20.0 As 50.1
\(O _{2} \text { in esters, acids }\) 60.0 Double bond: terminal
N 12.5 2,3-position 23.2
S 48.2 3,4-position
P 37.7 Triple bond 46.6
F 25.7 Rings
CI 54.3 3-membered 16.7
Br 68.0 4-membered 11.6
I 91.0 5-membered 8.5
Se 62.5 6-membered 6.1

 

Group Contribution No.
C 4.8 x 1 = 4.8
\(H – O\) 11.3 x 1 = 11.3
\(H – C\) 17.1 x 3 = 51.3
O 20.0 x 1 = 20.0
87.4

 

\(\sigma=\left[\frac{87.4 \times 791.7}{32.04}\right]^{4} \times 10^{-12}=\underline{\underline{21.8} mJ / m ^{2}}\) (8.23)

 

\(\text { Experimental value } 22.5 mJ / m ^{2} \text {. }\)
