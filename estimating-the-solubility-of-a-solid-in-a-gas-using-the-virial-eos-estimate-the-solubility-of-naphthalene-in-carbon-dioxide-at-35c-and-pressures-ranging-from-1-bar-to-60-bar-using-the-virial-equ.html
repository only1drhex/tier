Estimating the Solubility of a Solid in a Gas Using the Virial EOS

Estimate the solubility of naphthalene in carbon dioxide at 35°C and pressures ranging from 1 bar to 60 bar using the virial equation of state with the following values for the second virial coefficient

\(\begin{aligned}B_{ CO _{2}- N } &=-345 cc / mol \\B_{ CO _{2}- CO _{2}} &=-96.5 cc / mol \\B_{ N - N } &=-1850 cc / mol\end{aligned}\)
Admin
Solution

We assume that \(CO _{2}\) is insoluble in solid naphthalene, and therefore only equate the fugacities of naphthalene in the solid and vapor phases. The fugacity of solid naphthalene was calculated in . To calculate fugacity of naphthalene in the vapor phase, we use Eqs. 9.4-6 and 9.4-7. The procedure is to guess a vapor-phase mole fraction of naphthalene and then calculate its vapor-phase fugacity, which is compared with its solid-phase fugacity. If the two agree, the guessed composition is correct. If they do not, a new composition guess is made, and the calculation is repeated. The results are shown below.

 

\(\begin{aligned}\ln \frac{\bar{f}_{ i }^{ V }(T, P, y)}{y_{ i } P} &=\frac{2}{\underline{V}} \sum_{ j } y_{ j } B_{ ij }(T)-\ln Z_{ mix } \\&=\frac{2 P}{Z_{ mix } R T} \sum_{ j } y_{ j } B_{ ij }(T)-\ln Z_{ mix }\end{aligned}\) (9.4-6)

 

\(Z_{ mix }=\frac{1}{2}\left(1+\sqrt{1+\frac{4 B_{ mix } P}{R T}}\right)\) (9.4-7)
