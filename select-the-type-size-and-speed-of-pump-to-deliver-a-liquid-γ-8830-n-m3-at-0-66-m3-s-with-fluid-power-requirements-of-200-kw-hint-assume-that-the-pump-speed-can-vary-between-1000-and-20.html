Guest Question.

Select the type, size, and speed of pump to deliver a liquid \(\left(\gamma=8830 N / m ^{3}\right)\) at 0.66 \(m ^{3} / s\) with fluid power requirements of 200 kW. (Hint: Assume that the pump speed can vary between 1000 and 2000 rpm.)

Admin
Solution

Assume a pump speed N = 2000 rpm, or \(\omega=2000 \times \frac{\pi}{30}=209 rad / s\).

 

\(H_{P}=\dot{W}_{f} /(\gamma Q)=200 \times 10^{3} /(8830 \times 0.66)=34.3 m\),

 

\(\therefore \Omega_{P}=\frac{\omega \sqrt{Q}}{\left(g H_{P}\right)^{3 / 4}}=\frac{209 \sqrt{0.66}}{(9.81 \times 34.3)^{3 / 4}}=2.16\).

 

The specific speed suggests a mixed-flow pump. However, if N = 1000 rpm, a radial-flow pump may be appropriate. Consider both possibilities.

 

\(\underline{\text { Mixed flow: }}\) from Fig. 12.14, at best η ,:

 

\(C_{\dot{W}} \cong 0.0117, \quad C_{Q} \cong 0.148, \quad C_{H} \cong 0.067\).

 

Use

 

\(C_{Q}=\frac{Q}{\omega D^{3}} \quad \text { and } \quad C_{H}=\frac{g H_{P}}{\omega^{2} D^{2}}\)

 

Combining and solving for D and ω,

 

\(D=\sqrt{\frac{Q / C_{Q}}{\sqrt{g H_{P} / C_{H}}}} \text { and } \omega=\frac{Q}{C_{Q} D^{3}}\).

 

\(\therefore D=\sqrt{\frac{0.66 / 0.148}{\sqrt{9.81 \times 34.3 / 0.067}}}=\underline{0.251 m }\),

 

\(\omega=\frac{0.66}{0.148 \times 0.251^{3}}=\underline{282 rad / s }, \quad \text { or } N=282 \times 30 / \pi=\underline{2674 rpm }\).

 

\(\rho=\frac{\gamma}{g}=\frac{8830}{9.81}=900 kg / m ^{3}\),

 

\(\begin{aligned}\therefore \dot{W}_{P} &=C_{\dot{W}} \rho \omega^{3} D^{5}=0.0117 \times 900 \times 282^{3} \times 0.251^{5} \\&=\underline{2.35 \times 10^{5} W } \text {, or } \underline{235 kW} .\end{aligned}\)

 

\(\underline{\text {Radial flow}}\): \(\text { from Fig. } 12.12, \text { at best } \eta\),:

 

\(C_{W} \cong 0.027, \quad C_{Q} \cong 0.0165, \quad C_{H} \cong 0.125\).

 

\(\therefore D=\sqrt{\frac{0.66 / 0.0165}{\sqrt{9.81 \times 34.3 / 0.125}}}=\underline{0.878 m }\),

 

\(\omega=\frac{0.66}{0.0165 \times 0.878^{3}}=\underline{59.1 rad / s }, \quad \text { or } N=\frac{59.1 \times 30}{\pi}=\underline{564 rpm} .\)

 

\(\dot{W}_{P}=0.0027 \times 900 \times 59.1^{3} \times 0.878^{5}=\underline{2.62 \times 10^{5} W }, \quad \text { or } \underline{262 kW}\).

 

Hence, a \(\underline{\text {mixed-flow pump is preferred}}\).
