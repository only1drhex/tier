Guest Question.

The figure shows the single line diagram of a power system with a double circuit transmission line. The expression for electrical power is \(1.5 sin δ\), where \(δ\) is the rotor angle. The system is operating at the stable equilibrium point with mechanical power equal to \(1 pu\). If one of the transmission line circuits is removed, the maximum value of \(δ\), as the rotor swings, is \(1.221 radian\) . If the expression for electrical power with one transmission line circuit removed is \({ P }_{ max } sin δ\), the value of \({ P }_{ max }\), in \(pu \) is _____ (Give the answer up to three decimal places.)

Admin
Solution

With double circuit transmission line

\(\quad \quad \quad \quad { P }_{ eI }=1.5\sin { \delta }\)

with single line

\(\quad \quad \quad \quad { P }_{ eII }={ P }_{ max }\sin { \delta }\)

Here,

\(\quad \quad \quad \quad { \delta }_{ 1 }={ \sin { } }^{ -1 }\left[ \frac { 1 }{ 1.5 } \right] =41.81°\) or \(0.7297 radian\)

 

\(\quad \quad \quad \quad { \delta }_{ m }=1.221 radian\) or \( 69.96°\)

For stability

\(\quad \quad \quad \quad\quad \quad \quad { A }_{ 1 }={ A }_{ 2 }\)

 

\(\quad \quad \quad \quad \int _{ { \delta }_{ 1 } }^{ { \delta }_{ 2 } }{ \left( 1-{ P }_{ max }\sin { \delta } \right) d\delta } =\int _{ { \delta }_{ 2 } }^{ { \delta }_{ m } }{ \left( 1-{ P }_{ max }\sin { \delta } -1 \right) d\delta }\)

 

\(\quad \quad \quad \quad \delta { | }_{ { \delta }_{ 1 } }^{ { \delta }_{ 2 } }-{ P }_{ max }\left( -\cos { \delta } \right) { | }_{ { \delta }_{ 1 } }^{ { \delta }_{ 2 } }={ P }_{ max }\left( -\cos { \delta } \right) { | }_{ { \delta }_{ 2 } }^{ { \delta }_{ m } }-\delta { | }_{ { \delta }_{ 2 } }^{ { \delta }_{ m } }\)

 

\(\quad \quad \quad \quad { P }_{ max }=\left( \cos { { \delta }_{ m } } -\cos { { \delta }_{ 1 } } \right) ={ \delta }_{ 1 }-{ \delta }_{ m }\)

Substituting values and solving for \({ P }_{ max }\)

\({ P }_{ max } (cos 69.96° – cos 41.81°) = 41.81°– 69.96.°\)

 

\(\quad \quad \quad \quad { P }_{ max } = 1.220 pu\)

Hence, the correct answer is \((1.220 pu).\)
