Guest Question.

We wish to determine the steady-state voltage \(v_{o}\)(t) in Fig. 15.10 if the input voltage v(t) is given by the expression

v(t) = \(\sum\limits_{\begin{matrix}n=1 \\ n\,odd \end{matrix}}^{\infty }(\frac{20}{n\pi}sin\, 2nt -\frac{40}{n^{2}\pi^{2}}cos\,2nt) \) V

Admin
Solution

Note that this source has no constant term, and therefore its dc value is zero. The amplitude and phase for the first four terms of this signal are given in , and therefore the signal v(t) can be written as

v(t) = 7.5 cos (2t – 122°) + 2.2 cos (6t – 102°)
+ 1.3 cos (10t – 97°) + 0.91 cos (14t – 95°) + …

From the network we find that

I =\(\frac{V}{2 +\frac{2/ωj}{2 + 1/jω}}=\frac{V(1 + 2jω)}{4 + 4jω}\)

\(I_{1} =\frac{I(1/jω)}{2 + 1/jω}= \frac{I}{1 + 2jω}\)

\(V_{o} = (1)I_{1} = 1 .\frac{V(1 + 2jω)}{4 + 4jω}\frac{1}{1 + 2jω}=\frac{V}{4 + 4jω}\)

Therefore, since \(ω_{0}\) = 2,

\(V_{o}(n) =\frac{V(n)}{4 + j8n}\)

The individual components of the output due to the components of the input source are then

\(V_{o}(ω_{0}) =\frac{7.5\underline{/-122^\circ}}{4 + j8} = 0.84\underline{/-185.4^\circ}\) V

\(V_{o}(3ω_{0}) =\frac{2.2\underline{/-102^\circ}}{4 + j24} = 0.09\underline{/-182.5^\circ}\) V

\(V_{o}(5ω_{0}) =\frac{1.3\underline{/-97^\circ}}{4 + j40} = 0.03\underline{/-181.3^\circ}\) V

\(V_{o}(7ω_{0}) =\frac{0.91\underline{/-95^\circ}}{4 + j56} = 0.016\underline{/-181^\circ}\) V

Hence, the steady-state output voltage \(v_{o}\)(t) can be written as

\(v_{o}\)(t) = 0.84 cos (2t – 185.4°) + 0.09 cos (6t – 182.5°)
+ 0.03 cos (10t – 181.3°) + 0.016 cos (14t – 181°) + … V
