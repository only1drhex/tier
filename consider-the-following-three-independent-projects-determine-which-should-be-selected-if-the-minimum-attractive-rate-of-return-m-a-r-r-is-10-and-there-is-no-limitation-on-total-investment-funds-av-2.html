Guest Question.

Consider the following three independent projects. Determine which should be selected, if the minimum attractive rate of return (M.A.R.R.) is 10% and there is no limitation on total investment funds available.

 

Project Investment, P Life, n Salvage Value, S Net Annual Cash Flow, A
A \(₹ 5,00,000\) 5 years \(₹ 5,00,000\) \(+₹ 1,15,000\)
B \(₹ 6,00,000\) 5 years 0 \(+₹ 1,40,000\)
C \(₹ 7,50,000\) 5 years 0 \(+₹ 2,00,000\)
Admin
Solution

Solution of Example 6.7 by the Present Worth (P.W.) Method

 

Project P.W. of P P.W. of S P.W. of A Net P.W.
A \(-₹ 5,00,000\) \(\begin{array}{l}+₹ 5,00,000(P / F, 10 \%, 5) \\=+₹ 3,10,450\end{array}\) \(\begin{array}{l}+₹ 1,15,000(P / A, 10 \%, 5) \\=+₹ 4,35,942\end{array}\) \(\text { + ₹ } 2,46,392\)
B \(-₹ 6,00,000\) 0 \(\begin{array}{l}+₹ 1,40,000(P / A, 10 \%, 5) \\=+₹ 5,30,712\end{array}\) \(-₹ 69,288\)
C \(\text { – ₹ } 7,50,00\) 0 \(\begin{array}{l}+₹ 2,00,000(P / A, 10 \%, 5) \\=+₹ 7,58,160\end{array}\) \(+₹ 8,160\)

 

From the above calculations it is clear that projects A and B, having positive net P.W., would be satisfactory for investment, but project B would not be satisfactory.

Solution of Example 6.7 by the Future Worth (F.W.) Method

 

Project F.W. of P F.W. of S F.W. of A Net F.W.
A \(\begin{array}{l}-₹ 5,00,000(F / P, 10 \%, 5) \\=-₹ 8,05,250\end{array}\) \(+₹ 5,00,000\) \(\begin{array}{l}+₹ 1,15,000( F / A , 10 \%, 5) \\=+₹ 7,02,086.5\end{array}\) \(+₹ 396836.5\)
B \(\begin{array}{l}-₹ 6,00,000(F / P, 10 \%, 5) \\=-₹ 9,66,300\end{array}\) 0 \(\begin{array}{l}+₹ 1,40,000( F / A , 10 \%, 5) \\=₹ 8,54,714\end{array}\) \(\text { – ₹ } 1,11,586\)
C \(\begin{array}{l}-₹ 7,50,000(F / P, 10 \%, 5) \\=-₹ 12,07,875\end{array}\) 0 \(\begin{array}{l}+₹ 2,00,000( F / A , 10 \%, 5) \\=₹ 12,21,020\end{array}\) \(+₹ 13,145\)

 

From the above calculations, it is clear that projects A and B, having positive net F.W., would be satisfactory for investment, but project B would not be satisfactory.

Solution of Example 6.7 by the Internal Rate of Return (I.R.R.) Method

Project A

The equation for Net Present Worth (N.P.W.) at \( i^{\prime} \%\) is written as follows:

 

\(\text { N.P.W. }=-₹ 5,00,000+₹ 5,00,000\left(P / F, i^{\prime} \%, 5\right)+₹ 1,15,000\left(P / A, i^{\prime} \%, 5\right)\)

 

We have to find i¢ at which N.P.W. is zero. Linear interpolation between 20% and 25% results in \(i^{\prime}=23.13 \%\) , which is the I.R.R. Since 23.13% > 10%, investment in project A is justified. Thus, project A should be selected.

Project B

The equation for Net Present Worth (N.P.W.) at \( i^{\prime} \%\) is written as follows:

 

\(\text { N.P.W. }=-₹ 6,00,000+₹ 1,40,000\left(P / A, i^{\prime} \%, 5\right)\)

 

We have to find i¢ at which N.P.W. is zero. Linear interpolation between 5% and 8% results in \( i^{\prime}=5.39 \%\) , which is the I.R.R. Since 5.39% < 10%, investment in project B is not justified. Thus, project B should not be selected.

Project C

The equation for Net Present Worth (N.P.W.) at i’% is written as follows:

 

\(\text { N.P.W. }=-₹ 7,50,000+₹ 2,00,000\left(P / A, i^{\prime} \%, 5\right)\)

 

We have to find \(i^{\prime}\) which N.P.W. is zero. Linear interpolation between 10% and 12% results in \(i^{\prime}=11.92 \%\) which is the I.R.R. Since 11.929% > 10%, investment in project C is justified. Thus, project C should be selected.

From the above, we conclude that independent projects A and C should be selected, but project B should not be selected.

Solution of Example 6.7 by the External Rate of Return (E.R.R.) Method

Project A

 

\(₹ 5,00,000\left(F / P, i^{\prime} \%, 5\right)=₹ 5,00,000+₹ 1,15,000(F / A, 10 \%, 5)\)

 

Hit and trial method results in \(i^{\prime}=19.18 \%\), which is the value of E.R.R. Since 19.18% > 10%, investment in project A is justified. Thus, project A should be selected.

Project B

 

\(₹ 6,00,000\left(F / P, i^{\prime} \%, 5\right)=₹ 1,40,000(F / A, 10 \%, 5)\)

 

Hit and trial method results in \(i^{\prime}=7.33 \%\), which is the value of E.R.R. Since 7.33% < 10%, investment in project B is not justified. Thus, project B should not be selected.

Project C

 

\(₹ 7,50,000\left(F / P, i^{\prime} \%, 5\right)=₹ 2,00,000(F / A, 10 \%, 5)\)

 

Hit and trial method results in \(i^{\prime}=10.23 \%\) , which is the value of E.R.R. Since 10.23% > 10%, investment in project C is justified. Thus, project C should be selected.

From the above, we conclude that independent projects A and C should be selected, whereas, project B should not be selected.

Solution of Example 6.7 by the Explicit Reinvestment Rate of Return (E.R.R.R.) Method

Project A

 

\(\text { E.R.R.R. }=\frac{(₹ 1,15,000-0)-(₹ 5,00,000-5,00,000)(A / F, 10 \%, 10)}{₹ 5,00,000}\)

 

\(\text { E.R.R.R. }=23 \%\)

 

Since 23% > 10%, investment in project A is justified. Thus, project A should be selected.

Project B

 

$$\text { E.R.R.R. }=\frac{(₹ 1,40,000-0)-(₹ 6,00,000-0)(A / F, 10 \%, 10)}{₹ 6,00,000}$$E.R.R.R. $=6.95 \%$

 

Since 6.95% < 10%, investment in project B is not justified. Thus, project B should not be selected.

Project C

 

\(\text { E.R.R.R. }=\frac{(₹ 2,00,000-0)-(₹ 7,50,000-0)(A / F, 10 \%, 10)}{₹ 7,50,000}\)

 

\(\text { E.R.R.R. }=10.29 \%\)

 

Since 10.29% > 10%, investment in project C is justified economically. Thus, project C should be selected.

From the above, we conclude that independent projects A and C should be selected whereas, project B should not be selected.
