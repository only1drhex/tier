Non-Self-Starting Heun Method

Use the non-self-starting Heun method to perform the same computations as were performed previously in using Heun’s method. That is, integrate \(y^{\prime}=4 e^{0.8 x}-0.5 y\) from x = 0 to x = 4 using a step size of 1.0. As with, the initial condition at x = 0 is y = 2. However, because we are now dealing with a multistep method, we require the additional information that y is equal to 20.3929953 at x = – 1.

Admin
Solution

The predictor [Eq. (26.13)]

\(y_{i+1}^{0}=y_{i-1}^{m}+f\left(x_{i}, y_{i}^{m}\right) 2 h\) (26.13)

is used to extrapolate linearly from x = – 1 to x = 1.

\(y_{1}^{0}=-0.3929953+\left[4 e^{0.8(0)}-0.5(2)\right] 2=5.607005\)

The corrector [Eq. (26.14)]

\(y_{i+1}^{j}=y_{i}^{m}+\frac{f\left(x_{i}, y_{i}^{m}\right)+f\left(x_{i+1}, y_{i+1}^{j-1}\right)}{2} h\) (for j = 1, 2, . . . , m) (26.14)

is then used to compute the value:

\(y_{1}^{1}=2+\frac{4 e^{0.8(0)}-0.5(2)+4 e^{0.8(1)}-0.5(5.607005)}{2} 1=6.549331\)

which represents a percent relative error of – 5.73 percent (true value = 6.194631). This error is somewhat smaller than the value of – 8.18 percent incurred in the self-starting Heun. Now, Eq. (26.14) can be applied iteratively to improve the solution:

\(y_{1}^{2}=2+\frac{3+4 e^{0.8(1)}-0.5(6.549331)}{2} 1=6.313749\)

which represents an et of -1.92%. An approximate estimate of the error can also be determined using Eq. (26.15):

\(\left|\varepsilon_{a}\right|=\left|\frac{y_{i+1}^{j}-y_{i+1}^{j-1}}{y_{i+1}^{j}}\right| 100 \%\) (26.15)

 

\(\left|\varepsilon_{a}\right|=\left|\frac{6.313749-6.549331}{6.313749}\right| 100 \%=3.7 \%\)

Equation (26.14) can be applied iteratively until \(\varepsilon_{a}\) falls below a prespecifi ed value of \(\varepsilon_{s}\) As was the case with the Heun method (recall), the iterations converge on a value of 6.360865 \(\left(\varepsilon_{t}=-2.68 \%\right)\). However, because the initial predictor value is more accurate, the multistep method converges at a somewhat faster rate.

For the second step, the predictor is

\(y_{2}^{0}=2+\left[4 e^{0.8(1)}-0.5(6.360865)\right] 2=13.44346 \quad \varepsilon_{t}=9.43 \%\)

which is superior to the prediction of 12.08260 \(\left(\varepsilon_{t}=18 \%\right)\) that was computed with the original Heun method. The fi rst corrector yields 15.76693 \(\left(\varepsilon_{t}= 6.8\%\right)\), and subsequent iterations converge on the same result as was obtained with the self-starting Heun method: 15.30224\(\left(\varepsilon_{t}=-3.1 \%\right)\). As with the previous step, the rate of convergence of the corrector is somewhat improved because of the better initial prediction.
