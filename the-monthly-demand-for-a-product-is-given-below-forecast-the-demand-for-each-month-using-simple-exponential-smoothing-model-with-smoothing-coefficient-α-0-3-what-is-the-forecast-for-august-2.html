Guest Question.

The monthly demand for a product is given below. Forecast the demand for each month using simple exponential smoothing model with smoothing coefficient \(\alpha\) = 0.3. What is the forecast for August?

Month Actual Demand, \(D_{t}\)
Jan 200
Feb 260
Mar 210
Apr 220
May 230
Jun 270
Jul 290
Aug –
Admin
Solution

In order to forecast the demand for Jan i.e. FJan, we need to have the base, \(S_{t}\), or one period before (Initial) i.e. \(S_{\text {Initial }}\). Let us assume that the base for the initial period is equal to 220. The computation of forecast for other months is shown below in Table 14.1.

 

Month Actual Demand, \(D_{t}\) Base, \(S_{t}\) Forecast, \(F_{t}\)
Initial 220 (Assumed) –
Jan 200 \(\begin{aligned}S_{J a n} &=\alpha D_{J \text { an }}+(1-\alpha) S_{\text {Initial }} \\&=0.3 \times 200+0.7 \times 220 \\&=214\end{aligned}\) 220
Feb 260 \(\begin{aligned}S_{ Feb } &=\alpha D_{ Feb }+(1-\alpha) S_{ Jan } \\&=0.3 \times 260+0.7 \times 214 \\&=227.80\end{aligned}\) \(F_{ Feb }=S_{ Jan }=214\)
Mar 210 \(\begin{aligned}S_{ Mar } &=\alpha D_{ Mar }+(1-\alpha) S_{ Feb } \\&=0.3 \times 210+0.7 \times 227.80 \\&=222.46\end{aligned}\) \(F_{ Mar }=S_{ Feb }=227.80\)
Apr 220 \(\begin{aligned}S_{ Apr } &=\alpha D_{ Apr }+(1-\alpha) S_{ Mar } \\&=0.3 \times 220+0.7 \times 222.46 \\&=221.72\end{aligned}\) \(F_{ Apr }=S_{ Mar }=222.46\)
May 230 \(\begin{aligned}S_{\text {May }} &=\alpha D_{\text {May }}+(1-\alpha) S_{\text {Apr }} \\&=0.3 \times 230+0.7 \times 221.72 \\&=224.21\end{aligned}\) \(F_{\text {May }}=S_{\text {Apr }}=221.72\)
Jun 270 \(\begin{aligned}S_{J u n} &=\alpha D_{ Jun }+(1-\alpha) S_{ May } \\&=0.3 \times 270+0.7 \times 224.21 \\&=237.94\end{aligned}\) \(F_{\text {Jun }}=S_{\text {May }}=224.21\)
Jul 290 \(\begin{aligned}S_{J u l} &=\alpha D_{J u l}+(1-\alpha) S_{J u n} \\&=0.3 \times 290+0.7 \times 237.94 \\&=253.56\end{aligned}\) \(F_{ Jul }=S_{ Jun }=237.94\)
Aug – – \(F_{ Aug }=S_{ Jul }=253.56\)
