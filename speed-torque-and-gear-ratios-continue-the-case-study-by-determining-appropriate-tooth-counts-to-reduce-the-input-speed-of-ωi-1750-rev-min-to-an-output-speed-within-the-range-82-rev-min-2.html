Guest Question.

SPEED, TORQUE, AND GEAR RATIOS

Continue the case study by determining appropriate tooth counts to reduce the input speed of \(\omega_{i}=1750\) rev/min to an output speed within the range

\(82 rev / min <\omega_{0}<88 rev / min\)

Once final tooth counts are specified, determine values of

(a) Speeds for the intermediate and output shafts

(b) Torques for the input, intermediate and output shafts, to transmit 20 hp.

Admin
Solution

Use the notation for gear numbers from Fig. 18–1. Choose mean value for initial design, \(\omega_{5}=85\) rev/min.

 

\(H=T_{i} \omega_{i}=T_{o} \omega_{o}\) (18–2)

 

\(e=\frac{\omega_{5}}{\omega_{2}}=\frac{85}{1750}=\frac{1}{20.59}\)

 

For a compound reverted geartrain,

 

\(e=\frac{\text { product of driving tooth numbers }}{\text { product of driven tooth numbers }}\) (13–30)

 

\(e=\frac{1}{20.59}=\frac{N_{2}}{N_{3}} \frac{N_{4}}{N_{5}}\)

 

For smallest package size, let both stages be the same reduction. Also, by making the two stages identical, the in-line condition on the input and output shaft will automatically be satisfied.

 

\(\frac{N_{2}}{N_{3}}=\frac{N_{4}}{N_{5}}=\sqrt{\frac{1}{20.59}}=\frac{1}{4.54}\)

 

For this ratio, the minimum number of teeth from Eq. (13–11), p. 686, is 16.

 

\(N_{P}=\frac{2 k}{(1+2 m) \sin ^{2} \phi}\left(m+\sqrt{m^{2}+(1+2 m) \sin ^{2} \phi}\right)\) (13–11)

 

\(N_{2}=N_{4}=16 \text { teeth }\)

 

\(N_{3}=4.54\left(N_{2}\right)=72.64\)

 

Try rounding down and check if \(\omega_{5}\) is within limits.

 

\(\omega_{5}=\left(\frac{16}{72}\right)\left(\frac{16}{72}\right)(1750)=86.42 rev / min \quad \text { Acceptable }\)

 

Proceed with

 

\(\begin{array}{l}N_{2}=N_{4}=16 \text { teeth } \\N_{3}=N_{5}=72 \text { teeth }\end{array}\)

 

\(e=\left(\frac{16}{72}\right)\left(\frac{16}{72}\right)=\frac{1}{20.25}\)

 

\(\omega_{5}=86.42 rev / min\)

 

\(\omega_{3}=\omega_{4}=\left(\frac{16}{72}\right)(1750)=388.9 rev / min\)

 

To determine the torques, return to the power relationship,

 

\(H=T_{i} \omega_{i}=T_{o} \omega_{o}\) (18–1)

 

\(H=T_{2} \omega_{2}=T_{5} \omega_{5}\)

 

\(T_{2}=H / \omega_{2}=\left(\frac{20 hp }{1750 rev / min }\right)\left(550 \frac{ ft – lbf / s }{ hp }\right)\left(\frac{1 rev }{2 \pi rad }\right)\left(60 \frac{ s }{ min }\right)\)

 

\(T_{2}=60.0 lbf \cdot ft\)

 

\(T_{3}=T_{2} \frac{\omega_{2}}{\omega_{3}}=60.0 \frac{1750}{388.9}=270 lbf \cdot ft\)

 

\(T_{5}=T_{2} \frac{\omega_{2}}{\omega_{5}}=60.0 \frac{1750}{86.42}=1215 lbf \cdot ft\)
