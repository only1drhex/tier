Guest Question.

Show by means of matrix algebra how the input variables \(V_1\) and \(I_1\) can be described as functions of the output variables \(V_2\) and \(I_2\) in the cascade connection shown in Fig. 18.10.

Admin
Solution

We begin by expressing, in matrix notation, the relationship between the input and output variables of each two-port circuit. Thus

\(\left [ \begin{matrix} V_{1} \\ I_{1} \end{matrix} \right ] =\left [ \begin{matrix} a^′_{11} & -a^′_{12} \\ a^′_{21} & -a^′_{22} \end{matrix} \right ]\left [ \begin{matrix} V^′_{2} \\ I^′_{2} \end{matrix} \right ] \), (A.104)

and

\(\left [ \begin{matrix} V^′_{1} \\ I^′_{1} \end{matrix} \right ] =\left [ \begin{matrix} a^{′′}_{11} & -a^{′′}_{12} \\ a^{′′}_{21} & -a^{′′}_{22} \end{matrix} \right ]\left [ \begin{matrix} V_{2} \\ I_{2} \end{matrix} \right ] \), (A.105)

Now the cascade connection imposes the constraints

\(V^′_2= V^′_1 \) and \(I^′_2 = -I^′_1 \). (A.106)

These constraint relationships are substituted into Eq. A.104.Thus

\(\left [ \begin{matrix} V_{1} \\ I_{1} \end{matrix} \right ] =\left [ \begin{matrix} a^{′}_{11} & -a^{′}_{12} \\ a^{′}_{21} & -a^{′}_{22} \end{matrix} \right ]\left [ \begin{matrix} V^′_{1} \\ -I^′_{1} \end{matrix} \right ] \)

\(= \left [ \begin{matrix} a^{′}_{11} & a^{′}_{12} \\ a^{′}_{21} & a^{′}_{22} \end{matrix} \right ]\left [ \begin{matrix} V^′_{1} \\ I^′_{1} \end{matrix} \right ] \). (A.107)

The relationship between the input variables \(( V_1 ,I_1 )\) and the output variables \(( V_2 ,I_2 )\) is obtained by
substituting Eq. A.105 into Eq. A.107.The result is

\(\left [ \begin{matrix} V_{1} \\ I_{1} \end{matrix} \right ]= \left [ \begin{matrix} a^{′}_{11} & a^{′}_{12} \\ a^{′}_{21} & a^{′}_{22} \end{matrix} \right ]\left [ \begin{matrix} a^{′′}_{11} & -a^{′′}_{12} \\ a^{′′}_{21} & -a^{′′}_{22} \end{matrix} \right ]\left [ \begin{matrix} V_{2} \\ I_{2} \end{matrix} \right ]\) (A.108)

After multiplying the coefficient matrices, we have

\(\left [ \begin{matrix} V_{1} \\ I_{1} \end{matrix} \right ]= \left [ \begin{matrix} (a^\prime _{11}a^{\prime \prime } _{11}+ a^\prime _{12}a^{\prime \prime } _{21}) & -(a^\prime _{11}a^{\prime \prime } _{12}+ a^\prime _{12}a^{\prime \prime } _{22})\\ (a^\prime _{21}a^{\prime \prime } _{11} + a^\prime _{22}a^{\prime \prime } _{21} )& -(a^\prime _{21}a^{\prime \prime } _{12} + a^\prime _{22}a^{\prime \prime } _{22}) \end{matrix} \right ] \left [ \begin{matrix} V_{2} \\ I_{2} \end{matrix} \right ] \). (A.109)

Note that Eq.A.109 corresponds to writing Eqs. 18.72 and 18.73 in matrix form.

\(V_1=(a^\prime _{11}a^{\prime \prime } _{11}+ a^\prime _{12}a^{\prime \prime } _{21})V_2-(a^\prime _{11}a^{\prime \prime } _{12}+ a^\prime _{12}a^{\prime \prime } _{22})I_2,\) (18.72)

\(I_1=(a^\prime _{21}a^{\prime \prime } _{11}+ a^\prime _{22}a^{\prime \prime } _{21})V_2-(a^\prime _{21}a^{\prime \prime } _{12}+ a^\prime _{22}a^{\prime \prime } _{22})I_2, \) (18.73)
