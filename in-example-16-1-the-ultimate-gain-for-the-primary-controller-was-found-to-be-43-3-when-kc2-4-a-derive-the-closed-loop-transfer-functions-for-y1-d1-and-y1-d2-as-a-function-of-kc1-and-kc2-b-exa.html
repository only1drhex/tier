Guest Question.

In , the ultimate gain for the primary controller was found to be \(43.3\) when \(K_{c 2}=4\).

(a) Derive the closed-loop transfer functions for \(Y_{1} / D_{1}\) and \(Y_{1} / D_{2}\) as a function of \(K_{c 1}\) and \(K_{c 2}\).

(b) Examine the effect of \(K_{c 2}\) on the critical gain of \(K_{c 1}\) by varying \(K_{c 2}\) from 1 to 20 . For what values of \(K_{c 2}\) do the benefits of cascade control seem to be less important? Is there a stability limit on \(K_{c 2}\) ?

(c) Integral action was not included in either primary or secondary loops. First set \(K_{c 2}=5, \tau_{l 1}=\infty\), and \(\tau_{l 2}=5\) min. Find the ultimate controller gain using direct substitution. Then repeat the stability calculation for \(\tau_{I 1}=5 \min\) and \(\tau_{R 2}=\infty\) and compare the two results. Is offset for \(Y_{1}\) eliminated in both cases for step changes in \(D_{1}\) or \(D_{2}\) ?

Admin
Solution

a) The transfer function between \(Y_{1}\) and \(D_{1}\) is

\(\frac{Y_{1}}{D_{1}}=\frac{G_{d 1}}{1+G_{c 1}\left(\frac{G_{c 2} G_{v}}{1+G_{c 2} G_{v} G_{m 2}}\right) G_{p} G_{m 1}}\)

and that between \(Y_{1}\) and \(D_{2}\) is

\(\frac{Y_{1}}{D_{2}}=\frac{G_{p} G_{d 2}}{1+G_{c 2} G_{v} G_{m 2}+G_{c 2} G_{v} G_{m 1} G_{c 1} G_{p}}\)

 

\(\text { using } G_{v}=\frac{5}{s+1} \quad, \quad G_{d 2}=1, \quad G_{d 1}=\frac{1}{3 s+1}\)

\(G_{p}=\frac{4}{(2 s+1)(4 s+1)}, G_{m 1}=0.05, \quad G_{m 2}=0.2\)

For \(G_{c 1}=K_{c 1}\) and \(G_{c 2}=K_{c 2}\), we obtain

\(\frac{Y_{1}}{D_{1}}=\frac{8 s^{3}+\left(14+8 K_{c 2}\right) s^{2}+\left(7+6 K_{c 2}\right) s+K_{c 2}+1}{24 s^{4}+\left(50+24 K_{c 2}\right) s^{3}+\left[10+K_{c 2}\left(9+3 K_{c 1}\right)\right] s+\left(35+26 K_{c 2}\right) s^{2}+K_{c 2}\left(1+K_{c 1}\right)+1}\)

 

\(\frac{Y_{1}}{D_{2}}=\frac{4(s+1)}{8 s^{3}+\left(14+8 K_{c 2}\right) s^{2}+\left(7+6 K_{c 2}\right) s+K_{c 2}\left(1+K_{c 1}\right)+1}\)

The figures below show the step load responses for \(K_{c 1}=43.3\) and for \(K_{c 2}=25 .\) Note that both responses are stable. You should recall that the critical gain for \(K_{c 2}=5\) is \(K_{c 1}=43.3\). Increasing \(K_{c 2}\) stabilizes the controller, as is predicted.

b) The characteristic equation for this system is

\(1+G_{c 2} G_{v} G_{m 2}+G_{c 2} G_{v} G_{m 1} G_{c 1} G_{p}=0\) (1)

Let \(G_{c 1}=K_{c 2}\) and \(G_{c 2}=K_{c 2} .\) Then, substituting all the transfer functions into (1), we obtain

\(8 s^{3}+\left(14+8 K_{c 2}\right) s^{2}+\left(7+6 K_{c 2}\right) s+K_{c 2}\left(1+K_{c 1}\right)+1=0\) (2)

Now we can use the direct substitution:

\(\left[-8 \omega^{3}+\left(7+6 K_{c 2}\right)\omega\right] j-\left(14+8 K_{c 2}\right) \omega^{2}+K_{c2}\left(1+K_{c 1}\right)+1=0\) (3)

 

\(j:-8 \omega^{3}+\left(7+6 K_{c 2}\right) \omega=0\)

 

\(-\left(14+8 K_{c 2}\right) \omega^{2}+K_{c 2}\left(1+K_{c 1}\right)+1=0\)

 

Hence, for normal (positive) values of \(K_{c 1}\) and \(K_{c 2}\),

\(K_{c 1, u}=\frac{24 K_{c 2}^{2}+66 K_{c 2}+45}{4 K_{c 2}}\)

The results are shown in the table and figure below. Note the nearly linear variation of \(K_{c 1}\) ultimate with \(K_{c 2}\). This is because the right hand side is very nearly \(6 K_{c 2}+16.5\). For larger values of \(K_{c 2}\), the stability margin on \(K_{c 1}\) is higher. There don’t appear to be any nonlinear effects of \(K_{c 2}\) on \(K_{c 1}\), especially at high \(K_{c 2}\).

There is no theoretical upper limit for \(K_{c 2}\), except that large values may cause the valve to saturate for small set-point or load changes.

c) With integral action in the inner loop,

\(G_{c 1}=K_{c 1}\)

\(G_{c 2}=5\left(1+\frac{1}{5 s}\right)\)

Substitution of all the transfer functions into the characteristic equation yields

\(1+5\left(1+\frac{1}{5 s}\right) \frac{5}{s+1}(0.2)+5\left(1+\frac{1}{5 s}\right) \frac{5}{s+1}(0.05) K_{c 1}\)

 

\(\frac{4}{(4 s+1)(2 s+1)}=0\)

Rearrangement gives

\(8 s^{4}+54 s^{3}+45 s^{2}+\left(12+5 K_{c 1}\right) s+K_{c 1}+1=0\)

Now we can use the direct substitution:

\(\left[-54 \omega^{3}+\left(12+5 K_{c 1}\right) \omega\right] j+8 \omega^{4}-45 \omega^{2}+K_{c 1}+1=0\)

 

\(j:-54 \omega^{3}+\left(12+5 K_{c 1}\right) \omega=0\)

 

\(\omega^{4}-45 \omega^{2}+K_{c 1}+1=0\)

Solve the equations above, and we obtain:

\(K_{c 1, u}=44.2\)

The ultimate \(K_{c 1}\) is \(44.2\), which is close to the result as for proportional only control of the secondary loop.

With integral action in the outer loop only,

\(G_{c 1}=K_{c 1}\left(1+\frac{1}{5 s}\right)\)

 

\(G_{c 2}=5\)

Substituting the transfer functions into the characteristic equation.

\(1+5 \frac{5}{s+1}(0.2)+5 \frac{5}{s+1}(0.05) K_{c 1}\left(1+\frac{1}{5 s}\right) \frac{4}{(4 s+1)(2 s+1)}=0\)

 

\(\therefore 8 s^{4}+54 s^{3}+37 s^{2}+\left(6+5 K_{c 1}\right) s+K_{c 1}=0\)

Now we can use the direct substitution:

\(\left[-54 \omega^{3}+\left(6+5 K_{c 1}\right) \omega\right] j+8 \omega^{4}-37 \omega^{2}+K_{c 1}=0\)

\(j:-54 \omega^{3}+\left(6+5 K_{c 1}\right) \omega=0\)

\(8 \omega^{4}-37 \omega^{2}+K_{c 1}=0\)

Solve the equations above, and we obtain:

\(K_{c 1, u}=34.66\)

Hence, \(K_{c 1}<34.66\) is the limiting constraint. Note that due to integral action in the primary loop, the ultimate controller gain is reduced.

Calculation of offset:

For \(\quad G_{c 1}=K_{c 1}\left(1+\frac{1}{\tau_{I 1} s}\right) \quad, \quad G_{c 2}=K_{c 2} \quad, \quad\left(\tau_{I 2}=\infty\right)\)

 

\(\begin{aligned}&\frac{Y_{1}}{D_{1}}=\frac{G_{d 1}\left(1+K_{c 2} G_{v} G_{m 2}\right)}{1+K_{c 2} G_{v} G_{m 2}+K_{c 2} G_{v} G_{m 1} K_{c 1}\left(1+\frac{1}{\tau_{I 1} s}\right) G_{p}} \\&\frac{Y_{1}}{D_{1}}(s=0)=0\end{aligned}\)

Since \(G_{c 1}\) contains integral action, a step-change in \(D_{1}\) does not produce an offset in \(Y_{1}\).

\(\begin{aligned}&\frac{Y_{1}}{D_{2}}=\frac{G_{p} G_{d 2}}{1+K_{c 2} G_{v} G_{m 2}+K_{c 2} G_{v} G_{m 1} K_{c 1}\left(1+\frac{1}{\tau_{I 1} s}\right) G_{p}} \\&\frac{Y_{1}}{D_{2}}(s=0)=0\end{aligned}\)

Thus, for the same reason as before, a step-change in \(D_{2}\) does not produce an offset in \(Y_{1}\).

For \(\quad G_{c 1}=K_{c 1} \quad\) (ie. \(\left.\tau_{I 1}=\infty\right) \quad, \quad G_{c 2}=K_{c 2}\left(1+\frac{1}{\tau_{I 2} s}\right)\)

 

\(\frac{Y_{1}}{D_{1}}=\frac{G_{d 1}\left(1+K_{c 2}\left(1+\frac{1}{\tau_{I 2} s}\right) G_{v} G_{m 2}\right)}{1+K_{c 2}\left(1+\frac{1}{\tau_{I 2} s}\right) G_{v} G_{m 2}+K_{c 2} G_{v} G_{m 1} K_{c 1}\left(1+\frac{1}{\tau_{I 2} s}\right) G_{p}}\)

 

\(\frac{Y_{1}}{D_{1}}(s=0)â‰ 0\)

Therefore, when there is no integral action in the outer loop, a primary disturbance produces an offset.

Thus, there is no offset for a step-change in the secondary disturbance.

\(\frac{Y_{1}}{D_{2}}=\frac{G_{p} G_{d 2}}{1+K_{c 2}\left(1+\frac{1}{\tau_{I 2} s}\right) G_{v} G_{m 2}+K_{c 2} G_{v} G_{m 1} K_{c 1}\left(1+\frac{1}{\tau_{I 2} s}\right) G_{p}}\)

\(\frac{Y_{1}}{D_{2}}(s=0)=0\)

Thus, there is no offset for a step-change in the secondary disturbance.
