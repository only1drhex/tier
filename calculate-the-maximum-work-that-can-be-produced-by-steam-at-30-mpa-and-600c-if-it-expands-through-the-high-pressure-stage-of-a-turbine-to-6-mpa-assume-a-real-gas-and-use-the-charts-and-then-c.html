Guest Question.

Calculate the maximum work that can be produced by steam at 30 MPa and 600°C if it expands through the high-pressure stage of a turbine to 6 MPa. Assume a real gas, and use the charts, and then compare with tabulated values from the steam tables (Appendix C).

Admin
Solution

Maximum work occurs for an adiabatic reversible process, i.e., for Δs = 0. The reduced temperature and pressures are

\( T_{R 1}=\frac{T_{1}}{T_{C r}}=\frac{873}{647}=1.35 \quad P_{R 1}=\frac{30}{22.1}=1.36 \quad P_{R 2}=\frac{6}{22.1}=0.27 \).

\( \text { The isentropic process provides us with } T_{2} \text { by a trial-and-error procedure: } \)

\( \Delta s=0=\left(s_{1}^{*}-s_{1}\right)+\left(s_{2}-s_{2}^{*}\right)+\left(s_{2}^{*}-s_{1}^{*}\right)=\frac{4}{18}+s_{2}-s_{2}^{*}+1.872 \ln \frac{T_{2}}{873}-0.462 \ln \frac{3}{30} \).

\( \text { If } s_{2}-s_{2}^{*}=0 \text { we find } T_{2}=521 K \text { or } 248^{\circ} C \text {. Since } s_{2}-s_{2}^{*}<0 \text {, we try } T_{2}>521 K \text { : } \)

\( T_{2}=600 K , \quad T_{R 2}=0.93: \quad 0=\frac{?}{18}-\frac{3}{18}+1.872 \ln \frac{600}{873}-0.462 \ln \frac{6}{30}=0.097 \).

\( T_{2}=560 K , \quad T_{R 2}=0.87: \quad 0 \stackrel{?}{=} \frac{4}{18}-\frac{3.5}{18}+1.872 \ln \frac{560}{873}-0.462 \ln \frac{6}{30}=-0.060 \).

\( \text { Interpolation gives } T_{2}=575 K \text { or } 302^{\circ} C \text {. The work produced is then } \).

\( w=-\Delta h=\left(h_{1}-h_{1}^{*}\right)+\left(h_{2}^{*}-h_{2}\right)+\left(h_{1}^{*}-h_{2}^{*}\right) \).

\( =(-8)\left(\frac{647.4}{18}\right)+(4)\left(\frac{647.4}{18}\right)+\frac{30750-19500}{18}=481 kJ / kg \).

\( \text { where we have used the Ideal-Gas Table E. } 6 \text { to find } h_{1}^{*}-h_{2}^{*} \text {. A less accurate value would be found using } C_{p} \Delta T \text {. } \)

To compare with values obtained directly from the steam tables in Appendix C we use

\( \left.\begin{array}{c} s_{2}=s_{1}=6.2339 kJ / kg \cdot K \\ P_{2}=6 MPa \end{array}\right\} \quad \therefore h_{2}=2982 kJ / kg \).

\( \text { The work is } w=-\Delta h=h_{1}-h_{2}=3444-2982=462 kJ / kg \text {. } \).
