Guest Question.

We wish to compute the rms value of the waveform \( i(t)=I_{M} \cos (\omega t-\theta)\), which has a period of T = 2π/ω.

Admin
Solution

Substituting these expressions into Eq. (9.23) \( I_{ eff }=\sqrt{\frac{1}{T} \int_{t_{0}}^{t_{0}+T} i^{2}(t) d t}\) yields

\(I_{ rms }=\left[\frac{1}{T} \int_{0}^{T} I_{M}^{2} \cos ^{2}(\omega t-\theta) d t\right]^{1 / 2}\)

Using the trigonometric identity

\( \cos ^{2} \phi=\frac{1}{2}+\frac{1}{2} \cos 2 \phi\)

we find that the preceding equation can be expressed as

\( I_{ rms }=I_{M}\left\{\frac{\omega}{2 \pi} \int_{0}^{2 \pi / \omega}\left[\frac{1}{2}+\frac{1}{2} \cos (2 \omega t-2 \theta)\right] d t\right\}^{1 / 2}\)

Since we know that the average or mean value of a cosine wave is zero,

\(I_{ rms }=I_{M}\left(\frac{\omega}{2 \pi} \int_{0}^{2 \pi / \omega} \frac{1}{2} d t\right)^{1 / 2}\)
\(=I_{M}\left[\left.\frac{\omega}{2 \pi}\left(\frac{t}{2}\right)\right|_{0} ^{2 \pi / \omega}\right]^{1 / 2}=\frac{I_{M}}{\sqrt{2}}\) 9.24

Therefore, the rms value of a sinusoid is equal to the maximum value divided by \( \sqrt{2}\). Hence, asinusoidal current with a maximum value of \( I_{M} \) delivers the same average power to a resistor R as a dc current with a value of \( I_{M}/\sqrt{2}\). Recall that earlier a phasor X was defined as \( X_{M}\underline{ / \theta} \) for a sinusoidal wave of the form \(X_{M} \cos (\omega t+\theta)\). This phasor can also be represented as \( X_{M} / \sqrt{2}\underline{/\theta}\) if the units are given in rms. For example, \(120 \underline{/ 30^{\circ} V} \) rms is equivalent to \(170\underline{ / 30^{\circ}} V\).
