Guest Question.

Is \(\mathbb{S} = \left\{\left [ \begin{matrix} x_{1} \\ x_{2} \\ x_{1} + x_{2} \end{matrix} \right ] | x_{1}, x_{2} \in \mathbb{R} \right\}\) a vector space with standard addition and scalar multiplication in \(\mathbb{R} ^3 ?\)

Admin
Solution

Yes! To prove it we need to verify that all ten axioms hold. However, since axioms V2, V3, V7, V8, V9, and V10 refer only to the operations of addition and scalar multiplication, we know by Theorem 1.4.1

(For all \(\vec{w}, \vec{x}, \vec{y} \in \mathbb{R} ^n \) and \(s, t \in \mathbb{R}\) we have

V1 \( \vec{x} + \vec{y} \in \mathbb{R} ^n\) (closed under addition)

V2 \( \vec{x} + \vec{y} = \vec{y} + \vec{x} \) (addition is commutative)

V3 \( (\vec{x} + \vec{y}) + \vec{w} = \vec{x} + (\vec{y} + \vec{w}) \) (addition is associative)

V4 There exists a vector \(\vec{0} \in \mathbb{R} ^n\) such that \(\vec{z} + \vec{0} = \vec{z}\) for all \(\vec{z} \in \mathbb{R}^ n\) (zero vector)

V5 For each \(\vec{x} \in \mathbb{R}^ n\) there exists a vector \(– \vec{x} \in \mathbb{R}^ n\) such that \(\vec{x} + \ (-\vec{x}) = \vec{0}\) (additive inverses)

V6 \(t\vec{x} \in \mathbb{R}^ n\) (closed under scalar multiplication)

V7 \(s(t\vec{x}) = (st)\vec{x} \) (scalar multiplication is associative)

V8 \((s+t)\vec{x} = s\vec{x} + t\vec{x}\) (a distributive law)

V9 \(t (\vec{x} + \vec{y})= t\vec{x} + t\vec{y}\) (another distributive law)

V10 \(1 \vec{x} = \vec{x} \) (scalar multiplicative identity)

)that these all hold. We now prove that the remaining axioms also hold.

Let \(\vec{x} = \left[ \begin{matrix} x_{1} \\ x_{2} \\ x_{1} + x_{2} \end{matrix} \right ] \) and \(\vec{y} = \left[ \begin{matrix} y_{1} \\ y_{2} \\ y_{1} + y_{2} \end{matrix} \right ] \) be vectors in \(\mathbb{S}\).

V1: We have

\(\vec{x} + \vec{y} = \left[ \begin{matrix} x_{1} \\ x_{2} \\ x_{1} + x_{2} \end{matrix} \right ] + \left[ \begin{matrix} y_{1} \\ y_{2} \\ y_{1} + y_{2} \end{matrix} \right ] = \left [ \begin{matrix} x_{1} + y_{1} \\ x_{2} + y_{2} \\ x_{1} + y_{1} + x_{2} + y_{2} \end{matrix} \right ]\)

If we let \(z_{1} = x_{1} + y_{1}\) and \(z_{2} = x_{2} + y_{2} \), then \(z_{1} + z_{2} = x_{1} + y_{1} + x_{2} + y_{2}\) and hence

\(\vec{x} + \vec{y} = \left [ \begin{matrix} z_{1} \\ z_{2} \\ z_{1} + z_{2} \end{matrix} \right ] \in \mathbb{S}\)

V6: For any \(t \in \mathbb{R}, t\vec{x} = t \left [ \begin{matrix} x_{1} \\ x_{2} \\ x_{1} + x_{2} \end{matrix} \right ] = \left [ \begin{matrix} tx_{1} \\t x_{2} \\t x_{1} +t x_{2} \end{matrix} \right ] \in \mathbb{S}\).

V4: Since V6 holds, we have that \(\left [ \begin{matrix} 0 \\ 0 \\ 0 \end{matrix} \right ] = 0 \vec{x} \in \mathbb{S}\).

V5: Since V6 holds, we have that \(\left [ \begin{matrix} -x_{1} \\ -x_{2} \\ -x_{1} + (-x_{2}) \end{matrix} \right ] = (-1) \vec{x} \in \mathbb{S}\).

Thus, \(\mathbb{S}\)with these operators is a vector space as it satisfies all ten axioms.
