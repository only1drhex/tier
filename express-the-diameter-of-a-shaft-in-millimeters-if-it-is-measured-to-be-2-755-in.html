Guest Question.

Express the diameter of a shaft in millimeters if it is measured to be 2.755 in.

Admin
Solution

Table A18 gives the conversion factor for length to be 1.00 in = 25.4 mm. Then

Diameter =\(2.755 \ in\frac{25.4 \ mm}{1.00 \ in}=69.98 \ mm \)
