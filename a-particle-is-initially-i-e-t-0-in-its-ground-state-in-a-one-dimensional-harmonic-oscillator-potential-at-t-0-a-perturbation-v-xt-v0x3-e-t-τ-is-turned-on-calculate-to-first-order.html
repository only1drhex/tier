Guest Question.

A particle is initially (i.e., t = 0) in its ground state in a one-dimensional harmonic oscillator potential. At t = 0 a perturbation \(\hat{V}(x, t)=V_{0} \hat{x}^{3} e^{-t / \tau}\) is turned on. Calculate to first order the probability that, after a sufficiently long time (i.e., t → ∞), the system will have made a transition to a given excited state; consider all final states.

Admin
Solution

The transition probability from the ground state n = 0 to an excited state n is given by (10.41):

\(P_{i f}(t)=\left|-\frac{i}{\hbar} \int_{0}^{t}\left\langle\psi_{f}\left|\hat{V}\left(t^{\prime}\right)\right| \psi_{i}\right\rangle e^{i \omega_{f i} t^{\prime}} d t^{\prime}\right|^{2}\). (10.41)

 

\(P_{0 \rightarrow n}=\frac{1}{\hbar^{2}}\left|\int_{0}^{+\infty}\langle n|\hat{V}(t)| 0\rangle e^{i \omega_{n 0} t} d t\right|^{2}=\frac{V_{0}^{2}}{\hbar^{2}}\left|\left\langle n\left|\hat{x}^{3}\right| 0\right\rangle\right|^{2}\left|\int_{0}^{+\infty} e^{-(1 / \tau-i n \omega) t} d t\right|^{2}\), (10.201)

where \(\omega_{n 0}=\frac{E_{n}-E_{0}}{\hbar}=n \omega\) (since \(E_{n}=\hbar \omega\left(n+\frac{1}{2}\right)\)) and the time integration was calculated in (10.63):

\(\left|\int_{0}^{\infty} e^{-\left(1 / \tau-i \omega_{21}\right) t} d t\right|^{2}=\left[\omega_{21}^{2}+\frac{1}{\tau^{2}}\right]^{-1}=\left[\frac{9 \pi^{4} \hbar^{2}}{4 m^{2} a^{4}}+\frac{1}{\tau^{2}}\right]^{-1}\), (10.63)

 

\(\left|\int_{0}^{\infty} e^{-(1 / \tau-i n \omega) t} d t\right|^{2}=\frac{1}{n^{2} \omega^{2}+1 / \tau^{2}}\). (10.202)

Since \(\hat{a}|n\rangle=\sqrt{n}|n-1\rangle\) and \(\hat{a}^{\dagger}|n\rangle=\sqrt{n+1}|n+1\rangle\), and since \(\hat{X}^{3}=(\hbar / 2 m \omega)^{3 / 2}\left(\hat{a}^{\dagger}+\right.\hat{a})\left(\hat{a}^{2}+\hat{a}^{\dagger 2}+2 \hat{a}^{\dagger} \hat{a}+1\right)\), the only terms that survive in \(\left\langle n\left|\hat{x}^{3}\right| 0\right\rangle\) are

\(\left\langle n\left|\hat{X}^{3}\right| 0\right\rangle=\left(\frac{\hbar}{2 m \omega}\right)^{3 / 2}\left\langle n\left|\hat{a}^{\dagger 3}+\hat{a} \hat{a}^{\dagger 2}+\hat{a}^{\dagger}\right| 0\right\rangle=\left(\frac{\hbar}{2 m \omega}\right)^{3 / 2}\left(\sqrt{6} \delta_{n, 3}+3 \delta_{n, 1}\right) .\) (10.203)

This implies that the particle can be found after a long duration only either in the first or in the third excited state.

Inserting (10.202) and (10.203) into (10.201), we can verify that the probabilities corresponding to the transitions from the ground state to the first, the second and the third excited states are given, respectively, by

\(P_{0 \rightarrow 1}=\frac{V_{0}^{2}}{\hbar^{2}}\left|\left\langle 1\left|\hat{x}^{3}\right| 0\right\rangle\right|^{2}\left|\int_{0}^{+\infty} e^{-(1 / \tau-i \omega) t} d t\right|^{2}=\left(\frac{\hbar}{2 m \omega}\right)^{3} \frac{9 V_{0}^{2}}{(\hbar \omega)^{2}+\hbar^{2} / \tau^{2}},\), (10.204)

 

\(P_{0 \rightarrow 2}=0\) (10.205)

 

\(P_{0 \rightarrow 3}=\frac{V_{0}^{2}}{\hbar^{2}}\left|\left\langle 3\left|\hat{x}^{3}\right| 0\right\rangle\right|^{2}\left|\int_{0}^{+\infty} e^{-(1 / \tau-3 i \omega) t} d t\right|^{2}=\left(\frac{\hbar}{2 m \omega}\right)^{3} \frac{6 V_{0}^{2}}{(3 \hbar \omega)^{2}+\hbar^{2} / \tau^{2}}\). (10.206)

Therefore the system cannot undergo transitions to the second excited state nor to excited states higher than n = 3; that is, \(P_{0 \rightarrow 2}=0\), since \(\left\langle 2\left|\hat{X}^{3}\right| 0\right\rangle=0\) and \(P_{0 \rightarrow n}=0\) when n > 3, since \(\left\langle n\left|\hat{X}^{3}\right| 0\right\rangle=0\) for n > 3.
