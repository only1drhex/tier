Guest Question.

The shape shown in Figure is a W18 \(\times \)71 with holes in each flange for 1-inch-diameter bolts. The steel is A992. Compute the nominal flexural strength for an unbraced length of 10 feet. Use\( C_{b}\)= 1.0.

Admin
Solution

To determine the nominal flexural strength \(M_{n}\), all applicable limit states must be checked. From the \(Z_{x}\) table, a W18 \(\times\) 71 is seen to be a compact shape (no footnote to indicate otherwise). Also from the \(Z_{x}\) table, \(L_{p}\) = 6.00 ft and \(L_{r} \)= 19.6 ft. Therefore, for an unbraced length \(L_{b}\) = 10 ft,
\(L_{p} < L_{b} < L_{r}\)
and the beam is subject to inelastic lateral-torsional buckling. The nominal strength for this limit state is given by

\(M_{n}=C_{b}\left [M_{p} – (M_{p} – 0.7F_{y}S_{x}\left ( \frac{L_{b} -L_{p}}{L_{r} – L_{p}} \right ) \right ]\leq M_{p}\) (AISC Equation F2-2)

where
\(M_{p} = F_{y}Z_{x} = 50(146) = 7300 in.-kips\)

\(M_{n} = =1.0\left [7300- (7300 – 0.7\times 50 \times 127 \left ( \frac{10 – 6}{19.6 – 6} \right ) \right ]=6460 in.-kips\)

Check to see if the flange holes need to be accounted for. The gross area of one flange is
\(A_{fg} = t_{f}b_{f} = 0.810(7.64) = 6.188 in.^{2}\)
For a 1-inch diameter bolt, the standard hole size is
\(d_{h}=1+\frac{1}{8}=1\frac{1}{8}in.\)
The effective hole diameter is
\(d_{h}+\frac{1}{16}=1\frac{1}{8}+1\frac{1}{16}=1.188 in.\)
and the net flange area is
\(A_{fn} = A_{fg} – t_{f} \sum d_{h} = 6.188 – 0.810(2 \times 1.188) = 4.263 in.^{2}\)

\(F_{u}A_{fn}\) = 65(4.263) = 277.1 kips
Determine \(Y_{t}\). For A992 steel, the maximum \(F_{y}/F_{u}\) ratio is 0.85. Since this is greater than 0.8, use \(Y_{t} \)= 1.1.
\(Y_{t}F_{y}A_{fg} = 1.1(50)(6.188) = 340.3 kips\)

Since \(F_{u}A_{fn} < Y_{t}F_{y}A_{fg}\), the holes must be accounted for. From AISC Equation F13-1,

\(M_{n} = \frac{F_{u}A_{fn}}{A_{fg}}S_{x} = \frac{277.1}{6.188 (127)}=5687 in.-kips\)

This value is less than the LTB value of 6460 in.-kips, so it controls.
\(M_{n}\) = 5687 in.-kips = 474 ft-kips.
