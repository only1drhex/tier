Guest Question.

Find the rms value, half-cycle value, crest factor and form factor for a symmetric triangular waveform shown in Fig. 6.5-1.

Admin
Solution

Due to symmetry, the squared waveform needs to be integrated only from 0 to \(0.25T\). The integral for \(0\) to \(T\) range is four times this value.

\(V_{\mathrm{rms}}=\sqrt{\frac{4}{T} \int_{0}^{0.25 T}\left(\frac{4 V_{\mathrm{p}} t}{T}\right)^{2} d t}=\sqrt{\frac{64 V_{\mathrm{p}}^{2}}{3 T^{3}} \times(0.25 T)^{3}}=\frac{V_{\mathrm{p}}}{\sqrt{3}} \mathrm{~V}\)

 

\(V_{\text {caav }}=\frac{2}{T} \int_{0}^{0.5 T} v(t) d t=2 \times \frac{2}{T} \int_{0}^{0.25 T} \frac{4 V_{\mathrm{p}} t}{T} d t=\frac{V_{\mathrm{p}}}{2} \mathrm{~V}\)

 

\(\therefore \text { Form factor }=\frac{V_{\mathrm{p}} / \sqrt{3}}{V_{\mathrm{p}} / 2}=\frac{2}{\sqrt{3}}=1.1547\)

 

\(\text { Crest factor }=\frac{V_{\mathrm{p}}}{V_{\mathrm{p}} / \sqrt{3}}=\sqrt{3}=1.732\)
