Guest Question.

Estimate the minimum power needed to compress carbon dioxide in a steady-flow insulated compressor from 200 kPa and 20°C to 10 MPa. The inlet flow rate is 0.8 m³/min. Include real gas effects.

Admin
Solution

Minimum power is associated with a reversible process. Insulation results in negligible heat transfer. Consequently, an isentropic process is assumed. First, the reduced pressures and temperature are

\( P_{R 1}=\frac{P_{1}}{P_{c r}}=\frac{0.2}{7.39}=0.027 \quad P_{R 2}=\frac{10}{7.37}=1.37 \quad T_{R 1}=\frac{T_{1}}{T_{c r}}=\frac{293}{304.2}=0.963 \).

For the isentropic process Δs = 0:

\( \Delta s=0=\cancel {\left(s_{1}^{*}-{s_{1}}\right)}^{0}+\left(s_{2}-s_{2}^{*}\right)+\left(s_{2}^{*}-s_{1}^{*}\right)=0+\frac{\bar{s}^{2}-\bar{s}_{2}^{*}}{44}+0.842 \ln \frac{T_{2}}{293}-0.189 \ln \frac{10}{0.2} \).

\( \text { Since } \bar{s}_{2}-\bar{s}_{2}^{*} \text { depends on } T_{2} \text {, this equation has } T_{2} \text { as the only unknown. A trial-and-error procedure provides the } \) \( \text { solution. First, let } \bar{s}_{2}-\bar{s}_{2}^{*}=0 \text { and find } T_{2}=705 K \text {. Since } \bar{s}_{2}^{*}-\bar{s}^{2}>0, \text { we try the following: } \)

\( T_{2}=750 K , \quad T_{R 2}=2.47: \quad 0 \stackrel{?}{=}-\frac{2}{44}+0.842 \ln \frac{750}{293}-0.189 \ln \frac{10}{0.2}=0.0066 \).

\( T_{2}=730 K , \quad T_{R 2}=2.03: \quad 0 \stackrel{?}{=}-\frac{2}{44}+0.842 \ln \frac{730}{293}-0.189 \ln \frac{10}{0.2}=-0.016 \).

\( \text { Interpolating results in } T_{2}=744 K \text { or } 471^{\circ} C \text {. The work for this steady-flow process can now be found to be } \).

\( w=-\Delta h=\cancel {h_{1}-\hbar_{1}}^{0}+h_{2}^{*}-h_{2}+h_{1}^{*}-h_{2}^{*}=0+(2.0)\left(\frac{304.2}{44}\right)+(0.842)(20-471) \).

\( =-366 kJ / kg \).

\( \text { To find } \dot{W} \text {, we must know } \dot{m}=\left(\rho_{1}\right)(0.8 / 60) \text {. The density is found using } \)

\( \rho_{1}=\frac{P_{1}}{Z_{1} R T_{1}}=\frac{200}{(0.99)(0.189)(293)}=3.65 kg / m ^{3} \).

Finally

\( \dot{W}_{C}=\dot{m} w=\left[\frac{(3.65)(0.8)}{60}\right](-366)=-17.8 kW \).
