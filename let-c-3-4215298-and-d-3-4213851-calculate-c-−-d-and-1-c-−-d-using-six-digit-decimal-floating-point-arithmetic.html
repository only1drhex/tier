Guest Question.

Let c = 3.4215298 and d = 3.4213851. Calculate c − d and 1/(c − d), using six-digit decimal floating-point arithmetic.

Admin
Solution

I. The first step is to represent c and d by six-digit decimal floating-point num-bers:

\(c^{\prime} = 0.342153 × 10^{1}\)

 

\(d^{\prime} = 0.342139 × 10^{1}\)

The relative errors in c and d are, respectively,

\(\frac{c^{\prime} − c}{c} ≈ 0.6 × 10^{−7}\) and \(\frac{d^{\prime} − d}{d} ≈ 1.4 × 10^{−6}\)

II. \(f l(c^{\prime} − d^{\prime} ) = c^{\prime} − d^{\prime} = 0.140000 × 10^{−3}\). The actual value of c − d is \( 0.1447 × 10^{−3}\). The absolute and relative errors in approximating c − d by \( f l(c^{\prime} − d^{\prime})\) are, respectively,

\(f l(c^{\prime} − d^{\prime} ) − (c − d) = −0.47 × 10^{−5}\)

and

\( \frac{f l(c^{\prime} − d^{\prime}) − (c − d)}{c − d} ≈ −3.2 × 10^{−2}\)

Note that the magnitude of the relative error in the difference is more than \(10^{4}\) times the relative error in either c or d.

III. \( f l[1/(c^{\prime} − d^{\prime} )] = 0.714286 × 10^{4}\), and the correct answer, to six significant figures, is

\(\frac{1}{c − d} ≈ 0.691085 × 10^{4}\)

The absolute and relative errors are approximately 232 and 0.03, respectively.
