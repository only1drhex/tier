Guest Question.

Four resistors are connected as shown in Figure 28.9a.

(A) Find the equivalent resistance between points a and c.

(B) What is the current in each resistor if a potential difference f 42 V is maintained between a and c?

Admin
Solution

The combination of resistors can be reduced in steps, as shown in Figure 28.9. The 8.0-Ω and 4.0-Ω resistors are in series; thus, the equivalent resistance between a and b is 12.0 Ω (see Eq. 28.5). The 6.0-Ω and 3.0-Ω resistors are in parallel, so from Equation 28.7 we find that the equivalent resistance from b to c is 2.0 Ω. Hence, the equivalent resistance from a to c is 14.0Ω.

\( \Delta V=I R_{ eq }=I\left(R_{1}+R_{2}\right) \quad \longrightarrow \quad R_{ eq }=R_{1}+R_{2} \) (28.5).

\( \frac{1}{R_{\text {eq }}}=\frac{1}{R_{1}}+\frac{1}{R_{2}} \) (28.7).

(B) What is the current in each resistor if a potential difference of 42 V is maintained between a and c?

The currents in the 8.0-Ω and 4.0-Ω resistors are the same because they are in series. In addition, this is the same as the current that would exist in the 14.0-Ω equivalent resistor subject to the 42-V potential difference. Therefore, using Equation 27.8 (R =ΔV/I ) and the result from part (A), we obtain

\( R \equiv \frac{\Delta V}{I} \) (27.8).

\( I=\frac{\Delta V_{a c}}{R_{e q}}=\frac{42 V }{14.0 \Omega}=3.0 A \).

This is the current in the 8.0-Ω and 4.0-Ω resistors. When this 3.0-A current enters the junction at b, however, it splits, with part passing through the 6.0-Ω resistor \( \left(I_{1}\right) \) and part through the 3.0-Ω resistor \( \left(I_{2}\right) \). Because the potential difference is \(\Delta V_{b c}\) across each of these parallel resistors, we see that \( (6.0 \Omega) I_{1}=(3.0 \Omega) I_{2}, \text { or } I_{2}=2 I_{1} \). Using this result and the fact that \( I_{1}+I_{2}=3.0 A \text {, we find that } I_{1}=1.0 A \text { and } \)\( I_{2}=2.0 A \). We could have guessed this at the start by noting that the current in the 3.0-Ω resistor has to be twice that in the 6.0-Ω resistor, in view of their relative resistances and the fact that the same voltage is applied to each of them.

\(\text { As a final check of our results, note that } \Delta V_{b c}=\)\((6.0 \Omega) I_{1}=(3.0 \Omega) I_{2}=6.0 V \text { and } \Delta V_{a b}=(12.0 \Omega) I=36 V\)\( \text { therefore, } \Delta V_{a c}=\Delta V_{a b}+\Delta V_{b c}=42 V , \text { as it must. } \)
