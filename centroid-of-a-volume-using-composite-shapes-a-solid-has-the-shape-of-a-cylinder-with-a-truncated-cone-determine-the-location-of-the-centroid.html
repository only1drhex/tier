Guest Question.

Centroid of a Volume Using Composite Shapes

A solid has the shape of a cylinder with a truncated cone. Determine the location of the centroid.

Admin
Solution

Road Map This object consists of an arrangement of simple geometric shapes. Thus, compared to using integration, it will be easier to determine the position of the centroid by using composite shapes with Eq. (\(\overline{x} = \frac{\sum\limits_{i=1}^{n}{\widetilde{x}_i V_{i} } }{\sum\limits_{i=1}^{n}{V_i} } , \overline{y} = \frac{\sum\limits_{i=1}^{n}{\widetilde{y}_i V_{i} } }{\sum\limits_{i=1}^{n}{V_i} }\), and \(\overline{z} = \frac{\sum\limits_{i=1}^{n}{\widetilde{z}_i V_{i} } }{\sum\limits_{i=1}^{n}{V_i} }\)) on p. 399. This object is symmetric about the \(xz\) plane, and therefore its centroid must lie in the \(xz\) plane. The object is also symmetric about the \(yz\) plane, and therefore its centroid must also lie in the \(yz\) plane. The intersection of these planes is the \(z\) axis. Therefore, we can conclude that the \(x\) and \(y\) positions of the centroid are \(\overline{x} = \overline{y} = 0\).

Governing Equations & Computation In Fig. 2 the object is subdivided into three shapes. The volume and centroid position of each of the composite shapes are given in the Table of Properties of Solids on the inside back cover of this book, and these values are collected in Table 1. For example, for shape 2, the volume is \(V_2 = \pi r^2 h/3\) where

Table 1. Volumes and centroid positions for composite shapes in Fig. 2.
Shape no. \(V_i\) \(\overline{z}_i\)
1 \(\pi (2 in.)^2(3 in.) = 37.70 in.^3\) 1.5 in.
2 \(\frac{\pi}{3} (2 in.)^2(4 in.) = 16.76 in.^3\) \(3 in. + \frac{1}{4}(4 in.) = 4 in.\)
3 \(-\frac{\pi}{3} (1 in.)^2(4 in.) = -2.094 in.^3\) \(5 in. + \frac{1}{4}(2 in.) = 5.5 in.\)

r = 2 in. is the radius of the cone’s base and h = 4 in. is its height. The centroid for the cone measured from its base is h/4, and to this value, the distance from the origin of the coordinate system to the cone’s base (i.e., 3 in.) must be added. Evaluating the expression for \(\overline{z}_i\) in Eq. (\(\overline{x} = \frac{\sum\limits_{i=1}^{n}{\widetilde{x}_i V_{i} } }{\sum\limits_{i=1}^{n}{V_i} } , \overline{y} = \frac{\sum\limits_{i=1}^{n}{\widetilde{y}_i V_{i} } }{\sum\limits_{i=1}^{n}{V_i} }\), and \(\overline{z} = \frac{\sum\limits_{i=1}^{n}{\widetilde{z}_i V_{i} } }{\sum\limits_{i=1}^{n}{V_i} }\)) provides

\(\overline{z} = \frac{\sum\limits_{i=1}^{3}{\widetilde{z}_i V_{i} } }{\sum\limits_{i=1}^{3}{V_i} } \)

 

\(=\frac{(1.5 in.)(37.70 in.^3) + (4 in.)(16.76 in.^3) + (5.5 in.)(-2.094 in.^3)}{37.70 in.^3 + 16.76 in.3 – 2.094 in.^3}\)

 

\(= 2.14 in.\) (1)

Discussion & Verification Short of resolving this problem by using a different selection of composite shapes, or using a different method (e.g., integration), there are no definitive checks of accuracy for our solution. Nonetheless, we expect \(\overline{z}\) to be somewhat larger than 1.5 in., which is the centroid of volume \(V_1\) only, and in view of this, the value found in Eq. (1) is reasonable.
