Guest Question.

For the separation specified in, evaluate the effect of changes in reflux ratio on the number of stages required. This is an example of the application of the Erbar-Maddox method.

Solution

The relative volatilities estimated in , and the component distributions calculated in will be used for this example.

Minimum number of stages; Fenske equation, equation 11.58:

 

\(N_{m}=\frac{\log \left[\frac{x_{ LK }}{x_{ HK }}\right]_{d}\left[\frac{x_{ HK }}{x_{ LK }}\right]_{b}}{\log \alpha_{ LK }}\) (11.58)

 

\(N_{m}=\frac{\log \left[\left(\frac{24}{1}\right)\left(\frac{19}{1}\right)\right]}{\log 2}=\underline{\underline{8.8}}\)

 

Minimum reflux ratio; Underwood equations 11.60 and 11.61.

 

\(\sum \frac{\alpha_{i} x_{i, d}}{\alpha_{i}-\theta}=R_{m}+1\) (11.60)

 

\(\sum \frac{\alpha_{i} x_{i, f}}{\alpha_{i}-\theta}=0\) (11.61)
