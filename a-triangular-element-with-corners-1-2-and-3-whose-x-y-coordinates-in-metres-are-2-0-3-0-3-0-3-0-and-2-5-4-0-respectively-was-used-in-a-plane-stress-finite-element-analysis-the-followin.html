Guest Question.

A triangular element with corners 1, 2 and 3, whose x, y coordinates in metres are (2.0, 3.0), (3.0, 3.0) and (2.5, 4.0), respectively, was used in a plane stress finite element analysis. The following nodal displacements (mm) were obtained.

\(w_{1}=0.04, \quad v_{1}=0.08, \quad w_{2}=0.10, \quad v_{2}=0.12, \quad w_{3}=0.20, \quad v_{3}=0.18\)

Calculate the stresses in the element if Young’s modulus is \(200000 N / mm ^{2}\) and Poisson’s ratio is 0.3.
Ans. \(\sigma_{x}=25.4 N / mm ^{2}, \sigma_{y}=28.5 N / mm ^{2}, \tau_{x y}=13.1 N / mm ^{2}\).

Admin
Solution

The displacement functions are

\(w=\alpha_{1}+\alpha_{2} x+\alpha_{3} y\)

 

\(v=\alpha_{4}+\alpha_{5} x+\alpha_{6} y\)

Then

\(w_{1}=\alpha_{1}+2 \alpha_{2}+3 \alpha_{3}\)

 

\(w_{2}=\alpha_{1}+3 \alpha_{2}+3 \alpha_{3}\)

 

\(w_{3}=\alpha_{1}+2.5 \alpha_{2}+4 \alpha_{3}\)

Extracting \(\alpha_{1}, \alpha_{2} \text { and } \alpha_{3}\) from the above gives

\(\alpha_{1}=4.5 w_{1}-0.5 w_{2}-3 w_{3}\)

 

\(\alpha_{2}=w_{2}-w_{1}\)

 

\(\alpha_{3}=-0.5 w_{1}-0.5 w_{2}+w_{3}\)

Now substituting these expressions in the displacement function for w

\(w=(4.5-x-0.5 y) w_{1}+(-0.5+x-0.5 y) w_{2}+(-3+y) w_{3}\)

Similarly

\(v=(4.5-x-0.5 y) v_{1}+(-0.5+x-0.5 y) v_{2}+(-3+y) v_{3}\)

From Eqs (17.75)

\(\varepsilon_{x}=\frac{\partial w}{\partial x} \quad \varepsilon_{y}=\frac{\partial v}{\partial y} \quad \gamma_{x y}=\frac{\partial w}{\partial y}+\frac{\partial v}{\partial x}\) (17.75)

\(\varepsilon_{x}=\partial w / \partial x=-w_{1}+w_{2}\)

 

\(\varepsilon_{y}=\partial v / \partial y=-0.5 v_{1}-0.5 v_{2}+v_{3}\)

 

\(\gamma_{x y}=(\partial w / \partial y)+(\partial v / \partial x)=-0.5 w_{1}-0.5 w_{2}+w_{3}-v_{1}+v_{2}\)

Substituting for \(w_{1}, w_{2}\), etc. gives

\(\varepsilon_{x}=0.06 \times 10^{-3}, \quad \varepsilon_{y}=0.08 \times 10^{-3}, \quad \gamma_{x y}=0.17 \times 10^{-3}\)

Substituting these values in Eq. (17.80)

\(\{\sigma\}=\left\{\begin{array}{c}\sigma_{x} \\\sigma_{y} \\\tau_{x y}\end{array}\right\}=\frac{E(1-v)}{(1-v)(1-2 v)}\left[\begin{array}{ccc}1 & \frac{v}{1-v} & 0 \\\frac{v}{1-v} & 1 & 0 \\0 & 0 & \frac{(1-2 v)}{2(1-v)}\end{array}\right]\left\{\begin{array}{c}\varepsilon_{x} \\\varepsilon_{y} \\\gamma_{x y}\end{array}\right\}\) (17.80)

gives

\(\sigma_{x}=25.4 N / mm ^{2}, \quad \sigma_{y}=28.5 N / mm ^{2}, \tau_{x y}=13.1 N / mm ^{2}\).
