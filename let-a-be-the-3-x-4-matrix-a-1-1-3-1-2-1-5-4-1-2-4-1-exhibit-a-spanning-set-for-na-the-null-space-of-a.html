Guest Question.

Let A be the (3 × 4) matrix

\(A=\left[\begin{array}{rrrr}1 & 1 & 3 & 1 \\2 & 1 & 5 & 4 \\1 & 2 & 4 & -1\end{array}\right]\).

Exhibit a spanning set for N(A), the null space of A.

Admin
Solution

The first step toward obtaining a spanning set for N(A) is to obtain an algebraic specification for N(A) by solving the homogeneous system Ax = θ. For the given matrix A, this was done in of Section 3.3. Specifically,

N(A) = {x: x = \(\left[\begin{array}{c}-2 x_{3}-3 x_{4} \\-x_{3}+2 x_{4} \\x_{3} \\x_{4}\end{array}\right], \quad x_{3} \text { and } x_{4}\) any real numbers}.

Thus a vector x in N(A) is totally determined by the unconstrained parameters \(x_{3}\) and \(x_{4}\). Separating those parameters gives a decomposition of x:

\(x =\left[\begin{array}{c}-2 x_{3}-3 x_{4} \\-x_{3}+2 x_{4} \\x_{3} \\x_{4}\end{array}\right]=\left[\begin{array}{c}-2 x_{3} \\-x_{3} \\x_{3} \\0\end{array}\right]+\left[\begin{array}{c}-3 x_{4} \\2 x_{4} \\0 \\x_{4}\end{array}\right]=x_{3}\left[\begin{array}{r}-2 \\-1 \\1 \\0\end{array}\right]+x_{4}\left[\begin{array}{r}-3 \\2 \\0 \\1\end{array}\right]\). (8)

Let \(u _{1} \text { and } u _{2}\) be the vectors

\(u _{1}=\left[\begin{array}{r}-2 \\-1 \\1 \\0\end{array}\right] \quad \text { and } \quad u _{2}=\left[\begin{array}{r}-3 \\2 \\0 \\1\end{array}\right]\).

 

By setting \(x_{3}=1 \text { and } x_{4}=0\) in Eq. (8), we obtain \(u _{1}\), so \(u _{1}\) is in N(A). Similarly, \(u _{2}\) can be obtained by setting \(x_{3}=0 \text { and } x_{4}=1, \text { so } u _{2}\) is in N(A). Moreover, it is an immediate consequence of Eq. (8) that each vector x in N(A) is a linear combination of \(u _{1} \text { and } u _{2}\). Therefore, \(N (A)=\operatorname{Sp}\left\{ u _{1}, u _{2}\right\}\); that is, \(\left\{ u _{1}, u _{2}\right\}\) is a spanning set for N(A).
