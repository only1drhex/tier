Guest Question.

Block \(C\) has a mass \({ m }_{ c }\) and is confined between two walls by smooth rollers. If the block rests on to \(p\) of the spool of mass \({ m }_{ s }\), determine the minimum cable force \(P\) needed to move the spool. The cable is wrapped around the spool’s inner core. The coefficients of static friction at \({ \mu }_{ A }\) and \({ \mu }_{ B }\).

Given:

\(g\) = \(9.81\) \(\frac { m }{ { s }^{ 2 } }\)

\({ m }{ c }\) = \(50\) \(kg\)

\({ m }{ s }\) = \(40\) \(kg\)

\({ \mu }_{ A }\)= \(0.3\)

\({ \mu }_{ B }\) = \(0.6\quad { r }_{ 1 }\) = \(0.2\) \(m\quad { r }_{ 2 }\) = \(0.4\) \(m\)

Admin
Solution

Assume that the spool slips at \(A\) but not at \(B\).

The initial guesses are \(\quad { F }_{ B }=2N\quad \quad P=3N\quad \quad { N }_{ B }=1N\)

 

Given \(\quad P-{ F }_{ B }-{ \mu }_{ A }{ m }_{ c }g=0\)

 

\(\quad \quad \quad \quad { N }_{ B }-{ m }_{ s }g-{ m }_{ c }g=0\)

 

\(\quad \quad \quad \quad P\left( { r }_{ 1 }+{ r }_{ 2 } \right) -{ F }_{ B }2{ r }_{ 2 }=0\)

 

\(\left( \begin{matrix} { F }_{ B } \\ P \\ { N }_{ B } \end{matrix} \right) =Find\left( { F }_{ B },P,{ N }_{ B } \right) \quad \quad \quad \left( \begin{matrix} { F }_{ B } \\ P \\ { N }_{ B } \end{matrix} \right) =\left( \begin{matrix} 441.45 \\ 588.60 \\ 882.90 \end{matrix} \right) N\)

 

Now check the no slip assumption at \(B\)

\(\quad { F }_{ Bmax }=\mu B{ N }_{ B }\quad { F }_{ Bmax }=529.74N\) then our assumptions are correct

 

Since \(\quad { F }_{ B }=441N<\quad { F }_{ Bmax }=530N\quad \)

 

\(P=589N\)
