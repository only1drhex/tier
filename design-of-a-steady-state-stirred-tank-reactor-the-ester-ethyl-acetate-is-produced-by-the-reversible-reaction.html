Guest Question.

Design of a Steady-State Stirred-Tank Reactor The ester ethyl acetate is produced by the reversible reaction

\(CH _{3} COOH + C _{2} H _{5} OH \frac{k}{\overline{k^{\prime}}} CH _{3} COOC _{2} H _{5}+ H _{2} O\)

in the presence of a catalyst such as sulfuric or hydrochloric acid. The rate of ethyl acetate production has been found, from the analysis of chemical kinetics data, to be given by the following equation:

\(\frac{d C_{ EA }}{d t}=k C_{ A } C_{ E }-k^{\prime} C_{ EA } C_{ W }\)

where the subscripts EA, A, E, and W denote ethyl acetate, acetic acid, ethanol, and water, respectively. The values of the reaction rate constants at 100ºC and the catalyst concentration of interest are

\(k=4.76 \times 10^{-4} m ^{3} /( kmol min )\)

and

\(k^{\prime}=1.63 \times 10^{-4} m ^{3} /( kmol min )\)

The feed stream is an aqueous solution containing 250 kg of acetic acid and 500 kg of ethyl alcohol per \(m^{3}\) of solution (including catalyst). The density of the solution is 1040 kg/\(m^{3}\) and constant. The reaction will be carried out at 100ºC, the feed is at 100ºC, and the reactor will be operated at a sufficiently high pressure that a negligible amount of reactants or products vaporizes.

If a continuous-flow stirred-tank reactor is used for the reaction, determine

a. The size of the reactor needed to produce 1250 kg/hr of the ester, if 37.2 percent of the acid reacts.

b. The heat load on the reactor for this extent of reaction.

Admin
Solution

a. Using the equations in Illustration 13.1-9, and the fact that 37.2 percent of the acid (or 0.372 × 4.17 = 1.55 kmol/\(m^{3}\)) reacts, we have for the reactor contents and outlet concentrations

 

\(C_{ A }=4.17-1.55=2.62\) kmol / \(m ^{3}\)

 

\(C_{ E }=10.9-1.55=9.35\) kmol /\( m ^{3}\)

 

\(C_{ EA }=0+1.55=1.55\) kmol / \(m ^{3}\)

 

\(C_{ W }=16.1+1.55=17.65\) kmol / \(m ^{3}\)

 

Next, using Eq. 14.1-7 yields

 

\(C_{ i }=\left(C_{ i }\right)_{ in }+\frac{V}{q} \sum_{ j =1}^{ M } \nu_{ ij } r_{ j }\) (14.1-7)

 

\(C_{ EA , \text { in }}=\frac{V}{q} r=\frac{V}{q}\left\{k C_{ A } C_{ E }-k^{\prime} C_{ EA } C_{ W }\right\}\)

 

\(\begin{aligned}r &=\left\{4.76 \times 10^{-4} \times 2.62 \times 9.35-1.63 \times 10^{-4} \times 1.55 \times 17.65\right\} \\&=7.20 \times 10^{-3} \frac{ kmol }{ m ^{3} min }=7.20 \frac{ mol }{ m ^{3} min }\end{aligned}\)

 

and

 

\(1.55 \frac{ kmol }{ m ^{3}}=\frac{V}{q} 7.20 \times 10^{-3} \frac{ kmol }{ m ^{3} min }\)

 

so that

 

\(\tau=\frac{V}{q}=215.3 min =3.588 hr\)

 

is required to obtain the desired extent of reaction. (Note that τ has units of time and can be interpreted to be an average time that an element of fluid is in the reactor; this is referred to as the average residence time for fluid in the reactor.) The volumetric flow rate into and out of the reactor must be such that

 

\(q \times C_{ EA } \times mol \text { wt of ester }=1250 kg / hr\)

 

or

 

\(q=\frac{1250 kg / hr }{1.55 \frac{ kmol }{ m ^{3}} \times 88 \frac{ g }{ mol }}=9.164 m ^{3} / hr\)

 

Therefore, the reactor volume should be

 

\(V=\tau q=3.588 hr \times 9.164 m ^{3} / hr =32.88 m ^{3}\)

 

to obtain the desired production rate.

b. To determine the heat load on the reactor, we can use either Eqs. 14.1-9 or 14.1-10. Assuming that liquid-phase nonidealities are unimportant, and that the reactor effluent is at the reactor temperature (100ºC), these equations reduce to

 

\(\dot{Q}=q \sum_{ i }\left(C_{ i }\right)_{\text {in }}\left\{\bar{H}_{ i }-\left(\bar{H}_{ i }\right)_{\text {in }}\right\}+V \sum_{ j } r_{ j } \Delta_{ rxn , j } H\) (14.1-9a)

 

\(\dot{Q}=q \sum_{ i } C_{ i }\left\{\bar{H}_{ i }-\left(\bar{H}_{ i }\right)_{ in }\right\}+V \sum_{ j } r_{ j }\left(\Delta_{ rxn , j } H\right)_{ in }\) (14.1-9b)

 

\(\dot{Q}=q \sum_{ i =1}^{ C }\left(C_{ i }\right)_{ in }\left\{\underline{H}_{ i }(T)-\underline{H}_{ i }\left(T_{ in }\right)\right\}+V \sum_{ j =1}^{ M } \Delta_{ rxn , j } H(T) r_{ j }\) (14.1-10a)

 

\(\dot{Q}=q \sum_{ i } C_{ i }\left\{\underline{H}_{ i }(T)-H_{ i }\left(T_{ in }\right)\right\}+V \sum_{ j =1}^{ M } \Delta_{ rxn , j } H\left(T_{ in }\right) r_{ j }\) (14.1-10b)

 

\(Q=V \Delta_{ rxn } H \cdot r\)

 

since the inlet stream and reactor temperatures are equal. From part (a), \(V=32.88 m ^{3} \text { and }\) r = 7.20 mol/(\(m^{3}\) min), and from Appendix A.IV we have

 

\(\begin{array}{l}\Delta_{ f } \underline{H}_{ A }^{\circ}\left(T=25^{\circ} C \right)=-486.1 kJ / mol\\\Delta_{ f } \underline{H}_{ E }^{\circ}\left(T=25^{\circ} C \right)=-277.7 kJ / mol\\\Delta_{ f } \underline{H}_{ EA }^{\circ}\left(T=25^{\circ} C \right)=-463.3 kJ / mol\\\Delta_{ f } \underline{H}_{ W }^{\circ}\left(T=25^{\circ} C \right)=-285.8 kJ / mol\end{array}\)

 

Therefore,

 

\(\begin{aligned}\Delta_{ rxn } H^{\circ}\left(T=25^{\circ} C \right) &=\sum \nu_{ i } \Delta_{ f } \underline{H}_{ i }^{\circ}=(-463.3)+(-285.8)-(-486.1)-(-277.7) \\&=14.7 kJ / mol \text { ester produced }\end{aligned}\)

 

and

 

\(\Delta_{ rxn } H^{\circ}\left(T=100^{\circ} C \right)=\Delta_{ rxn } H^{\circ}\left(T=25^{\circ} C \right)+\int_{T=25^{\circ} C }^{T=100^{\circ} C } \Delta_{ rxn } C_{ P } d T\)

 

\(\text { where } \Delta_{ rxn } C_{ P }=-13.39 J /( mol K ) \text {. Therefore, }\)

 

\(\begin{aligned}\Delta_{ rxn } H^{\circ}\left(T=100^{\circ} C \right) &=14700 J / mol -13.39 J /( mol K ) \times 75 K \\&=14700-1004 \cong 13700 J / mol\end{aligned}\)

 

Neglecting solution nonidealities, we have \(\Delta_{ rxn } H\left(T=100^{\circ} C \right)=\Delta H_{ rxn }^{\circ}\left(T=100^{\circ} C \right)\) so that

 

\(\dot{Q}=32.88 m ^{3} \times 7.20 mol /\left( m ^{3} min \right) \times 13.7 kJ / mol =3243 kJ / min\)

 

Note: The fact that \(\dot{Q}\) is positive indicates that the reaction is endothermic, so that heat must be supplied to the reactor to maintain isothermal operation.)
