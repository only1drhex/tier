Guest Question.

Show how 1-butyne could be converted to 3-hexyne using a nucleophilic sub stitution reaction.

Admin
Solution

Compare the starting material with the product.

\( \underset{1-butyne}{CH_{3}CH_{2}C\equiv CH}\) \( \underset{3-hexyne}{CH_{3}CH_{2}C\equiv CCH_{2}CH_{3}} \)

From Table 6.1, entry 15, we see that acetylides react with alkyl halides to give acetylenes. We therefore need to convert 1-butyne to an acetylide (review eq. 3.53 \(R-CC-\underset{\underset{\begin{matrix} this \ hydrogen \ is \\ weakly \ acidic \end{matrix} }{\uparrow } }{H} + \underset{sodium \ amide}{Na^{+} NH_{2}^{-}} \overset{liquid \ NH_{3}}{\longrightarrow } R-CC:^{-}Na^{+} + NH_{3} \) ), then treat it with a 2-carbon alkyl halide.
