Guest Question.

Using the indicated state variables, write the state equations for each of the systems shown in Fig. 7.85. Find the transfer function for each system using both block-diagram manipulation and matrix algebra [as in Eq. (7.48) \(G(s)=\frac{Y(s)}{U(s)}= H (s I - F )^{-1} G +J\) ].

Admin
Solution

(a) Performing a partial fraction expansion on (s + 2)/(s + 4), Fig. 7.85(a) can be redrawn as shown below.

By inspection of the block diagram, the state equations are,

\(\dot{ x }=\left[\begin{array}{ccc}-5 & 1 & 1 \\0 & -3 & -2 \\0 & 0 & -4\end{array}\right] x +\left[\begin{array}{l}0 \\1 \\1\end{array}\right] u\),

\(y=\left[\begin{array}{lll}1 & 0 & 1\end{array}\right] x \).

Computing \(G(s)= H (s I – F )^{-1} G\) the following transfer function is obtained:

\(\frac{Y(s)}{U(s)}=\frac{s^{2}+10 s+20}{s^{3}+12 s^{2}+47 s+60}\).

We can use the MATLAB ss2tf command to verify this result.

(b) Using the second block diagram given in Fig. 7.85(b), we can write,

\(\dot{ x }=\left[\begin{array}{cccc}-3 & 1 & 0 & 0 \\0 & 0 & 4 & 0 \\0 & -1 & -2 & 1 \\0 & 0 & 0 & -10\end{array}\right] x +\left[\begin{array}{l}0 \\0 \\1 \\1\end{array}\right] u\),

\(y=\left[\begin{array}{llll}1 & 0 & 1 & 0\end{array}\right] x\).

Computing \(H (s I – F )^{-1} G\) (using MATLABâ€™s ss2tf), we obtain the following transfer function,

\(\frac{Y(s)}{U(s)}=\frac{s^{3}+14 s^{2}+37 s+44}{s^{4}+15 s^{3}+60 s^{2}+112 s+120}\).
