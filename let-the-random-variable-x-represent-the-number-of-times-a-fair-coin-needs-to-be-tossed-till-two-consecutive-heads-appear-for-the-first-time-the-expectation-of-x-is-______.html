Guest Question.

Let the random variable X represent the number of times a fair coin needs to be tossed till two consecutive heads appear for the first time. The expectation of X is ______.

Admin
Solution

Given that X is a random variable representing “The number of times a fair coin needs to be tossed till two consecutive heads appear for the first time”.
 ∴ The possible values of X are 2, 3, 4, 5, ……

\(\begin{aligned}& P (X=2)= P ( HH )= P ( H ) P ( H )=\frac{1}{2} \times \frac{1}{2}=\frac{1}{2^{2}} \\& P (X=3)= P ( THH )= P ( T ) P ( H ) P ( H ) \\&\quad=\frac{1}{2} \times \frac{1}{2} \times \frac{1}{2}=\frac{1}{2^{3}} \\& P (X=4)= P ( TTHH )= P ( T ) P ( T ) P ( H ) P ( H ) \\&\quad=\frac{1}{2} \times \frac{1}{2} \times \frac{1}{2} \times \frac{1}{2}=\frac{1}{2^{4}} \\&\therefore \quad \text { Expectation of } X=E(x)=\sum_{x=2}^{\infty} x P(X=x) \\&\quad=2 P (X=2)+3 P (X=3)+4 P (X=4)+\ldots \infty \\&\quad=2 \times \frac{1}{2^{2}}+3 \times \frac{1}{2^{3}}+4 \times \frac{1}{2^{4}}+\ldots . \infty\end{aligned}\)

 

\(\therefore \quad E(X)=\frac{1}{2^{2}}\left[2+3 \times \frac{1}{2}+4 \times \frac{1}{2^{2}}+5 \times \frac{1}{2^{3}}+\ldots \infty\right]\) (1)

Consider \(2+3 \times \frac{1}{2}+4 \times \frac{1}{2^{2}}+5 \times \frac{1}{2^{3}}+\ldots . \infty\)

which is an arithmetic geometric progression with \(a=2, d=1 \text { and } r=\frac{1}{2}\)

∴ The sum of infinite number of terms in AGP

\(=\frac{a}{1-r}+\frac{d r}{(1-r)^{2}}\)

 

\(\begin{aligned}&=\frac{2}{(1-1 / 2)}+\frac{1 \times(1 / 2)}{(1-1 / 2)^{2}} \\&=4+\frac{(1 / 2)}{(1 / 4)}\end{aligned}\)

= 4 + 2
 = 6
∴ Substituting in (1), we have

\(E(X)=\frac{1}{4} \times 6=\frac{3}{2}=1.5\)

Hence, the correct Answer is (1.5).
