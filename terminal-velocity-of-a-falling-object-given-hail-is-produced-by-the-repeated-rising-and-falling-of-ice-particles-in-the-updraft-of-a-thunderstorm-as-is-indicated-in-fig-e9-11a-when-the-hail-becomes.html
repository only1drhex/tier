Guest Question.

Terminal Velocity of a Falling Object

GIVEN Hail is produced by the repeated rising and falling of ice particles in the updraft of a thunderstorm, as is indicated in Fig. E9.11a. When the hail becomes large enough, the aerodynamic drag from the updraft can no longer support the weight of the hail, and it falls from the storm cloud.

FIND Estimate the velocity, U, of the updraft needed to make D = 1.5-in.-diameter (i.e., “golf ball-sized”) hail.

Admin
Solution

As is discussed in, for steady-state conditions a force balance on an object falling through a fluid at its terminal velocity, U, gives

 

\(W = D +F_{B}\)

 

\(\text { where } F_{B}=\gamma_{ air } \forall i\) is the buoyant force of the air on the particle, \(W=\gamma_{\text {ice }} \forall\) is the particle weight, and D is the aerodynamic drag. This equation can be rewritten as

 

\(\frac{1}{2} \rho_{ air } U^{2} \frac{\pi}{4} D^{2} C_{D}=q W-F_{B}\) (1)

 

\(\text { With } \forall=\pi D^{3} / 6 \text { and since } \gamma_{\text {ice }} \gg \gamma_{\text {air }} \text { (i.e., } W \gg \gg F_{B} \text { ), }\) Eq. 1 can be simplified to

 

\(U=\left(\frac{4}{3} \frac{\rho_{\text {ice }}}{\rho_{\text {air }}} \frac{g D}{C_{D}}\right)^{1 / 2}\)

 

\(\text { By using } \rho_{\text {ice }}=1.84 \text { slugs } / ft ^{3}, \rho_{\text {air }}=2.38 \times 10^{-3} \text { slugs } / ft ^{3} \text {, and }\) \(D=1.5 \text { in. }=0.125 ft \text {, Eq. } 2 \text { becomes }\)

 

\(U=\left[\frac{4\left(1.84 \text { slugs } / ft ^{3}\right)\left(32.2 ft / s ^{2}\right)(0.125 ft )}{3\left(2.38 \times 10^{-3} \text { slugs } / ft ^{3}\right) C_{D}}\right]^{1 / 2}\)

 

or

 

\(U=\frac{64.5}{\sqrt{C_{D}}}\) (3)

 

where U is in ft/s. To determine U, we must know \(C_{D}\). nately, \(C_{D}\) is a function of the Reynolds number (see Fig. 9.21), which is not known unless U is known. Thus, we must use an iterative technique similar to that done with the Moody chart for certain types of pipe flow problems (see Section 8.5).

From Fig. 9.21 we expect that \(C_{D}\)is on the order of 0.5. Thus,we assume \(C_{D}\) = 0.5 and from Eq. 3 obtain

 

\(U=\frac{64.5}{\sqrt{0.5}}=91.2 ft / s\)

 

The corresponding Reynolds number (assuming v = 1.57 x \(\left.10^{-4} ft ^{2} / s \right) \text { is }\)

 

\(\operatorname{Re}=\frac{U D}{\nu}=\frac{91.2 ft / s (0.125 ft )}{1.57 \times 10^{-4} ft ^{2} / s }=7.26 \times 10^{4}\)

 

For this value of Re we obtain from Fig. 9.21, \(C_{D}=0.5\). Thus, our assumed value of \(C_{D}=0.5\) was correct. The corresponding value of U is

 

\(U=91.2 ft / s =62.2 mph\)

 

COMMENTS By repeating the calculations for various altitudes, z, above sea level (using the properties of the U.S. Standard Atmosphere given in Appendix C), the results shown in Fig. E9.11b are obtained. Because of the decrease in density with altitude, the hail falls even faster through the upper portions of the storm than when it hits the ground.

Clearly, an airplane flying through such an updraft would feel its effects (even if it were able to dodge the hail). As seen from Eq. 2, the larger the hail, the stronger the necessary updraft.

Hailstones greater than 6 in. in diameter have been reported. In reality, a hailstone is seldom spherical and often not smooth. However, the calculated updraft velocities are in agreement with measured values.
