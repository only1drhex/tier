Guest Question.

Use the Runge-Kutta method to solve y” – 3y + 2y = 0; y(0) = -1, y'(0) = 0 on the interval [0,1] with h = 0.1.

Admin
Solution

Using the results of , we have \(f(x, y, z)=z, g(x, y, z)=3 z-2 y, x_{0}=0, y_{0}=-1, \text { and } z_{0}=0\). Then, using (20.4), we compute:

\(\begin{aligned}n=0: k_{1} &=h f\left(x_{0}, y_{0}, z_{0}\right)=h f(0,-1,0)=(0.1)(0)=0 \\l_{1} &=h g\left(x_{0}, y_{0}, z_{0}\right)=h g(0,-1,0)=(0.1)[3(0)-2(-1)]=0.2 \\k_{2} &=h f\left(x_{0}+\frac{1}{2} h, y_{0}+\frac{1}{2} k_{1}, z_{0}+\frac{1}{2} l_{1}\right) \\&=h f\left[0+\frac{1}{2}(0.1),-1+\frac{1}{2}(0), 0+\frac{1}{2}(0.2)\right] \\&=h f(0.05,-1,0.1)=(0.1)(0.1)=0.01 \\l_{2} &=h g\left(x_{0}+\frac{1}{2} h, y_{0}+\frac{1}{2} k_{1}, z_{0}+\frac{1}{2} l_{1}\right) \\&=h g(0.05,-1,0.1)=(0.1)[3(0.1)-2(-1)]=0.23 \\k_{3} &=h f\left(x_{0}+\frac{1}{2} h, y_{0}+\frac{1}{2} k_{2}, z_{0}+\frac{1}{2} l_{2}\right) \\&=h f\left[0+\frac{1}{2}(0.1),-1+\frac{1}{2}(0.01), 0+\frac{1}{2}(0.23)\right] \\&=h f(0.05,-0.995,0.115)=(0.1)(0.115)=0.012 \\l_{3} &=h g\left(x_{0}+\frac{1}{2} h, y_{0}+\frac{1}{2} k_{2}, z_{0}+\frac{1}{2} l_{2}\right) \\&=h g(0.05,-0.995,0.115)=(0.1)[3(0.115)-2(-0.995)] \\&=0.234\end{aligned}\)
