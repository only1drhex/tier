Guest Question.

Consider the following circuit in steady state.

If the complex power of the current source is \(-144-j 112\), find the complex power of each component.

Admin
Solution

First, we can find the impedance seen by the source as

\(\begin{aligned}Z_{\text {in }} &=20\|j 10\|(16-j 12)=\frac{j 20}{2+j} \|(16-j 12) \\&=(4+j 8) \|(16-j 12)=\frac{(4+j 8)(16-j 12)}{20-j 4} \\&=\frac{4(1+j 2)(4-j 3)}{5-j}=20 \frac{2+j}{5-j} \Omega .\end{aligned}\)

Using this impedance and the given complex power, we can find the absolute value of the source voltage tro be

\(s_{s}=\underline{v}_{s}\underline{i}_{s}^{*} =\frac{\left|\underline{v}_{s}\right|^{2}}{-Z_{\mathrm{in}}^{*}}=-144-j 112\)

\(\longrightarrow\left|\underline{v}_{s}\right|^{2}=(-144-j 112) \times(-20) \frac{2-j}{5+j}=1600\),

leading to \(\left|\underline{v}_{s}\right|=40 \mathrm{~V}\). Then, using this voltage value, we obtain

  • \(s_{20 \Omega}=1600 / 20=80 \mathrm{~W}\),
  • \(s_{j 10 \Omega}=1600 /(-j 10)=j 160 \mathrm{VA}\),
  • \(s_{16 \Omega}=\operatorname{Re}\{1600 /(16+12 j)\}=64 \mathrm{~W}\),
  • \(s_{-j 12 \Omega}=\operatorname{Im}\{1600 /(16+12 j)\}=-j 48 \mathrm{VA}\).

We also note that

\(s_{20 \Omega}+s_{j 10 \Omega}+s_{16 \Omega}+s_{-j 12 \Omega}=144+j 112=-s_{s},\)

as expected from the conservation of energy.
