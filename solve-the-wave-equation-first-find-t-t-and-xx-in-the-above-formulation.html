Guest Question.

Solve the wave equation. First find \(T(t)\)and \(X(x)\) in the above formulation.

Admin
Solution

The solution to \(\ddot{T}+\omega^{2} T=0\) is of the form \(T(t)=A \cos \omega t+B \sin \omega t\) Note that ω will be determined from the solution of the differential equation for \(X(x)\).

The initial conditions are used to evaluate the integration constants \(A\) and \(B\) only after the complete solution is determined.

The solution to \(X^{\prime \prime}+\beta^{2} X=0 \text { is } X(x)=C \cos \beta x+\) \(D \sin \beta x .\) The boundary conditions are used to evaluate the integration constants \(C\) and \(D\),

 

\(\begin{aligned}&X(0)=C \\&X(L)=C \cos \beta L+D \sin \beta L .\end{aligned}\)

 

Solving the second equation for \(D\) results in

 

\(D=\frac{X(L)-X(0) \cos \beta L}{\sin \beta L}\)

 

For specificity, suppose \(X(0)=0 \text { and } X(L)=0\). Then \(C=0\), and it must be that \(D \neq 0 \text { so that } X(x) \neq 0\) to avoid the trivial solution. It follows that sin \(\sin \beta L=0\), from which we find

 

\(\beta_{n} L=n \pi \quad \text { or } \quad \beta_{n}=\frac{n \pi}{L}, \quad n=0,1,2, \ldots\)

 

or, using the above definition for β,

 

\(\frac{\omega_{n}}{c}=\frac{n \pi}{L} \quad \text { or } \quad \omega_{n}=\frac{n \pi c}{L}, \quad n=0,1,2, \ldots\)

 

giving the values of \(\omega_{n}\). The complete solution is the sum of all solutions,

 

\(\begin{aligned}y(x, t) &=\sum_{n=0}^{\infty} X_{n}(x) T_{n}(t) \\&=\sum_{n=0}^{\infty} D_{n} \sin \beta_{n} x\left(A_{n} \cos \omega_{n} t+B_{n} \sin \omega_{n} t\right)\end{aligned}\)

 

where the values of \(D_{n}\) are generally obtained in ways customary to the application. In vibration, we will see that amplitude \(D\) is found via a normalization procedure and coefficients \(A_{n}\) and \(B_{n}\) are found by satisfying the initial conditions.
