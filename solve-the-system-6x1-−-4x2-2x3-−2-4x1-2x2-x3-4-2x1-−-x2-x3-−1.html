Guest Question.

Solve the system

\(6x_{1} − 4x_{2} + 2x_{3} = −2\)

 

\(4x_{1}+ 2x_{2} + x_{3} = 4\)

 

\(2x_{1} − x_{2} + x_{3} = −1\)
Admin
Solution

The coefficient matrix of this system is the matrix A from Example 1. P, L, and U

have already been determined, and they can be used to solve the system as follows:

Step 1. \(c = Pb = (−1, −2, 4)^{T}\)

Step 2. \(y_{1} = −1\) \(y_{1} = −1\)
\(3y_{1} + y_{2} = −2\) \(y_{2} = −2 + 3 = 1\)
\(2y_{1} − 4y_{2} + y_{3} = 4\) \( y_{3} = 4 + 2 + 4 = 10\)

Step 3. \(2x_{1} − x_{2} + x_{3} = −1\) \( x_{1} = 1\)
\(− x_{2} − x_{3} = 1\) \( x_{2} = 1\)
\(− 5x_{3} = 10\) \(x_{3} = −2\)

The solution of the system is \(x = (1, 1, −2)^{T}\) .
