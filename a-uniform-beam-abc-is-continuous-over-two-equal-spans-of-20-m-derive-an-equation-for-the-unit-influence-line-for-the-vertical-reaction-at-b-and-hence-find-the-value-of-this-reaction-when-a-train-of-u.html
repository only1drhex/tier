Guest Question.

A uniform beam ABC is continuous over two equal spans of 20 m. Derive an equation for the unit influence line for the vertical reaction at B and hence find the value of this reaction when a train of uniform load intensity 30 kN/m covers both spans.
Ans. 750 kN.

Admin
Solution

Referring to Fig. S.20.17, the elastic curve for P causing unit displacement is the influence line for \(R_{ B }\)

Using Macauley’s method (see Section 13.2)

\(E I \frac{ d ^{2} v}{ d x^{2}}=\frac{P}{2} x-P[x-L]\) (i)

\(E I \frac{ d v}{ d x}=\frac{P}{4} x^{2}-\frac{P}{2}[x-L]^{2}+C_{1}\) (ii)

\(E I v=\frac{P}{12} x^{3}-\frac{P}{6}[x-L]^{3}+C_{1} x+C_{2}\) (iii)

When x = 0, v = 0 so that \(C_{2}=0\). When x = L, dv/dx = 0, which gives \(C_{1}=-P L^{2} / 4\).

When x = L, v = -1 so that, from Eq. (iii)

\(-E I 1=\frac{P L^{3}}{12}-\frac{P L^{3}}{4}\),

which gives

\(\frac{P}{E I}=\frac{6}{L^{3}}\)

The equation for the \(R_{ B }\) influence line is then

\(R_{ B }(\text { unit load })=1 / 2(x / L )^{3}-[(x-L) / L]^{3}-(3 / 2)(x / L)\)

For a uniformly distributed load of intensity w covering both spans

\(R_{ B }=w \int_{0}^{2 L}\left\{(1 / 2)(x / L)^{3}-[(x-L) / L]^{3}-(3 / 2)(x / L)\right\} d x\),

which gives \(R_{ B }=-5 w L / 4\) (i.e. in the negative direction of P)

Then, for w = 30 kN/m and L = 20 m

\(R_{ B }=750 kN\) (upwards)
