Guest Question.

Determine the deflected shape of the beam shown in Fig. 13.12.

Admin
Solution

In this problem an external moment \(M_{0}\) is applied to the beam at B. The support reactions are found in the normal way and are

\(R_{ A }=-\frac{M_{0}}{L} \text { (downwards) } R_{ C }=\frac{M_{0}}{L} \text { (upwards) }\)

The bending moment at any section X between B and C is then given by

\(M=R_{ A } x+M_{0}\) (i)

Equation (i) is valid only for the region BC and clearly does not contain a singularity function which would cause \(M_{0}\) to vanish for x ≤ b. We overcome this difficulty by writing

\(M=R_{ A } x+M_{0}[x-b]^{0} \quad\left(\text { Note }:[x-b]^{0}=1\right)\) (ii)

Equation (ii) has the same value as Eq. (i) but is now applicable to all sections of the beam since \([x-b]^{0}\) disappears when x ≤ b. Substituting for M from Eq. (ii) in Eq. (13.3)

\(\frac{ d ^{2} v}{ d x^{2}}=\frac{M}{E I}\) (13.3)

we obtain

\(E I \frac{ d ^{2} v}{ d x^{2}}=R_{ A } x+M_{0}[x-b]^{0}\) (iii)

Integration of Eq. (iii) yields

\(E I \frac{ d v}{ d x}=R_{ A } \frac{x^{2}}{2}+M_{0}[x-b]+C_{1}\) (iv)

and

\(E I v=R_{ A } \frac{x^{3}}{6}+\frac{M_{0}}{2}[x-b]^{2}+C_{1} x+C_{2}\) (v)

where \(C_{1}\) and \(C_{2}\) are arbitrary constants. The boundary conditions are \(v=0\) when x = 0 and x = L. From the first of these we have \(C_{2}=0\) while the second gives

\(0=-\frac{M_{0}}{L} \frac{L^{3}}{6}+\frac{M_{0}}{2}[L-b]^{2}+C_{1} L\)

from which

\(C_{1}=-\frac{M_{0}}{6 L}\left(2 L^{2}-6 L b+3 b^{2}\right)\)

The equation of the deflection curve of the beam is then

\(v=\frac{M_{0}}{6 E I L}\left\{x^{3}+3 L[x-b]^{2}-\left(2 L^{2}-6 L b+3 b^{2}\right) x\right\}\) (vi)
