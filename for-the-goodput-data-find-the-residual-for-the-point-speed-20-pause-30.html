Guest Question.

For the goodput data, find the residual for the point Speed \(=20\), Pause \(=30\).

Admin
Solution

The observed value of goodput (Table 8.1) is \(y=84.937\). The predicted value \(\hat{y}\) is found by substituting Speed \(=20\) and Pause \(=30\) into the fitted model presented in the solution to . This yields a predicted value for goodput of \(\widehat{y}=86.350\). The residual is given by \(y-\widehat{y}=84.937-86.350=-1.413\).
