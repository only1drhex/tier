Frictionless, Constant Area Compressible Flow with Heat Transfer (Rayleigh Flow)

GIVEN Air (k = 1.4 ) enters [section 112] a frictionless, constant flow cross-sectional area duct with the following properties (the same as in ):

\(T_{0}=518.67^{\circ} R\) \(T_{1}=514.55^{\circ} R\) \(p_{1}=14.3 \text { psia }\)

FIND For Rayleigh flow, determine corresponding values of fluid temperature and entropy change for various levels of downstream pressure and plot the related Rayleigh line.

Admin
Solution

To plot the Rayleigh line asked for, use Eq. 11.111

 

\(p+\frac{(\rho V)^{2} R T}{p}= constant\) (11.111)

 

\(p+\frac{(\rho V)^{2} R T}{p}=\text { constant }\) (1)

 

and Eq. 11.76

 

\(s-s_{1}=c_{p} \ln \frac{T}{T_{1}}-R \ln \frac{p}{p_{1}}\) (11.76)

 

\(s-s_{1}=c_{p} \ln \frac{T}{T_{1}}-R \ln \frac{p}{p_{1}}\) (2)

 

to construct a table of values of temperature and entropy change corresponding to different levels of pressure downstream in a Rayleigh flow.

Use the value of ideal gas constant for air from Table 1.7

\(R=1716( ft \cdot lb ) /\left(\text { slug } \cdot{ }^{\circ} R \right)\)

 

or in EE system units

 

\(R=53.3( ft \cdot lb ) /\left( lbm \cdot{ }^{\circ} R \right)\)

 

and the value of specific heat at constant pressure for air from , namely,

 

\(c_{p}=187( ft \cdot lb ) /\left( lbm \cdot{ }^{\circ} R \right)\)

 

Also, from Example 11.11, \(\rho V=16.7 lbm /\left( ft ^{2} \cdot s \right)\). For the given inlet [section (1)] conditions, we get

 

\(\begin{aligned}\frac{R T_{1}}{p_{1}} &=\frac{\left[53.3( ft \cdot lb ) /\left( lbm \cdot{ }^{\circ} R \right)\right]\left(514.55{ }^{\circ} R \right)}{14.3 psia \left(144 in ^{2} / ft ^{2}\right)} \\&=13.3 ft ^{3} / lbm\end{aligned}\)

 

Thus, from Eq. 1 we get

 

\(\begin{aligned}p+\frac{(\rho V)^{2} R T}{p} &=14.3 psia +\left[16.7 lbm /\left( ft ^{2} \cdot s \right)\right]^{2}\left(13.3 ft ^{3} / lbm \right) \\&=14.3 psia +3720 lbm /\left( ft \cdot s ^{2}\right)=\text { constant }\end{aligned}\)

 

\(\text { or, } \quad \text { since } 1 lbm /\left( ft \cdot s ^{2}\right)=\left[(1 / 32.2)\left( lb \cdot s ^{2} / ft \right)\right] /\left( ft \cdot s ^{2}\right)=(1 /\) \(\text { 32.2) } lb / ft ^{2}\)

 

\(\begin{aligned}p+\frac{(\rho V)^{2} R T}{p} &=14.3 psia +\left[(3720 / 32.2) lb / ft ^{2}\right]\left(1 ft ^{2} / 144 in ^{2}\right) \\&=15.10 \text { psia }=\text { constant }\end{aligned}\) (3)

 

With the downstream pressure of p = 13.5 psia, we can obtain the downstream temperature by using Eq. 3 with the fact that

 

\(\begin{aligned}\frac{(\rho V)^{2} R}{p} &=\frac{\left[16.7 lbm /\left( ft ^{2} \cdot s \right)\right]^{2}\left[53.3( ft \cdot lb ) /\left( lbm \cdot{ }^{\circ} R \right)\right]}{\left(144 in .{ }^{2} / ft ^{2}\right) 13.5 psia } \\&=7.65\left[ lbm /\left( ft \cdot s ^{2}\right)\right] /{ }^{\circ} R \left[1 lb /\left(32.2 lbm \cdot ft / s ^{2}\right)\right] \\&=0.238\left( lb / ft ^{2}\right) /{ }^{\circ} R \left(1 ft ^{2} / 144 in .^{2}\right) \\&=1.65 \times 10^{-3}\left( lb / in .^{2}\right) /{ }^{\circ} R\end{aligned}\)

 

Hence, from Eq. 3,

 

\(13.5 psia +\left[1.65 \times 10^{-3}\left( lb / \text { in. }^{2}\right) /{ }^{\circ} R \right] T=15.10 psia\)

 

or

 

\(T=969^{\circ} R\)

 

From Eq. 2 with the downstream pressure p = 13.5 psia and temperature T = 969 °R we get

 

\(\begin{aligned}s-s_{1}=& {\left[187( ft \cdot lb ) /\left( lbm \cdot{ }^{\circ} R \right)\right] \ln \left(\frac{969{ }^{\circ} R }{514.55{ }^{\circ} R }\right) } \\&-\left[53.3( ft \cdot lb ) /\left( lbm \cdot{ }^{\circ} R \right)\right] \ln \left(\frac{13.5 psia }{14.3 psia }\right) \\s-s_{1}=& 121( ft \cdot lb ) /\left( lbm \cdot{ }^{\circ} R \right)\end{aligned}\)
