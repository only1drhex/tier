Guest Question.

(a) Show that \(\alpha_{0}\) in Eq. (6.2) is given by

\(\alpha_{0} = \left[G(s) \frac{U_{0} \omega}{s - j\omega}\right]_{s = - j\omega} = - U_{0}G(- j\omega) \frac{1}{2j}\)

and

\(\alpha^{\ast}_{0} = \left[G(s) \frac{U_{0}\omega}{s + j \omega}\right]_{s = +j \omega} = U_{0}G(j \omega) \frac{1}{2j}\) .

(b) By assuming the output can be written as

\(y(t) = \alpha_{0}e^ {- j \omega t} + \alpha^{\ast}_{0} e ^{j \omega t}\),

derive Eqs. (6.4) – (6.6).

\(y(t) = A M \cos(\omega_{o}t + \phi)\) (6.4)

\(\phi = \tan^{- 1} \left[\frac{Im[G(j \omega_{o})]}{Re[G(j \omega_{o})]}\right] = \angle G(j \omega_{o})\). (6.6)

Admin
Solution

(a) Eq. (6.2):
\(Y(s) = \frac{\alpha_{1}}{s – p_{1}} + \frac{\alpha_{2}}{s – p_{2}} + … + \frac{\alpha_{n}}{s – p_{n}} + \frac{\alpha_{o}}{s + j \omega_{o}} + \frac{\alpha^{\ast}_{o}}{s – j \omega_{o}}\)

Multiplying this by \((s + j \omega)\) :

\(Y(s) (s + j \omega) = \frac{\alpha_{1}}{s + a_{1}}(s + j \omega) + … + \frac{\alpha_{n}}{s + a_{n}}(s +j \omega) + \alpha_{o} + \frac{\alpha^{\ast}_{o}}{s – j \omega} (s + j \omega)\)

 

\(\Rightarrow \alpha_{o} = Y(s)(s + j \omega) – \frac{\alpha_{1}}{s + a_{1}}(s + j \omega) – … – \frac{\alpha_{n}}{s + a_{n}}(s +j \omega) – \frac{\alpha^{\ast}_{o}}{s – j \omega} (s + j \omega)\)

 

\(\alpha_{o} = \alpha_{o}|_{s = – j_{\omega}} = \left[ Y(s)(s + j \omega) – \frac{\alpha_{1}}{s + a_{1}}(s + j \omega) – … – \frac{\alpha^{\ast}_{o}}{s – j \omega} (s + j \omega)\right]_{s = – j_{\omega}} = Y(s)(s + j \omega)|_{s = – j_{\omega}} = G(s) \frac{U_{o} \omega}{s^{2} + \omega^{2}} (s + j \omega)|_{s = – j_{\omega}}\) \( = G(s) \frac{U_{o} \omega}{s – j \omega}|_{s = – j _{\omega}} = – U_{o} G(- j \omega) \frac{1}{2 j}\)

Similarly, multiplying Eq. (6.2) by \((s – j \omega)\) :

\(Y(s)(s – j \omega) = \frac{\alpha_{1}}{s + a_{1}}(s – j \omega) + … + \frac{\alpha_{n}}{s + a_{n}}(s – j \omega) + \frac{\alpha_{o}}{s + j \omega}(s – j \omega) + \alpha^{\ast}_{o}\)

 

\(\alpha^{\ast}_{o} = \alpha^{\ast}_{o}|_{s = j \omega} = Y(s)(s – j \omega)|_{s = j_{\omega}} = G(s) \frac{U_{o} \omega}{s^{2} + \omega^{2}}(s – j \omega)|_{s = j_{\omega}} = G(s) \frac{U_{o} \omega}{s + j \omega} |_{s = j_{\omega}} = U_{o} G(j \omega)\frac{1}{2 j}\)

(b)

\(y(t) = \alpha_{o} e^{- j \omega t} + \alpha^{\ast}_{o} e^{j \omega t}\)

 

\(y(t) = – U_{o} G(- j \omega) \frac{1}{2 j} e^{- j \omega t} + U_{o} G(j \omega) \frac{1}{2 j} e^{j \omega t} = U_{o} \left[\frac{G(j \omega) e^{j \omega t} – G(- j \omega) e^{- j \omega t}}{2 j}\right]\)

 

\(\left|G(j \omega)\right| = \left\{Re[G(j \omega)]^{2} + Im[G(j \omega)]^{2}\right\}^{\frac{1}{2}} = A\)

 

\(\angle G(j \omega) = \tan^{- 1} \frac{Im[G(j \omega)]}{Re [G(j \omega)]} = \phi\)

 

\(\left|G(- j \omega)\right| = \left\{Re [G(- j \omega)]^{2} + Im [G(- j \omega)]^{2}\right\}^{\frac{1}{2}} = \left|G(j \omega)\right| = \left\{Re [G(j \omega)]^{2} + Im [G(j \omega)]^{2}\right\}^{\frac{1}{2}} = A\)

 

\(\angle G(- j \omega) = \tan^{- 1} \frac{Im [G(- j \omega)]}{Re [G(- j \omega)]} = \tan^{- 1} \frac{- Im [G(j \omega)]}{Re [G(j \omega)]} = – \phi\)

 

\(\Rightarrow G(j \omega) = A e^{j \phi} , G(- j \omega) = A e^{- j \phi}\)

Thus ,

\(y(t) = U_{o}\left[\frac{A e^{j \phi} e ^{j \omega t} – Ae^{- j \phi} e^{- j \omega t}}{2 j}\right] = U_{o} A \left[\frac{e^{j(\omega t + \phi)} – e^{- j(\omega t + \phi)}}{2 j}\right] \)

 

\(y(t) = U_{o} A \sin(\omega t + \phi)\)

where

\(A = \left|G(j \omega)\right| , \phi = \tan^{- 1} \frac{Im [G(j \omega)]}{Re[G(j \omega)]} = \angle G(j \omega)\)
