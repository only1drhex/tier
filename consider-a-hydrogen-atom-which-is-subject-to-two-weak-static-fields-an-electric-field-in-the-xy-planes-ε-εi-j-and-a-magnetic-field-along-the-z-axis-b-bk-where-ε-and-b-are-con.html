Guest Question.

Consider a hydrogen atom which is subject to two weak static fields: an electric field in the xy planes \(\vec {\varepsilon } =\varepsilon (\vec{i} +\vec{j} )\) and a magnetic field along the z-axis \(\vec{B} =B\vec{K} \), where \(\varepsilon\) and B are constant. Neglecting the spin–orbit interaction, calculate the energy levels of the n = 2 states to first-order perturbation.

Admin
Solution

In the absence of any external field, and neglecting spin–orbit interactions, the energy of the n = 2 state is fourfold degenerate: four different states \(|nlm 〉=|200 〉,|211 〉,|210 〉\), and \( |21-1 〉\) correspond to the same energy \(E_{2} =-R/4 \), where \(R=m_{e} e^{4} /(2\hbar ^{2} )=13.6 eV\) is the Rydberg constant.

When the atom is placed in an external electric field \(\vec {\varepsilon } =\varepsilon (\vec{i} +\vec{j} )\), the energy of interaction between the electron’s dipole moment \((\vec{d}=-e \vec{r} )\) and \(\varepsilon\) is given by \( -\vec {d}.\vec{\varepsilon }=e\varepsilon (x+y)=e\varepsilon r\sin \theta (\cos \phi +\sin \phi )\). On the other hand, when subjecting the atom to an external magnetic field \(\vec{B} =B\vec{K}\), the linear momentum of the electron becomes \(\vec{p} \longrightarrow (\vec {p} -e\vec{A} /c)\), where \(\vec{A}\) is the vector potential corresponding to \(\vec{B}\). So when subjecting a hydrogen atom to both \(\varepsilon\) and \(\vec{B}\), its Hamiltonian is given by

\(\hat{H} =\frac{1}{2\mu } \left(\vec{p} -\frac{e}{c} \vec{A} \right) ^{2} -\frac{e^{2} }{r} +e\varepsilon r\sin \theta (\cos \phi + \sin \phi )\)

 

\(=\frac{\vec{p}^{2} }{2\mu } -\frac{e^{2} }{r} -\frac{e}{e\mu c} \vec{B} .\vec{L} +\frac{e}{2\mu c} A^{2} +e\varepsilon r\sin \theta (\cos \phi +\sin \phi )\). (9.297)

Since the magnetic field is weak, we can ignore the term \(e A^{2} /(2\mu c)\); hence we can write \(\hat{H}\) as \( \hat{H} =\hat{H}_{0} +\hat{H}_{p} \), where \(\hat {H}_{0}\) is the Hamiltonian of an unperturbed hydrogen atom, while \(\hat{H}_{p} \) can be treated as a perturbation:

\(\hat{H}_{0} =\frac{\vec{p}^{2} }{2\mu }-\frac{e^{2} }{r}, \hat{H}_{p} =-\frac{eB}{2\mu c} \hat{L}_{z} +e\varepsilon r \sin \theta (\cos \phi +\sin \phi )\). (9.298)

To calculate the energy levels of the n = 2 state, we need to use degenerate perturbation theory, since the n = 2 state is fourfold degenerate; for this, we need to diagonalize the matrix

\(\left(\begin{matrix} 〈1|\hat{H}_{p} |1 〉 & 〈1|\hat{H}_{p} |2 〉 & 〈1|\hat{H}_{p} |3 〉 & 〈1|\hat{H}_{p} |4 〉 \\ 〈2|\hat{H}_{p} |1 〉 & 〈2|\hat{H}_{p} |2 〉 & 〈2|\hat{H}_{p} |3 〉 & 〈2|\hat{H}_{p} |4 〉\\ 〈3|\hat{H}_{p} |1 〉 & 〈3|\hat{H}_{p} |2 〉 & 〈3|\hat{H}_{p} |3 〉 & 〈3|\hat{H}_{p} |4 〉 \\ 〈4|\hat{H}_{p} |1 〉 & 〈4|\hat{H}_{p} |2 〉 & 〈4|\hat{H}_{p} |3 〉 & 〈4|\hat{H}_{p} |4 〉\end{matrix} \right) \), (9.299)

where \(|1 〉=|200 〉, |2 〉=|211 〉,|3 〉=|210 〉\), and \(|4 〉=|21-1 〉\). We therefore need to calculate the term

\(〈2l^{\prime } m^{\prime } |\hat{H}_{p}|2lm 〉=-\frac{eB}{2\mu c} m\hbar \delta _{l^{\prime },l}\delta _{m^{\prime },m} +e\varepsilon 〈2l^{\prime } m^{\prime } |r\sin \theta (\cos \phi +\sin \phi )|2lm 〉\). (9.300)

Since \(x=r\sin \theta \cos \phi\) and \(y=r\sin \theta \sin \phi\) are both odd, the only terms that survive among \(〈2l^{\prime } m^{\prime } |x|2lm 〉\) and \(〈2 l^{\prime } m^{\prime } |y|2lm 〉\) are \(〈200|x|21\pm 1 〉,〈200|y|21\pm 1 〉\), and their complex conjugates. That is, x and y can couple only states of different parities \((l^{\prime }-l=\pm 1)\) and whose azimuthal quantum numbers satisfy this condition: \(m^{\prime }-m=\pm 1\). So we need to calculate only

\(〈200|x|21\pm 1 〉=∫^{+\infty }_{0} R^{*}_{20} (r)R_{21} (r)r^{3} dr ∫ Y^{*}_{00}(\Omega )\sin \theta \cos \phi Y_{1\pm 1} (\Omega ) d\Omega \), (9.301)

 

\(〈200|y|21\pm 1 〉=∫^{+\infty }_{0} R^{*}_{20} (r)R_{21} (r)r^{3} dr ∫ Y^{*}_{00}(\Omega )\sin \theta \sin \phi Y_{1\pm 1} (\Omega ) d\Omega \), (9.302)

where

\(∫^{+\infty }_{0} R^{*}_{20} (r)R_{21} (r)r^{3} dr=-3\sqrt{3} a_{0} \); (9.303)

\(a_{0} \) is the Bohr radius, \(a_{0}=\hbar ^{2} /(m_{e} e^{2})\). Using the relations

\(\sin \theta \cos \phi=\sqrt{\frac{2\pi }{3} } [Y_{1- 1} (\Omega )-Y_{1 1} (\Omega )], \sin \theta \sin \phi =i\sqrt{\frac{2\pi }{3} }[Y_{1- 1} (\Omega )-Y_{1 1} (\Omega )]\), (9.304)

along with

\(∫ Y^{*}_{l^{\prime } m^{\prime }}(\Omega )Y_{lm} (\Omega ) d\Omega =\delta _{l^{\prime },l}\delta _{m^{\prime },m}\), (9.305)

we obtain

\(∫ Y^{*}_{00}(\Omega )\sin \theta \cos \phi Y_{1 1}(\Omega ) d\Omega=\frac{1}{\sqrt{4\pi } } ∫ \sin \theta \cos \phi Y_{1 1}(\Omega ) d\Omega=\frac{1}{\sqrt{6} } ∫ Y_{1- 1}(\Omega )Y_{1 1}(\Omega )d\Omega \)

 

\(=-\frac{1}{\sqrt{6} } \), (9.306)

 

\(∫ Y^{*}_{00}(\Omega )\sin \theta \sin \phi Y_{1 1}(\Omega ) d\Omega=\frac{i}{\sqrt{6} } ∫ Y_{1- 1}(\Omega )Y_{1 1}(\Omega )d\Omega =-\frac{i}{\sqrt{6} } \). (9.307)

Similarly, we have

\(∫ Y^{*}_{00}(\Omega )\sin \theta \cos \phi Y_{1-1}(\Omega ) d\Omega=\frac{1}{\sqrt{6} }\). (9.308)

 

\(∫ Y^{*}_{00}(\Omega )\sin \theta \sin \phi Y_{1-1}(\Omega ) d\Omega=-\frac{i}{\sqrt{6} }\). (9.309)

Now, substituting (9.303), (9.306), and (9.308) into (9.301), we end up with

\(〈200|x|21\pm 1 〉=\pm \frac{3}{\sqrt{2} } a_{0}, 〈200|y|21\pm 1 〉=\frac{3i}{\sqrt{2} } a_{0}\); (9.310)

hence

\(〈21\pm 1 |x|200 〉=\pm \frac{3}{\sqrt{2} } a_{0}, 〈21\pm 1|y|200 〉=-\frac{3i}{\sqrt{2} } a_{0}\). (9.311)

The matrix (9.299) thus becomes

\(\left(\begin{matrix} 0 & \alpha +i\alpha & 0 & -\alpha +i\alpha \\ \alpha -i\alpha & -\beta & 0 & 0 \\ 0 & 0 & 0 & 0 \\ -\alpha -i\alpha & 0 & 0 & \beta \end{matrix} \right) \), (9.312)

where α and β stand for \(\alpha =3e\varepsilon a_{0}/\sqrt{2} \) and \(\beta =e\hbar B/(2\mu c)\).

The diagonalization of (9.312) yields the following eigenvalues:

\(\lambda _{1} =-\sqrt{\frac{e^{2} \hbar ^{2} B^{2} }{4\mu ^{2} c^{2} } +18e^{2} \varepsilon ^{2} a^{2}_{0} } , \lambda _{2} =\lambda _{3} =0, \lambda _{4} =\sqrt{\frac{e^{2} \hbar ^{2} B^{2} }{4\mu ^{2} c^{2} } +18e^{2} \varepsilon ^{2} a^{2}_{0} }\). (9.313)

Finally, the energy levels of the n = 2 states are given to first-order approximation by

\(E^{(1)} _{2_{1} } =-\frac{R}{4} -\sqrt{\frac{e^{2} \hbar ^{2} B^{2} }{4\mu ^{2} c^{2} } +18e^{2} \varepsilon ^{2} a^{2}_{0} }, E^{(1)} _{2_{2} } =-\frac{R}{4}\), (9.314)

 

\( E^{(1)} _{2_{3} } =-\frac{R}{4}, E^{(1)} _{2_{4} } =-\frac {R}{4} +\sqrt{\frac{e^{2} \hbar ^{2} B^{2} }{4\mu ^{2} c^{2} } +18 e^{2} \varepsilon ^{2} a^{2}_{0} }\). (9.315)

So the external electric and magnetic fields have lifted the degeneracy of the n = 2 level only partially.
