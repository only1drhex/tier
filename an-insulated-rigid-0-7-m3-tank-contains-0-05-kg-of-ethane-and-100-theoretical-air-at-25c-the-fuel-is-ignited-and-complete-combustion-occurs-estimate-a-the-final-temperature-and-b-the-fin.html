Guest Question.

An insulated, rigid 0.7-m³ tank contains 0.05 kg of ethane and 100% theoretical air at 25°C. The fuel is ignited and complete combustion occurs. Estimate (a) the final temperature and (b) the final pressure.

Admin
Solution
\( \text { With } 100 \% \text { theoretical air, } C _{2} H _{6}+3.5\left( O _{2}+3.76 N _{2}\right) \rightarrow 2 CO _{2}+3 H _{2} O +13.16 N _{2} \text {. } \)

(a) The first law, with Q = W = 0, is written for this constant-volume process using (13.11):

\( Q=\sum_{\text {prod }} N_{i}\left(\bar{h}_{f}^{\circ}+\bar{h}-\bar{h}^{\circ}-R_{u} T\right)_{i}-\sum_{\text {react }} N_{i}\left(\bar{h}_{f}^{\circ}+\bar{h}-\bar{h}^{\circ}-R_{u} T\right)_{i} \) (13.11).

\( \sum_{\text {react }} N_{i}\left(\bar{h}_{f}^{\circ}+\bar{h}-\bar{h}^{\circ}-R_{u} T\right)_{i}-\sum_{\text {prod }} N_{i}\left(\bar{h}_{f}^{\circ}+\bar{h}-\bar{h}^{\circ}-R_{u} T\right)_{i} \).

The reactants are at 25°C (the initial pressure is unimportant if not extremely large) and the products are at \( T_{P} \); therefore,

\( \text { L.H.S. }=(1)[-84680-(8.314)(298)]+(3.5)[(-8.314)(298)]+(13.16)[(-8.314)(298)] \).

\( \text { R.H.S. }=(2)\left[-393520+\bar{h}_{ CO _{2}}-9360-8.314 T_{P}\right] \)

\( +(3)\left[\left(-241820+\bar{h}_{ H _{2} O }-9900-8.314 T_{P}\right)+(13.16)\left(\bar{h}_{ N _{2}}-8670-8.314 T_{P}\right)\right] \).

or

\( 1579000=2 \bar{h}_{ CO _{2}}+3 \bar{h}_{ H _{2} O }+13.16 \bar{h}_{ N _{2}}-151 T_{P} \).

\( \text { We solve for } T_{P} \text { by trial and error: } \)

\( T_{P}=2600 K : 1579000 \stackrel{?}{=}(2)(137400)+(3)(114300)+(13.16)(86850)-(151)(2600)=1365000 \).

\( T_{P}=2800 K : 1579000 \stackrel{?}{=}(2)(149810)+(3)(125200)+(13.16)(94010)-(151)(2800)=1490000 \).

\( T_{P}=3000 K : 1579000 \stackrel{?}{=}(2)(162230)+(3)(136260)+(13.16)(101410)-(151)(3000)=1615000 \).

Interpolation provides a temperature between 2800 K and 3000 K: \( T_{P}=2942 K \).

\( \text { (b) We have } N_{\text {fuel }}=0.05 / 30=0.001667 kmol \text {; therefore, } N_{\text {prod }}=(18.16)(0.001667)=0.03027 kmol \text {. The pressure } \) in the products is then

\( P_{\text {prod }}=\frac{N_{\text {prod }} R_{u} T_{\text {prod }}}{V}=\frac{(0.03027)(8.314)(2942)}{0.7}=1058 kPa \).
