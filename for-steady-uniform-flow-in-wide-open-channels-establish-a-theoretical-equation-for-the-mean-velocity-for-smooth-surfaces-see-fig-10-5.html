Guest Question.

For steady uniform flow in wide, open channels, establish a theoretical equation for the mean velocity for smooth surfaces. See Fig. 10-5.

Admin
Solution

For turbulent flow in general, shear stress τ can be expressed as

\(\tau=\rho l^{2}(d v / d z)^{2}\).

where \( l \) is the mixing length and a function of z. (See Chapter 8.)

Also, from , expression (A), \(\tau_{o}=\gamma R S=\gamma h S\) since hydraulic radius R for wide channels approximates the depth.

\(\text { In the boundary layer, since } y \text { is very small, } z \cong h \text { and } \tau \cong \tau_{o} \). Hence we may equate the values of \( \tau_{o}, or\)

\(\rho l^{2}(d v / d z)^{2}=\gamma z S \quad \text { or } \quad d v / d z=\pm \sqrt{g z S / l^{2}}\).

\(\text { To integrate this expression, try a value of } l=k(h-z)(z / h)^{1 / 2} \text {. Then }\)

\(-\frac{d v}{d z}=\sqrt{g S}\left[\frac{z^{1 / 2}}{k(h-z)(z / h)^{1 / 2}}\right]=\frac{\sqrt{g S h}}{k}\left(\frac{1}{h-z}\right)\).

\(\text { Let } y=(h-z) \text { and } d y=-d z \text {; then }\)

\(+y\left(\frac{d v}{d y}\right)=\frac{\sqrt{g S h}}{k} \quad \text { and } \quad d v=\frac{\sqrt{g S h}}{k}\left(\frac{d y}{y}\right)\).

\(\text { Since } \tau_{o} / \rho=\gamma h S / \rho=g S h\).

\(d v=\frac{1}{k} \sqrt{\tau_{o} / \rho}\left(\frac{d y}{y}\right) \quad \text { or } \quad v=\frac{1}{k} \sqrt{\tau_{o} / \rho} \ln y+C\).

\(\text { For } y=y_{o}, v \cong 0 \text {, then } C=(-1 / k) \sqrt{\tau_{o} / \rho} \ln y_{o} \text { and }\)

\(v=\frac{1}{k} \sqrt{\tau_{o} / \rho} \ln \left(y / y_{o}\right)\) (A).

Note: Neglecting the log curve to the left of \(y_{o}\), while producing an approximation, gives satisfactory results well within the limits of expected accuracy because \(y_{o}\) is very small. See for the value of \(y_{o}\).

\(\text { In this expression }(A), k \cong 0.40\) and is called the von Karman constant. Inasmuch as the term \(\sqrt{\tau_{o} / \rho}\) has the dimensions of ft/sec or m/s, this term is called the shear velocity and is designated by \(v_{*}\). Thus

\(v=2.5 v_{*} \ln \left(y / y_{o}\right)\) (B).

\(\text { From } Q=A V=(h \times 1) V=\int v(d y \times 1) \text {, we obtain the value of mean velocity } V \text {. Thus }\)

\(V=\frac{\int v(d y \times 1)}{(h \times 1)}=\frac{2.5 v_{*}}{h} \int_{0}^{h}\left(\ln y-\ln y_{o}\right) d y\).

Using L’Hospital’s rule from calculus, the mean velocity for smooth surfaces where a boundary layer exists can be evaluated as

\(V=2.5 v_{*}\left[\ln h-\ln y_{o}-1\right]\) (C).

It will be shown in that \(y_{0} \text { equals } v / 9 v_{*} \). Therefore equations (B) and (C) may be written

\(v=2.5 v_{*} \ln \left(9 v_{*} y / v\right)\) (D).

and

\(V=2.5 v_{*}\left[\ln h-\ln \left(v / 9 v_{*}\right)-1\right]\). (E).

Frequently the average velocity in an open channel is taken as the velocity observed at a point 0.6 of the depth (measured from the surface). If we accept this value of \(\bar{y}\), then we may write, from (B) above, average velocity V as

\(V=2.5 v_{*} \ln \left(0.4 h / y_{o}\right)\).

From , \(y_{o}=\delta / 103\). Then for wide channels this average velocity becomes, since hydraulic radius R = h,

\(V=2.5 v_{*} \ln (41.2 R / \delta)\) (F).
