Guest Question.

A balanced delta-connected load contains a 10-Ω resistor in series with a 20-mH inductor in each phase. The voltage source is an abc-sequence three-phase 60-Hz, balanced wye with a voltage \(V_{an} = 120\underline{/30^\circ}\) V rms. We wish to determine all Δ currents and line currents.

Admin
Solution

The impedance per phase in the delta load is \(Z_{Δ}\) = 10 + j7.54 Ω. The line voltage \(V_{ab} =120\sqrt{3}\underline{/60^\circ}\) V rms. Since there is no line impedance, \(V_{AB} = V_{ab} = 120\sqrt{3}\underline{/60^\circ}\) V rms.
Hence,

\(I_{AB} = \frac{120\sqrt{3}\underline{/60^\circ}}{10 + j7.5}\)
= 16.60\(\underline{/+22.98^\circ}\) A rms

If \(Z_{Δ}\) = 10 + j7.54 Ω, then

\(Z_{Y} = \frac{1}{3}Z_{Δ}\)
= 3.33 + j2.51 Ω

Then the line current

\(I_{aA} = \frac{V_{an}}{Z_{Y}}= \frac{120\underline{/30^\circ}}{3.33 + j2.51}\)
= \(\frac{120\underline{/30^\circ}}{4.17\underline{/37.01^\circ}}\)
= 28.78\(\underline{/-7.01^\circ}\) A rms

Therefore, the remaining phase and line currents are

\(I_{BC} = 16.60\underline{/-97.02^\circ}\) A rms \(I_{bB} = 28.78\underline{/-127.01^\circ}\) A rms
\(I_{CA} = 16.60\underline{/+142.98^\circ}\) A rms \(I_{cC} = 28.78\underline{/+112.99^\circ}\) A rms
