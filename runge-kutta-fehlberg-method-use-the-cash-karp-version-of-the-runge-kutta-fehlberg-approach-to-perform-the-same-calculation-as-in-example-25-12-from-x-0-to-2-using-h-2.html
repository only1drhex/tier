Runge-Kutta Fehlberg Method

Use the Cash-Karp version of the Runge-Kutta Fehlberg approach to perform the same calculation as in from x = 0 to 2 using h = 2.

Admin
Solution

The calculation of the k’s can be summarized in the following table:

x y f(x, y)
\(k_{1}\) 0 2 3
\(k_{2}\) 0.4 3.2 3.908511
\(k_{3}\) 0.6 4.20883 4.359883
\(k_{4}\) 1.2 7.228398 6.832587
\(k_{5}\) 2 15.42765 12.09831
\(k_{6}\) 1.8 12.17686 10.13237

These can then be used to compute the fourth-order prediction

\(y_{1}=2+\left(\frac{37}{378} 3+\frac{250}{621} 4.359883+\frac{125}{594} 6.832587+\frac{512}{1771} 10.13237\right) 2=14.83192\)

along with a fi fth-order formula:

\(y_{1}=2+\left(\frac{2825}{27,648} 3+\frac{18,575}{48,384} 4.359883+\frac{13,525}{55,296} 6.832587\right.\left.+\frac{227}{14.336} 12.09831+\frac{1}{4} 10.13237\right) 2=14.83677\)
The error estimate is obtained by subtracting these two equations to give

\(E_{a}=14.83677-14.83192=0.004842\)
