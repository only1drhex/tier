\( \bar{x} = \frac{\sum{x} }{n} = \frac{100.06 + 100.01 + 100.02 + 99.99 + 100.01}{5}\)

 

\(= \frac{500.09}{5} = 100.02 \)

 

\( R = X_{h} — x_{l} = 100.06 — 99.99 = 0.07\)

The next step is to calculate the process mean and the mean range and draw them on the appropriate chart:

\(\overset{-}{\bar{ x} } = \frac{\sum{\bar{x}} }{N} = \frac{2499.85}{25} = 99.99\)

 

\(\bar{R} = \frac{\sum{R} }{N} = \frac{1.6}{25} = 0.064\)

It is also useful to calculate the standard deviation:

\(s = \frac{\bar{R}}{d_{2}} = \frac{0.064}{2.326} = 0.0275 \)

Calculate the control limits and plot them:

\(UCL_{\bar{x} } = \overset{-}{\bar{ x} } + A_{2} \bar{R}\) , where \(A_{2} = 0.577\) for n = 5

\(UCL_{\bar{x} } = 99.99 + (0.577 × 0.064)\) \( LCL_{\bar{x} } = \overset{-}{\bar{ x} } – A_{2} \bar{R}\)

\(UCL_{\bar{x} } = 99.99 + 0.037\) \( LCL_{\bar{x} } = 99.99 – 0.037 \)

\(UCL_{\bar{x} } = 100.03\) \( LCL_{\bar{x} } = 99.95 \)

\( UCL_{R} = D_{4} \bar{R}\) \( LCL_{R} = D_{3} \bar{R}\)

\( UCL_{R} = 2.114 × 0.064\) \( LCL_{R} =0 × 0.064 \)

\( UCL_{R} = 0.135 \) \( LCL_{R} = 0\)
