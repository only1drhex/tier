Guest Question.

The displacements at three points, 1, 2, and 3 on a beam were measured when loads were applied separately at each point; the flexural rigidity of the beam is EI. Initially, when a load \(W_{1}=400 N\) was applied at the point 1 the displacements were \(\Delta_{1}=0.75 mm , \Delta_{2}=-0.25 mm\), and \(\Delta_{3}=-1.25 mm\). Find

(i) \(\Delta_{1}\) given that \(\Delta_{2}=1 mm , \Delta_{3}=0.875 mm\) for \(W_{2}=300 N\).

(ii) \(\Delta_{1}\) and \(\Delta_{2}\) given that \(\Delta_{3}=2.5 mm\) for \(W_{3}=500 N\).

(iii) \(\Delta_{1}, \Delta_{2}\), and \(\Delta_{3}\) when \(W_{1}=W_{2}=W_{3}=600 N\) are applied simultaneously.
Ans. (i) -0.1875 mm, (ii) -1.5625 mm, 1.4583 mm, (iii) -1.125 mm, 3.375 mm, 2.875 mm.

Admin
Solution

From Section 15.4

\(\Delta_{1}\)=\(a_{11} W_{1}+a_{12} W_{2}+a_{13} W_{3}\)

\(\Delta_{2}=a_{21} W_{1}+a_{22} W_{2}+a_{23} W_{3}\) (i)

\(\Delta_{3}=a_{31} W_{1}+a_{32} W_{2}+a_{33} W_{3}\)

For the initial condition Eqs (i) reduce to

\(0.75=a_{11} 400, \quad-0.25=a_{21} 400, \quad-1.25=a_{31} 400\)

so that

\(a_{11}=0.75 / 400, \quad a_{21}=-0.25 / 400=a_{12}, \quad a_{31}=-1.25 / 400=a_{13}\)

Case (i)

\(\Delta_{1}=a_{12} 300=(-0.25 / 400) \times 300=-0.1875\)

Also from Eqs (i)

\(\Delta_{2}=1=a_{22} 300 \text { so that } a_{22}=1 / 300\)

and

\(\Delta_{3}=0.875=a_{32} 300 \text { so that } a_{32}=0.875 / 300\)

Case (ii)
From Eqs (i)

\(\Delta_{1}=(-1.25 / 400) \times 500=-1.5625\)

 

\(\Delta_{2}=(0.875 / 300) \times 500=1.4583\)

and

\(2.5=a_{33} 500 \text { so that } a_{33}=2.5 / 500\)

Case (iii)

\(\Delta_{1}=(0.75 / 400) \times 600-(0.25 / 400) \times 600-(1.25 / 400) \times 600=-1.125 mm\)

 

\(\Delta_{2}=(-0.25 / 400) \times 600+(1 / 300) \times 600+(0.875 / 300) \times 600=3.375 mm\)

 

\(\Delta_{3}=(-1.25 / 400) \times 600+(0.875 / 300) \times 600+(2.5 / 500) \times 600=2.875 mm\)
