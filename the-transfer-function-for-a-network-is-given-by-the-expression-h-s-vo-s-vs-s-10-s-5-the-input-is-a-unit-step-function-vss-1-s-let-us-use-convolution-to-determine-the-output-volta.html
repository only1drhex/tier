Guest Question.

The transfer function for a network is given by the expression

\(H (s)=\frac{ V _{o}(s)}{ V _{s}(s)}=\frac{10}{s+5}\)

The input is a unit step function \(V _{S}(s)=\frac{1}{s}\). Let us use convolution to determine the output voltage \(v_{o}(t)\).

Admin
Solution

Since \(H (s)=\frac{10}{(s+5)}, h(t)=10 e^{-5 t}\) and therefore
\(v_{o}(t)=\int_{0}^{t} 10 u(\lambda) e^{-5(t-\lambda)} d \lambda\)
\(=10 e^{-5 t} \int_{0}^{t} e^{5 \lambda} d \lambda\)
\(=\frac{10 e^{-5 t}}{5}\left[e^{5 t}-1\right]\)
\(=2\left[1-e^{-5 t}\right] u(t) V\)
For comparison, let us determine \(v_{o}(t) \text { from } H (s) \text { and } V _{s}(s)\) using the partial fraction expansion method. \(V _{o}(s)\) can be written as
\(V _{o}(s)= H (s) V _{s}(s)\)
\(=\frac{10}{s(s+5)}=\frac{K_{0}}{s}+\frac{K_{1}}{s+5}\)
Evaluating the constants, we obtain \(K_{0}=2 \text { and } K_{1}=-2\). Therefore,
\(V _{o}(s)=\frac{2}{s}-\frac{2}{s+5}\)
and hence
\(v_{o}(t)=2\left[1-e^{-5 t}\right] u(t) V\)
