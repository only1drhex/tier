Guest Question.

Design a shell-and-tube exchanger for the following duty.

20,000 kg/h of kerosene (42º API) leaves the base of a kerosene side-stripping column at 200ºC and is to be cooled to 90ºC by exchange with 70,000 kg/h light crude oil (34º API) coming from storage at 40ºC. The kerosene enters the exchanger at a pressure of 5 bar and the crude oil at 6.5 bar. A pressure drop of 0.8 bar is permissible on both streams. Allowance should be made for fouling by including a fouling factor of 0.0003 \(\left( W / m ^{2}{ }^{\circ} C \right)^{-1}\) on the crude stream and 0.0002 \(\left( W / m ^{2}{ }^{\circ} C \right)^{-1}\) on the kerosene stream.

Admin
Solution

The solution to this example illustrates the iterative nature of heat exchanger design calculations. An algorithm for the design of shell-and-tube exchangers is shown in Figure A (see p. 684). The procedure set out in this figure will be followed in the solution.

Step 1: Specification

The specification is given in the problem statement.

20,000 kg/h of kerosene (42º API) at 200ºC cooled to 90ºC, by exchange with 70,000 kg/h light crude oil (34º API) at 40ºC.

The kerosene pressure 5 bar, the crude oil pressure 6.5 bar.

Permissible pressure drop of 0.8 bar on both streams.

 

Fouling factors: crude stream 0.00035 \(\left( W / m ^{2}{ }^{\circ} C \right)^{-1}\), kerosene stream 0.0002 \(\left( W / m ^{2}{ }^{\circ} C \right)^{-1}\).

 

To complete the specification, the duty (heat transfer rate) and the outlet temperature of the crude oil needed to be calculated.

 

\(\text { The mean temperature of the kerosene }=(200+90) / 2=145^{\circ} C \text {. }\)

 

\(\text { At this temperature the specific heat capacity of } 42^{\circ} \text { API kerosene is } 2.47 kJ / kg { }^{\circ} C\)

 

(physical properties from D. Q. Kern, Process Heat Transfer, McGraw-Hill).

 

\(\text { Duty }=\frac{20,000}{3600} \times 2.47(200-90)=1509.4 kW\)

 

As a first trial take the mean temperature of the crude oil as equal to the inlet temperature, \(40^{\circ} C\); specific heat capacity at this temperature \(=2.01 kJ / kg ^{\circ} C\).

An energy balance gives:

 

\(\frac{7000}{3600} \times 2.01\left(t_{2}-40\right)=1509.4\)

 

\(t_{2}=78.6^{\circ} C \text { and the stream mean temperature }=(40+78.6) / 2=59.3^{\circ} C\)

 

The specific heat at this temperature is \(2.05 kJ / kg ^{\circ} C\). A second trial calculation using this value gives \(t_{2}=77.9^{\circ} C\) and a new mean temperature of \(58.9^{\circ} C\). There is no significant change in the specific heat at this mean temperature from the value used, so take the crude stream outlet temperature to be \(77.9^{\circ} C \text {, say } 78^{\circ} C\).

Step 4: Exchanger type and dimensions

An even number of tube passes is usually the preferred arrangement, as this positions the inlet and outlet nozzles at the same end of the exchanger, which simplifies the pipework. Start with one shell pass and 2 tube passes.

 

\(\Delta T_{l m}=\frac{(200-78)-(90-40)}{\ln \frac{(200-78)}{(90-40)}}=80.7^{\circ} C\) (12.4)

 

\(R=\frac{(200-90)}{(78-40)}=2.9\) (12.6)

 

\(S=\frac{(78-40)}{(200-40)}=0.24\) (12.7)

 

\(\text { From Figure } 12.19, F_{t}=0.88 \text {, which is acceptable. }\)

 

So,

 

\(\Delta T_{m}=0.88 \times 80.7=71.0^{\circ} C\)

 

Step 5: Heat transfer area

 

\(A_{o}=\frac{1509.4 \times 10^{3}}{300 \times 71.0}=70.86 m ^{2}\)

 

Step 6: Layout and tube size

Using a split-ring floating head exchanger for efficiency and ease of cleaning. Neither fluid is corrosive, and the operating pressure is not high, so a plain carbon steel can be used for the shell and tubes.

The crude is dirtier than the kerosene, so put the crude through the tubes and the kerosene in the shell.

Use 19.05 mm (3/4 inch) outside diameter, 14.83 mm inside diameter, 5 m Long tubes (a popular size) on a triangular 23.81 mm pitch (pitch/dia. = 1.25).

Step 7: Number of tubes

Area of one tube (neglecting thickness of tube sheets)

 

\(=\pi \times 19.05 \times 10^{-3} \times 5=0.2992 m ^{2}\)

 

\(\text { Number of tubes }=70.89 / 0.2992=237 \text {, say } 240\)

 

So, for 2 passes, tubes per pass = 120

Check the tube-side velocity at this stage to see if it looks reasonable.

 

\(\text { Tube cross-sectional area }=\frac{\pi}{4}\left(14.83 \times 10^{-3}\right)^{2}=0.0001727 m ^{2}\)

 

\(\text { So area per pass }=120 \times 0.0001727=0.02073 m ^{2}\)

 

\(\text { Volumetric flow }=\frac{70,000}{3600} \times \frac{1}{820}=0.0237 m ^{3} / s\)

 

\(\text { Tube-side velocity, } u_{t}=\frac{0.0237}{0.02073}=1.14 m / s\)

 

The velocity is satisfactory, between 1 to 2 m/s, but may be a little low. This will show up when the pressure drop is calculated.

Step 8: Bundle and shell diameter

 

\(\text { From Table } 12.4 \text {, for } 2 \text { tube passes, } K_{1}=0.249, n_{1}=2.207 \text {, }\)

So,

 

\(D_{b}=19.05\left(\frac{240}{0.249}\right)^{1 / 2.207}=428 mm (0.43 m )\) (12.3b)

 

For a split-ring floating head exchanger the typical shell clearance from Figure 12.10 is 56 mm, so the shell inside diameter,

 

\(D_{s}=428+56=484 mm\)

 

Step 9: Tube-side heat transfer coefficient

 

\(R e=\frac{820 \times 1.14 \times 14.83 \times 10^{-3}}{3.2 \times 10^{-3}}=4332,\left(4.3 \times 10^{3}\right)\)

 

\(\operatorname{Pr}=\frac{2.05 \times 10^{3} \times 3.2 \times 10^{-3}}{0.134}=48.96\)

 

\(\frac{L}{d_{i}}=\frac{5000}{14.83}=337\)

 

\(\text { From Figure } 12.23, j_{h}=3.2 \times 10^{-3 \mid}\)

 

\(N u=3.2 \times 10^{-3}(4332)(48.96)^{0.33}=50.06\) (12.15)

 

\(h_{i}=50.06 \times\left(\frac{0.134}{14.83 \times 10^{-3}}\right)=452 W / m ^{2}{ }^{\circ} C\)

 

This is clearly too low if \(U_{o} \text { is to be } 300 W / m ^{2}{ }^{\circ} C \text {. }\) The tube-side velocity did look low, so increase the number of tube passes to 4. This will halve the cross-sectional area in each pass and double the velocity.

New

 

\(u_{t}=2 \times 1.14=2.3 m / s\)

 

and

 

\(R e=2 \times 4332=8664\left(8.7 \times 10^{3}\right)\)

 

\(j_{h}=3.8 \times 10^{-3}\)

 

\(\begin{aligned}h_{i} &=\left(\frac{0.134}{14.83 \times 10^{-3}}\right) \times 3.8 \times 10^{-3}(8664)(48.96)^{0.33} \\&=1074 W / m ^{2{ }^{\circ} C }\end{aligned}\)

 

Step 10: Shell-side heat transfer coefficient

Kern’s method will be used.

With 4 tube passes the shell diameter will be larger than that calculated for 2 passes.

 

\(\text { For } 4 \text { passes } K_{1}=0.175 \text { and } n_{1}=2.285\)

 

\(D_{b}=19.05\left(\frac{240}{0.175}\right)^{1 / 2.285}=450 mm ,(0.45 m )\) (12.3b)

 

The bundle to shell clearance is still around 56 mm, giving:

 

\(D_{s}=506 mm \text { (about } 20 \text { inches) }\)

 

As a first trial take the baffle spacing \(=D_{s} / 5\), say 100 mm. This spacing should give good heat transfer without too high a pressure drop.

\(A_{s}=\frac{(23.81-19.05)}{23.81} 506 \times 100=10,116 mm ^{2}=0.01012 m ^{2}\) (12.21)

 

\(d_{e}=\frac{1.10}{19.05}\left(23.81^{2}-0.917 \times 19.05^{2}\right)=13.52 mm\) (12.23)

 

\(\text { Volumetric flow-rate on shell-side }=\frac{20,000}{3600} \times \frac{1}{730}=0.0076 m ^{3} / s\)

 

\(\text { Shell-side velocity }=\frac{0.076}{0.01012}=0.75 m / s\)

 

\(R e=\frac{730 \times 0.75 \times 13.52 \times 10^{-3}}{0.43 \times 10^{-3}}=17,214,\left(1.72 \times 10^{4}\right)\)

 

\(P r=\frac{2.47 \times 10^{3} \times 0.43 \times 10^{-3}}{0.132}=8.05\)

 

Use segmental baffles with a 25% cut. This should give a reasonable heat transfer coefficient without too large a pressure drop.

 

\(\text { From Figure } 12.29, j_{h}=4.52 \times 10^{-3} \text {. }\)

 

Neglecting the viscosity correction:

 

\(h_{s}=\left(\frac{0.132}{13.52} \times 10^{3}\right) \times 4.52 \times 10^{-3} \times 17,214 \times 8.05^{0.33}=1505 W / m ^{2}{ }^{\circ} C\) (12.25)

 

Step 11: Overall coefficient

 

\(\begin{aligned}\frac{1}{U_{o}} &=\left(\frac{1}{1074}+0.00035\right) \frac{19.05}{14.83}+\frac{19.05 \times 10^{-3} \operatorname{Ln}\left(\frac{19.05}{14.83}\right)}{2 \times 55}+\frac{1}{1505}+0.0002 \\U_{o} &=386 W / m ^{2{ }^{\circ} C }\end{aligned}\) (12.2)

 

This is above the initial estimate of \(300 W / m ^{2}{ }^{\circ} C\). The number of tubes could possibly be reduced, but first check the pressure drops.

Step 12: Pressure drop

Tube-side

 

\(240 \text { tubes, } 4 \text { passes, tube i.d. } 14.83 mm , u_{t} 2.3 m / s \text {, }\)

 

\(\operatorname{Re}=8.7 \times 10^{3} . \text { From Figure } 12.24, j_{f}=5 \times 10^{-3}\).

 

\(\begin{aligned}\Delta P_{t} &=4\left(8 \times 5 \times 10^{-3}\left(\frac{5000}{14.83}\right)+2.5\right) \frac{\left(820 \times 2.3^{2}\right)}{2} \\&=4(13.5+2.5) \frac{\left(820 \times 2.3^{2}\right)}{2} \\&=138,810 N / m ^{2}, 1.4 bar\end{aligned}\) (12.20)

 

This exceeds the specification. Return to step 6 and modify the design.

Modified design

The tube velocity needs to be reduced. This will reduce the heat transfer coefficient, so the number of tubes must be increased to compensate. There will be a pressure drop across the inlet and outlet nozzles. Allow 0.1 bar for this, a typical figure (about 15% of the total); which leaves 0.7 bar across the tubes. Pressure drop is roughly proportional to the square of the velocity and \(u_{t}\) is proportional to the number of tubes per pass. So the pressure drop calculated for 240 tubes can be used to estimate the number of tubes required.

 

\(\text { Tubes needed }=240 /(0.6 / 1.4)^{0.5}=365\)

 

Say, 360 with 4 passes.

Retain 4 passes as the heat transfer coefficient will be too low with 2 passes.

Second trial design: 360 tubes 19.05 mm o.d., 14.83 mm i.d., 5 m long, triangular pitch 23.81 mm.

 

\(D_{b}=19.05\left(\frac{360}{0.175}\right)^{1 / 2.285}=537 mm ,(0.54 m )\) (12.3b)

 

From Figure 12.10 clearance with this bundle diameter = 59 mm

 

\(D_{s}=537+59=596 mm\)

 

\(\text { Cross-sectional area per pass }=\frac{360}{4}\left(14.83 \times 10^{-3}\right)^{2} \frac{\pi}{4}=0.01555 m ^{2}\)

 

\(\text { Tube velocity } u_{t}=\frac{0.0237}{0.01555}=1.524 m / s\)

 

\(R e=\frac{820 \times 1.524 \times 14.83 \times 10^{-3}}{3.2 \times 10^{-3}}=5792\)

 

L/d is the same as the first trial, 337

 

\(j_{h}=3.6 \times 10^{-3}\)

 

\(h_{i}=\left(\frac{0.134}{14.83} \times 10^{-3}\right) 3.6 \times 10^{-3} \times 5792 \times 48.96^{0.33}=680 W / m ^{2}{ }^{\circ} C\) (12.15)

 

This looks satisfactory, but check the pressure drop before doing the shell-side calculation.

 

\(j_{f}=5.5 \times 10^{-3}\)

 

\(\Delta P_{t}=4\left(8 \times 5.5 \times 10^{-3}\left(\frac{5000}{14.83}\right)+2.5\right) \frac{\left(820 \times 1.524^{2}\right)}{2}=66,029 N / m ^{2}, 0.66 \text { bar }\) (12.20)

 

Well within specification.

Keep the same baffle cut and spacing.

 

\(A_{s}=\frac{(23.81-19.05)}{23.81} 596 \times 100=11,915 mm ^{2}, 0.01192 m ^{2}\) (12.21)

 

\(u_{s}=\frac{0.0076}{0.01193}=0.638 m / s\)

 

\(d_{e}=13.52 mm \text {, as before }\)

 

\(R e=\frac{730 \times 0.638 \times 13.52 \times 10^{-3}}{0.43 \times 10^{-3}}=14,644,\left(1.5 \times 10^{4}\right)\)

 

\(P r=8.05\)

 

\(j_{h}=4.8 \times 10^{-3}, \quad j_{f}=4.6 \times 10^{-2}\)

 

\(h_{s}=\left(\frac{0.132}{13.52 \times 10^{-3}}\right) 4.8 \times 10^{-3} \times 14,644 \times(8.05)^{0.33}=1366 W / m ^{2}{ }^{\circ} C , \text { looks } OK\) (12.25)

 

\(\Delta P_{s}=8 \times 4.6 \times 10^{-2}\left(\frac{596}{13.52}\right)\left(\frac{5000}{100}\right) \frac{\left(730 \times 0.638^{2}\right)}{2}=120,510 N / m ^{2}, 1.2 bar\) (12.26)

 

Too high; the specification only allowed 0.8 overall, including the loss over the nozzles.

Check the overall coefficient to see if there is room to modify the shell side design.

 

\(\frac{1}{U_{0}}=\left(\frac{1}{683}+0.00035\right) \frac{19.05}{14.83}+\frac{19.05 \times 10^{-3} \ln \left(\frac{19.05}{14.88}\right)}{2 \times 55}+\frac{1}{1366}+0.0002\) \(U_{o}=302 W / m ^{2 \circ} C\) (12.2)

 

\(U_{o} \text { required }=\frac{Q}{\left(A_{o} \Delta T_{\operatorname{lm}}\right)}, \quad A_{o}=360 \times 0.2992=107.7 m ^{2}\)

 

\(\text { so } U_{o} \text { required }=\frac{1509.4 \times 10^{3}}{(107.7 \times 71)}=197 W / m ^{2}{ }^{\circ} C\)

 

The estimated overall coefficient is well above that required for design, 302 compared to 192 \(W / m ^{2}{ }^{\circ} C\), which gives scope for reducing the shell-side pressure drop.

Allow a drop of 0.1 bar for the shell inlet and outlet nozzles, leaving 0.7 bar for the shell-side flow. So, to keep within the specification, the shell-side velocity will have to be reduced by around \(\sqrt{(1 / 2)}=0.707\). To achieve this the baffle spacing will need to be increased to 100/0.707 = 141, say 140 mm.

 

\(A_{s}=\frac{(23.81-19.05)}{23.81} 596 \times 140=6681 mm ^{2}, 0.167 m ^{2}\) (12.21)

 

\(u_{s}=\frac{0.0076}{0.0167}=0.455 m / s\),

 

\(\text { Giving: } R e=10,443, h_{s}=1177 W / m ^{2}{ }^{\circ} C , \Delta P_{s}=0.47 \text { bar, and } U_{o}=288 Wm ^{-2{ }^{\circ} C ^{-1}} \text {. }\)

 

The pressure drop is now well within the specification.

Step 14: Optimisation

There is scope for optimising the design by reducing the number of tubes, as the pressure drops are well within specification and the overall coefficient is well above that needed. However, the method used for estimating the coefficient and pressure drop on the shell-side (Kern’s method) is not accurate, so keeping to this design will give some margin of safety.

Viscosity correction factor

The viscosity correction factor \(\left(\mu / \mu_{w}\right)^{0.14}\) was neglected when calculating the heat transfer coefficients and pressure drops. This is reasonable for the kerosene as it has a relatively low viscosity, but it is not so obviously so for the crude oil. So, before firming up the design, the effect of this factor on the tube-side coefficient and pressure drop will be checked.

First, an estimate of the temperature at the tube wall, \(t_{w}\) is needed.

 

\(\text { The inside area of the tubes }=\pi \times 14.83 \times 10^{-3} \times 5 \times 360=83.86 m ^{2}\)

 

\(\text { Heat flux }=Q / A=1509.4 \times 10^{3} / 83.86=17,999 W / m ^{2}\)

 

As a rough approximation

 

\(\left(t_{w}-t\right) h_{i}=17,999\)

 

where t is the mean bulk fluid temperature = \(59^{\circ} C\).

So,

 

\(t_{w}=\frac{17,999}{680}+59=86^{\circ} C\)

 

\(\text { The crude oil viscosity at this temperature }=2.1 \times 10^{-3} Ns / m ^{2} \text {. }\)

 

\(\text { Giving }\left(\frac{\mu}{\mu_{w}}\right)^{0.14}=\left(\frac{3.2 \times 10^{-3}}{2.1 \times 10^{-3}}\right)^{0.14}=1.06\)

 

Only a small factor, so the decision to neglect it was justified. Applying the correction would increase the estimated heat transfer coefficient, which is in the right direction. It would give a slight decrease in the estimated pressure drop.

Optimisation using a CAD program

The use of a proprietary computer program (HTFS, M-TASC) to find the lowest cost design that meets the specification resulted in the design set out below. The program selected longer tubes, to minimise the cost. This has resulted in an exchanger with a shell length to diameter ratio of greater than 10 : 1. This could cause problems in supporting the shell, and in withdrawing the tube bundle for maintenance.

The CAD program was rerun with the tube length restricted to 3500 mm, to produce a more compact design. This gave a design with 349 tubes, 4 passes, in a shell 540 mm diameter. The setting plan for this design is shown in Figure B.
