Guest Question.

Consider the thermal control system shown in Figure 9.65. The physical plant can be a room, an oven, etc.

(a) What is the limit cycle period?

(b) If \(T_{r}\) is commanded as a slowly increasing function, sketch the output of the system, T.

Show the solution for \(T_{r}\) “large”.

Admin
Solution

(a) This is a first order system so use (T, t) plot. For an oven, it is piecewise linear

\(\dot{T}+a T=B N s g n(e)\) with hysteresis

with

\(T_{r}=800^{\circ} C , \quad \frac{B N}{a} \approx 1000^{\circ} C \) above T = 0 (say room temperature)

\(a \simeq 0.01 sec ^{-1}\) , plot vs at, \(h \sim 100^{\circ} C , T_{o}=0\).

(b)

limit cycle period \(\triangleq P=t_{o n}+t_{o f f}\),

\(T_{r} e^{-a t}=\left(T_{r}-h\right)\)

gives \(t_{off}\) ,

\(\left[\frac{B N}{a}-\left(T_{r}-h\right)\right] e^{-a t}=\left[\frac{BN}{a}-T_{r}\right]\)

gives \(t_{on}\) ,

\(a P=-\ln \left(\frac{700}{800}\right)-\ln \left(\frac{200}{300}\right)\),

\(P=100(0.058+0.176)=23.4 sec\).
