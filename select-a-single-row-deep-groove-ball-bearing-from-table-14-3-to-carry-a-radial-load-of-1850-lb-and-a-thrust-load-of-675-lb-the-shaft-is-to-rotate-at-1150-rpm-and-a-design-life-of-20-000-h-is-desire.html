Solution

Use the procedure outlined above.
Step 1. Assume Y = 1.50.
Step 2. P = VXR + YT= (1.0)(0.56)( 1850) + (1.50)(675) = 20491b.
Step 3. From Figure 14-12. the speed factor/v = 0.30. and the life factor \(f_L = 3.41.\)
Then the required basic dynamic load rating C is

\(C = Pf_L/f_N = 2049(3.41 )/(0.30) = 23 300\) lb

Step 4. From Table 14-3. we could use either bearing number 6222 or 6318. The 6318 has a bore of 3.5433 in and is well suited to this application.

Step 5. For bearing number 6318,\( C_o= 22 500\) lb.
Step 6. \( T/C_o = 675/22 500 = 0.03\) .
Step 7. From Table 14-5, e = 0.22 (approximately).

Step 8. T/R = 675/1850 = 0.36. Because T/R > e, we can find Y = 1.97 from Table 14-5 by interpolation based on \(T/C_o= 0.03.\)
Step 9. Recompute P = (1.0)(0.56)( 1850) + (l.97)(675) = 23661b:
C = 2366(3.41)/ (0.30) = 26 900 lb

The bearing number 6318 is not satisfactory at this load. Let’s choose bearing number 6320 and repeat the process from Step 5.

Step 5.\(C_o = 29 800\) lb.
Step 6. \(T/C_o= 675/29 800 = 0.023\).
Step 7.e = 0.20.

Step 8. T/R > e. Then Y = 2.10 using \(T/C_o = 0.023\)

Step 9. P = (1.0)(0.56)(1850) + (2.10)(675) = 2454 lb. Thus,
C = 2454(3.41 )/(0.30) = 27 900 lb

Because bearing number 6320 has a value of C = 30 000 lb, it is satisfactory.
