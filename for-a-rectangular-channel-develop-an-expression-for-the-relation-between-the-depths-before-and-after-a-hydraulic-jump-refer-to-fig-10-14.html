Guest Question.

For a rectangular channel, develop an expression for the relation between the depths before and after a hydraulic jump. Refer to Fig. 10-14.

Admin
Solution

For the free body between sections 1 and 2, considering a unit width of channel and unit flow q,

\(P_{1}=\gamma \bar{h} A=\gamma\left(\frac{1}{2} y_{1}\right) y_{1}=\frac{1}{2} \gamma y_{1}^{2}, \quad \text { and similarly } \quad P_{2}=\frac{1}{2} \gamma y_{2}^{2}\).

From the principle of impulse and momentum,

\(\Delta P_{x} d t=\Delta \text { linear momentum }=\frac{W}{g}\left(\Delta V_{x}\right)\).

\(\frac{1}{2} \gamma\left(y_{2}^{2}-y_{1}^{2}\right) d t=\frac{\gamma q d t}{g}\left(V_{1}-V_{2}\right)\).

\(\text { Since } V_{2} y_{2}=V_{1} y_{1} \text { and } V_{1}=q / y_{1} \text {, the above equation becomes }\)

\(q^{2} / g=\frac{1}{2} y_{1} y_{2}\left(y_{1}+y_{2}\right)\).

\(\text { Since } q^{2} / g=y_{c}^{3} \text {, }\) \(y_{c}^{3}=\frac{1}{2} y_{1} y_{2}\left(y_{1}+y_{2}\right)\).

\(\text { The length of the jump has been found to vary between } 4.3 y_{2} \text { and } 5.2 y_{2} \).

The hydraulic jump is an energy dissipator. In designing hydraulic jump stilling basins, knowledge of jump length and depth \(y_{2} \text { is important. Good energy dissipation occurs when } V_{1}^{2} / g y_{1}=20 \text { to } 80 \).
