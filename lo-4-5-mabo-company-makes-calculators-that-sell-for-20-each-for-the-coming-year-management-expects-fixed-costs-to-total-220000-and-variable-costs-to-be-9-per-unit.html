Guest Question.

(LO 4, 5) Mabo Company makes calculators that sell for $20 each. For the coming year, management expects fixed costs to total $220,000 and variable costs to be $9 per unit.

 

Instructions
a. Compute break-even point in units using the mathematical equation.
b. Compute break-even point in dollars using the contribution margin (CM) ratio.
c. Compute the margin of safety percentage assuming actual sales are $500,000.
d. Compute the sales required in dollars to earn net income of $165,000.

 

Compute break-even point, contribution margin ratio, margin of safety, and sales for target net income.

Admin
Solution

a.

\(\begin{aligned}\text { Sales }-\text { Variable costs }-\text { Fixed costs } &=\text { Net income } \\\$ 20 \mathrm{Q}-\$ 9 \mathrm{Q}-\$ 220,000 &=\$ 0 \\\$ 11 \mathrm{Q} &=\$ 220,000 \\\mathrm{Q} &=20,000 \text { units }\end{aligned}\)

 

b.

\(\begin{aligned}\text { Unit contribution margin } &=\text { Unit selling price }-\text { Unit variable costs } \\\$ 11 &=\$ 20-\$ 9 \\\text { Contribution margin ratio } &=\text { Unit contribution margin } \div \text { Unit selling price } \\55 \% &=\$ 11 \div \$ 20 \\\text { Break-even point in dollars } &=\text { Fixed costs } \div \text { Contribution margin ratio } \\&=\$ 220,000 \div 55 \% \\&=\$ 400,000\end{aligned}\)

 

c.

\(\begin{aligned}\text { Margin of safety } &=\frac{\text { Actual sales }-\text { Break-even sales }}{\text { Actual sales }} \\&=\frac{\$ 500,000-\$ 400,000}{\$ 500,000} \\&=20 \%\end{aligned}\)

 

d.

Sales \(–\) Variable costs \(–\) Fixed costs \(=\) Net income

\(\begin{aligned}\$ 20 \mathrm{Q}-\$ 9 \mathrm{Q}-\$ 220,000 &=\$ 165,000 \\\$ 11 \mathrm{Q} &=\$ 385,000 \\\mathrm{Q} &=35,000 \text { units }\end{aligned}\)

35,000 units \(\times \$ 20=\$ 700,000\) required sales
OR
(Fixed costs + Target net income) \(\div\) Contribution margin ratio \(=\) Sales in dollars \((\$ 220,000+\$ 165,000) \div .55=\$ 700,000\)
