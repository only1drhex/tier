Guest Question.

Find a Householder matrix S such that SBS = H, where H is a Hessenberg matrix and where B is given by

\(B=\left[\begin{array}{rrrr}1 & 2 & 4 & 2 \\3 & 3 & -4 & 2 \\0 & 3 & 9 & -1 \\0 & -4 & -2 & 8\end{array}\right]\).

Also, calculate the matrix SB.

Admin
Solution

We seek a Householder matrix S such that the vector \(S B _{2}\) has a zero in the fourth component, where \(B _{2}=[2,3,3,-4]^{T}\).
Using k = 3 in Algorithm 2, we define a vector u, where

\(u =\left[\begin{array}{r}0 \\0 \\8 \\-4\end{array}\right]\).

The appropriate Householder matrix is \(S=I-b u u ^{T}, b=2 /\left( u ^{T} u \right)=1 / 40\).
Next, we calculate the matrix SB by using \(S B=\left[S B _{1}, S B _{2}, S B _{3}, S B _{4}\right]\), where

\(S B _{i}= B _{i}-\gamma_{i} u ; \quad \gamma_{i}=2 u ^{T} B _{i} /\left( u ^{T} u \right)= u ^{T} B _{i} / 40\).

The details are:

\(\text { (a) } \gamma_{1}= u ^{T} B _{1} / 40=0, \quad \text { so } \quad S B _{1}= B _{1}=[1,3,0,0]^{T}\).
\(\text { (b) } \gamma_{2}= u ^{T} B _{2} / 40=1, \text { so } S B _{2}= B _{2}- u =[2,3,-5,0]^{T} \text {. }\)
\(\text { (c) } \gamma_{3}= u ^{T} B _{3} / 40=2, \quad \text { so } \quad S B _{3}= B _{3}-2 u =[4,-4,-7,6]^{T} .\)
\(\text { (d) } \gamma_{4}= u ^{T} B _{4} / 40=-1, \text { so } S B _{4}= B _{4}+ u =[2,2,7,4]^{T} \text {. }\)

Thus the matrix SB is given by

\(S B=\left[\begin{array}{rrrr}1 & 2 & 4 & 2 \\3 & 3 & -4 & 2 \\0 & -5 & -7 & 7 \\0 & 0 & 6 & 4\end{array}\right]\).
