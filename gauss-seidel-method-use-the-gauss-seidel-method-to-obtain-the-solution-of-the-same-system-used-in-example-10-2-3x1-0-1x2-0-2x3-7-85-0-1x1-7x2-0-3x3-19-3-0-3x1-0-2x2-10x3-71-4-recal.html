Gauss-Seidel Method

Use the Gauss-Seidel method to obtain the solution of the same system used in :

\(3x_1 - 0.1x_2 - 0.2x_3 = 7.85\)

 

\(0.1x_1 + 7x_2 - 0.3x_3 = -19.3\)

 

\(0.3x_1 - 0.2x_2 + 10x_3 = 71.4\)

Recall that the true solution is \(x_1 = 3, x_2 = - 2.5\), and \(x_3 = 7\).

Admin
Solution

First, solve each of the equations for its unknown on the diagonal.

\(x_1= \frac{7.85 + 0.1x_2 + 0.2x_3}{3}\) (E11.3.1)

\(x_2= \frac{- 19.3 – 0.1x_1 1 0.3x_3}{7} \) (E11.3.2)

\(x_3= \frac{71.4 – 0.3x_1 + 0.2x_2}{10} \) (E11.3.3)

By assuming that \(x_2\) and \(x_3\) are zero, Eq. (E11.3.1) can be used to compute

\(x_1= \frac{7.85 + 0 + 0}{3}= 2.616667\)

This value, along with the assumed value of \(x_3 = 0\), can be substituted into Eq. (E11.3.2) to calculate

\(x_2= \frac{-19.3 – 0.1(2.616667) + 0}{7}= – 2.794524\)

The first iteration is completed by substituting the calculated values for \(x_1\) and \(x_2\) into Eq. (E11.3.3) to yield

\(x_3= \frac{71.4 – 0.3(2.616667) + 0.2(22.794524)}{10}= 7.005610\)

For the second iteration, the same process is repeated to compute

\(x_1= \frac{7.85 + 0.1(- 2.794524) + 0.2(7.005610)}{3}= 2.990557 |\varepsilon _t| =0.31\% \)

 

\(x_2= \frac{- 19.3 – 0.1(2.990557) + 0.3(7.005610)}{7}= – 2.499625 |\varepsilon _t| =0.015\% \)

 

\(x_3= \frac{71.4 – 0.3(2.990557) + 0.2(- 2.499625)}{10}= 7.000291 |\varepsilon _t| =0.0042\% \)

The method is, therefore, converging on the true solution. Additional iterations could be applied to improve the answers. However, in an actual problem, we would not know the true answer a priori. Consequently, Eq. (11.6)

\(|\varepsilon _{a,i}|= \left|\frac{x^{j}_i – x^{j-1}_i}{x^j_i} \right|100\% \lt \varepsilon _ s\) (11.6)

provides a means to estimate the error. For example, for \(x_1\),

\(|\varepsilon _{a,1}|= \left|\frac{2.990557 – 2.616667}{2.990557} \right|100\%= 12.5\% \)

For \(x_2\) and \(x_3\), the error estimates are \(|\varepsilon _{a,2}|= 11.8\% \) and \(|\varepsilon _{a,3}|= 0.076\% \). Note that, as was the case when determining roots of a single equation, formulations such as Eq. (11.6) usually provide a conservative appraisal of convergence. Thus, when they are met, they ensure that the result is known to at least the tolerance specified by \(\varepsilon_s \).
