Solution

a . Using KVL, the voltage across R is 19 V. Then using Ohm’s law, the current in the loop is I = 5/ 3.9k = 1.282 mA

 

The required value of R is then

R = 19/0.001282 = 14.82 kΩ

 

b. As shown in Table 2.1, the nearest standard 10 % tolerance resistor is 15 kΩ.

 

C. Using the standard 15-kΩ resistor, the actual current in the circuit is

I =24 / 18.9 k=1.2698 mA

and the voltage across the 3.9-k Ω resistor is

V=I R=(0.0012698)(3.9 k)=4.952 V

 

d. The percent error involved in using the standard resistor is

% Error =(4.952-5) / 5 × 100=-0.96 %

 

e. The power absorbed by the resistor R is then

P = I R =(0.0012698)\(^{2}\)(15 k)=24.2 mW

Therefore, even a quarter-watt resistor is adequate in this application.
