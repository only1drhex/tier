Guest Question.

Let the pipe in Problem 7.87 be a cast iron pipe (Fig. 7.13 provides a value for e). Estimate the maximum velocity using the logarithmic velocity profile.

Admin
Solution

We must find \(u_τ \) :

\(V=\frac{Q}{A}= \frac{1.2}{\pi \times 0.4^2} =19.63 m/s . Re = \frac{19.63\times 0.8}{2.2\times 10^{-4}} =71400\) .

\(\frac{e}{D}=\frac{0.26}{800}=0.000325\) . ∴ Moody diagram ⇒ f = 0.021 .

\(τ_0= \frac{1}{8} fρ V^2 = \frac{1}{8}\times 0.021× 917 × 19.63^2 = 928 Pa\) .

\(u_τ = \sqrt{τ_0 /ρ } = \sqrt{928/917} =1.006 m/s\).

∴ \(u_{max}=1.006 \left[2.44ln \frac{1.006 ×0.4}{2.2×10^{-4}}+ 5.7\right] =\underline{ 24.2 m/s}\)
