Guest Question.

Design a skirt support for the column specified in .

Admin
Solution

Try a straight cylindrical skirt \(\left(\theta_{s}=90^{\circ}\right)\) of plain carbon steel, design stress 135 \(N / mm ^{2}\) and Young’s modulus 200,000 \(N / mm ^{2}\) at ambient temperature.

The maximum dead weight load on the skirt will occur when the vessel is full of water.

 

\(\text { Approximate weight }=\left(\frac{\pi}{4} \times 2^{2} \times 50\right) 1000 \times 9.81\) = 1,540,951 N = 1541 kN

 

Weight of vessel, from = 842 kN

Total weight = 1541 + 842 = 2383 kN

Wind loading, from Example 13.4 = 2.79 kN/m

 

\(\text { Bending moment at base of skirt }=2.79 \times \frac{53^{2}}{2}\) = 3919 kNm (13.77)

 

As a first trial, take the skirt thickness as the same as that of the bottom section of the vessel, 18 mm.

 

\(\sigma_{b s}=\frac{4 \times 3919 \times 10^{3} \times 10^{3}}{\pi(2000+18) 2000 \times 18}\) \(=68.7 N / mm ^{2}\) (13.88)

 

\(\sigma_{w s}(\text { test })=\frac{1543 \times 10^{3}}{\pi(2000+18) 18}=13.5 N / mm ^{2}\) (13.89)

 

\(\sigma_{w s}(\text { operating })=\frac{842 \times 10^{3}}{\pi(2000+18) 18}=7.4 N / mm ^{2}\) (13.89)

 

Note: the “test” condition is with the vessel full of water for the hydraulic test. In estimating total weight, the weight of liquid on the plates has been counted twice. The weight has not been adjusted to allow for this as the error is small, and on the “safe side”.

 

\(\text { Maximum } \hat{\sigma}_{s}(\text { compressive })=68.7+13.5=82.2 N / mm ^{2}\) (13.87)

 

\(\text { Maximum } \hat{\sigma}_{s}(\text { tensile })=68.7-7.4=61.3 N / mm ^{2}\) (13.86)

 

Take the joint factor J as 0.85.

Criteria for design:

 

\(\hat{\sigma}_{s}(\text { tensile }) \ngtr f_{s} J \sin \theta\) (13.90)

 

\(\begin{array}{l}61.3 \ngtr 0.85 \times 135 \sin 90 \\61.3 \ngtr 115\end{array}\)

 

\(\hat{\sigma}_{s} \text { (compressive) } \ngtr 0.125 E\left(\frac{t_{s}}{D_{s}}\right) \sin \theta\) (13.91)

 

\(\begin{array}{l}82.2 \ngtr 0.125 \times 200,000\left(\frac{18}{2000}\right) \sin 90 \\82.2 \ngtr 225\end{array}\)

 

Both criteria are satisfied, add 2 mm for corrosion, gives a design thickness of 20 mm

Base ring and anchor bolts

Approximate pitch circle dia., say, 2.2 m

Circumference of bolt circle = 2200π

Number of bolts required, at minimum recommended bolt spacing

 

\(=\frac{2200 \pi}{600}=11.5\)

 

Closest multiple of 4 = 12 bolts

 

\(\text { Take bolt design stress }=125 N / mm ^{2}\)

 

\(M_{s}=3919 kN m\)

 

Take W = operating value = 842 kN.

 

\(A_{b}=\frac{1}{12 \times 125}\left[\frac{4 \times 3919 \times 10^{3}}{2.2}-842 \times 10^{3}\right]\) \(=4190 mm ^{2}\) (13.92)

 

\(\text { Bolt root dia. }=\sqrt{\frac{4190 \times 4}{\pi}}=73 mm \text {, looks too large. }\)

 

Total compressive load on the base ring per unit length

 

\(F_{b}=\left[\frac{4 \times 3919 \times 10^{3}}{\pi \times 2.0^{2}}+\frac{842 \times 10^{3}}{\pi \times 2.0}\right]\) \(=1381 \times 10^{3} N / m\) (13.93)

 

\(\text { Taking the bearing pressure as } 5 N / mm ^{2}\)

 

\(L_{b}=\frac{1381 \times 10^{3}}{5 \times 10^{3}}=276 mm\) (13.94)

 

\(\text { Rather large — consider a flared skirt. }\)

 

Take the skirt bottom dia. as 3 m

 

\(\text { Skirt base angle } \theta_{s}=\tan ^{-1} \frac{3}{\frac{1}{2}(3-2)}=80.5^{\circ}\)

 

Keep the skirt thickness the same as that calculated for the cylindrical skirt. Highest stresses will occur at the top of the skirt; where the values will be close to those calculated for the cylindrical skirt. \(\operatorname{Sin} 80.5^{\circ}\) = 0.99, so this term has little effect on the design criteria.

Assume bolt circle dia. = 3.2 m.

Take number of bolts as 16.

 

\(\text { Bolt spacing }=\frac{\pi \times 3.2 \times 10^{3}}{16}=628 mm \text { satisfactory }\)

 

\(A_{b}=\frac{1}{16 \times 125}\left[\frac{4 \times 3919 \times 10^{3}}{3.2}-842 \times 10^{3}\right]\) \(=2029 mm ^{2}\)

 

Use M56 bolts (BS 4190:1967) root area = 2030 \(mm ^{2}\)

 

\(F_{b}=\left[\frac{4 \times 3919 \times 10^{3}}{\pi \times 3.0^{2}}+\frac{842 \times 10^{3}}{\pi \times 3.0}\right]\) = 644 kN/m.

 

\(L_{b}=\frac{644 \times 10^{3}}{5 \times 10^{3}}=129 mm\)

 

This is the minimum width required; actual width will depend on the chair design.

Actual width required (Figure 13.30):

 

\(\begin{array}{l}=L_{r}+t_{s}+50 mm \\=150+20+50=220 mm\end{array}\)

 

Actual bearing pressure on concrete foundation:

 

\(f_{c}^{\prime}=\frac{644 \times 10^{3}}{220 \times 10^{3}}=2.93 N / mm ^{2}\)

 

\(t_{b}=150 \sqrt{\frac{3 \times 2.93}{140}}=37.6 mm\) (13.95)

 

round off to 40 mm

Chair dimensions from Figure 13.30 for bolt size M56.

Skirt to be welded flush with outer diameter of column shell.
