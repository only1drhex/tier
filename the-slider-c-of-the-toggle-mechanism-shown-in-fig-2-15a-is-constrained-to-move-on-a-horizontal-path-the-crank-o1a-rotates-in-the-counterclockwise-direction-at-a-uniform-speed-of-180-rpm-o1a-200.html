Guest Question.

The slider C of the toggle mechanism shown in Fig.2.15(a) is constrained to move on a horizontal path. The crank \( O_{1} A \) rotates in the counterclockwise direction at a uniform speed of 180 rpm.

\( O_{1} A=200 mm , A B=400 mm , O_{2} B=300 mm , \text { and } B C=600 mm \).

Determine (a) velocity of slider C, (b) angular velocity of links AB, \( O_{2} B \) , and BC, (c) rubbing velocities on the pins of 25 mm diameter at A and C, and (d) torque required at the crank \( O_{1} A \) for a force of 2 kN at C.

Admin
Solution

Given: N = 180 rpm, radius of pin, r = 12.5 mm, lengths of links.

1. Draw the configuration diagram to a convenient scale as shown in Fig.2.15(a).

2. Calculate the angular velocity of crank \( O_{1} A \).

\( \omega=2 \pi \times \frac{180}{60}=18.85 rad / s \).

3. Calculate the linear velocity of point A.

\( v_{a}=\omega \times O_{1} A=18.85 \times 0.2=3.77 m / s \).

4. Draw the velocity diagram as shown in Fig.2.15(b) to a scale of 1 cm = 0.5 m/s.

\( \text { 5. Draw } v_{a}=o, a \perp O_{1} A \text { such that } o_{1} a=7.54 cm \text {. } \)

\( \text { 6. From ' } a \text { ' draw a line perpendicular to } A B \text { and another line from } o_{2} \text { perpendicular to } o_{2} B \text { to intersect } \text { at } b \text {. Then } o_{2} b=v_{b} \text { and } ab =v_{b a}\).

7. From b, draw a line perpendicular to BC and another line from \( o_2\) parallel to the path of motion of the slider C to meet at c. Then \( o_{2} c=v_{c} \). By measurement, we get

\( \text { (a) Velocity of the slider, } v_{c}=o_{2} c=4.2 cm =2.1 m / s \).

\( \text { (b) Velocity of link } A B A B=v_{b a}=a b=2.4 cm =1.2 m / s \).

\( \text { Velocity of point } B, v_{b}=o_{2} b=6.2 cm =3.1 m / s \).

\( \text { Velocity of link } B C, v_{c b}=b c=5.7 cm =2.85 m / s \).

\( \text { Angular velocity of } B \text { about } A, \omega_{a b}=\frac{v_{\text {ba }}}{A B}=\frac{1.2}{0.4}=3 rad / s ( ccw \text { about } A) \).

\( \text { Angular velocity of } B \text { about } O_{2}, \omega_{b}=\frac{v_{b}}{O_{2} B}=\frac{3.1}{0.3}=10.33 rad / s \left( cc w \text { about } O_{2}\right) \).

\( \text { Angular velocity of } C \text { about } B, \omega_{b c}=\frac{v_{c b}}{B C}=\frac{2.85}{0.6}=4.75 rad / s (c w \text { about } B) \).

For direction of angular velocities, refer to Fig.2.15(c).

\( \text { (c) Radius of pin, } r=\frac{25}{2}=12.5 mm \).

\( \text { Rubbing velocity on pin at } C=\omega_{c b} \cdot r =4.75 \times 0.0125=0.0594 m / s \).

\( \text { Relative angular velocity at } A=\omega_{b}-\omega_{b a}+\omega_{c b} \).

\( =10.33-3+4.75=12.08 rad / s \).

Rubbing velocity on pin at A = 12.08 Ã— 0.0125 = 0.151 m/s.

\( \text { (d) Let } T \text { be the torque required at the crank } O_{1} A \text {. } \)

\( T \omega=F_{c} v_{c} \).

\( 18.85 T=2000 \times 2.1 \).

\( T=222.81 Nm \).
