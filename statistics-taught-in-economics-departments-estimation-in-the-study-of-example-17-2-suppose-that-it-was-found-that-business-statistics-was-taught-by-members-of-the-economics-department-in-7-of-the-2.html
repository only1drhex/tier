Statistics Taught in Economics Departments (Estimation)

In the study of , suppose that it was found that business statistics was taught by members of the economics department in 7 of the 2-year colleges and 13 of the 4-year colleges in the sample. Estimate the proportion of all colleges in which this course is taught in the economics department.

Admin
Solution

We know that

\(N_{1} = 364 n_{1} = 40 \hat{p} _{1} = \frac{7}{40} = 0.175\)

 

\(N_{2} = 1,031 n_{2} = 60 \hat{p} _{2} = \frac{13}{60} = 0.217\)

Our estimate of the population proportion is as follows:

\(\hat{p} _{st} = \frac{1}{N}\sum\limits_{j=1}^{k}{N_{j}\hat{p} _{j} } = \frac{\left(364\right)\left(0.175\right) + \left(1,031\right)\left(0.217\right) }{1,395} = 0.206\)

Thus, it is estimated that in 20.6% of all colleges, the economics department teaches the course.

Next,

\(\hat{\sigma } ^{2}_{p_{1} } = \frac{\hat{p} _{1}\left(1 – \hat{p} _{1} \right) }{n_{1} – 1 } \times \frac{\left(N_{1} – n_{1} \right) }{N_{1} – 1} = \frac{\left(0.175\right)\left(0.825\right) }{39} \times \frac{324}{363} = 0.003304\)

 

\(\hat{\sigma } ^{2}_{p_{2} } = \frac{\hat{p} _{2}\left(1 – \hat{p} _{2} \right) }{n_{2} – 1 } \times \frac{\left(N_{2} – n_{2} \right) }{N_{2} – 1} = \frac{\left(0.217\right)\left(0.783\right) }{59} \times \frac{971}{1,030} = 0.002715\)

Together with the individual stratum sample proportions, these values can be used to calculate confidence intervals for the two stratum population proportions. However, here, focus is given to interval estimation for the overall population proportion, for which

\(\hat{\sigma } ^{2}_{\hat{p} _{st} } = \frac{1}{N^{2} }\sum\limits_{j=1}^{k}{N^{2}_{j}\hat{\sigma } ^{2}_{\hat{p} _{j} } } = \frac{\left(364\right) ^{2}\left(0.003304\right) + \left(1,031\right) ^{2}\left(0.002715\right) }{\left(1,395\right) ^{2} } = 0.001708\)

so taking the square root yields the following:

\(\hat{\sigma }_{\hat{p} _{st}} = 0.0413\)

For a 90% confidence level,

\(z_{\alpha /2} = z_{0.05} = 1.645\)

and the 90% confidence interval for the population proportion from a stratified random sample is

(0.206) Â± (1.645)(0.0413)

This interval runs from 13.8% to 27.4% of all colleges.
