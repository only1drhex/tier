Two-Point Gauss-Legendre Formula

Use Eq. (22.23)

\(I \cong f\left(\frac{-1}{\sqrt{3}}\right)+f\left(\frac{1}{\sqrt{3}}\right)\) (22.23)

to evaluate the integral of

\(f(x)=0.2+25 x-200 x^{2}+675 x^{3}-900 x^{4}+400 x^{5}\)

between the limits x = 0 to 0.8. Recall that this was the same problem that we solved in Chap. 21 using a variety of Newton-Cotes formulations. The exact value of the integral is 1.640533.

Admin
Solution

Before integrating the function, we must perform a change of variable so that
the limits are from 21 to 11. To do this, we substitute a = 0 and b = 0.8 into Eq. (22.29)

\(x=\frac{(b+a)+(b-a) x_{d}}{2}\) (22.29)

to yield

\(x=0.4+0.4 x_{d}\)

The derivative of this relationship is [Eq. (22.30)]

\(d x=\frac{b-a}{2} d x_{d}\) (22.30)

\(d x=0.4 d x_{d}\)

Both of these can be substituted into the original equation to yield

\(\int_{0}^{0.8}\left(0.2+25 x-200 x^{2}+675 x^{3}-900 x^{4}+400 x^{5}\right) d x\)

 

\(=\int_{-1}^{1}\left[0.2+25\left(0.4+0.4 x_{d}\right)-200\left(0.4+0.4 x_{d}\right)^{2}+675\left(0.4+0.4 x_{d}\right)^{3}\right.\left.-900\left(0.4+0.4 x_{d}\right)^{4}+400\left(0.4+0.4 x_{d}\right)^{5}\right] 0.4 d x_{d}\)

Therefore, the right-hand side is in the form that is suitable for evaluation using Gauss quadrature. The transformed function can be evaluated at \(-1 / \sqrt{3}\) to be equal to 0.516741 and at \(1 / \sqrt{3}\) to be equal to 1.305837. Therefore, the integral according to Eq. (22.23)

\(I \cong f\left(\frac{-1}{\sqrt{3}}\right)+f\left(\frac{1}{\sqrt{3}}\right)\) (22.23)
