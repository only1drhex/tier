Guest Question.

A three-phase full-bridge quasi square-wave inverter is supplying a Y-connected symmetrical load with R = 10 Ω and L = 20 mH per phase. If the line-to-line voltage fundamental component is 250 V rms 50 Hz. Perform the following:

a) Calculate the inverter input dc voltage.

b) Calculate the THD factors of the output line-to-line voltage and line current taking into consideration the first two high harmonic components.

c) Calculate the power factor and the value of the dc input current.

d) Verify the theoretical results with the respective simulation ones.

Admin
Solution

a) Using Eq. (\(\widehat{V}_{ab,n}\)=amplitude of the nth harmonic component of \(v_{ab}\)

 

\(=\frac{2}{T}\int_{-T/2}^{T/2}{v_{ab} \sin(n\omega t)dt} = \frac{1}{\pi}\int_{-\pi}^{\pi}{v_{ab} \sin(n\omega t)d(\omega t)}\)

 

\(=\frac{4}{\pi}\int_{\pi/6}^{\pi/2}{v_{ab} \sin(n\omega t)d(\omega t)}=\frac{4}{\pi}\int_{\pi/6}^{\pi/2}{V_{in} \sin(n\omega t)d(\omega t)}\)

 

\(=\frac{4 V_{in}}{n \pi} \cos(\frac{n\pi}{6})\) n = 1, 3, 5, 7, 9, . . . ) the required dc input voltage is:

\(V_{in} = \frac{\widehat{V}_{ab,1} \times \pi }{4\cos (\frac{\pi}{6})} = \frac{250\sqrt{2} \times \pi}{4\cos (\frac{\pi}{6})} = 320.48 V\)

b) \(THD_{v_{ab}}\% = \frac{\sqrt{\widehat{V}^2_{ab,5} + \widehat{V}^2_{ab,7}}}{\widehat{V}_{ab,1}} \times 100\) \(THD_{i_a}\% = \frac{\sqrt{\widehat{I}^2_{ab,5} + \widehat{I}^2_{ab,7}}}{\widehat{I}_{ab,1}} \times 100\)

Using Eq. (\(\widehat{V}_{ab,n}\)=amplitude of the nth harmonic component of \(v_{ab}\)

 

\(=\frac{2}{T}\int_{-T/2}^{T/2}{v_{ab} \sin(n\omega t)dt} = \frac{1}{\pi}\int_{-\pi}^{\pi}{v_{ab} \sin(n\omega t)d(\omega t)}\)

 

\(=\frac{4}{\pi}\int_{\pi/6}^{\pi/2}{v_{ab} \sin(n\omega t)d(\omega t)}=\frac{4}{\pi}\int_{\pi/6}^{\pi/2}{V_{in} \sin(n\omega t)d(\omega t)}\)

 

\(=\frac{4 V_{in}}{n \pi} \cos(\frac{n\pi}{6})\) n = 1, 3, 5, 7, 9, . . .) and knowing that the impedance magnitude at the nth harmonic frequency is \(|Z_{o,n}| = \sqrt{R^2 + (n\omega L)^2}\) , then the following results are obtained:

\(\widehat{V}_{ab,5} = \frac{4 \times 320.48}{5\pi} \cos(\frac{5 \pi}{6}) = \frac{\widehat{V}_{ab,1}}{5} = 70.71 V\)

 

\(\widehat{V}_{ab,7} = \frac{4 \times 320.48}{7\pi} \cos(\frac{7 \pi}{6}) = \frac{\widehat{V}_{ab,1}}{7} = 50.51 V\)

 

\(|Z_{o,1}| = \sqrt{10^2 + (2\pi \times 50 \times 0.02)^2} = \sqrt{100 + 39.48} = 11.81 \Omega\)

 

\(|Z_{o,5}| = \sqrt{10^2 + (5 \times 2\pi \times 50 \times 0.02)^2} = \sqrt{100 + 985.96} = 32.95 \Omega\)

 

\(|Z_{o,7}| = \sqrt{10^2 + (7 \times 2\pi \times 50 \times 0.02)^2} = \sqrt{100 +1932.48} = 45.083 \Omega\)

 

\(\widehat{I}_{a,1} = \frac{\widehat{V}_{aN,1}}{|Z_{o,1}|} = \frac{\frac{250 \sqrt{2}}{\sqrt{3}}}{11.81} = 17.28 A\)

 

\(\widehat{I}_{a,5} = \frac{\widehat{V}_{aN,5}}{|Z_{o,5}|} = \frac{\frac{70.71}{\sqrt{3}}}{32.95} = 1.24 A\)

 

\(\widehat{I}_{a,7} = \frac{\widehat{V}_{aN,7}}{|Z_{o,7}|} = \frac{\frac{50.51}{\sqrt{3}}}{45.083} = 0.65 A\)

Therefore: \(THD_{v_{ab}}\% = \frac{\sqrt{(70.71)^2 + (50.51)^2}}{250 \sqrt{2}} \times 100 = 24.58\%\)

 

\(THD_{i_{a}}\% = \frac{\sqrt{(1.24)^2 + (0.65)^2}}{17.28} \times 100 = 8.10\%\)

c) \(\widetilde{I}_a = \sqrt{\widetilde{I}^2_{a,1} + \widetilde{I}^2_{a,5} +\widetilde{I}^2_{a,7} } = \sqrt{(\frac{17.28}{\sqrt{2}})^2 + (\frac{1.24}{\sqrt{2}})^2 + (\frac{0.65}{\sqrt{2}})^2} = 12.25 A\)

 

\(\varphi_1 = \tan^{-1}(\frac{\omega L}{R}) = \tan^{-1}(\frac{2\pi \times 50 \times 0.02}{10}) = 32.14°\)

 

\(\widetilde{V}_{aN,1} = \frac{\widetilde{V}_{ab,1}}{\sqrt{3}} = \frac{250}{\sqrt{3}} = 144.33 V\)

Therefore, using the previous results the power factor is calculated:

\(\lambda =\frac{P_{load-phase}}{S_{load-phase}} = \frac{\widetilde{V}_{aN,1} \widetilde{I}_{a,1}\cos\varphi_1}{\widetilde{V}_{aN,1} \widetilde{I}_{a}} = \frac{(144.33)(\frac{17.28}{\sqrt{2}}) \cos 32.14°}{(123.35)(12.25)} = \frac{1493.28 W}{1511.04 V A}\)

 

\(=0.988\)

Assuming that the inverter is ideal then the following results are obtained:

\(P_o = P_i\) or \(3P_{load-phase} = V_{in}\overline{I}_i\) or \(\overline{I}_i = \frac{3P_{load-phase}}{V_{in}} = \frac{3 \times 1493.28}{320.48}=13.98 A\)
