Guest Question.

A mixture of \( 40 \% N _{2} \text { and } 60 \% O _{2} \) by weight is compressed from 70째F and 14.7 psia to 60 psia. Estimate the horsepower required by an 80 percent-efficient compressor and the entropy change, if the mass flux is 10 lbm /min.

Admin
Solution
\( \text { The efficiency of a compressor is based on an isentropic process. Let us find } k \text { and } C_{p} \text { : } \)

\( C_{p}=x_{1} C_{p, 1}+x_{2} C_{p, 2}=(0.4)(0.248)+(0.6)(0.219)=0.231 Btu / lbm -{ }^{\circ} R \).

\( C_{v}=x_{1} C_{v, 1}+x_{2} C_{v, 2}=(0.4)(0.177)+(0.6)(0.157)=0.165 Btu / lbm -{ }^{\circ} R \).

\( k=\frac{C_{p}}{C_{v}}=\frac{0.231}{0.165}=1.4 \).

The isentropic relation provides

\( T_{2}=T_{1}\left(\frac{P_{2}}{P_{1}}\right)^{(k-1) / k}=(530)\left(\frac{60}{14.7}\right)^{(1.4-1) / 1.4}=792^{\circ} R \).

\( \text { For an ideal compressor, } \dot{W}_{\text {comp }}=\dot{m} \Delta h=\dot{m} C_{p} \Delta T=(10 / 60)(0.231)(778)(792-530)=7850 ft – lbf / s \text {, where the } \) factor 778 ft-lbf/Btu provides us with the desired units. If the compressor is 80 percent efficient, the actual power \( \text { is } \dot{W}_{\text {comp }}=7850 / 0.8=9810 ft – lbf / s \text { or } 17.8 hp \text {. } \)

To find the entropy change we need the actual outlet temperature. Using the definition of compressor efficiency,

\( \eta_{\text {comp }}=\frac{w_{s}}{w_{a}}=\frac{C_{p}(\Delta T)_{s}}{C_{p}(\Delta T)_{a}} \).

\( \text { we find } 0.8=(792-530) /\left(T_{2}-530\right) \text { and } T_{2}=857.5^{\circ} R \text {. The change in entropy is then } \).

\( \Delta s=C_{p} \ln \frac{T_{2}}{T_{2}}-R \ln \frac{P_{2}}{P_{1}}=0.231 \ln \frac{857.5}{530}-0.066 \ln \frac{60}{14.7}=0.0183 Btu / lbm -{ }^{\circ} R \).

\( \text { where we have used } R=C_{p}-C_{v}=0.231-0.165=0.066 Btu / lbm -{ }^{\circ} R \text {. } \)
