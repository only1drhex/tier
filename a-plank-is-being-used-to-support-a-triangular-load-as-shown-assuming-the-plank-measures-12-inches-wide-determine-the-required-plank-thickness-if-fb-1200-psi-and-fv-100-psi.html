Guest Question.

A plank is being used to support a triangular load as shown. Assuming the plank measures 12 inches wide, determine the required plank thickness if Fb = 1,200 psi and Fv = 100 psi.

Admin
Solution

Fb = 1200 psi
Fv = 100psi

\(I_{x} = \frac{bh^{3}}{12} = \frac{(12)h^{3}}{12} = h^{3}\)

A = 12h

Bending :

\(f_{b} = \frac{M_{c}}{I_{x}};\)

 

\(1200 = \frac{(980 \times 12)(h/2)}{h^{3}} = \frac{500}{h^{2}} = 4.9in.^{2}\)

h = 2.2″

Shear:

\(f_{v} = \frac{1.5V}{A}; 100 = \frac{1.5(1280)}{12h}\)

 

\(h = \frac{1.5(1280)}{1200} = 1.6^{"}\)
