Guest Question.

Electrical sources such as batteries, solar panels, and fuel cells produce a dc output voltage. An electrical load requiring an ac voltage can be powered from a dc source using a device called an inverter, which converts a dc voltage to an ac voltage. Inverters can produce single-phase or three-phase ac voltages. Single-phase inverters are often classified as pure or true sine wave inverters or modified sine wave inverters. The output from a pure sine wave inverter is shown in Fig. 15.21. This waveform was discussed in Chapter 8 and could be described by v(t) = 170 sin 377t volts. Fig. 15.22 is the output voltage from a modified sine wave inverter. Note that this waveform is more square wave than sine wave.

Admin
Solution

Let’s determine the Fourier components of the modified sine wave inverter output voltage using the waveform in Fig. 15.23. Note that this waveform consists of one positive pulse of width \(t_{δ}\) centered about T/4 and a negative pulse of the same width centered about 3T/4. Close examination of this waveform reveals that it is an odd function with half-wave symmetry. Therefore,

\(a_{0}\) = 0
\(a_{n}\) = 0 for all n
\(b_{n}\) = 0 for n even

We can find \(b_{n}\) for n odd using

\(b_{n} =\frac{4}{T_{0}}\int^{T_{0}/2}_{0} f(t) sin\,nω_{0}\)t dt

The waveform has a value of \(V_{in}\) between t = T/4 – δ/2 and t = T/4 + δ/2 and zero elsewhere over the interval from 0 to T/2. Therefore,

\(b_{n} =\frac{4}{T}\int^{T/4+t_{δ}/2}_{T/4-t_{δ}/2} V_{in} sin\,nω_{0}\) t dt

\(b_{n} =\frac{4V_{in}}{T} \int^{T/4+t_{δ}/2}_{T/4-t_{δ}/2} sin \,nω_{0}\) t dt

Integrating yields

\(b_{n} =\frac{4V_{in}}{nω_{0}T}[-cos nω_{0}t ]^{T/4+t_{δ}/2}_{T/4-t_{δ}/2}\)

Recalling that \(ω_{0}\)T = 2π and evaluating the function at the limits produces

\(b_{n} =\frac{2V_{in}}{\pi} \Big[-cos ( \frac{nω_{0}T}{4} +\frac{nω_{0}t_{δ}}{2} ) + cos (\frac{nω_{0}T}{4} -\frac{nω_{0}t_{δ}}{2} ) \Big] \)

The expression in brackets is -cos(α + β) + cos(α – β). Using the appropriate trigonometric identities, we have

-cos(α + β) + cos(α – β) = -cos α cos β + sin α sin β + cos α cos β+ sin α sin β
-cos(α + β) + cos(α – β) = 2 sin α sin β

The expression for \(b_{n}\), which is valid n odd, becomes

\(b_{n} = \frac{4V_{in}}{n\pi} \Big[ sin (\frac{nω_{0}T}{4} ) sin (\frac{nω_{0}t_{δ}}{2}) \Big]\)

Let’s define \(ω_{0}t_{δ} \) = δ and again utilize \(ω_{0}\)T = 2π:

\(b_{n} = \frac{4V_{in}}{n\pi} \Big[ sin (\frac{nπ}{2} )sin (\frac{nδ}{2} ) \Big] \)

Using this expression,

\(b_{1} =\frac{4V_{in}}{π}sin (\frac{π}{2} )sin (\frac{δ}{2} ) =\frac{4V_{in}}{π}sin (\frac{δ}{2})\)

\(b_{3} =\frac{4V_{in}}{3π}sin (\frac{3π}{2} )sin (\frac{3δ}{2} ) =\frac{4V_{in}}{3π}sin (\frac{3δ}{2})\)

\(b_{5} =\frac{4V_{in}}{5π}sin (\frac{5π}{2} )sin (\frac{5δ}{2} ) =\frac{4V_{in}}{5π}sin (\frac{5δ}{2})\)

Now let’s plot the absolute value of \(b_{1}, b_{3}\), and \(b_{5}\) as δ varies between 0º and 180º for \(V_{in}\) = 1 volt as shown in Fig. 15.24. Note that \(b_{1}\)—the coefficient of the first harmonic or fundamental—is zero for δ = 0° and reaches a maximum value of 4/π = 1.273 volts for δ = 180°. Examination of this plot reveals that the absolute value of the third harmonic is zero for δ = 120°. The expression for \(b_{3}\) contains the term sin(3δ/2), which has a value of zero for δ = 120°. If we chose δ = 72°, the amplitude of the fifth harmonic would be zero. This example illustrates that it is possible to eliminate one harmonic from the Fourier series for the output voltage by proper selection of the angle δ.
