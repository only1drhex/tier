Guest Question.

Consider now the network in Fig. 5.4a. Let us use superposition to find \(V_o\).

Admin
Solution

The contribution of the 6-V source to \(V_{o}\) is found from the network in Fig. 5.4 b, which is redrawn in Fig. 5.4 c. The 2 kΩ+6 kΩ= 8-kΩ resistor and 4-kΩ resistor are in parallel, and their combination is an 8/3-kΩ resistor. Then, using voltage division,

 

\(V_{1}=6\left(\frac{\frac{8}{3} k}{\frac{8}{3} k+2 k}\right)=\frac{24}{7} \mathrm{~V}\)

 

Applying voltage division again,

 

\(V_{o}^{\prime}=V_{1}\left(\frac{6 \mathrm{k}}{6 \mathrm{k}+2 \mathrm{k}}\right)=\frac{18}{7} \mathrm{~V}\)

 

The contribution of the 2-mA source is found from Fig. 5.4 d, which is redrawn in Fig. 5.4 e. \(V_{o}^{*}\) is simply equal to the product of the current source and the parallel combination of the resistors; that is,

 

\(V_{0}^{\prime \prime}=\left(2 \times 10^{-3}\right)\left(\frac{10}{3} \mathrm{k} / / 6 \mathrm{k}\right)=\frac{30}{7} \mathrm{~V}\)

 

Then

 

\(V_{o}=V_{o}^{\prime}+V_{o}^{\prime \prime}=\frac{48}{7} \mathrm{~V}\)

 

A nodal analysis of the network can be performed using Fig. 5.4f. The equation for the supernode is

 

\(-2 \times 10^{-3}+\frac{\left(V_{o}-6\right)-V_{1}}{2 \mathrm{k}}+\frac{V_{o}-V_{1}}{4 \mathrm{k}}+\frac{V_{o}}{6 \mathrm{k}}=0\)

 

The equation for the node labeled \(V_{1}\) is

 

\(\frac{V_{1}-V_{o}}{4 \mathrm{k}}+\frac{V_{1}-\left(V_{o}-6\right)}{2 \mathrm{k}}+\frac{V_{1}}{2 \mathrm{k}}=0\)

 

Solving these two equations, which already contain the constraint equation for the supernode, yields \(V_{o}=48 / 7 \mathrm{~V}\).

 

Once again, referring to the network in Fig. 5.4f, the mesh equations for the network are

 

\(-6+4 \mathrm{k}\left(I_{1}-I_{3}\right)+2 \mathrm{k}\left(I_{1}-I_{2}\right) =0 \)

 

\(I_{2} =2 \times 10^{-3} \)

 

\(2 \mathrm{k}\left(I_{3}-I_{2}\right)+4 \mathrm{k}\left(I_{3}-I_{1}\right)+6 \mathrm{k} I_{3} =0\)

 

Solving these equations, we obtain \(I_{3}=8 / 7 \mathrm{~mA}\) and, hence, \(V_{o}=48 / 7 \mathrm{~V}\).
