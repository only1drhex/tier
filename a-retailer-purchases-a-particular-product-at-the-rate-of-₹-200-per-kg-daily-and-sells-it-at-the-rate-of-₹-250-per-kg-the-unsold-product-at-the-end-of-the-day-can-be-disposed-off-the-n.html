Guest Question.

A retailer purchases a particular product at the rate of ₹ 200 per kg daily and sells it at the rate of ₹ 250 per kg. The unsold product at the end of the day can be disposed off the next day at the salvage value of ₹ 170 per kg. Past sales have ranged from 25 to 28 kg per day. Table 16.8 shows the sales record for the past 100 days.

Table 16.8 Sales record
Amount sold (kg) 25 26 27 28
Number of days 30 40 20 10

Determine the quantity of the product in kg which the retailer should purchase to optimize the
profit.

Admin
Solution

Let \(A _{i}(i=1,2,3,4)\) be the possible alternative strategies, i.e., the quantity of the product purchased in kg and \(S _{j}(j=1,2,3,4)\) be the possible states of nature, i.e., daily demand.

Profit per kg can be defined as:

 

\(\begin{aligned}\text { Profit per } kg &=\text { selling price per } kg -\text { cost per } kg \\&=₹ 250-₹ 200=₹ 50 \text { per kg }\end{aligned}\)

 

In case of unsold product, there would be loss which is:

 

\(\text { Loss per } kg =₹ 200-₹ 170=₹ 30 \text { per } kg\)

 

\(\text { Conditional Profit }=\text { Profit } \times \text { amount sold }( kg )-\text { Loss } \times \text { amount unsold }( kg )\)

 

\(=\left\{\begin{array}{ll}50 A & \text { if } S \geq A \\(250-200) S-30(A-S)=80 S-30 A & \text { if } A>S\end{array}\right.\)

 

The pay-off table is shown in Table 16.9.

Expected monetary value (EMV) is 1,276 because this is the maximum profit among the profits from various alternative strategies. Thus, the daily purchase should be 26kg.
