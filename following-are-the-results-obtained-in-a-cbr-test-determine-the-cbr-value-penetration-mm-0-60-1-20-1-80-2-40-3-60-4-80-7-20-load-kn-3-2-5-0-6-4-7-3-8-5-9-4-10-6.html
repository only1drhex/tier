Guest Question.

Following are the results obtained in a CBR test. Determine the CBR value:

Penetration (mm) 0.60 1.20 1.80 2.40 3.60 4.80 7.20

Load (kN) 3.2 5.0 6.4 7.3 8.5 9.4 10.6

Admin
Solution

Load versus penetration curve is drawn as shown in Fig. 17.7.

Standard Load:
At 2.5 mm … 13.44 kN
At 5.0 mm … 20.16 kN
CBR at 2.5 mm penetration \(=\frac{\text { Test load at } 2.5 mm }{\text { Standard load at } 2.5 mm } \times 100\)

 

\(=\frac{7.4}{13.44} \times 100=55 \%\)

CBR at 5.0 mm penetration \(=\frac{\text { Test load at } 5.0 mm }{\text { Standard load at } 5.0 mm } \times 100\)

 

\(=\frac{9.6}{20.16} \times 100=47.6\)

Since, the CBR is greater at 2.5 mm penetration, it is to be taken.

\(\therefore \quad CBR = 5 5 \% \text {. }\)
