Guest Question.

Consider two particles of mass \( m_{1} \text { and } m_{2} \) (in one dimension) that interact via a potential that depends only on the distance between the particles \( V\left(\left|x_{1}-x_{2}\right|\right) \) , so that the Hamiltonian is

\( \hat{H}=-\frac{\hbar^{2}}{2 m_{1}} \frac{\partial^{2}}{\partial x_{1}^{2}}-\frac{\hbar^{2}}{2 m_{2}} \frac{\partial^{2}}{\partial x_{2}^{2}}+V\left(\left|x_{1}-x_{2}\right|\right) \).

Acting on a two-particle wave function the translation operator would be

\( \hat{T}(a) \psi\left(x_{1}, x_{2}\right)=\psi\left(x_{1}-a, x_{2}-a\right) \).

(a) Show that the translation operator can be written

\( \hat{T}(a)=e^{-\frac{i a}{\hbar} \hat{P}} \),

where \( \hat{P}=\hat{p}_{1}+\hat{p}_{2} \) is the total momentum.

(b) Show that the total momentum is conserved for this system.

Admin
Solution

(a) We can expand \( \hat{T} \) in a Taylor series just as we did in the single-particle case:

\( \hat{T}(a) \psi\left(x_{1}, x_{2}\right)=\psi\left(x_{1}-a, x_{2}-a\right) \).

\( =\sum_{m, n} \frac{1}{m ! n !} \frac{\partial^{m+n}}{\partial x_{1}^{m} \partial x_{2}^{n}} \psi\left(x_{1}, x_{2}\right)(-a)^{m+n} \).

We can write this in terms of the two momenta as

\( \hat{T}(a) \psi\left(x_{1}, x_{2}\right)=\sum_{m n} \frac{1}{m ! n !}\left(-\frac{i a}{\hbar}\right)^{m+n} \hat{p}_{1}^{m} \hat{p}_{2}^{n} \psi\left(x_{1}, x_{2}\right) \).

and then split up the sums as

\( \hat{T}(a) \psi\left(x_{1}, x_{2}\right)=\sum_{m} \frac{1}{m !}\left(-\frac{i a}{\hbar}\right)^{m} \hat{p}_{1}^{m} \sum_{n} \frac{1}{n !}\left(-\frac{i a}{\hbar}\right)^{n} \hat{p}_{2}^{n} \psi\left(x_{1}, x_{2}\right) \).

\( =e^{-i a \hat{p}_{1} / \hbar} e^{-i a \hat{p}_{2} / \hbar} \psi\left(x_{1}, x_{2}\right) \).

\( \text { Finally, since } \hat{p}_{1} \text { and } \hat{p}_{2} \text { commute we can write } \).

\( \hat{T}(a)=e^{-i a \hat{p}_{1} / \hbar} e^{-i a \hat{p}_{2} / \hbar}=e^{-i a\left(\hat{p}_{1}+\hat{p}_{2}\right) / \hbar}=e^{-i a \hat{P} / \hbar} \).

(b) We first need to show that the Hamiltonian is translationally invariant. Acting on a test function \( f\left(x_{1}, x_{2}\right) \):

\( \hat{H}^{\prime} f\left(x_{1}, x_{2}\right)=\hat{T}^{\dagger}(a) \hat{H} \hat{T}(a) f\left(x_{1}, x_{2}\right) \).

\( =\hat{T}(-a)\left[-\frac{\hbar^{2}}{2 m} \frac{\partial^{2}}{\partial x_{1}^{2}}-\frac{\hbar^{2}}{2 m} \frac{\partial^{2}}{\partial x_{2}^{2}}\right. \)

\( \left.+V\left(\left|x_{1}-x_{2}\right|\right)\right] f\left(x_{1}-a, x_{2}-a\right) \).

\( =\left[-\frac{\hbar^{2}}{2 m} \frac{\partial^{2}}{\partial x_{1}^{\prime 2}}-\frac{\hbar^{2}}{2 m} \frac{\partial^{2}}{\partial x_{2}^{\prime 2}}\right. \)

\( \left.+V\left(\left|x_{1}+a-\left(x_{2}+a\right)\right|\right)\right] f\left(x_{1}+a-a, x_{2}+a-a\right) \).

\( \text { where } x_{1}^{\prime}=x_{1}+a \text { and } x_{2}^{\prime}=x_{2}+a \text {. Since } \)

\( \frac{\partial}{\partial x_{1}^{\prime}}=\frac{\partial x_{1}}{\partial x_{1}^{\prime}} \frac{\partial}{\partial x_{1}}=\frac{\partial}{\partial x_{1}} \).

(and similarly for \( x_{2}^{\prime} \) ) we have

\( \hat{H}^{\prime} f(x)=\left[-\frac{\hbar^{2}}{2 m} \frac{\partial^{2}}{\partial x_{1}^{2}}-\frac{\hbar^{2}}{2 m} \frac{\partial^{2}}{\partial x_{2}^{2}}+V\left(\left|x_{1}-x_{2}\right|\right)\right] f\left(x_{1}, x_{2}\right) \).

so that \( \hat{H}^{\prime}=\hat{H} . \) Of course, this is all rather pedantic. Just as in the single-particle case the momentum (and therefore kinetic energy) is invariant under a translation and since the potential in this case depends only on the distance between particles, it is invariant as well.

\( \hat{T}^{\dagger}(\delta) \hat{H} \hat{T}(\delta)=\hat{H} \).

\( \hat{H}+i \frac{\delta}{\hbar} \hat{P} \hat{H}-i \frac{\delta}{\hbar} \hat{H} \hat{P}=\hat{H} \).

\( [\hat{H}, \hat{P}]=0 \).

and therefore, according to the generalized Ehrenfest theorem, the total momentum is conserved.
