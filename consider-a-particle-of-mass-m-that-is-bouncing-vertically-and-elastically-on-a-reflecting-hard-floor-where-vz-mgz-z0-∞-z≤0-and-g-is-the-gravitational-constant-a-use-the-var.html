Guest Question.

Consider a particle of mass m that is bouncing vertically and elastically on a reflecting hard floor where \(V(z)= \begin{cases} mgz, & z>0, \\ +\infty, & z\leq 0,\end{cases} \), and g is the gravitational constant.

(a) Use the variational method to estimate the ground state energy of this particle.

(b) Use the WKB method to estimate the ground state energy of this particle.

(c) Compare the results of (a) and (b) with the exact ground state energy.

Admin
Solution

(a) The ground state wave function of this particle has no nodes and must vanish at z = 0 and be finite as \(z\rightarrow +\infty\). The following trial function satisfies these conditions:

\(\psi _{0} (z,\alpha )=Aze^{-\alpha z} \), (9.363)

where α is a parameter and A is the normalization constant. We can show that \(A=2\alpha ^{3/2} \) and hence

\(\psi _{0} (z,\alpha )=2\sqrt{\alpha ^{3} } ze^{-\alpha z}\) . (9.364)

The energy is given by

\(E^{VM}_{0} (\alpha )=4\alpha ^{3} \int_{0}^{+\infty} ze^{-\alpha z}\left[-\frac{\hbar ^{2} }{2m}\frac{d^{2} }{dz^{2} } +mgz \right] ze^{-\alpha z} dz \)

 

\(=4\alpha ^{3} \frac{\hbar ^{2} }{2m} \int_{0}^{+\infty} \left (2\alpha z-\alpha ^{2}z^{2} \right) e^{-2\alpha z} dz+4\alpha ^{3} mg \int_{0}^{+\infty} z^{3} e^{-2\alpha z}dz \)

 

\(=2\alpha ^{3} \frac{\hbar ^{2} }{m} \left(\frac{1}{2\alpha } -\frac{1}{4\alpha } \right) +4mg\alpha ^{3} \frac{3}{8\alpha ^{4} } \), (9.365)

or

\(E^{V M}_{0} (\alpha )=\frac{\hbar ^{2} }{2m} \alpha ^{2} + \frac{3}{2\alpha } mg\). (9.366(

The minimization of \(E_{0} (\alpha )\) yields \(\alpha _{0} =(3m^{2} g/2\hbar ^{2} )^{1/3} \) and hence

\(E^{V M}_{0} (\alpha _{0} )=\frac{3}{2} \left(\frac{9}{2} \right) ^{1/3} \left(\frac{1}{2}mg^{2} \hbar ^{2} \right) ^{1/3} \). (9.367)

(b) Since this potential has one rigid wall at x = 0, the correct quantization rule is given by (9.224): \(\int_{0}^{E/mg} pdz=(n+\frac{3}{4})\pi \hbar \);

\(\int_{x_{1} }^{x_{2} } p(x)dx=\left(n+\frac{3}{4} \right) \pi \hbar , n=0, 1, 2, 3, lcdots\). (9.224)

the turning point occurs at E = mgz and hence z = E/mg. Now, since \(E=p^{2} /2m+mgz\) we have \(p(E,z)=\sqrt{2mE} \sqrt{1-mgz/E} \), and therefore

\(\int_{0}^{E/mg} p(E_{z} ,z)dz=\sqrt{2mE} \int_{0}^{E/mg} \sqrt{1-\frac{mg}{E} z} dz=\sqrt{2mE} \frac{2E}{3mg} \sqrt{\frac {8E^{3} }{9mg^{2} } } \). (9.368)

Inserting this relation into the quantization condition \(\int_{0 }^{E/mg} pdz=(n+\frac{3}{4})\pi \hbar \) gives

\(\sqrt{\frac{8E^{3} }{9mg^{2} } } =\left(n+\frac{3}{4} \right) \pi \hbar \), (9.369)

and we obtain the WKB approximation for the energy:

\(E^{W K B}_{n} =\left[\frac{9\pi ^{2} }{8} mg^{2} \pi ^{2} \hbar ^{2} \left(n+\frac{3}{4} \right)^{2} \right] ^{1/3} \). (9.370)

Hence the ground state energy is given by

\(E^{W K B}_{0} =\frac{3}{4} \left(3\pi ^{2} \right) ^{1/3} \left (\frac{1}{2}mg^{2} \hbar ^{2} \right) ^{1/3} \). (9.371)

(c) Recall that the exact ground state energy, calculated in , page 275, for a particle of mass m moving in the potential V(z) = mgz is given by

\(E^{exact}_{0} =2.338\left(\frac{1}{2}mg^{2} \hbar ^{2} \right) ^{1/3} \). (9.372)

Combining this relation with (9.367) and (9.371), we see that the variational method overestimates the energy by a 5.9% error, while the WKB method underestimates it by a 0.8% error:

\(E^{V M}_{0} =\frac{3}{2} \left(\frac{9}{2} \right) ^{1/3} \frac {E^{exact}_{0} }{2.338} \simeq 1.059 E^{exact}_{0} \), (9.373)

 

\(E^{W K B}_{0} =\frac{3}{4} \left(3\pi ^{2} \right) ^{1/3} \frac{E^{exact}_{0} }{2.338} \simeq 0.992 E^{exact}_{0} \). (9.374)

The variational method has given a reasonably accurate result because we succeeded quite well in selecting the trial function. As for the WKB method, it has given a very accurate result because we have used the correct quantization rule (9.224). Had we used the quantization rule (9.210),

\(\oint p(x,E_{n} ) dx=2\int_{x_{1} }^{x_{2} } \sqrt{2m(E_{n} -V(x))}dx= \left(n+\frac{1}{2} \right)h\), (9.210)

which contains a factor of \(\frac{1}{2} \) instead of \( \frac{1}{4} \) in (9.224), the WKB method would have given a very inaccurate result with a 24.3% error, \(i.e.,E^{WKB}_{0} \simeq 0.757 E^{exact}_{0} \).
