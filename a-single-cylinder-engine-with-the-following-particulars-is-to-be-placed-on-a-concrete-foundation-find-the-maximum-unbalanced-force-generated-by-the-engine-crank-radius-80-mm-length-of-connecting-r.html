Guest Question.

A single cylinder engine with the following particulars is to be placed on a concrete foundation. Find the maximum unbalanced force generated by the engine:

crank radius = 80 mm

Length of connecting rod = 280 mm

operating frequency = 1800 rpm

Weight of reciprocating parts = 49 N

Admin
Solution
\(P_{z}=\left(M_{\text {rec }}+M_{\text {rot }}\right) R \omega^{2} \cos \omega t+M_{\text {rec }} \frac{R^{2} \omega^{2}}{L} \cos 2 \omega t\)

The maximum value is

\(\begin{aligned}P_{z} &=M_{ rec } R \omega^{2}+M_{ rec } \frac{R^{2} \omega^{2}}{L}, \text { approximately } \\&=M_{ rec } R \omega^{2}\left(1+\frac{R}{L}\right) \\f_{n} &=1800 rpm =\frac{1800}{60} cps =30 cps \\\omega_{n} &=2 \pi f_{n}=2 \pi \times 30 rad / s\end{aligned}\)

Substituting this and \(R=80 \times 10^{-3} m , L=280 \times 10^{-3} m\) , and

\(M_{ rec }=\frac{49}{9.81} kN \sec ^{2} / m\)

we have \(P_{z}=\frac{49 \times 80}{9.81 \times 10^{3}}\left(1+\frac{80}{280}\right)(2 \times 30 \pi)^{2} N\)

= 18,254 N = 18.25 kN.
