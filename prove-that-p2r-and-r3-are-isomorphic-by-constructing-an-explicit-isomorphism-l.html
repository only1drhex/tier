Guest Question.

Prove that \(P_{2} \left(\mathbb{R} \right)\) and \(\mathbb{R}^3\) are isomorphic by constructing an explicit isomorphism L.

Admin
Solution

We define \(L :P_{2} \left(\mathbb{R} \right) \rightarrow \mathbb{R}^3\) by \(L(a_{0} + a_{1}x + a_{2}x^2) = \left [ \begin{matrix} a_{0} \\ a_{1} \\ a_{2} \end{matrix} \right ]\).

We need to prove that L is linear, one-to-one, and onto.

Linear: Let \(\mathbf{p} (x) = a_{0} + a_{1}x + a_{2}x^2, \mathbf{q}(x) = b_{0} + b_{1}x + b_{2}x^2 \in P_{2} \left(\mathbb{R} \right)\), and let \(s, t \in\mathbb{R} \).
Then, L is linear since

\(L(s\mathbf{p} + t\mathbf{q}) = L(sa_{0} + tb_{0} + (sa_{1} + tb_{1})x + (sa_{2} + tb_{2})x^2)\)

\(= \left [ \begin{matrix} sa_{0} + tb_{0} \\ sa_{1} + tb_{1} \\ sa_{2} + tb_{2} \end{matrix} \right ]\)

\(= s \left [ \begin{matrix} a_{0} \\ a_{1} \\ a_{2} \end{matrix} \right ] + t \left [ \begin{matrix} b_{0} \\ b_{1} \\ b _{2} \end{matrix} \right ] = sL(\mathbf{p}) + tL(\mathbf{q})\)

One-to-one: Let \(a_{0} + a_{1}x + a_{2}x^2 \in Null(L)\). Then, \(\left [ \begin{matrix} 0 \\ 0 \\ 0 \end{matrix} \right ] = L(a_{0} + a_{1}x + a_{2}x^2) = \left [ \begin{matrix} a_{0} \\ a_{1} \\ a_{2} \end{matrix} \right ]\).

Hence, \(a_{0} = a_{1} = a_{2} = 0\), so \(Null(L) = \left\{0\right\}\) and thus L is one-to-one by Theorem 4.7.1 (A linear mapping \( L : \mathbb{U} \rightarrow \mathbb{V}\) is one-to-one if and only if \(Null(L) = \left\{0\right\})\).

Onto: For any \(\left [ \begin{matrix} a_{0} \\ a_{1} \\ a_{2} \end{matrix} \right ] \in \mathbb{R} ^3\), we have \(L(a_{0} + a_{1}x + a_{2}x^2) = \left [ \begin{matrix} a_{0} \\ a_{1} \\ a_{2} \end{matrix} \right ]\). Hence, L is onto.

Thus, L is an isomorphism from \(P_{2}(\mathbb{R} )\) to \(\mathbb{R} ^3\). Hence \(P_{2}(\mathbb{R} )\) and \(\mathbb{R} ^3\) are isomorphic.
