Guest Question.

Consider the experimental observations given in the following table:

t 1 4 8 11
y 1 2 4 5

Find the least-squares linear fit to the data.

Admin
Solution

For the function defined by y = mt + c, the data lead to the overdetermined system
m + c = 1
4m + c = 2
8m + c = 4
11m + c = 5.
In matrix terms, the system is Ax = b, where

\(A=\left[\begin{array}{ll}1 & 1 \\4 & 1 \\8 & 1 \\11 & 1\end{array}\right], \quad x =\left[\begin{array}{c}m \\c\end{array}\right], \quad \text { and } \quad b =\left[\begin{array}{c}1 \\2 \\4 \\5\end{array}\right]\).

The least-squares solution, \(X ^{*}\), is found by solving \(A^{T} A x =A^{T} b\), where

\(A^{T} A=\left[\begin{array}{rr}202 & 24 \\24 & 4\end{array}\right] \text { and } A^{T} b =\left[\begin{array}{l}96 \\12\end{array}\right]\).

There is a unique solution to \(A^{T} A x =A^{T} b\) because A has rank 2. The solution is

\(x ^{*}=\left[\begin{array}{l}12 / 29 \\15 / 29\end{array}\right]\).

Thus the least-squares linear fit is defined by

\(y=\frac{12}{29} t+\frac{15}{29}\).
