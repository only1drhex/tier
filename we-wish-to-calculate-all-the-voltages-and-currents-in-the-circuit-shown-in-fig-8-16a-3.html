Guest Question.

We wish to calculate all the voltages and currents in the circuit shown in Fig. 8.16a,

Admin
Solution

Our approach will be as follows. We will calculate the total impedance seen by the source \(\mathbf{V}_{S}\). Then we will use this to determine \(\mathbf{I}_{1}\). Knowing \(\mathbf{I}_{1}\), we can compute \(\mathbf{V}_{1}\) using KVL. Knowing \(\mathbf{V}_{1}\), we can compute \(\mathbf{I}_{2}\) and \(\mathbf{I}_{3}\), and so on.

The total impedance seen by the source \(\mathbf{V}_{S}\) is
\(\begin{aligned}\mathbf{Z}_{\mathrm{eq}} &=4+\frac{(j 6)(8-j 4)}{j 6+8-j 4} \\\\&=4+\frac{24+j 48}{8+j 2} \\\\&=4+4.24+j 4.94 \\\\&=9.61 \underline{/30.94^{\circ}} \Omega\end{aligned}\)

 

Then

 

\(\begin{aligned}\mathbf{I}_{1} &=\frac{\mathbf{V}_{S}}{\mathbf{Z}_{\mathrm{eq}}}=\frac{24 \underline{/ 60^{\circ}}}{9.61 \underline{/ 30.94^{\circ}}} \\\\&=2.5 \underline{/ 29.06^{\circ}} \mathrm{A}\end{aligned}\)

 

\(\mathbf{V}_{1}\) can be determined using KVL:

 

\(\begin{aligned}\mathbf{V}_{1} &=\mathbf{V}_{S}-4 \mathbf{I}_{1} \\\\&=24 \underline{/60^{\circ}}-10 \underline{/ 29.06^{\circ}} \\\\&=3.26+j 15.93 \\\\&=16.26 \underline{/ 78.43^{\circ}} \mathrm{V}\end{aligned}\)

 

Note that \(\mathbf{V}_{1}\) could also be computed via voltage division:

 

\(\mathbf{V}_{1}=\frac{\mathbf{V}_{S} \frac{(j 6)(8-j 4)}{j 6+8-j 4}}{4+\frac{(j 6)(8-j 4)}{j 6+8-4}} \mathrm{~V}\)

 

which from our previous calculation is

 

\(\begin{aligned}\mathbf{V}_{1} &=\frac{\left(24 \underline{/ 60^{\circ}}\right)\left(6.51 \underline{/ 49.36^{\circ}}\right)}{9.61 \underline{/ 30.94^{\circ}}} \\\\&=16.26 \underline{/ 78.42^{\circ}} \mathrm{V}\end{aligned}\)

 

Knowing \(\mathbf{V}_{1}\), we can calculate both \(\mathbf{I}_{2}\) and \(\mathbf{I}_{3}\) :

 

\(\begin{aligned}\mathbf{I}_{2} &=\frac{\mathbf{V}_{1}}{j 6}=\frac{16.26 \underline{/78.43^{\circ}}}{6 \underline{/ 90^{\circ}}} \\\\&=2.71 \underline{/-11.58^{\circ}} \mathrm{A}\end{aligned}\)

 

and

 

\(\begin{aligned}\mathbf{I}_{3} &=\frac{\mathbf{V}_{1}}{8-j 4} \\\\&=1.82 \underline{/ 105^{\circ}} \mathrm{A}\end{aligned}\)

 

Note that \(\mathbf{I}_{2}\) and \(\mathbf{I}_{3}\) could have been calculated by current division. For example, \(\mathbf{I}_{2}\) could be determined by

 

\(\begin{aligned}\mathbf{I}_{2} &=\frac{\mathbf{I}_{1}(8-j 4)}{8-j 4+j 6} \\\\&=\frac{\left(2.5 \underline{/ 29.06^{\circ}}\right)\left(8.94 \underline{/-26.57^{\circ}}\right)}{8+j 2} \\\\&=2.71 \underline{/-11.55^{\circ}} \mathrm{A}\end{aligned}\)

 

Finally, \(\mathbf{V}_{2}\) can be computed as

 

\(\begin{aligned}\mathbf{V}_{2} &=\mathbf{I}_{3}(-j 4) \\\\&=7.28 \underline{/ 15^{\circ}} \mathrm{V}\end{aligned}\)

 

This value could also have been computed by voltage division. The phasor diagram for the currents \(\mathbf{I}_{1}, \mathbf{I}_{2}\), and \(\mathbf{I}_{3}\) is shown in Fig. 8.16b and is an illustration of KCL.

 

Finally, the reader is encouraged to work the problem in reverse; that is, given \(\mathbf{V}_{2}\), find \(\mathbf{V}_{S}\). Note that if \(\mathbf{V}_{2}\) is known, \(\mathbf{I}_{3}\) can be computed immediately using the capacitor impedance. Then \(\mathbf{V}_{2}+\mathbf{I}_{3}\) (8) yields \(\mathbf{V}_{1}\). Knowing \(\mathbf{V}_{1}\) we can find \(\mathbf{I}_{2}\). Then \(\mathbf{I}_{2}+\mathbf{I}_{3}=\mathbf{I}_{1}\), and so on. Note that this analysis, which is the subject of Learning Assessment E8.12, involves simply a repeated application of Ohm’s law, KCL, and KVL.

 

 

[ hint ]

Technique
1. Compute \(I_1\).

 

2. Determine \(\mathbf{V}_{1}=\mathbf{V}_{s}-\mathbf{I}_{1} \mathbf{Z}_{1}\)

Then \(I_{2}=\frac{V_{1}}{Z_{2}}\) and \(I_{3}=\frac{V_{1}}{Z_{3}}\)

 

Current and voltage division are also applicable.
