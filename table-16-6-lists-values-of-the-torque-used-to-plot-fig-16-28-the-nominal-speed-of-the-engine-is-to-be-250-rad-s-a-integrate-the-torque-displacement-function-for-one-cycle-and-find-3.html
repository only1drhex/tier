Guest Question.

Table 16–6 lists values of the torque used to plot Fig. 16–28. The nominal speed of the engine is to be 250 rad/s.

(a) Integrate the torque-displacement function for one cycle and find the energy that can be delivered to a load during the cycle.

(b) Determine the mean torque \(T_{m}\) (see Fig. 16–28).

(c) The greatest energy fluctuation is approximately between θ = 15◦ and θ = 150◦ on the torque diagram; see Fig. 16–28 and note that \(T_{o}=-T_{m}\). Using a coefficient of speed fluctuation \(C_{S}=0.1\), find a suitable value for the flywheel inertia.

(d) Find \(\omega_{2} \text { and } \omega_{1}\).

Admin
Solution

(a) Using n = 48 intervals of \(\Delta \theta=4 \pi / 48\), numerical integration of the data of Table 16–6 yields E = 3368 in · lbf. This is the energy that can be delivered to the load.

(b)

 

\(T_{m}=\frac{3368}{4 \pi}=268 lbf \cdot in\)

 

(c) The largest positive loop on the torque-displacement diagram occurs between \(\theta=0^{\circ} \text { and } \theta=180^{\circ}\). We select this loop as yielding the largest speed change.

Subtracting 268 lbf · in from the values in Table 16–6 for this loop gives, respectively, −268, 2532, 1822, 2162, 1892, 1572, 1322, 942, 798, 535, 264, −84, and −268 lbf · in.

Numerically integrating \(T-T_{m}\) with respect to θ yields \(E_{2}-E_{1}\) = 3531 lbf · in. We now solve Eq. (16–64) for I. This gives

 

\(E_{2}-E_{1}=C_{s} I \omega^{2}\) (16–64)

 

\(I=\frac{E_{2}-E_{1}}{C_{s} \omega^{2}}=\frac{3531}{0.1(250)^{2}}=0.565 lbf \cdot s ^{2} \text { in }\)

 

(d) Equations (16–62) and (16–63) can be solved simultaneously for \(\omega_{2} \text { and } \omega_{1}\).

 

\(C_{s}=\frac{\omega_{2}-\omega_{1}}{\omega}\) (16–62)

 

\(\omega=\frac{\omega_{2}+\omega_{1}}{2}\) (16–63)

 

Substituting appropriate values in these two equations yields

 

\(\omega_{2}=\frac{\omega}{2}\left(2+C_{s}\right)=\frac{250}{2}(2+0.1)=262.5 rad / s\)

 

\(\omega_{1}=2 \omega-\omega_{2}=2(250)-262.5=237.5 rad / s\)

 

These two speeds occur at \(\theta=180^{\circ} \text { and } \theta=0^{\circ}\), respectively.
