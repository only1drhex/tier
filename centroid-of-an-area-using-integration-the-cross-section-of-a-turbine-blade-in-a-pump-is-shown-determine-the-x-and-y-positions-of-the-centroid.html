Centroid of an Area Using Integration

The cross section of a turbine blade in a pump is shown. Determine the x and y positions of the centroid.

Admin
Solution

Road Map The cross section shape of the turbine blade is an area, and thus Eq. (7.9) on p. 397 will be used to determine the x and y positions of its centroid. In Eq. (7.9), three ingredients are needed: expressions for \(dA, \widetilde{x}\), and \(\widetilde{y}\). These expressions can be developed using either a vertical area element or a horizontal area element, and solutions using both of these are demonstrated.

Solution 1 â€“ vertical area element

Governing Equations & Computation To evaluate Eq. (7.9), expressions for \(dA, \widetilde{x}\), and \(\widetilde{y}\) are needed, and these can be developed using the vertical area element shown in Fig. 2 as follows.

\(dA = (y_t – y_b) dx = (3\sqrt{3} – \frac{3}{5}x ) dx\) , (1)

\( \widetilde{x} = x \) (2)

\(\widetilde{y} = \frac{1}{2}(y_t + y_b) = \frac{1}{2}(3\sqrt{3} + \frac{3}{5}x )\). (3)

Substituting Eqs. (1) through (3) into Eq. (7.9), the position of the centroid is given by

\(\overline{x} = \frac{\int{\widetilde{x} dA} }{\int{dA} } = \frac{\int_{0}^{25 mm}{x (3\sqrt{3} – \frac{3}{5}x ) dx} }{\int_{0}^{25 mm}{(3\sqrt{3} + \frac{3}{5}x )} } = \frac{(\frac{6x^{5/2}}{5} – \frac{x^3}{5})|^{25mm}_0}{(2x^{3/2} – \frac{3x^2}{10})|^{25mm}_{0}}\)

 

\(=\frac{625 mm^3}{62.5 mm^2} = 10 mm\), (4)

\(\overline{y} = \frac{\int{\widetilde{y} dA} }{\int{dA} } = \frac{\int_{0}^{25 mm}{\frac{1}{2}(3\sqrt{3} + \frac{3}{5}x )(3\sqrt{3} – \frac{3}{5}x ) dx} }{\int_{0}^{25 mm}{(3\sqrt{3} + \frac{3}{5}x )} } = \frac{(\frac{9x^{2}}{4} – \frac{3x^3}{50})|^{25mm}_0}{(2x^{3/2} – \frac{3x^2}{10})|^{25mm}_{0}}\)

 

\(=\frac{468.75 mm^3}{62.5 mm^2} = 7.5 mm\). (5)

Solution 2 â€“ horizontal area element

Governing Equations & Computation The centroid can also be determined using the horizontal area element shown in Fig. 3. Since integrations will be over y, it is necessary to express the shape of the object as functions of y. Thus, the equation for the bottom curve \(y_b = (3/5)x\) is rearranged to obtain \(x_r = (5/3)y\). Similarly, the equation for the top curve \(y_t = 3\sqrt{x}\) is rearranged to obtain \(x_l = y^2/9\). Then expressions for \(dA, \widetilde{x}\), and \(\widetilde{y}\) can be written as follows:

\(dA = (x_r – x_l) dy = (\frac{5}{3}y – \frac{y^2}{9}) dy\) , (6)

\(\widetilde{x} = \frac{1}{2} (x_r + x_l ) = \frac{1}{2}(\frac{5}{3}y + \frac{y^2}{9})\) , (7)

\(\widetilde{y} = y\). (8)

\(\overline{x} = \frac{\int{\widetilde{x}dA } }{\int{dA} } = \frac{\int\limits_{0}^{15 mm}{\frac{1}{2}(\frac{5}{3}y+\frac{y^{2} }{9})(\frac{5}{3}y-\frac{y^{2} }{9} )dy } }{\int\limits_{0}^{15 mm}{(\frac{5}{3}y-\frac{y^{2} }{9} )dy }} =\frac{(\frac{25y^{3} }{54}-\frac{y^5}{810} )|^{15 mm}_{0} }{(\frac{5y^{2} }{6}-\frac{y^3}{27})|^{15 mm}_{0}} \)

 

\(=\frac{625 mm^3}{62.5 mm^2} = 10 mm\), (9)

\(\overline{y} = \frac{\int{\widetilde{y}dA } }{\int{dA} } = \frac{\int\limits_{0}^{15 mm}{y(\frac{5}{3}y-\frac{y^{2} }{9} )dy } }{\int\limits_{0}^{15 mm}{(\frac{5}{3}y-\frac{y^{2} }{9} )dy }} =\frac{(\frac{5y^{3} }{9}-\frac{y^4}{36} )|^{15 mm}_{0} }{(\frac{5y^{2} }{6}-\frac{y^3}{27})|^{15 mm}_{0}} \)

 

\(=\frac{468.75 mm^3}{62.5 mm^2} = 7.5 mm\). (10)

As expected, the solutions for both \(\overline{x}\) and \(\overline{y}\) agree with those obtained earlier in Eqs. (4) and (5).

Discussion & Verification

â€¢ The location of the centroid is shown as point C in Fig. 4, and this location appears to be reasonable.

â€¢ The solution procedure outlined here can be used for a wide variety of area centroid problems by simply using the appropriate expressions for the bounding curves \(y_t\) and \(y_b\), or \(x_r\) and \(x_l\) . Specifically:

If a vertical area element is used, then expressions for the top and bottom curves \(y_t\) and \(y_b\), respectively, as functions of x are needed, and Eqs. (1) through (3) can be used to obtain expressions for \(dA, \widetilde{x}\), and \(\widetilde{y}\) .

If a horizontal area element is used, then expressions for the right and left curves \(x_r\) and \(x_l\) , respectively, as functions of y are needed, and Eqs. (6) through (8) can be used to obtain expressions for \(dA, \widetilde{x}\), and \(\widetilde{y}\) .

â€¢ For some problems, a vertical area element may be more convenient than a horizontal area element, or vice versa. For example, for the area shown in Fig. 5, a vertical area element will be more convenient because both \(y_t\) and \(y_b\) are given by single equations. To use a horizontal area element, two different equations are needed for \(x_r\) , depending on where the area element is located.

â€¢ Software such as Mathematica and Maple can make evaluation of integrals easy, but obtaining correct results with such software requires that you provide the correct integrand and limits of integration.
