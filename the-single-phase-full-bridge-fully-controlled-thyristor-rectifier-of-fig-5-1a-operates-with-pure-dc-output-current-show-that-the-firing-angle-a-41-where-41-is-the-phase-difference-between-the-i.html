Guest Question.

The single-phase full-bridge fully controlled thyristor rectifier of Fig. 5.1(a) operates with pure dc output current. Show that the firing angle \(\alpha=\varphi_{1}, \text { where } \varphi_{1}\) is the phase difference between the input voltage and current fundamental components.

Admin
Solution

Solution

According to Fig. 5.7 the power supply voltage and current waveforms will be:

Since the output current is pure dc, the average output voltage is given by:

 

\(\overline{ V }_{ o }=\frac{2 \sqrt{2} \widetilde{ V }_{ i }}{\pi} \cos \alpha\) (1)

 

Furthermore, according to input current waveform, which is a square wave function, the rms value of the fundamental harmonic component is:

 

\(\widetilde{ I }_{ i , 1}=\frac{4 \overline{ I }_{ o }}{\pi \sqrt{2}}=\frac{2 \sqrt{2} \overline{ I }_{ o }}{\pi}\) (2)

 

Assuming that the rectifier is ideal then:

 

\(P _{ i }=\overline{ P }_{ o } \text { or } P _{ i }=\widetilde{ V }_{ i } \widetilde{ I }_{ i , 1} \cos \varphi_{1}=\overline{ V }_{ o } \overline{ I }_{ o }\) (3)

 

where

 

\(\overline{ I }_{ O }=\frac{\overline{ V }_{ o }}{ R }\) (4)

 

\(\widetilde{ I }_{ i , 1}=\) rms value of the input current fundamental component

Substituting Eqs. (1) and (2) into Eq. (3) yields:

 

\(\begin{aligned}&\widetilde{ V }_{ i }\left(\frac{2 \sqrt{2} \overline{ I }_{ o }}{\pi}\right) \cos \varphi_{1}=\left(\frac{2 \sqrt{2} \widetilde{ V }_{ i }}{\pi} \cos \alpha\right)\left(\overline{ I }_{ o }\right) \\&\text { or } \quad \cos \varphi_{1}=\cos \alpha \quad \text { or } \quad \varphi_{1}=\alpha\end{aligned}\) (5)
