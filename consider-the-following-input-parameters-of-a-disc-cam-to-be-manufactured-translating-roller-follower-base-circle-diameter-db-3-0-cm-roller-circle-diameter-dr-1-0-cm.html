• translating roller follower
• base circle diameter db = 3.0 cm
• roller circle diameter dr = 1.0 cm
• offset S = 0.6 cm

\(r_{b}= \frac{d_{b}}{2} = \frac{3.0}{2} = 1.5 cm \); \(r_{r}= \frac{d_{r}}{2} = \frac{1.0}{2} =0.5 cm\)

(a) the cam profile
