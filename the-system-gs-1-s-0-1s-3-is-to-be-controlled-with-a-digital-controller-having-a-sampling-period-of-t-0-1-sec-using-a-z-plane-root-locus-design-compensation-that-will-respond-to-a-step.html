Guest Question.

The system

\(G(s)=\frac{1}{(s+0.1)(s+3)}\)

is to be controlled with a digital controller having a sampling period of T = 0.1 sec. Using a z-plane root locus, design compensation that will respond to a step with a rise time \(t_{r} \leq 1 sec\) and an overshoot \(M_{p} \leq 5 \%\). What can be done to reduce the steady-state error?

Admin
Solution

(a) Continuous plant :

\(G(s)=\frac{1}{(s+0.1)(s+3)}\), Type 0 system

Discrete model of G(s) preceeded by a ZOH (T = 0.1 sec) :

\(G(z)=0.0045 \frac{z+0.9019}{(z-0.7408)(z-0.99)}\)

Specifications :

\(t_{r} \leq 1 sec \longrightarrow \omega_{n} \geq 1.8 rad / sec\)

 

\(M_{p} \leq 5 \% \longrightarrow \zeta \geq 0.7\)

Discrete design : A simple proportional feedback, D(z) = K = 4.0, will bring the closed-loop poles to :

\(z=0.8564 \pm 0.1278 j\)

which are inside the specs region.

\(\omega_{n}=2.07 rad / sec , \zeta=0.70\)

Root locus :

Step response :
The step response shows that :

\(t_{r} \cong 1.02 sec\)

 

\(M_{p} \cong 4.7 \%\)

However, since the system is type 0, steady-state error exists and is 7% in this case. An integral control of the form,

\(D(z)=\frac{K}{T_{I}} \frac{T z}{z-1}\)

can be added to the proportional control to reduce the steady-state error, but this typically occurs at the cost of reduced stability.
