Guest Question.

Work Moody’s problem, , backward to find the wall roughness ε if everything else is known: V = 6 ft/s, d = 0.5 ft, L = 200 ft, ρ = 1.94 slug/\(ft^3\), µ = 2.09E-5 slug/ft-s, \(h_f\) = 4.5 ft.

Admin
Solution

• Analytic solution: This is not as bad as having the diameter unknown, because ε appears in only one parameter, ε/d. We can immediately calculate Q, \(Re_d\), and the friction factor:

\(Q = V \pi R^2 = (6.0 ft/s)\pi(0.25 ft)^2 = 1.18 ft^3/s\)

 

\(Re_d = \frac{\rho Vd}{\mu} = \frac{(1.94 slug/ft^3)(6 ft/s)(0.5 ft)}{2.09e – 5 slug/ft – s} = 278,500\)

 

\(f = \frac{h_f}{(L/d)(V^2/2g)} = \frac{4.5 ft}{(200 ft/0.5 ft)[(6 ft/s)^2/2/(32.2 ft/s^2)]} = 0.0201\)

With f and \(Re_d\) known, we look on the Moody chart or solve Eq. (6.48) for the roughness ratio:

\(\frac{1}{\sqrt{f}} = – 2.0 \log_{10} \left(\frac{\epsilon /d}{3.7} + \frac{2.51}{Re_d \sqrt{f}}\right) or \frac{1}{\sqrt{0.0201}} = -2.0 \log_{10} \left(\frac{\epsilon /d}{3.7} + \frac{2.51}{278500 \sqrt{0.0201}}\right)\)

After a bit of ugly manipulation, we calculate ε/d = 0.000871, or ε ≈ 0.000435 ft.

• EES solution: Simply type in the data, in BG units (ft, s, lbf, slugs):

rho = 1.94 mu = 2.09E-5 d = 0.5 V = 6.0 L = 200 hf = 8.0 g = 32.2

Then type in the same five defining formulas for pipe flow that we used in Example 6.11:

Re = rho*V*d/mu
Q = V*pi*d^2/4
f = (-2.0*log10 (epsod/3.7 + 2.51/Re/f^.5))^(-2)
hf = f*L/d*V^2/2/g
epsod = eps/d

With any reasonable guess for ε > 0, EES promptly returns ε ≈ 0.000435 ft.

• Comments: Finding the roughness is not as hard as finding the diameter. The discrepancy from Moody’s value of ε = 0.00040 ft was caused by rounding off \(h_f\) to 4.5 ft.
