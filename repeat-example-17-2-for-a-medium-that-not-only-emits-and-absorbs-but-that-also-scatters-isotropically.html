Guest Question.

Repeat for a medium that not only emits and absorbs, but that also scatters isotropically

Admin
Solution

We shall still limit ourselves to a single gas zone even though this will no longer render the analysis exact: For that, the radiative source term, 4(1-w)\(E_{b}\) +wG, must be constant across the volume zone.
It follows immediately that the direct exchange areas remain unchanged (except that \(τ_{L}\) =βL is now based on the extinction coefficient). From equations (17.80) \(Q_{si}=\epsilon _{i}A_{i}E_{bsi}-\sum\limits_{j=1}^{N}{\overline{S_{i}S_{j}}E_{bsj} }- \sum\limits_{k=1}^{K}{\overline{S_{i}G_{k}}E_{bgk} }, i= 1,2,…,N\) and (17.86) \(\sum\limits_{j=1}^{N}{}\overline{S_{i}S_{j}}+ \sum\limits_{k=1}^{K}{}\overline{S_{i}G_{k}} =\epsilon _{i}A_{i} , i= 1,2,…,N\) it follows that the nondimensional heat flux at the surface is still evaluated as

\(\Psi =\frac{q_{w}}{\sigma (T^{4}_{w}-T^{4}_{m} )} =\widetilde{SG},\)

and that only the evaluation of \(\widetilde{SG}\) from the direct exchange areas is different from the previous example.
While the tensors T and Q remain unchanged, we still need to evaluate U, V, W, and X for the evaluation of \(\widetilde{SG}\). Thus,

\(U=\frac{\widetilde{sg}w}{4\tau _{L}(1-w)}\left\lgroup\begin{matrix}1 \\ 1 \end{matrix} \right\rgroup ;\) \(V = \overline{sg}(1-w) \left\lgroup\begin{matrix}1 \\ 1 \end{matrix} \right\rgroup ;\)
\(W= \left(\frac{4\tau _{L}-w\widetilde{gg} }{4\tau _{L}(1-w)} \right) ;\) \(X=\left(\widetilde{gg}(1-w)\right) ;\)

where W and X are single-term 1 × 1 matrices. Thus,

\(W^{-1}= (1/W) and D_{ij}=V_{ij}+\sum\limits_{k=1}^{1}{}\sum\limits_{l=1}^{1}{}U_{ik}W^{-1}_{kl}X_{ij},\)

or

\(D= \left[\overline{sg}(1-w)+\frac{\widetilde{sg}w}{4\tau _{L}(1-w)}\times \frac{4\tau _{L}(1-w)}{4\tau _{L}-w\widetilde{gg} } \times \widetilde{gg} (1-w) \right] \left\lgroup\begin{matrix} 1 \\ 1 \end{matrix} \right\rgroup = \frac{4\tau _{L}(1-w)\widetilde{sg}}{4\tau _{L}-w\widetilde{gg} } \left\lgroup\begin{matrix} 1 \\ 1 \end{matrix} \right\rgroup\)

Similarly,

\(P_[ij]=T_{ij}-\frac{U_{i1} Q_{1j}}{W} = T_{ij} – \frac{\widetilde{sg}w}{4\tau _{L}(1-w)}\times \frac{4\tau _{L}(1-w)}{4\tau _{L}-w\widetilde{gg}}\times \frac{\rho }{\epsilon } \widetilde{sg}\)
\(=T_{ij}-\frac{\rho }{\epsilon } \frac{w(\widetilde{sg} )^{2}}{4\tau _{L}-w\widetilde{gg}},\)

or

\(P=\left\lgroup\begin{matrix} \frac{1}{\epsilon }- \frac{\rho }{\epsilon } \frac{w(\widetilde{sg} )^{2}}{4\tau _{L}-w\widetilde{gg}} &-\frac{\rho }{\epsilon } \left[\widetilde{s_{1}s_{2}} + \frac{w(\widetilde{sg} )^{2}}{4\tau _{L}-w\widetilde{gg}} \right] \\-\frac{\rho }{\epsilon } \left[\widetilde{s_{1}s_{2}} + \frac{w(\widetilde{sg} )^{2}}{4\tau _{L}-w\widetilde{gg}} \right] & \frac{1}{\epsilon }- \frac{\rho }{\epsilon } \frac{w(\widetilde{sg} )^{2}}{4\tau _{L}-w\widetilde{gg}} \end{matrix} \right\rgroup ,\)
\(P^{-1}=\frac{1}{P} \left\lgroup\begin{matrix} \frac{1}{\epsilon }- \frac{\rho }{\epsilon } \frac{w(\widetilde{sg} )^{2}}{4\tau _{L}-w\widetilde{gg}} &\frac{\rho }{\epsilon } \left[\widetilde{s_{1}s_{2}} + \frac{w(\widetilde{sg} )^{2}}{4\tau _{L}-w\widetilde{gg}} \right] \\ \frac{\rho }{\epsilon } \left[\widetilde{s_{1}s_{2}} + \frac{w(\widetilde{sg} )^{2}}{4\tau _{L}-w\widetilde{gg}} \right] & \frac{1}{\epsilon } -\frac{\rho }{\epsilon } \frac{w(\widetilde{sg} )^{2}}{4\tau _{L}-w\widetilde{gg}} \end{matrix} \right\rgroup\)

and

\(P=P^{2}_{11}-P^{2}_{12}=\frac{1}{\epsilon ^{2}} \left[1-2\rho \frac{w(\widetilde{sg} )^{2}}{4\tau _{L}-w\widetilde{gg}} -\rho ^{2}(\widetilde{s_{1}s_{2}} )^{2}-2\rho ^{2}\widetilde{s_{1}s_{2}} \frac{w(\widetilde{sg} )^{2}}{4\tau _{L}-w\widetilde{gg}} \right]\)
\(=\frac{1}{\epsilon ^{2}}\left[1-(\rho \widetilde{s_{1}s_{2}} )^{2}-2\rho (1-\rho \widetilde{s_{1}s_{2}} ) \frac{w(\widetilde{sg} )^{2}}{4\tau _{L}-w\widetilde{gg}} \right] .\)

Finally,

\(\widetilde{SG}=P^{-1}\cdot D=\left\lgroup\begin{matrix} P^{-1}_{11}D_{11}+ P^{-1}_{12}D_{21} \\ P^{-1}_{21}D_{11}+ P^{-1}_{22}D_{21} \end{matrix}\right\rgroup\)
\(=\epsilon \frac{(1+\rho \widetilde{s_{1}s_{2}})4\tau _{L}(1-w)\widetilde{sg}/(4\tau _{L}-w\widetilde{gg} ) }{1-(\rho \widetilde{s_{1}s_{2}})^{2}-2\rho (1+\rho \widetilde{s_{1}s_{2}}) \frac{w(\widetilde{sg} )^{2}}{4\tau _{L}-w\widetilde{gg}}} \left\lgroup\begin{matrix} 1 \\ 1 \end{matrix} \right\rgroup\)
\(=\frac{4\tau _{L}(1-w)\epsilon \widetilde{sg} }{(1-\rho \widetilde{s_{1}s_{2}})(4\tau _{L}-w\widetilde{gg} )-2\rho w (\widetilde{sg} )^{2}}\left\lgroup\begin{matrix} 1 \\ 1 \end{matrix} \right\rgroup .\)

Substituting for \(\widetilde{s_{1}s_{2}}\), \(\widetilde{sg}\) and \(\widetilde{gg}\), which have already been evaluated in (although T/. is now based on the extinction coefficient), this reduces to

\(\Psi =\frac{q_{w}}{\sigma (T^{4}_{1}-T^{4}_{2} )} =\widetilde{SG} = \frac{\epsilon [1-2E_{3}(\tau _{L})]}{1-2\rho E_{3}(\tau _{L})+\frac{2\epsilon w}{4\tau _{L}(1-w)}[1-2E_{3}(\tau _{L})] } .\)
