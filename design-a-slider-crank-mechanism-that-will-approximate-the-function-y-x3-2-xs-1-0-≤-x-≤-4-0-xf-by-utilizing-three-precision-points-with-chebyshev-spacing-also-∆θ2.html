Guest Question.

Design a slider crank mechanism that will approximate the function

\(Y = X^{3/2} , X_{s} = 1.0 ≤ X ≤ 4.0 = X_{f}\)

by utilizing three precision points with Chebyshev spacing. Also

\(∆θ_{2} = -120°; ∆s = 5.0 cm\)
Admin
Solution

We make the selection

\( (θ_{2})_{s} = 150° ; s = 2.000 cm\)

Then from the function graph shown in Figure 11.15

\( X_{1} = 1.201; Y_{1} = 1.316; (θ_{2})_{1} = 141.96°; s_{1} = 2.226 cm\)

 

\( X_{2} = 2.500; Y_{2} = 3.953 ; (θ_{2})_{2} = 90.00°; s_{2} = 4.109 cm\)

 

\( X_{3} =3.799; Y_{3} = 7.405; (θ_{2})_{3} = 38.04°; s_{3} = 6.575 cm\)

Substituting the above values in Equation (11.4-7) \(s_{i}^{2} = k_{1} s_{i} \cos (\theta _{2})_{i} + k_{2} \sin (\theta _{2})_{i} -k_{3}\), \( i=1,2,3\), and solving for the unknowns,

\(k_{1} = 5.522 cm; k_{2} = 5.865 cm^{2}; k_{3} = -11.02 cm^{2}\)

Using Equations (11.4-8) \(r_{2} = \frac{k_{1}}{2} ; r_{1} = \frac{k_{2}}{2r_{2}} ; r_{3} = (r_{1}^{2} + r_{2}^{2} -k_{3})^{1/2}\), the dimensions of the mechanism are

\(r_{1} =1.062 cm; r_{2} = 2.761 cm; r_{3} = 4.446 cm\)

Figure 11.16 shows the mechanism in three configurations corresponding to the precision points.
Having the dimensions of the slider crank mechanism, we then evaluate the function that would actually be produced by the mechanism and compare it with the desired function to determine the structural error as a function of the rotation of link 2. The actual positions of the slider are evaluated using Equations (4.3-77) \(\theta _{3} = 180° – \theta _{3} ^{*} = 180° – \sin ^{-1} \left\lgroup\frac{-r_{1} + r_{2} \sin \theta _{2}}{r_{3}} \right\rgroup \) and (4.3-78) \(r_{4} = r_{2} \cos \theta _{2} – r_{3} \cos \theta _{3}\).
