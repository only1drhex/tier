Guest Question.

A pair of identical straight-tooth miter gears listed in a catalog has a diametral pitch of 5 at the large end, 25 teeth, a 1.10-in face width, and a 20° normal pressure angle; the gears are grade 1 steel through-hardened with a core and case hardness of 180 Brinell.
The gears are uncrowned and intended for general industrial use. They have a quality number of \(Q_{v}\) = 7. It is likely that the application intended will require outboard mounting of the gears. Use a safety factor of 1, a \(10^{7}\) cycle life, and a 0.99 reliability.
(a) For a speed of 600 rev/min find the power rating of this gearset based on AGMA bending strength.
(b) For the same conditions as in part (a) find the power rating of this gearset based on AGMA wear strength.
(c) For a reliability of 0.995, a gear life of \(10^{9}\) revolutions, and a safety factor of \(S_{F} = S_{H}\) = 1.5, find the power rating for this gearset using AGMA strengths.

Admin
Solution

From Figs. 15–14 and 15–15,

\(d_{P} = N_{P}/P = 25/5 = 5.000 in\)
\(v_{t} = πd_{P}n_{P}/12 = π(5)600/12 = 785.4 ft/min\)

Overload factor: uniform-uniform loading, Table 15–2, \(K_{o}\) = 1.00.

Table 15–2
Overload Factors \(K_{o} (K_{A})\) Source: ANSI/AGMA 2003-B97.

Character of Load on Driven Machine Character of Prime Mover
Heavy Shock Medium Shock Light Shock Uniform
1.75 or higher 1.50 1.25 1.00 Uniform
1.85 or higher 1.60 1.35 1.10 Light shock
2.00 or higher 1.75 1.50 1.25 Medium Shock
2.25 or higher 2.00 1.75 1.50 Heavy Shock

Note: This table is for speed-decreasing drives. For speed-increasing drives, add \(0.01(N/n)^{2} or 0.01(z_{2} /z_{1})^{2}\) to the above factors.

Safety factor: \(S_{F} = 1, S_{H} = 1\).

Dynamic factor \(K_{v}\) : from Eq. (15–6),

\(A = 50 + 56(1 − B)\)
\(B = 0.25(12 − Q_{v})^{2/3}\) (15–6)

\(B = 0.25(12 − 7)^{2/3} = 0.731\)
\(A = 50 + 56(1 − 0.731) = 65.06\)
\(K_{v} = \left( \frac{65.06 + \sqrt{785.4}}{65.06}\right)^{0.731}= 1.299\)

From Eq. (15–8),

\(v_{t max} = [A + (Q_{v} − 3)]^{2}\) (U.S. customary units)
\(v_{te max} =\frac{[A + (Q_{v} − 3)]^{2}}{200}\) (SI units) (15–8)

\(v_{t max} = [65.06 + (7 − 3)]^{2} = 4769 ft/min\)

\(v_{t} < v_{t max}\), that is, 785.4 < 4769 ft/min, therefore \(K_{v}\) is valid. From Eq. (15–10),

\(K_{s} =\begin{cases} 0.4867 + 0.2132/P_{d}&0.5 ≤ P_{d} ≤ 16 in^{−1}\\ 0.5&Pd > 16 in^{−1}\end{cases}\) (U.S. customary units)

\(Y_{x} =\begin{cases} 0.5&m_{et} < 1.6 mm\\ 0.4867 + 0.008 339m_{et}&1.6 ≤ m_{et} ≤ 50 mm\end{cases}\) (SI units) (15–10)

\(K_{s} = 0.4867 + 0.2132/5 = 0.529\)

From Eq. (15–11),

\(K_{m} = K_{mb} + 0.0036F^{2}\) (U.S. customary units)
\(K_{Hβ} = K_{mb} + 5.6(10−6)b^{2}\) (SI units) (15–11)

\(K_{mb} = 1.25 and K_{m} = 1.25 + 0.0036(1.10)^{2} = 1.254\)

From Eq. (15–13), \(K_{x}\) = 1. From Fig. 15–6, I = 0.065; from Fig. 15–7, \(J_{P} = 0.216, J_{G} = 0.216\). From Eq. (15–15),

\(K_{x} = Y_{β} = 1 \) (15–13)

\(K_{L} =\begin{cases} 2.7&10^{2} ≤ N_{L} < 10^{3}\\6.1514N^{−0.1182}_{L}&10^{3} ≤ N_{L} < 3(10^{6})\\1.6831N^{−0.0323}_{L}&3(10^{6}) ≤ N_{L} ≤ 10^{10} general\\1.3558N^{−0.0178}_{L}&33(10^{6}) ≤ N_{L} ≤ 10^{10} critical\end{cases}\)

\(Y_{NT} =\begin{cases} 2.7&10^{2} ≤ n_{L} < 10^{3}\\6.1514n^{−0.1182}_{L}&10^{3} ≤ n_{L} < 3(10^{6})\\1.6831n^{−0.0323}_{L}&3(10^{6}) ≤ n_{L} ≤ 10^{10} general\\1.3558n^{−0.0323}_{L}&33(10^{6}) ≤ n_{L} ≤ 10^{10} critical\end{cases}\) (15-15)

\(K_{L} = 1.683(10^{7})^{−0.0323} = 0.999 96\dot{=}1\)

From Eq. (15–14),

\(C_{L} =\begin{cases} 2&10^{3} ≤ N_{L} < 10^{4}\\3.4822N^{−0.0602}_{L}&10^{4} ≤ N_{L} ≤ 10^{10}\end{cases}\)

\(Z_{NT}=\begin{cases} 2&10^{3} ≤ n_{L} < 10^{4}\\3.4822n^{−0.0602}_{L}&10^{4} ≤ n_{L} ≤ 10^{10}\end{cases}\) (15-14)

\(C_{L} = 3.4822(10^{7})^{−0.0602} = 1.32\)

Since \(H_{BP}/H_{BG}\) = 1, then from Fig. 15–10, \(C_{H}\) = 1. From Eqs. (15–13) and (15–18), \(K_{x}\) = 1 and \(K_{T}\) = 1, respectively. From Eq. (15–20),

\(K_{T} =\begin{cases}1&32°F ≤ t ≤ 250°F\\(460 + t)/710&t > 250°F\end{cases}\)

\(K_{θ} =\begin{cases}1&0°C ≤ θ ≤ 120°C\\(273 + θ)/393&θ > 120°C\end{cases}\) (15-18)

\(Y_{Z} = K_{R} =0.70 − 0.15 log(1 − R) 0.90 ≤ R < 0.99\) (15–20)

\(K_{R} = 0.70 − 0.15 log(1 − 0.99) = 1, C_{R} =\sqrt{K_{R}} =\sqrt{1} = 1\)

(a) Bending: From Eq. (15–23),

\(s_{at} = 44H_{B} + 2100\) psi grade 1
\(σ_{F lim} = 0.30H_{B} + 14.48\) MPa grade 1
\(s_{at} = 48H_{B} + 5980\) psi grade 2
\(σ_{H lim} = 0.33H_{B} + 41.24\) MPa grade 2
(15–23)

\(s_{at} = 44(180) + 2100 = 10 020\) psi

From Eq. (15–3),

\(s_{t} =\frac{W^{t}}{F}P_{d}K_{o}K_{v} \frac{K_{s}K_{m}}{K_{x} J}\) (U.S. customary units)
\(σ_{F} =\frac{1000W^{t}}{b} \frac{K_{A}K_{v}}{m_{et}} \frac{Y_{x} K_{Hβ}}{Y_{β}Y_{J}}\) (SI units)
(15–3)

\(s_{t}= σ =\frac{W^{t}}{F}P_{d}K_{o}K_{v} \frac{K_{s}K_{m}}{K_{x} J} =\frac{W^{t}}{1.10}(5)(1)1.299\frac{0.529(1.254)}{(1)0.216}\)

\(= 18.13W^{t}\)

From Eq. (15–4),

\(s_{wt} =\frac{s_{at} K_{L}}{S_{F} K_{T} K_{R}}\) (U.S. customary units)
\(σ_{F P} =\frac{σ_{F lim}Y_{NT}}{S_{F} K_{θ}Y_{z}}\) (SI units)
(15–4)

\(s_{wt} =\frac{s_{at} K_{L}}{S_{F} K_{T} K_{R}} =\frac{10 020(1)}{(1)(1)(1) }= 10 020 psi\)

Equating \(s_{t} and s_{wt}\) ,

\(18.13W^{t} = 10 020 W^{t} = 552.6\) lbf

\(H =\frac{W^{t}v_{t}}{33 000 }=\frac{552.6(785.4)}{33 000} =13.2\) hp

(b) Wear: From Fig. 15–12,

\(C_{xc} = Z_{xc} =\begin{cases}1.5&properly crowned teeth\\2.0&or larger uncrowned teeth\end{cases}\) (15-12)

\(s_{ac} = 341(180) + 23 620 = 85 000\) psi

From Eq. (15–2),

\(s_{wc} = (σ_{c})_{all} =\frac{s_{ac}C_{L}C_{H}}{S_{H} K_{T}C_{R}}\) (U.S. customary units)
\(σ_{H P} =\frac{σ_{H lim}Z_{NT} Z_{W}}{S_{H} K_{θ} Z_{Z}}\) (SI units)
(15–2)

\(σ_{c,all} =\frac{s_{ac}C_{L}C_{H}}{S_{H} K_{T}C_{R}} =\frac{85 000(1.32)(1)}{(1)(1)(1)} = 112 200 psi\)

Now \(C_{p} = 2290\sqrt{psi}\) from definitions following Eq. (15–21). From Eq. (15–9),

\(C_{s} =\begin{cases} 0.5&F < 0.5 in\\0.125F + 0.4375&0.5 ≤ F ≤ 4.5 in\\1&F > 4.5 in \end{cases}\) (U.S. customary units)

\(Z_{x} =\begin{cases} 0.5& b < 12.7 mm\\0.004 92b + 0.4375 &12.7 ≤ b ≤ 114.3 mm\\1& b > 114.3 mm\end{cases}\) (SI units) (15.9)

\(C_{p} =\sqrt{\frac{1}{π [1 − ν^{2}_{P}]E_{P} +[1 − ν^{2}_{G}]E_{G}}}\)

\(Z_{E} =\sqrt{\frac{1}{π [1 − ν^{2}_{1}]E_{1} +[1 − ν^{2}_{2}]E_{2}}}\) (15-21)

\(C_{s} = 0.125(1.1) + 0.4375 = 0.575\)

From Eq. (15–12), \(C_{xc}\)= 2. Substituting in Eq. (15–1) gives

\(s_{c} = σ_{c} = C_{p} \left( \frac{W^{t}}{Fd_{P} I}K_{o}K_{v}K_{m}C_{s}C_{xc}\right)^{1/2}\) (U.S. customary units)
\(σ_{H} = Z_{E} \left(\frac{1000W^{t}}{bd Z_{1}}K_{A}K_{v}K_{Hβ} Z_{x} Z_{xc}\right)^{1/2}\) (SI units)
(15–1)

\(σ_{c} = C_{p} \left( \frac{W^{t}}{Fd_{P} I}K_{o}K_{v}K_{m}C_{s}C_{xc}\right)^{1/2}\)

\(= 2290 \left[ \frac{W^{t}}{1.10(5)0.065}(1)1.299(1.254)0.575(2)\right]^{1/2}= 5242\sqrt{W^{t}}\)

Equating \(σ_{c} and σ_{c,all}\) gives

\(5242\sqrt{W^{t}}= 112 200, \ \ \ \ W^{t} = 458.1 lbf\)

\(H =\frac{458.1(785.4)}{33 000} = 10.9 hp\)

Rated power for the gearset is

\(H = min(12.9, 10.9) = 10.9 hp\)

(c) Life goal \(10^{9}\) cycles, R = 0.995, \(S_{F} = S_{H} = 1.5\), and from Eq. (15–15),

\(K_{L} = 1.683(10^9)^{−0.0323} = 0.8618\)

From Eq. (15–19),

\(Y_{Z} = K_{R} =0.50 − 0.25 log(1 − R) 0.99 ≤ R ≤ 0.999\) (15.19)

\(K_R = 0.50 − 0.25 log(1 − 0.995) = 1.075, C_{R} =\sqrt{K_{R}} =\sqrt{1.075} = 1.037\)

From Eq. (15–14),

\(C_{L} = 3.4822(10^9)^{−0.0602} = 1\)

Bending: From Eq. (15–23) and part (a), \(s_{at}\) = 10 020 psi. From Eq. (15–3),

\(s_{t} = σ =\frac{W^{t}}{1.10} 5(1)1.299 \frac{0.529(1.254)}{(1)0.216} = 18.13W^{t}\)

From Eq. (15–4),

\(s_{wt} =\frac{s_{at} K_{L}}{S_{F} K_{T} K_{R}} =\frac{10 020(0.8618)}{1.5(1)1.075} = 5355 psi\)

Equating \(s_{t} to s_{wt}\) gives

\(18.13W^{t} = 5355 W^{t} = 295.4 lbf\)

\(H =\frac{295.4(785.4)}{33 000 }= 7.0 hp\)

Wear: From Eq. (15–22), and part (b), \(s_{ac}\) = 85 000 psi.

\(s_{ac} = 341H_{B} + 23 620 psi grade 1\)
\(σ_{H lim} = 2.35H_{B} + 162.89 MPa grade 1\)
\(s_{ac} = 363.6H_{B} + 29 560 psi grade 2\)
\(σ_{H lim} = 2.51H_{B} + 203.86 MPa grade 2\)
(15–22)

Substituting into Eq. (15–2) gives

\(σ_{c,all} =\frac{s_{ac}C_{L}C_{H}}{S_{H} K_{T}C_{R}} =\frac{85 000(1)(1)}{1.5(1)1.037 }= 54 640 psi\)

Substituting into Eq. (15–1) gives, from part (b), \(σ_{c} = 5242\sqrt{W^{t}}\) .

Equating \(σ_{c} to σ_{c,all}\) gives

\(σ_{c} = σ_{c,all} = 54 640 = 5242\sqrt{W^{t}} \ \ \ \ W^{t} = 108.6 lbf\)

The wear power is

\(H =\frac{108.6(785.4)}{33 000} = 2.58 hp\)

The mesh rated power is H = min(7.0, 2.58) = 2.6 hp.
