Guest Question.

Methane is compressed isothermally in a steady-flow compressor from 100 kPa and 20°C to 20 MPa. Calculate the minimum power required if the mass flux is 0.02 kg/s. Include real gas effects.

Admin
Solution

The reduced temperatures and pressures are

\( T_{R 2}=T_{R 1}=\frac{T_{1}}{T_{c r}}=\frac{293}{191.1}=1.53 \quad P_{R 1}=\frac{0.1}{4.64}=0.02 \quad P_{R 2}=\frac{20}{4.64}=4.31 \).

Minimum power is required for an isothermal process if the process is ideal, so that the heat transfer is given by q = TΔs. The entropy change is

\( \Delta s=\cancel {\left(s_{1}^{*}-s_{1}\right)}^{0}+\left(s_{2}-s_{2}^{*}\right)+\left(s_{2}^{*}-s_{1}^{*}\right)=0-\frac{7}{16}+2.25 \ln 1-0.518 \ln \frac{20}{01}=-3.18 kJ / kg \cdot K \).

\( \text { so that } q=T \Delta s=(293)(-3.18)=-932 kJ / kg \text {. The first law, } q-w=\Delta h \text {, requires that we find } \Delta h \text {. } \) \( \text { We find } \bar{h}_{2}^{*}-\bar{h}_{2}=14 kJ / kmol \cdot K \text {, so that } \)

\( \Delta h=\cancel {\left(h_{1}^{*}-\hbar_{1}\right)}^{0}+\left(h_{2}-h_{2}^{*}\right)+\cancel {\left(h_{2}^{*}-\hbar_{1}\right)}^{0}=(-14)\left(\frac{191.1}{16}\right)=-167 kJ / kg \).

Finally, the compressor power is

\( \dot{W}_{C}=(q-\Delta h) \dot{m}=[-932-(-167)](0.02)=-15.3 kW \).
