Guest Question.

Convert 4.00 oz to kg.

Admin
Solution

Based on the English system relationships provided in Tables 1.1 and 1.3, the data given in \(\mathrm{oz}\) should be converted to a relating unit in lb, so lb can be converted to \(\mathrm{g}\). Then, using the prefix equalities in Table 1.2, the initial data result in \(\mathrm{g}\) can be converted to the desired result in \(\mathrm{kg}\). The possible conversion factors are:

table 1.1

Weight 1 pound (lb)=16 ounces (oz)
1 ton (t)=2000 pounds (lb)
Length 1 foot (ft)=12 inches ( in )
1 yard (yd)=3 feet (ft)
1 mile (mi)=5280 feet (ft)
Volume 1 quart (qt)=32 fluid ounces (floz)
1 quart (qt)=2 pints (pt)
1 gallon (gal)=4 quarts (qt)

table 1.3

Quantity English Metric
Mass 1 pound = 454 grams
Length 1 yard = 0.914 meter
Volume 1 quart = 0.946 liter

table 1.2

Prefix Abbreviation Meaning Decimal Equivalent Equality with major metric units (g, m, or L are represented by x in each)
mega M 10\(^{6}\) 1,000,000.00 1Mx=10\(^{6}\)x
kilo k 10\(^{3}\) 1000 1kx=10\(^{3}\)x
deka da 10\(^{1}\) 10 1dax=10\(^{1}\)x
deci d 10\(^{-1}\) 0.1 1 dx=10\(^{-1}\)x
centi c 10\(^{-2}\) 0.01 1cx=10\(^{-2}\)x
milli m 10\(^{-3}\) 0.001 1 mx=10\(^{-3}\)x
micro μ 10\(^{-6}\) 0.000001 1μx=10\(^{-6}\)x
nano n 10\(^{-9}\) 0.000000001 1nx=10\(^{-9}\)x
\(\frac{16 \mathrm{oz}}{1 \mathrm{lb}}=\frac{1 \mathrm{lb}}{16 \mathrm{oz}} \text { and } \frac{454 \mathrm{~g}}{1 \mathrm{lb}}=\frac{1 \mathrm{lb}}{454 \mathrm{~g}} \text { and } \frac{10^{3} \mathrm{~g}}{1 \mathrm{~kg}}=\frac{1 \mathrm{~kg}}{10^{3} \mathrm{~g}}\)

Step 1. Since the initial value is in oz, the conversion factor with oz in the denominator should be used first because it relates the data given to a relating unit.

\(4.00 \cancel{oz} \times \frac{1 \mathrm{lb}}{16 \cancel{oz}}=0.250 \mathrm{lb}\)

Data Given \(\times\) Conversion Factor \(=\) Relating Unit
If the other conversion factor relating \(o z\) and \(l b\) was used, the resulting units would have been \(o z^{2} / l b\), and the answer would have been incorrect.

Step 2. Now that \(4.00 \mathrm{oz}\) has been converted to \(0.250 \mathrm{lb}\), the conversion factor relating \(\mathrm{lb}\) to \(\mathrm{g}\) is used. The conversion factor with \(\mathrm{lb}\) in the denominator and \(\mathrm{g}\) in the numerator is the only one that leads to the correct answer.

\(0.250 \cancel{lb} \times \frac{454 \mathrm{~g}}{1 \cancel{~lb}}=114 \mathrm{~g}\)

Relating Unit \(\times\) Conversion Factor \(=\) Initial Data Result
Step 3. In the final step of this conversion, the conversion is within the desired system of units required by the problem. The conversion factor relating \(\mathrm{g}\) and \(\mathrm{kg}\) with \(\mathrm{g}\) in the denominator and \(\mathrm{kg}\) in the numerator is the only one that leads to the correct answer.

\(114 \cancel{g} \times \frac{1 \mathrm{~kg}}{10^{3} \cancel{g}}=0.114 \mathrm{~kg}\)

Initial Data Result \(\times\) Conversion Factor \(=\) Desired Result
This calculation may also be done in a single step by arranging the conversion factors in a chain:

\(4.00 . \cancel{oz} \times \frac{1 \cancel{lb}}{16 \cancel{oz}} \times \frac{454 \cancel{~g}}{1 \cancel{~lb}} \times \frac{1 \mathrm{~kg}}{10^{3} \cancel{~g}}=0.114 \mathrm{~kg}\)

Data Given \(\times\) Conversion Factor \(\times\) Conversion Factor \(\times\) Conversion Factor \(=\) Desired Result
Helpful Hint: After the conversion factors have been selected and set up in the solution to the problem, it is important to also cancel the units that can be canceled. This process will allow for you to verify that you have set up the problem correctly. In addition, the unit oz represents a smaller quantity than the unit kg. Therefore, one would expect a large number of oz to equal a small number of \(\mathrm{kg}\).
