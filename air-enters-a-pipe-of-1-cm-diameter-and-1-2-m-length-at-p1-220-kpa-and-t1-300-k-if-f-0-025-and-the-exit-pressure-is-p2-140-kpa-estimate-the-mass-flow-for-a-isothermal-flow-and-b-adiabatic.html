Guest Question.

Air enters a pipe of 1-cm diameter and 1.2-m length at \(p_1 = 220 kPa and T_1 = 300 K. If \bar{f} = 0.025\) and the exit pressure is \(p_2\) = 140 kPa, estimate the mass flow for (a) isothermal flow and (b) adiabatic flow.

Admin
Solution

Part (a)

For isothermal flow Eq. (9.73) applies without iteration:

\(G^2 = \left(\frac{\dot{m}}{A}\right)^2 = \frac{p^2_1 – p^2_2}{RT [\bar{f}L/D + 2 \ln (p_1/p_2)]}\) (9.73)

\(\frac{\bar{f}L}{D} + 2 \ln \frac{p_1}{p_2} = \frac{(0.025)(1.2 m)}{0.01 m} + 2 \ln \frac{220}{140} = 3.904\)

 

\(G^2 = \frac{(220,000 Pa)^2 – (140,000 Pa)^2}{[287 m^2/(s^2 \cdot K)](300 K)(3.904)} = 85,700 or G = 293 kg/(s \cdot m^2)\)

Since \(A = (\pi /4)(0.01 m)^2 = 7.85 E-5 m^2\), the isothermal mass flow estimate is

\(\dot{m}\) = GA = (293)(7.85 E-5) ≈ 0.0230 kg/s

Check that the exit Mach number is not choked:

\(\rho_2 = \frac{p_2}{RT} = \frac{140,000}{(287)(300)} = 1.626 kg/m^3 V_2 = \frac{G}{\rho_2} = \frac{293}{1.626} = 180 m/s\)

or \(Ma_2 = \frac{V_2}{\sqrt{kRT}} = \frac{180}{[1.4(287)(300)^{1/2}} = \frac{180}{347} \approx 0.52\)

This is well below choking, and the isothermal solution is accurate.

EES Part (b)

For adiabatic flow, we can iterate by hand, in the time-honored fashion, using Eqs. (9.74) and (9.75) plus the definition of stagnation speed of sound. A few years ago the author would have done just that, laboriously. However, EES makes handwork and manipulation of equations unnecessary, although careful programming and good guesses are required. If we ignore superfluous output such as \(T_2 and V_2\), 13 statements are appropriate. First, spell out the given physical properties (in SI units):

k = 1.4
P1 = 220000
P2 = 140000
T1 = 300

\(V^2_1 = \frac{a^2_0 [1 – (V_1/V_2)^2]}{k\bar{f}L/D (k + 1) \ln (V_2/V_1)}\) (9.74)

\(\frac{V_1}{V_2} = \frac{p_2}{p_1} \frac{T_1}{T_2} = \frac{p_2}{p_1} \left[\frac{2a^2_0 – (k – 1)V^2_1}{2a^2_0 – (k – 1)V^2_2}\right]\) (9.75)

Next, apply the adiabatic friction relations, Eqs. (9.66) and (9.67), to both points 1 and 2:

fLD1 = (1 – Ma1^2)/k/Ma1^2 + (k + 1)/2/k*LN((k + 1)*Ma1^2/(2
+ (k – 1)*Ma1^2))
fLD2 = (1 – Ma2^2)/k/Ma2^2 + (k + 1)/2/k*LN((k + 1)*Ma2^2/(2 + (k – 1)*Ma2^2))
DeltafLD = 0.025*1.2/0.01
fLD1 = fLD2 + DeltafLD

\(\frac{\bar{f}L^*}{D} = \frac{1 – Ma^2}{k Ma^2} + \frac{k + 1}{2k} \ln \frac{(k + 1) Ma^2}{2 + (k – 1) Ma^2}\) (9.66)

\(\bar{f} \frac{\Delta L}{D} = \left(\frac{\bar{f}L^*}{D}\right)_1 – \left(\frac{\bar{f}L^*}{D}\right)_2\) (9.67)

Then apply the pressure ratio formula (9.68a) to both points 1 and 2:

P1/Pstar = ((k + 1)/(2 + (k – 1)*Ma1^2))^0.5/Ma1
P2/Pstar = ((k + 1)/(2 + (k – 1)*Ma2^2))^0.5/Ma2

\(\frac{p}{p^*} = \frac{1}{Ma} \left[\frac{k + 1}{2 + (k – 1)Ma^2}\right]^{1/2}\) (9.68a)

These are adiabatic relations, so we need not further spell out quantities such as \(T_0 or a_0\) unless we want them as additional output.

The above 10 statements are a closed algebraic system, and EES will solve them for \(Ma_1 and Ma_2\). However, the problem asks for mass flow, so we complete the system:

V1 = Ma1*sqrt(1.4*287*T1)
Rho1 = P1/287/T1
Mdot = Rho1*(pi/4*0.01^2)*V1

If we apply no constraints, EES reports “cannot solve” because its default allows all variables to lie between -∞ and +∞. So we enter Variable Information and constrain \(Ma_1 and Ma_2\) to lie between 0 and 1 (subsonic flow). EES still complains that it “cannot solve” but hints that “better guesses are needed.” Indeed, the default guesses for EES variables are normally 1.0, too large for the Mach numbers. Guess the Mach numbers equal to 0.8 or even 0.5, and EES still complains, for a subtle reason: Since fΔL/D = 0.025(1.2/0.01) = 3.0, \(Ma_1\) can be no larger than 0.36 (see Table B.3). Finally, then, we guess \(Ma_1 and Ma_2\) = 0.3 or 0.4, and EES reports the solution:

\(Ma_1 = 0.3343 Ma_2 = 0.5175 \frac{fL}{D_1} = 3.935 \frac{fL}{D_2} = 0.9348\)

\(p^*\) = 67,892 Pa \(\dot{m}\) = 0.0233 kg/s
