Guest Question.

A 350-km, 500-kV, 60-Hz, three-phase uncompensated line has a positivesequence series reactance \(x=0.34 \Omega / km\) and a positive-sequence shunt admittance \(y=j 4.5 \times 10^{-6} S / km\). Neglecting losses, calculate: (a) \(Z _{c}\), (b) \(\gamma l \text { , }\) (c) the ABCD parameters, (d) the wavelength λ of the line in kilometers, and (e) the surge impedance loading in MW.

Admin
Solution

(a) \(\bar{Z}_{C}=\sqrt{\frac{\bar{z}}{\bar{y}}}=\sqrt{\frac{j 0.34}{j 4.5 \times 10^{-6}}}=274.9 \angle 0^{\circ}=274.9 \Omega\)

(b) \(\bar{\gamma} l=\sqrt{\bar{z} \bar{y}}(l)=\sqrt{(j 0.34)\left(j 4.5 \times 10^{-6}\right)}(350)=j 0.4330 pu\)

(c) \(\bar{\gamma} l=j(\beta l) ; \quad \beta l=0.4330 pu\)

\(\begin{aligned}A=D=\cos (\beta l) &=\cos (0.4330 \text { radians })=0.9077 \angle 0^{\circ} pu \\\bar{B}=j Z_{C} \sin (\beta l) &=j(274.9) \sin (0.4330 \text { radians }) \\&=j 115.35 \Omega\end{aligned}\)

 

\(\begin{aligned}\bar{C}=j\left(\frac{1}{Z_{C}}\right) \sin (\beta l) &=j\left(\frac{1}{274.9}\right) \sin (0.4330 \text { radians }) \\&=j 1.516 \times 10^{-3} S\end{aligned}\)

 

(d)

\(\begin{aligned}&\beta=0.4330 / 350=1.237 \times 10^{-3} \text { radians } / km \\&\lambda=2 \pi / \beta=5079 km\end{aligned}\)

 

(e) \(SIL =\frac{V_{\text {rated } L-L}^{2}}{Z_{C}}=\frac{(500)^{2}}{274.9}=909.4 MW (3 \phi)\)
