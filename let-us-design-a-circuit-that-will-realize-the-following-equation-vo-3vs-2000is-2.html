Guest Question.

Let us design a circuit that will realize the following equation.

 

\(V_o = -3V_s - 2000I_s\)
Admin
Solution

An examination of this equation indicates that we need to add two terms, one of which is from a voltage source and the other from a current source. Since the terms have negative signs, it would appear that the use of an inverting op-amp stage would be useful. Thus, one possible circuit for this application appears to be that shown in Fig. 5.57.

The Norton equivalent circuit at the terminals A-B will provide a composite view of the op-amp’s input. Superposition can also be used in conjunction with the Norton equivalent to simplify the analysis. Using the network in Fig. 5.58 a, we can determine the contribution of \(V_{S}\) to the short-circuit current, \(I_{\mathrm{sc}}\), which we call \(I_{\mathrm{sc}_{1}}\).

 

\(I_{\mathrm{sc}_{1}}=\frac{V_{S}}{R_{1}}\)

 

 

In a similar manner, using Fig. 5.58 b, we find that the contribution of \(I_{S}\) to the short-circuit current is

 

\(I_{\mathrm{sc}_{2}}=I_{S}\)

 

Employing superposition, the sum of these two currents yields the actual short-circuit current

 

\(I_{\mathrm{sc}}=\frac{V_{s}}{R_{1}}+I_{s}\)

 

The Thévenin equivalent resistance at nodes A-B is obtained from the network in Fig. 5.58 c as

 

\(R_{\mathrm{Th}}=R_{1}\)

 

The equivalent circuit is now redrawn in Fig. 5.59 where we have employed the ideal opamp conditions (i.e., \(V_{\text {in }}=0\) ), and the current into the op-amp terminals is zero. Since \(V_{\text {in }}\) is directly across \(R_{\mathrm{Th}}\), the current in this resistor is also zero. Hence, all the current \(I_{\mathrm{sc}}\) will flow through \(R_{2}\), producing the voltage

 

\(V_{o}=-R_{2}\left[\frac{V_{S}}{R_{1}}+I_{S}\right]=-\frac{R_{2}}{R_{1}} V_{S}-I_{S} R_{2}\)

 

A comparison of this equation with the design requirement specifies that

 

\(\frac{R_{2}}{R_{1}}=3 \quad \text { and } \quad R_{2}=2000 \Omega\)

 

which yields \(R_{1}=667 \Omega\). Combining a 1-kΩ and a 2-kΩ resistor in parallel will yield the necessary 667 Ω exactly.
