Guest Question.

Gas oil at \(200^{\circ} C\) is to be cooled to \(40^{\circ} C\). The oil flow-rate is 22,500 kg/h. Cooling water is available at \(30^{\circ} C\) and the temperature rise is to be limited to \(20^{\circ} C\). The pressure drop allowance for each stream is 100 \(kN / m ^{2}\).

Design a suitable exchanger for this duty.

Admin
Solution

Only the thermal design will be carried out, to illustrate the calculation procedure for an exchanger with a divided shell.

 

\(\Delta T_{l m}=\frac{(200-40)-(40-30)}{\operatorname{Ln} \frac{(200-50)}{(40-30)}}=51.7^{\circ} C\) (12.4)

 

\(R=(200-50) /(50-30)=8.0\) (12.6)

 

\(S=(50-30) /(200-30)=0.12\) (12.7)

 

These values do not intercept on the figure for a single shell-pass exchanger, Figure 12.19, so use the figure for a two-pass shell, Figure 12.20, which gives

 

\(F_{t}=0.94, \text { so }\)

 

\(\Delta T_{m}=0.94 \times 51.7=48.6^{\circ} C\)

 

Physical properties

Water, from steam tables:

 

\(\text { Temperature }{ }^{\circ} C\) 30 40 50
\(C \cdot k J k o^{-1 \circ} C^{-1}\) 4.18 4.18 4.18
\(k, kWm ^{-1}{ }^{\circ} C ^{-1}\) \(618 \times 10^{-6}\) \(631 \times 10^{-6}\) \(643 \times 10^{-6}\)
\(\mu, mNm ^{-2} s\) \(797 \times 10^{-3}\) \(671 \times 10^{-3}\) \(544 \times 10^{-3}\)
\(\rho, kg m ^{-3}\) 995.2 992.8 990.1

 

Gas oil, from Kern, Process Heat Transfer, McGraw-Hill:

 

\(\text { Temperature }{ }^{\circ} C\) 200 120 40
\(C \cdot k J k o^{-1 \circ} C^{-1}\) 2.95 2.28 1.97
\(k, kWm ^{-1}{ }^{\circ} C ^{-1}\) 0.13 0.125 0.12
\(\mu, mNm ^{-2} s\) 0.06 0.17 0.28
\(\rho, kg m ^{-3}\) 830 850 870

 

\(\text { Oil flow-rate }=22,500 / 3600=6.25 kg / s\)

 

\(Q=6.25 \times 2.28 \times(200-40)=2280 kW\)

 

\(\text { Water flow-rate }=\frac{2280}{4.18(50-30)}=27.27 kg / h\)

 

From Figure 12.1, for cooling tower water and heavy organic liquid, take

 

\(U=500 Wm ^{-2} C ^{-1}\)

 

\(\text { Area required }=\frac{2280 \times 10^{3}}{500 \times 48.6}=94 m ^{2}\)

 

Tube-side coefficient

 

\(\text { Select } 20 mm \text { o.d., } 16 mm \text { i.d. tubes, } 4 m \text { long, triangular pitch } 1.25 d_{o} \text {, carbon steel. }\)

 

\(\text { Surface area of one tube }=\Pi \times 20 \times 10^{-3} \times 4=0.251 m ^{2}\)

 

\(\text { Number of tubes required }=94 / 0.251=375 \text {, say } 376 \text {, even number }\)

 

\(\text { Cross-sectional area, one tube }=\frac{\Pi}{4}\left(16 \times 10^{-3}\right)^{2}=2.011 \times 10^{-4} m ^{2}\)

 

\(\text { Total tube area }=376 \times 2.011 \times 10^{-4}=0.0756 m ^{2}\)

 

Put water through tube for ease of cleaning.

Tube velocity, one pass = 27.27/(992.8 ð 0.0756) = 0.363 m/s

Too low to make effective use of the allowable pressure drop, try 4 passes.

 

\(u_{t}=4 \times 0.363=1.45 m / s\)

 

A floating head will be needed due to the temperature difference. Use a pull through type.

Tube-side heat transfer coefficient

 

\(h_{i}=\frac{4200(1.35+0.02 \times 40) 1.45^{0.8}}{16^{0.2}}=6982 Wm ^{-2 \circ} C ^{-1}\) (12.17)

 

Shell-side coefficient

 

\(\text { From Table } 12.4 \text { and equation } 12.3 b \text {, for } 4 \text { passes, } 1.25 d_{o} \text { triangular pitch }\)

 

Table 12.4. Constants for use in equation 12.3
\(\text { Triangular pitch, } p_{t}=1.25 d_{o}\)
No. passes 1 2 4 6 8
\(K_{1}\) 0.319 0.249 0.175 0.0743 0.0365
\(n_{1}\) 2.142 2.207 2.285 2.499 2.675
\(\text { Square pitch, } p_{t}=1.25 d_{o}\)
No. passes 1 2 4 6 8
\(K_{1}\) 0.215 0.156 0.158 0.0402 0.0331
\(n_{1}\) 2.207 2.291 2.263 2.617 2.643

 

\(\text { Bundle diameter, } D_{b}=20(376 / 0.175)^{1 / 2.285}=575 mm\)

 

From Figure 12.10, for pull through head, clearance D 92 mm

 

\(\text { Shell diameter. } D_{c}=575+92=667 mm \text { (26 in pipe) }\)

 

Use 25 per cent cut baffles, baffle arrangement for divided shell as shown below:Take baffle spacing as 1/5 shell diameter = 667/5 = 133 mm

 

\(\text { Tube pitch, } p_{t}=1.25 \times 20=25 mm\)

 

Area for flow, As, will be half that given by equation 12.21

 

\(\text { Cross-flow area } A_{s}=\frac{(25-20)}{25} 894 \times 178 \times 10^{-6}=0.032 m ^{2}\) (12.21)

 

\(A_{s}=0.5 \times\left(\frac{25-20}{25} \times 0.667 \times 0.133\right)=0.00887 m ^{2}\)

 

\(G_{s}=6.25 / 0.00887=704.6 kg / s\)

 

\(u_{s}=704.6 / 850=0.83 m / s \text {, looks reasonable }\)

 

\(d_{e}=\frac{1.10}{20}\left(25^{2}-0.917 \times 20^{2}\right)=14.2 mm\) (12.23)

 

\(\operatorname{Re}=\frac{0.83 \times 14.2 \times 10^{-3} \times 850}{0.17 \times 10^{-3}}=58,930\)

 

\(\text { From Figure } 12.29, j_{h}=2.6 \times 10^{-3}\)

 

\(\operatorname{Pr}=\left(2.28 \times 10^{3} \times 0.17 \times 10^{-3}\right) / 0.125=3.1\)

 

\(Nu =2.6 \times 10^{-3} \times 58,930 \times 3.1^{1 / 3}=223.4\) (12.25)

 

\(h_{s}=(223.4 \times 0.125) /\left(14.2 \times 10^{-3}\right)=1967 Wm ^{-2 \circ} C ^{-1}\)

 

Overall coefficient

Take fouling factors as 0.00025 for cooling tower water and 0.0002 for gas oil (light organic). Thermal conductivity for carbon steel tubes 45 \(Wm ^{-1}{ }^{\circ} C ^{-1}\).

 

\(\begin{array}{c}1 / U_{o}=1 / 1967+0.0002+\frac{20 \times 10^{-3} \ln (20 / 16)}{2 \times 45} \\+20 / 16(1 / 6982+0.00025)=0.00125 \\U_{o}=1 / 0.00125=800 Wm ^{-2{ }^{\circ} C ^{-1}}\end{array}\) (12.2)

 

Well above the initial estimate of 500 \(Wm ^{-1}{ }^{\circ} C ^{-1}\), so design has adequate area for the duty required.

Pressure drops

Tube-side

 

\(\operatorname{Re}=\frac{1.45 \times 16 \times 10^{-3} \times 992.8}{670 \times 10^{-6}}=34,378 \quad\left(3.4 \times 10^{-4}\right)\)

 

\(\text { From Figure } 12.24, j_{f}=3.5 \times 10^{-3} \text {. Neglecting the viscosity correction }\)

 

\(\begin{aligned}\Delta P_{t} &=4\left[8 \times 3.5 \times 10^{-3} \times\left(\frac{4}{16 \times 10^{-3}}\right)+2.5\right] 992.8 \times \frac{1.45^{2}}{2}=39,660 \\&=40 kN / m ^{2}\end{aligned}\) (12.20)

 

Well within the specification, so no need to check the nozzle pressure drop.

Shell-side

 

\(\text { From Figure } 12.30, \text { for } \operatorname{Re}=58,930, j_{s}=3.8 \times 10^{-2}\)

 

\(\text { With a divided shell, the path length }=2 \times\left(L / l_{b}\right)\)

 

Neglecting the viscosity correction factor,

 

\(\begin{aligned}\Delta P_{s} &=8 \times 3.8 \times 10^{-2}\left(\frac{662 \times 10^{-3}}{14.2 \times 10^{-3}}\right) \times\left(\frac{2 \times 4}{132 \times 10^{-3}}\right) \times 850 \times \frac{0.83^{2}}{2}=251,481 \\&=252 kN / m ^{2}\end{aligned}\) (12.26)

 

Well within the specification, no need to check nozzle pressure drops.

So the proposed thermal design is satisfactory. As the calculated pressure drops are below that allowed, there is some scope for improving the design.
