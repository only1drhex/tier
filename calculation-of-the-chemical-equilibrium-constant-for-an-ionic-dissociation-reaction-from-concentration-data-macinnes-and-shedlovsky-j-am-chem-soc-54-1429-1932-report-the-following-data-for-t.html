Solution

The ionization reaction is

 

\(CH _{3} COOH = CH _{3} COO ^{-}+ H ^{+}\)

 

so that \(\nu_{+}=\nu_{-}=1, z_{+}=1, z_{-}=-1\) and

 

\(\ln K_{c}=\ln K_{c}^{\circ}+2.356 \sqrt{I}\)

 

Also, \(C_{ H ^{+}}=C_{ CH _{3} COO ^{-}}, C_{ CH _{3} COOH }=C_{ T }-C_{ CH _{3} COO ^{-}}\), and

 

\(I=\frac{1}{2} \sum z_{ i }^{2} C_{ i }=\frac{1}{2}\left\{C_{ H ^{+}}+C_{ CH _{3} COO ^{-}}\right\}=C_{ CH _{3} COO ^{-}}\)

 

The dissociation constant \(K_{c}\) is related to ionic strength as follows

 

\(K_{c}=\frac{C_{ CH _{3} COO ^{-}} C_{ H ^{+}}}{C_{ CH _{3} COOH }}=\frac{\left(C_{ CH _{3} COO ^{-}}\right)^{2}}{C_{ T }-C_{ CH _{3} COO ^{-}}}=\frac{I^{2}}{C_{ T }-I}\)

 

The dissociation constant and ionic strength are tabulated and plotted here:

\(0.028 \times 10^{-3}\) \(0.1511 \times 10^{-4}\) \(1.768 \times 10^{-5}\) -10.943
\(0.1532 \times 10^{-3}\) \(0.4405 \times 10^{-4}\) \(1.778 \times 10^{-5}\) -10.938

From Fig. 13.1-3 we see that Eq. 13.1-32 does fit the low-ionic-strength data well, and that \(\ln K_{c}^{\circ}=-10.9515 \text {. Thus, } K_{c}^{\circ}=1.753 \times 10^{-5} kmol / m ^{3} \text { and } K_{a}=1.753 \times 10^{-5}\). Finally, since \(\Delta_{ rxn } G^{\circ}=-R T \ln K_{a}\), we have that at 25째C

 

\(\begin{aligned}\Delta_{ rxn } G^{\circ}=& \bar{G}_{ CH _{3} COO ^{-}}(\text {ideal } 1 \text { molal solution })+\bar{G}_{ H +} \text { (ideal } 1 \text { molal solution) } \\&-\bar{G}_{ CH _{3} COOH }(\text { ideal } 1 \text { molal solution })=27.15 kJ / mol\end{aligned}\)
