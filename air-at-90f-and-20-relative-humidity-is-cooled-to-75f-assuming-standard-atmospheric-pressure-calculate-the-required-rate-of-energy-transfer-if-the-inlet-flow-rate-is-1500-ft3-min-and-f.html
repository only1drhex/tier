Guest Question.

Air at 90°F and 20% relative humidity is cooled to 75°F. Assuming standard atmospheric pressure, calculate the required rate of energy transfer, if the inlet flow rate is 1500 ft³/min, and find the final humidity, using (a) the psychrometric chart and (b) the equations.

Admin
Solution
\( \text { (a) The partial pressure is } P_{a 1}=P-P_{v 1}=P-\phi P_{g 1}=14.7-(0.2)(0.6988)=14.56 \text { psia; hence, } \)

\( \rho_{a 1}=\frac{P_{a 1}}{R_{a} T_{1}}=\frac{(14.56)(144)}{(53.3)(550)}=0.0715 lbm / ft ^{3} \).

\( \text { and } \dot{m}_{a}=(1500 / 60)(0.0715)=1.788 lbm / s \text {. The psychrometric chart at state } 1 \text { provides } h_{1}=28.5 Btu / lbm \text { dry air. } \) \( \text { With } \omega=\text { const., state } 2 \text { is located by following an } A-C \text { curve in Fig. } 12.4 \text {; we find } h_{2}=24.5 Btu / lbm dry \text { air. } \)

This gives

\( \dot{Q}_{a}=\dot{m}\left(h_{2}-h_{1}\right)=(1.788)(24.5-28.5)=-7.2 Btu / s \).

\( \text { The relative humidity is found on the psychrometric chart at state } 2 \text { to be } \phi_{2}=32.5 \% \text {. } \)

(b) The equations provide more accurate results than can be obtained by using the psychrometric chart. The value for \( \dot{m}_{a} \) a from part (a) has been calculated so we’ll simply use that number. The rate of heat transfer is

\( \dot{Q}=\dot{m}_{a}\left(h_{a 2}-h_{a 1}\right)+\dot{m}_{v}\left(h_{v 2}-h_{v 1}\right)=\dot{m}_{a} C_{p}\left(T_{2}-T_{1}\right)+\dot{m}_{v}\left(h_{v 2}-h_{v 1}\right) \).

\( \text { We find } \dot{m}_{v} \text { as follows: } \)

\( \dot{m}_{v}=\omega \dot{m}_{a}=0.622 \frac{\phi P_{g}}{P_{a}} \dot{m}_{a}=(0.622)(0.2)\left(\frac{0.6988}{14.56}\right)(1.788)=0.01067 lbm / s \).

\( \text { Thus } \dot{Q}=(1.788)(0.24)(75-90)+(0.01067)(1094.2-1100.7)=-6.51 Btu / s . \)

\( \text { To find the relative humidity using (12.22) we must find } P_{v 2} \text { and } P_{g 2} \text {. The final temperature is } 75^{\circ} F \text {; Table } C .1 E \) \( \text { gives, by interpolation, } P_{g 2}=0.435 \text { psia. Since the mass of vapor and the mass of dry air remain constant, the } \) partial pressure of vapor and dry air remain constant. Hence,

\( \phi=\frac{P_{v} V / R_{v} T}{P_{g} V / R_{v} T}=\frac{P_{v}}{P_{g}} \) (12.22).

\( P_{v 2}=P_{v 1}=\phi P_{g 1}=(0.2)(0.6988)=0.1398 psia \).

\( \text { The final relative humidity is } \phi_{2}=P_{v 2} / P_{g 2}=0.1398 / 0.435=0.321 \text { or } 32.1 \% \text {. } \)

\( \text { The values found in part }(b) \text { are more accurate than those of part }(a) \text {, especially for } Q \text {, since it is difficult to read } h_{1} \) \( \text { and } h_{2} \text { accurately. } \)
