Guest Question.

Find an expression for the change in internal energy if \( P=R T /(v-b)-\left(a / v^{2}\right) \text { and } C_{v}=A+B T \). Simplify the expression for an ideal gas with constant specific heats.

Admin
Solution

We integrate (11.41) as follows:

\( d u=C_{v} d T+\left[T\left(\frac{\partial P}{\partial T}\right)_{v}-P\right] d v \) (11.41).

\( \Delta u=\int C_{v} d T+\int\left[T\left(\frac{\partial P}{\partial T}\right)_{v}-P\right] d v \).

\( =\int(A+B T) d T+\int\left[T \frac{R}{v-b}-\frac{R T}{v-b}+\frac{a}{v^{2}}\right] d v \).

\( =\int_{T_{1}}^{T_{2}}(A+B T) d T+\int_{v_{1}}^{v_{2}} \frac{a}{v^{2}} d v \).

\( A\left(T_{2}-T_{1}\right)+\frac{1}{2} B\left(T_{2}^{2}-T_{1}^{2}\right)-a\left(\frac{1}{v_{2}}-\frac{1}{v_{1}}\right) \).

\( \text { For an ideal gas } P=R T / v \text { so that } a=b=0 \text {, and if } C_{v}=\text { const., we set } B=0 \text {. Then the above expression simplifies to } \) \( \Delta u=A\left(T_{2}-T_{1}\right)=C_{v}\left(T_{2}-T_{1}\right) . \)
