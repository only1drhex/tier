Guest Question.

Consider the circuit shown in Fig. 7.13a. The input function is the voltage pulse shown in Fig. 7.13b. Since the source is zero for all negative time, the initial conditions for the network are zero [ i.e., \(v_C\)(0-)=0 ]. The response \(v_o\)(t) for 0 < t < 0.3 s is due to the application of the constant source at t = 0 and is not influenced by any source changes that will occur later. At t =0.3 s the forcing function becomes zero, and therefore \(v_{o}\)(t) for t > 0.3 s is the source-free or natural response of the network.
Let us determine the expression for the voltage \(v_{o}(t)\).

Admin
Solution

Since the output voltage \(v_{o}(t)\) is a voltage division of the capacitor voltage, and the initial voltage across the capacitor is zero, we know that \(v_{o}\)(0+) = 0, as shown in Fig. 7.13c.

If no changes were made in the source after t = 0, the steady-state value of \(v_{o}\)(t)[ i.e., \(v_{o}(\infty)\)] due to the application of the unit step at t = 0 would be

\(v_{o}(\infty) =\frac{9}{6 \mathrm{k}+4 \mathrm{k}+8 \mathrm{k}}(8 \mathrm{k}) \)

 

\(=4 \mathrm{~V}\)

 

as shown in Fig. 7.13d.

 

The Th√©venin equivalent resistance is

\(R_{\mathrm{Th}} =\frac{(6 \mathrm{k})(12 \mathrm{k})}{6 \mathrm{k}+12 \mathrm{k}} \)

 

\(=4 \mathrm{k} \Omega\)

 

as illustrated in Fig. 7.13e.

 

Therefore, the circuit time constant \(\tau\) is

 

\(\tau =R_{\mathrm{Th}} C \)

 

\(=(4)\left(10^{3}\right)(100)\left(10^{-6}\right) \)

 

=0.4 s

 

Therefore, the response \(v_{o}(t)\) for the period 0 < t < 0.3 s is

 

\(v_{o}(t)=4-4 e^{-t / 0.4} \mathrm{~V} \qquad 0<t<0.3 \mathrm{~s}\)

 

 

The capacitor voltage can be calculated by realizing that using voltage division, \(v_{o}(t)=2 / 3 v_{C}(t)\). Therefore,

 

\(v_{C}(t)=\frac{3}{2}\left(4-4 e^{-t / 0.4}\right) \mathrm{V}\)

 

Since the capacitor voltage is continuous,

 

\(v_{C}(0.3-)=v_{C}(0.3+)\)

 

and therefore,

 

\(v_{o}(0.3+) =\frac{2}{3} v_{C}(0.3-) \)

 

\(=4\left(1-e^{-0.3 / 0.4}\right) \)

 

=2.11 V

 

Since the source is zero for t > 0.3 s, the final value for \(v_{o}(t)\) as \(t \rightarrow \infty\) is zero. Therefore, the expression for \(v_{o}(t)\) for t > 0.3 s is

 

\(v_{o}(t)=2.11 e^{-(t-0.3) / 0.4} \mathrm{~V} \quad t>0.3 \mathrm{~s}\)

 

The term \(e^{-(t-0.3) / 0.4}\) indicates that the exponential decay starts at t = 0.3 s. The complete solution can be written by means of superposition as

 

\(v_{o}(t)=4\left(1-e^{-t / 0.4}\right) u(t)-4\left(1-e^{-(t-0.3) / 0.4}\right) u(t-0.3) \mathrm{V}\)

 

 

or, equivalently, the complete solution is

 

\(v_{o}(t)=\left\{\begin{array}{ll}0 & t<0 \\4\left(1-e^{-t / 0.4}\right) \mathrm{V} & 0<t<0.3 \mathrm{~s} \\2.11 e^{-(t-0.3) / 0.4} \mathrm{~V} & 0.3 \mathrm{~s}<t\end{array}\right\}\)

 

which in mathematical form is

 

\(v_{o}(t)=4\left(1-e^{-t / 0.4}\right)[u(t)-u(t-0.3)]+2.11 e^{-(t-0.3) / 0.4} u(t-0.3) \mathrm{V}\)

 

Note that the term \([u(t)-u(t-0.3)]\) acts like a gating function that captures only the part of the step response that exists in the time interval 0 < t < 0.3 s. The output as a function of time is shown in Fig. 7.13f.
