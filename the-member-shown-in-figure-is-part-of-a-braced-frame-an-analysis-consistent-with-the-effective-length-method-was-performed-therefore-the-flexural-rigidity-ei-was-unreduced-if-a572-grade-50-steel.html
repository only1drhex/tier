Guest Question.

The member shown in Figure is part of a braced frame. An analysis consistent with the effective length method was performed; therefore, the flexural rigidity, EI, was unreduced. If A572 Grade 50 steel is used, is this member adequate? \(L_={cx} = L_{cy} = L.\)

Admin
Solution

The factored loads, computed from load combination 2, are shown in Figure 6.11. Determine which interaction formula to apply. The required compressive strength is
\(P_{r} = P_{u} = P_{nt} + B_{2}P_{ℓt} = 420 + 0 = 420 kips\) (\(B_{2}\) = 0 for a braced frame)
From Manual Table 6-2, for \(L_{c}\) = 14 feet, the axial compressive strength of a
W12 \(\times\) 65 is
\(\phi _{c}P_{n} = 685\) kips

\(\frac{P_{u}}{\phi _{c}p_{n}} = \frac{420}{685} = 0.6131 > 0.2 \therefore\) Use Equation 6.3 (AISC Equation H1-1a).
In the plane of bending,

\(P_{e1} = =\frac{\pi ^{2}EI}{(L_{c1})^{2}}=\frac{\pi ^{2}EI_{x}}{(L_{cx})^{2}}=\frac{\pi ^{2}(29,000)(533)}{(14 \times 12)} = 5405 kips\)

 

\(C_{m} = 0.6 – 0.4\frac{M_{1}}{M_{2}} = 0.6 – 0.4 \left ( – \frac{70.8}{82.4} \right )= 0.9437\)

 

\(B_{1} = \frac{C_{m}}{1 – (\alpha P_{r}/P_{e1})}=\frac{C_{m}}{1 – (1.00P_{u}/P_{e1})}= \frac{0.9437}{1 – (420/5405)}= 1.023\)
From Manual Table 6-2 with C_{b}= 1.0 and L_{b} = 14 feet, the moment strength is
\(\phi _{b}M_{n} = 345\) ft-kips
For the actual value of C_{b}, refer to the moment diagram of Figure 6.11:
\(C_{b} = \frac{12.5M_{max}}{2.5M_{max} + 3M_{A} + 4M_{B} + 3M_{C}}\)

 

= \(\frac{12.5(82.4)}{2.5(82.4) + 3(73.7) + 4(76.6) + 3(79.5)}=1.060\)

 

\(\therefore \phi _{b}M_{n} = C_{b} (345) = 1.060(345) = 366 ft-kips.\)
But \(\phi _{b}M_{p} = 356\) ft-kips (obtained from Table 6-2 as \(\phi _{b}M_{n} for L_{b} = 0\)) , 366 ft-kips, therefore use \(\phi _{b}M_{n} = 356\) ft-kips. (Since a W12 \(\times\) 65 is noncompact for \(F_{y}= 50\) ksi, 356 ft-kips is the design strength based on FLB rather than full yielding of the cross section.) The factored load moments are
\(M_{nt} = 82.4 ft-kips M_{ℓt} = 0\)
From AISC Equation A-8-1, the required moment strength is
\(M_{r} = M_{u} = B_{1}M_{nt} + B_{2}M_{ℓt} = 1.023(82.4) + 0 = 84.30 ft-kips = M_{ux}\)
From Equation 6.3 (AISC Equation H1-1a),

\(\frac{P_{u}}{\phi _{c}P_{n}}+\frac{8}{9}\left ( \frac{M_{ux}}{\phi _{b}M_{nx}}+\frac{M_{uy}}{\phi _{b}M_{ny}} \right ) = 0.6131 +\frac{8}{9}\left (\frac{84.30}{356}+0 \right ) = 0.824 < 1.0\) (OK)
The member is satisfactory.
The service loads, computed from load combination 2, are shown in Figure 6.12. Determine which interaction formula to apply. The required compressive strength is
\(P_{r} = P_{a} = P_{nt} + B_{2}P_{ℓt} = 280 + 0 = 280 kips (B_{2} = 0\) for a braced frame)
From Table 6-2, for \(L_{c}\) = 14 feet, the axial compressive strength of a W12 \(\times\) 65 is
\(\frac{P_{n}}{\Omega _{c}}= 456 kips\)

 

\(\frac{P_{a}}{p_{n}/\Omega _{c}}= \frac{280}{456} = 0.6140 > 0.2 \therefore\) Use Equation 6.5 (AISC Equation H1-1a).
In the plane of bending,
\(P_{e1} = =\frac{\pi ^{2}EI}{(L_{c1})^{2}}=\frac{\pi ^{2}EI_{x}}{(L_{cx})^{2}}=\frac{\pi ^{2}(29,000)(533)}{(14 \times 12)} =5405 kips\)

 

\(C_{m} = 0.6 – 0.4\frac{M_{1}}{M_{2}} = 0.6 – 0.4 \left ( – \frac{47}{55} \right )= 0.9418\)

 

\(B_{1} = \frac{C_{m}}{1 – (\alpha P_{r}/P_{e1})}=\frac{C_{m}}{1 – (1.60P_{a}/P_{e1})}= \frac{0.9418}{1 – (1.60 \times 280 /5405)}= 1.027\)
Next, from Table 6-2 with \(C_{b}\) = 1.0 and \(L_{b}\) = 14 feet, the moment strength is
\(\frac{M_{n}}{\Omega _{b}}= 230 \)ft-kips
For the actual value of \(C_{b}\), refer to the moment diagram of Figure 6.12:
\(C_{b} = \frac{12.5M_{max}}{2.5M_{max} + 3M_{A} + 4M_{B} + 3M_{C}}\)

 

\(= \frac{12.5(55)}{2.5(55) + 3(49) + 4(51) + 3(53)}=1.062\)

 

\(\therefore \frac{M_{n}}{\Omega _{b}} = C_{b}(230) = 1.062(230) = 244.3\) ft­kips.
But \(M_{p}/\Omega _{b} = 237 ft-kips\) (from Table 6-2 with \(L_{b}\) = 0) < 244.3 ft-kips, so use \(M_{n}/\Omega _{b}\) =237 ft-kips.
(Since a W12 \(\times\) 65 is noncompact for \(F_{y} \)= 50 ksi, 237 ft-kips is the strength based on
FLB rather than full yielding of the cross section.) The unamplified moments are
\(M_{nt} = 55 ft-kips M_{ℓt} = 0\)
From AISC Equation A-8-1, the required moment strength is
\(M_{r} = M_{a} = B_{1}M_{nt} + B_{2}M_{ℓt} = 1.027(55) + 0 = 56.49 ft-kips = M_{ax}\)
From Equation 6.5 (AISC Equation H1-1a),

\(\frac{P_{a}}{P_{n}/\Omega _{c}}+\frac{8}{9}\left ( \frac{M_{ax}}{M_{nx}/\Omega _{b}}+\frac{M_{uy}}{M_{ny}\Omega _{b}} \right ) =\frac{280}{456}+\frac{8}{9}\left (\frac{56.49}{237}+0 \right ) = 0.826 < 1.0\) (OK)
The member is satisfactory.
