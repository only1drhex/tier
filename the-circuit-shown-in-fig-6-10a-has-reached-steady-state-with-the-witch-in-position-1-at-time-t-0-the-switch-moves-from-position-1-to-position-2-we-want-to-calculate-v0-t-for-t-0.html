Guest Question.

The circuit shown in Fig. 6.10a has reached steady state with the witch in position 1 . At time t = 0 the switch moves from position 1 to position 2. We want to calculate \(v_{o}\)( t) for t > 0.

Admin
Solution

Step 1. \(v_{o}\)(t)is of the form \( K_{1} + K_{2} e^{-t/\tau }\)

Step 2. Using the circuit in Fig. 6.10b, we can calculate\( i_{L}\)(0-)

\(i_{A}=\frac{12}{4}= 3 \ A \)

Then

\(i_{L}(0-)= \frac{12+2i_{A}}{6}=\frac{18}{6}= 3 \ A\)

Step 3. The new circuit, valid only for t = 0+, is shown in Fig. 6.10c. The value of the current source that replaces the inductor is \( i_{L}\)(0-) = \( i_{L}\)(0+)= 3 A. Because of the current source \( v_{o}\)(0-) (0+) = (3)(6) = 18 V

Step 4. The equivalent circuit, for the steady-state condition after switch closure, is given in Fig. 6.10d. Using the voltages and currents defined in the figure, we can compute \(v_{o}(\infty )\)in a variety of ways. For example, using node equations we can find \(v_{o}(\infty )\) from

\(\frac{v_{B}-36}{2}+\frac{v_{B}}{4}+\frac{v_{B}}{4} + \frac{v_{B}+2i^{'}_{A}}{6}=0\)

 

\(i^{'}_{A}=\frac{v_{B}}{4}\)

 

\(v_{o}(\infty )= v_{B}+i^{'}_{A}\)

or, using loop equations,

\(36= 2(i_{1}+i_{2})+4i_{1}\)

 

\(36=2(i_{1}+i_{2})+6i_{2}-2i_{1}\)

 

\(v_{o}(\infty )=6i_{2}\)

Using either approach, we find that \(v_{o}(\infty )\)= 27 V.
Step 5. The Thevenin equivalent resistance can be obtained via \(v_{oc}\) and \(i_{sc}\) because of the presence of the dependent source. From Fig. 6.10e we note that

\(i^{'}_{A}=\frac{36}{2+4}= 6 \ A \)

Therefore

\(v_{\infty }=(4)(6)+2(6)\)

 

\(\ \ \ \ \ = 36 \ V\)

From Fig. 6.10f we can write the following loop equations:

\(36= 2(i^{'''}_{A}+i_{sc})+4i^{'''}_{A}\)

 

\(36=2(i^{'''}_{A}+i_{sc})+6i_{sc}-2i^{'''}_{A}+i_{sc}\)

Solving these equations for \(i_{sc}\) yields

\(i_{sc}=\frac{9}{2}A\)

Therefore,

\(R_{Th}= \frac{v_{oc}}{i_{sc}}=\frac{36}{9/2}= 8 \\Omega \)

Hence, the circuit time constant is

\(\tau =\frac{L}{R_{Th}}=\frac{3}{8}s\)

Step 6. Using the information just computed, we can derive the final equation for \(v_{o}\) ( t):

\(K_{1}\) = \(v_{o}(\infty )\)= 27
\(K_{1}\) = \(v_{o}(o+ )\) – \(v_{o}(\infty )\) = 18 – 27 = – 9

Therefore,

\(v_{o}(t)= 27-9e^{-1/(3/8)}V\)
