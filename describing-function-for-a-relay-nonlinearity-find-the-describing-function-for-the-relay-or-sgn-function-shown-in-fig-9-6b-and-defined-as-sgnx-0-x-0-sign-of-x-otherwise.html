Guest Question.

Describing Function for a Relay Nonlinearity
Find the describing function for the relay or sgn function shown in Fig. 9.6(b) and defined as

\( \text{sgn}(x) = 0, \ \ \ \ \ \ \ \ \ x = 0, \)

\( \ \ \ \ \ \ \ \ \ \ = \text{sign of } x, \ \ \ \text{ otherwise}.\)

Admin
Solution

The output is a square wave of amplitude \( N \) for every size input; thus \( Y_1 = \frac{4N}{π} \) and \( K_{eq} = \frac{4N}{πa} .\) The solution can also be obtained from Eq. (9.29) if we let \( k → ∞. \) For small angles,

\( K_{e q}(a)=\left\{\begin{array}{cc} \frac{2}{\pi}\left(k \sin ^{-1}\left(\frac{N}{a k}\right)+\frac{N}{a} \sqrt{1-\left(\frac{N}{k a}\right)^{2}}\right), & \frac{k a}{N}>1 \\ k, & \frac{k a}{N} \leq 1 \end{array}\right. \) (9.29)

\( \sin^{−1} \left( \frac{N}{ak} \right) ≅ \frac{N}{ak} ,\)

and thus, from Eq. (9.27), we have

\( DF = K_{eq}(a) = \frac{b_1}{a} \) (9.27)

\( K_{eq}(a) = \frac{2}{π} \left( k \left( \frac{N}{ak}\right) + \frac{N}{a} \right) = \frac{4N}{πa}. \) (9.30)
