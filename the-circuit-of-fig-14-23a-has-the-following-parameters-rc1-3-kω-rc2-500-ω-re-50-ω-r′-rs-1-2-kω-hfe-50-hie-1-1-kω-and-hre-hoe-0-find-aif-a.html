Guest Question.

The circuit of Fig. 14.23(a) has the following parameters: \(R_{c1} = 3 kΩ, R_{c2} = 500 Ω, R_{E} = 50 Ω, R^{\prime} = R_{s} = 1.2 kΩ, h_{fe} = 50, h_{ie}\) = 1.1 kΩ, and \(h_{re} = h_{oe}\) = 0. Find \(A_{If}, A_{Vf}, R_{if}, R_{of}\) and \(R^{\prime}_{of}\).

Admin
Solution

Referring to Fig. 14.23(a), the current gain without feedback is given by

\(A_{I} =\frac{– I_{c2}}{I_{s}}=\frac{– I_{c2}}{I_{b2}}\frac{I_{b2}}{I_{c1}}\frac{I_{c1}}{I_{b1}}\frac{I_{b1}}{I_{s}}\)

where \(\frac{–I_{c2}}{I_{b2}}= – h_{fe} = –50\frac{I_{c1}}{I_{b1}}= +h_{fe}\) = +50

\(R_{i2} = h_{ie} + (1 + h_{fe}) (R_{E} || R^{\prime}) = 1.14 + (51) (\frac{50 × 1.2 × 10^{3}}{1.25 × 10^{3}})\) = 3.55 kΩ

\(\frac{I_{b2}}{I_{c1}}=\frac{–R_{c1}}{R_{c1}+ R_{i2}}=\frac{–3}{3 + 3.55}\)= – 0.458

R ≡ \(R_{s} || (R^{\prime}+ R_{E}) =\frac{(1.2 × 10^{3}) (1.25 × 10^{3})}{(1.2 + 1.25) × 10^{3}}\) = 0.612 kΩ

\(\frac{I_{b1}}{I_{s}}=\frac{R}{R + h_{ie}}=\frac{0.61 × 10^{3}}{(0.61 + 1.1) × 10^{3}}\)= 0.358

Substituting all the above current ratio numerical values in \(A_{I}\) equation results in

\(A_{I}\) = (–50) (– 0.458) (50) (0.358) = 410

The feedback factor is given by

β =\(\frac{R_{E}}{R^{\prime} + R_{E}}=\frac{50}{1,250}\)= 0.04

The desensitivity is given by

D = 1 + β\(A_{I}\) = 1 + (0.040) (410) = 17.4

Current gain and voltage gain with feedback is given by

\(A_{If} =\frac{A_{I}}{D}=\frac{410}{17.4}\)= 23.6

\(A_{Vf} =\frac{V_{o}}{V_{s}}=\frac{–I_{c2}R_{c2}}{I_{s}R_{s}}=\frac{A_{If}R_{c2}}{R_{s}}=\frac{(23.6) (0.5)}{1.2}\) = 9.83

The approximate expression for voltage gain results in

\(A_{Vf} ≈ \frac{R_{c2}}{βR_{s}}=\frac{0.5}{(0.04) (1.2)}\) = 10.4

which is in error by 6% with the actual value.
The input resistance without feedback, seen by the current source is

\(R_{i} = R || h_{ie} =\frac{(0.61 × 10^{3}) (1.1 × 10^{3})}{1.71 × 10^{3}}\)= 0.394 kΩ

and the resistance \(R_{if}\) with feedback, seen by the current source is

\(R_{if} =\frac{R_{i}}{D}=\frac{394}{17.4}\)= 22.6 Ω

If \(R_{c2}\) is considered as an external load, then \(R_{o}\) is the resistance seen looking into the collector of \(Q_{2}\).
Since \(h_{0e}\) = 0, then \(R_{0}\) = ∞. The output resistance with feedback without considering the load, is written as \(R_{0f} = R_{0} (1 + βA_{i})\) = ∞.
We know that, for current-shunt feedback, \(A_{I}\) is independent of the load (\(R_{L} = R_{c2}\)).
Hence \(A_{i} =\underset{R_{c2} → 0}{lim} A_{I} = A_{I}\). Since \(R^{\prime}_{o} = R_{o} || R_{c2} = R_{c2}\), then the output resistance with feedback, by considering the load, \(R^{\prime}_{of} = R^{\prime}_{o}\frac{1 +βA_{i}}{1 + βA_{I}}= R^{\prime}_{o}= R_{c2}\) = 500 Ω.
