Guest Question.

Calculate the electric and magnetic fields of an oscillating magnetic dipole without using approximation 3. [Do they look familiar? Compare .] Find the Poynting vector, and show that the intensity of the radiation is exactly the same as we got using approximation 3.

Admin
Solution

Go back to Eq. 11.33:

\(A (r, \theta, t)=\frac{\mu_{0} m_{0}}{4 \pi}\left(\frac{\sin \theta}{r}\right)\left\{\frac{1}{r} \cos [\omega(t-r / c)]-\frac{\omega}{c} \sin [\omega(t-r / c)]\right\} \hat{ \phi }\) (11.33)

\(A =\frac{\mu_{0} m_{0}}{4 \pi}\left(\frac{\sin \theta}{r}\right)\left\{\frac{1}{r} \cos [\omega(t-r / c)]-\frac{\omega}{c} \sin [\omega(t-r / c)]\right\} \hat{\phi}\).

Since V = 0 here,

\(E =-\frac{\partial A }{\partial t}=-\frac{\mu_{0} m_{0}}{4 \pi}\left(\frac{\sin \theta}{r}\right)\left\{\frac{1}{r}(-\omega) \sin [\omega(t-r / c)]-\frac{\omega}{c} \omega \cos [\omega(t-r / c)]\right\} \hat{\phi}\)

 

\(=\frac{\mu_{0} m_{0} \omega}{4 \pi}\left(\frac{\sin \theta}{r}\right)\left\{\frac{1}{r} \sin [\omega(t-r / c)]+\frac{\omega}{c} \cos [\omega(t-r / c)]\right\} \hat{\phi}\) .

\(B = \nabla \times A =\frac{1}{r \sin \theta} \frac{\partial}{\partial \theta}\left(A_{\phi} \sin \theta\right) \hat{ r }-\frac{1}{r} \frac{\partial}{\partial r}\left(r A_{\phi}\right) \hat{ \theta }\)

 

\(=\frac{\mu_{0} m_{0}}{4 \pi}\left\{\frac{1}{r \sin \theta} \frac{2 \sin \theta \cos \theta}{r}\left[\frac{1}{r} \cos [\omega(t-r / c)]-\frac{\omega}{c} \sin [\omega(t-r / c)]\right] \hat{ r }\right.\)

 

\(\left.-\frac{\sin \theta}{r}\left[-\frac{1}{r^{2}} \cos [\omega(t-r / c)]+\frac{\omega}{r c} \sin [\omega(t-r / c)]-\frac{\omega}{c}\left(-\frac{\omega}{c}\right) \cos [\omega(t-r / c)]\right] \hat{\theta}\right\}\)

 

\(=\begin{aligned}\frac{\mu_{0} m_{0}}{4 \pi} &\left\{\frac{2 \cos \theta}{r^{2}}\left[\frac{1}{r} \cos [\omega(t-r / c)]-frac{\omega}{c} \sin [\omega(t-r / c)]\right] \hat{ r }\right.\\&\left.-\frac{\sin \theta}{r}\left[-\frac{1}{r^{2}} \cos [\omega(t-r /c)]+\frac{\omega}{r c} \sin [\omega(t-r / c)]+\left(\frac{\omega}{c}\right)^{2} \cos [\omega(t-r / c)]\right] \hat{ \theta }\right\}\end{aligned}\)
