Guest Question.

Given:

The void ratio versus pressure data shown below. The initial void ratio is 0.725, and the existing vertical effective overburden pressure is 130 kPa.

Void Ratio Pressure (kPa)

0.445 1600
0.460 400
0.492 100
0.530 25

Required:

a. Plot the data as e versus \(\log \sigma_{v c}^{\prime}\).

b. Evaluate the overconsolidation ratio.

c. Determine the field compression index using the Schmertmann procedure:

d. If this consolidation test is representative of a 12 m thick clay layer, compute the settlement of this layer if an additional stress of 220 kPa were added.

Admin
Solution

a. The data is plotted in Fig. Ex. 8.17.

b. The given value of \(\sigma_{v o}^{\prime}\) is plotted on the graph, and the Casagrande construction performed to evaluate \(\sigma_{p}^{\prime}\). A value of 190 kPa is found

 

\(OCR =\frac{\sigma_{p}^{\prime}}{\sigma_{v o}^{\prime}}=\frac{190}{130}=1.46\)

 

Thus the soil is slightly overconsolidated.

c. Using the Schmertmann procedure for overconsolidated clays, control points 1,2, and 3 are established, as shown in Fig. Ex. 8.17. The values of \(C_{r} \text { and } C_{c}\) are evaluated directly from Fig. Ex. 8.17 over one log cycle. \(C_{r}=0.611-0.589=0.022, \text { and } C_{c}=0.534-0.272=0.262\). (Note that \(C_{r} \approx 10 \% \text { of } C_{c}\).)

d. Using Eq. (8.18b), the settlement is computed:

 

\(s_{c}=C_{r} \frac{H_{o}^{ }}{1+e_{o}^{\prime}} \log \frac{\sigma_{p}^{\prime}}{\sigma_{v o}^{\prime}}+C_{c} \frac{H_{o}}{1+e_{o}} \log \frac{\sigma_{v o}^{\prime}+\Delta \sigma_{v}}{\sigma_{p}^{\prime}}\) (8.18b)

 

\(\begin{aligned}s_{c} &=\frac{C_{r}}{1+e_{o}} H_{o} \log \frac{\sigma_{p}^{\prime}}{\sigma_{v o}^{\prime}}+\frac{C_{c}}{1+e_{o}} H_{o} \log \frac{\sigma_{v o}^{\prime}+\Delta \sigma}{\sigma_{p}^{\prime}} \\&=\frac{0.022}{1+0.725}(12 m ) \log \frac{190}{130}+\frac{0.262}{1+0.725}(12 m ) \log \frac{130+220}{190} \\&=0.025 m +0.484 m \\&=0.509 m \approx 0.5 m\end{aligned}\)
