Guest Question.

A short rectangular post supports a compressive load of P = 2,500 lb as shown in Figure P15.34a. A top view of the post showing the location where load P is applied to the top of the post is shown in Figure P15.34b. The cross-sectional dimensions of the post are b = 5 in. and d = 10 in. The load P is applied at offset distances of \(y_{P}\) = 3 in. and \(z_{P}\) = 2 in. from the centroid of the post. Determine the normal stresses at corners A, B, C, and D of the post.

Admin
Solution

Section properties:

\(\begin{aligned}&A=(5 in. )(10 in .)=50 in. ^{2} \\&I_{y}=\frac{(10 \text { in. })(5 in .)^{3}}{12}=104.167 in .^{4} \\&I_{z}=\frac{(5 \text { in. })(10 in .)^{3}}{12}=416.667 in .^{4}\end{aligned}\)

Equivalent forces at base of post:

\(\begin{aligned}&F=2,500 lb \\&M_{y}=(2,500 lb )(-2 in .)=-5,000 lb – in. \\&M_{z}=(2,500 lb )(-3 in .)=-7,500 lb – in.\end{aligned}\)

Axial stress due to F:

\(\sigma_{\text {axial }}=\frac{2,500 lb }{50 in .^{2}}=50 psi ( C )\)

Bending stress due to \(M _{ y }\):

\(\sigma_{\text {bend } y}=\frac{M_{y} z}{I_{y}}=\frac{(5,000 lb – in. )(\pm 2.5 in .)}{104.167 in.^{4}}=\pm 120 psi\)

Bending stress due to \(M _{z}\):

\(\sigma_{\text {bend } z}=\frac{M_{z} y}{I_{z}}=\frac{(7,500 lb – in .)(\pm 5 in .)}{416.667 in. ^{4}}=\pm 90 psi\)

 

Normal stresses at corners A, B, C, and D: The normal stresses acting at each of the four corners of the post can be determined by superimposing the results obtained above. In all instances, the normal stresses act in the vertical direction; that is, the y direction. The sense of the stress, either tension or compression, can be determined by inspection.

__________________________________________________________________________________________________

Corner A:

\(\begin{aligned}\sigma_{A} &=50 psi ( C )+120 psi ( C )+90 psi ( C ) \\&=-50 psi -120 psi -90 psi \\&=-260 psi =260 psi ( C )\end{aligned}\)

__________________________________________________________________________________________________

Corner B:

\(\begin{aligned}\sigma_{B} &=50 psi ( C )+120 psi ( T )+90 psi ( C ) \\&=-50 psi +120 psi -90 psi \\&=-20 psi =60 psi ( C )\end{aligned}\)

__________________________________________________________________________________________________

Corner C:

\(\begin{aligned}\sigma_{C} &=50 psi ( C )+120 psi ( T )+90 psi ( T ) \\&=-50 psi +120 psi +90 psi \\&=160 psi =160 psi ( T )\end{aligned}\)

__________________________________________________________________________________________________

Corner D:

\(\begin{aligned}\sigma_{D} &=50 psi ( C )+120 psi ( C )+90 psi ( T ) \\&=-50 psi -120 psi +90 psi \\&=-80 psi =80 psi ( C )\end{aligned}\)
