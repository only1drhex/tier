Guest Question.

For the constant elevations of the water surfaces shown in Fig. 9-14(a), what flows will occur?

Admin
Solution

Because the elevation of the hydraulic grade line at C cannot be computed (all flows unknown), the problem will be solved by successive trials. A convenient assumption is to choose the elevation of the hydraulic grade line at C at 190.0′. By so assuming, flow to or from reservoir B will be zero, thus reducing the number of calculations.
For elevation of hydraulic grade line at C = 190.0′,

\(S_{24}\) = (212 – 190)/8 = 2.75 ft/1000 ft and Q = 7.15 mgd to C.

\(S_{12}\) = (190 – 100)/4 = 22.5 ft/1000 ft and Q = 3.60 mgd away from C.

Examination of these values of flow indicates that the grade line at C must be higher, thereby reducing the flow from A and increasing the flow to D as well as adding flow to B. In an endeavor to “straddle” the correct elevation at C, assume a value of 200.0′. Thus, for elevation at C = 200.0′,

\(\begin{aligned}&S_{24}=(212-200) / 8=1.50 ft / 1000 ft \quad \text { and } \quad Q=5.15 mgd \text { to } C \\&S_{16}=(200-190) / 4=2.50 ft / 1000 ft \quad \text { and } \quad Q=2.82 mgd \text { away from } C \\&S_{12}=(200-100) / 4=25.0 ft / 1000 ft \quad \text { and } \quad Q=3.80 mgd \text { away from } C\end{aligned}\)

The flow away from C = 6.62 mgd against the flow to C of 5.15 mgd. Using Fig. 9-l4(b) to obtain a guide regarding a reasonable third assumption, connect plotted points R and S. The line so drawn intersects the \(\left(Q_{\text {to }}-Q_{\text {away }}\right) \text { zero abscissa at } Q_{\text {to }}=5.7\) mgd (scaled). Inasmuch as the values plotted do not vary linearly, use a flow to C slightly larger, say 6.0 mgd.
\(\text { For } Q=6.0 mgd \text { to } C, S_{24}=1.98 ft / 1000 ft , LH _{A-C}=1.98 \times 8=15.8 ft\), and the hydraulic grade line at C is at elevation (212 – 15.8) = 196.2′. Then

\(\begin{array}{ll}S_{16}=6.2 / 4=1.55 ft / 1000 ft , & Q=2.17 mgd \text { from } C \\S_{12}=96.2 / 4=24.05 ft / 1000 ft , & Q=3.73 mgd \text { from } C\end{array}\)

total Q from C = 5.90 mgd.

These resulting flows agree sufficiently well as not to require further calculations. (An elevation of the hydraulic grade line at C of about 196.5′ would probably prove correct, with flows to C and from C of about 5.95 mgd.)
