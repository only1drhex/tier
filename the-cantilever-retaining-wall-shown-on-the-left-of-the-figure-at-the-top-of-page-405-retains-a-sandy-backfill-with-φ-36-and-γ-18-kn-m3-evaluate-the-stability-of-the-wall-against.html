Guest Question.

The cantilever retaining wall shown on the left of the figure at the top of page 405 retains a sandy backfill with \(\phi^{\prime}=36^{\circ} \text { and } \gamma=18\) kN / m \(^{3}\). Evaluate the stability of the wall against sliding, overturning, and bearing capacity failure. Assume that the same sand exists on the passive side and below the wall. \(\gamma_{\text {concrete }}=24\) kN / m \(^{3}\). Draw the pressure distribution beneath the retaining wall.

Admin
Solution
\(\phi^{\prime}=36^{\circ} \rightarrow K_{A}=0.260 \text { and } K_{P}=3.852\)

 

The free body diagram is shown on the right of the figure on page 405:

 

\(P_{A}=0.5 K_{A} \gamma H^{2}=0.5 \times 0.260 \times 18 \times 5.5^{2}=70.8 kN \text { per m width }\)

 

\(P_{P}=0.5 K_{P} \gamma h^{2}=0.5 \times 3.852 \times 18 \times 0.5^{2}=8.7 kN \text { per m width }\)

 

Let’s tabulate the values of \(W_{i}, x_{i}\), and Wixi as below.

 

Block No. \(\text { Weight } W_{i}( kN \text { per m) }\) \(\text { Hor. Distance } x_{i}( m )\) \(W_{i} x_{i}( kN – m \text { per } m )\)
1 \(0.5 \times 5 \times 24=60.0\) 1.5 90.0
2 \(3.25 \times 0.5 \times 24=39.0\) 1.625 63.4
3 \(1.5 \times 5 \times 18=135.0\) 2.5 337.5
\(\Sigma W_{i}=234.0 kN \text { per } m\) \(\Sigma W_{i} x_{i}=490.9 kNm \text { per } m\)

 

\(\text { Let's take } \delta \text { as } 2 / 3 \phi^{\prime} \rightarrow \delta=24^{\circ} \text { : }\)

 

\(F_{\text {sliding }}=\frac{\text { Maximum resistance available }}{\text { Driving force }}=\frac{P_{P}+S_{\max }}{P_{A}}=\frac{8.7+234.0 \tan 24}{70.8}=1.59>1.5\)

 

\(F_{\text {overturning }}=\frac{\text { Resisting moment }}{\text { Driving force }}=\frac{\left(\sum_{i=1}^{n} W_{i} x_{i}\right)+P_{P} \frac{h}{3}}{P_{A} \frac{H}{3}}=\frac{490.9+8.7 \times 0.17}{70.8 \times 1.83}=3.80>2\)

 

Bearing pressures:

 

\(R=\Sigma W_{i}=234.0 kN \text { per m; } S=P_{A}-P_{P}=70.8-8.7=62.1 kN \text { per m }\)

 

Note: We are assuming that the passive resistance is fully mobilized while the sliding resistance is only partially mobilized.

Inclination of the load to vertical:

 

\(\alpha=\tan ^{-1} \frac{S}{R}=\tan ^{-1} \frac{62.1}{234}=14.9^{\circ}\)

 

\(\bar{x}=\frac{\left(\sum_{i=1}^{n} W_{i} x_{i}\right)+P_{P} \frac{h}{3}-P_{A} \frac{H}{3}}{R}=\frac{490.9+8.7 \times 0.167-70.8 \times 1.83}{234}=1.550 m\)

 

\(\therefore \text { eccentricity, } e=1.625-1.550=0.075 m \text { or } 75 mm \leftarrow \text { well within }\) the middle third of the base

 

\(q_{\max }=\frac{Q}{B}\left(1+\frac{6 e}{B}\right)=\frac{234}{3.25}\left(1+\frac{6 \times 0.075}{3.25}\right)=82.0 kPa\)

 

\(q_{\min }=\frac{Q}{B}\left(1-\frac{6 e}{B}\right)=\frac{234}{3.25}\left(1-\frac{6 \times 0.075}{3.25}\right)=62.0 kPa\)

 

\(\text { Bearing capacity calculations: } B^{\prime}=B-2 e=3.25-2 \times 0.075=3.10 m\)

 

\(\text { Plane-strain correction: } \phi^{\prime}=1.1 \times 36=39.6^{\circ} \rightarrow N_{q}=60, N_{\gamma, \text { Meyerhof }}=86\)

 

\(\text { Shape factors: } s_{q}=s_{\gamma}=1\)

 

Depth factors:

 

\(d_{q}=d_{\gamma}=1+0.1 \frac{D_{f}}{B} \tan \left(45+\frac{\phi}{2}\right)=1+0.1 \times \frac{0.5}{3.25} \times \tan 64.8=1.04\)

 

Inclination factors:

 

\(i_{q}=\left(1-\frac{\alpha^{\circ}}{90}\right)^{2}=\left(1-\frac{14.9}{90}\right)^{2}=0.70\)

 

\(i_{\gamma}=\left(1-\frac{\alpha}{\phi}\right)^{2}=\left(1-\frac{14.9}{39.6}\right)^{2}=0.39\)

 

Applying Equation 12.7:

 

\(q_{\text {ult }}=s_{c} d_{c} i_{c} c N_{c}+s_{q} d_{q} i_{q} \gamma_{1} D_{f} N_{q}+s_{\gamma} d_{\gamma} i_{\gamma} 0.5 B \gamma_{2} N_{\gamma}\) (12.7)

 

\(q_{\text {ult }}=s_{q} d_{q} i_{q} \gamma_{1} D_{f} N_{q}+s_{\gamma} d_{\gamma} i_{\gamma} 0.5 B \gamma_{2} N_{\gamma}\)

 

\(\begin{aligned}q_{\text {ult,gross }}=& 1.0 \times 1.04 \times 0.70 \times 18 \times 0.5 \times 60+1.0 \times 1.04 \times 0.39 \times 0.5 \\& \times 3.10 \times 18 \times 86=1366.3 kPa\end{aligned}\)

 

\(q_{ ult , \text { net }}=1366.3-0.5 \times 18=1357.3 kPa\)

 

From Equation 14.11:

 

\(F_{\text {bearing capacity }}=\frac{q_{\text {ult, net }}}{q_{\max }}\) (14.11)

 

\(F_{\text {bearing capacity }}=\frac{1357.3}{82.0}=16.6>3\)

 

The retaining wall is very safe with respect to sliding, overturning, and bearing capacity.
