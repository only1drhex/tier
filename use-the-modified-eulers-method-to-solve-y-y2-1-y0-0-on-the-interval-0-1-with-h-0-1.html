Guest Question.

Use the modified Euler’s method to solve \(y^{\prime}=y^{2}+1 ; y(0)=0\) on the interval [0, 1] with h = 0.1.

Admin
Solution

Here \(f(x, y)=y^{2}+1, x_{0}=0, \text { and } y_{0}=0\). From (19.2) we have \(y_{0}^{\prime}=f(0,0)=(0)^{2}+1=1\). Then using (19.4) and (19.3), we compute

\(\begin{array}{ll}n=0: & x_{1}=0.1 \\& p y_{1}=y_{0}+h y_{0}^{\prime}=0+0.1(1)=0.1 \\& p y_{1}^{\prime}=f\left(x_{1}, p y_{1}\right)=f(0.1,0.1)=(0.1)^{2}+1=1.01 \\& y_{1}=y_{0}+(h / 2)\left(p y_{1}^{\prime}+y_{0}^{\prime}\right)=0+0.05(1.01+1)=0.1005 \\& y_{1}^{\prime}=f\left(x_{1}, y_{1}\right)=f(0.1,0.1005)=(0.1005)^{2}+1=1.0101003 \\n= 1 : & x_{2}=0.2 \\& p y_{2}=y_{1}+h y_{1}^{\prime}=0.1005+0.1(1.0101003)=0.2015100 \\& p y_{2}^{\prime}=f\left(x_{2}, p y_{2}\right)=f(0.2,0.2015100)=(0.2015100)^{2}+1=1.0406063 \\& y_{2}=y_{1}+(h / 2)\left(p y_{2}^{\prime}+y_{1}^{\prime}\right)=0.1005+0.05(1.0406063)+1.0101002=0.2030353 \\& y_{2}^{\prime}=f\left(x_{2}, y_{2}\right)=f(0.2,0.2030353)=(0.2030353)^{2}+1=1.0412233 \\n=2: & x_{3}=0.3 . \\& p y_{3}=y_{2}+h y_{2}^{\prime}=0.2030353+0.1(1.0412233)=0.3071577 \\& p y_{3}^{\prime}=f\left(x_{3}, p y_{3}\right)=f(0.3,0.3071577)=(0.3071577)^{2}+1=1.0943458 \\& y_{3}+y_{2}+(h / 2)\left(p y_{3}^{\prime}+y_{2}^{\prime}\right)=0.2030353+0.05(1.0943458+1.0412233)=0.3098138 \\& y_{3}^{\prime}=f\left(x_{3}, y_{3}\right)=f(0.3,0.3098138)=(0.3098138)^{2}+1=1.0959846\end{array}\)

Continuing in this manner, we generate Table 19-2. Compare it to Table 18-3.
