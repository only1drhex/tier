Guest Question.

A challenging problem need not involve a large circuit. Consider the following circuit.

Find \(i_{x}\).

Admin
Solution

This example contains only three loops but needs a careful arrangement of equations for the solution. Using mesh analysis, we define three different meshes.

We apply KVL in all three meshes to derive three equations:

  • \(\operatorname{KVL}(a):-24+10\left(i_{a}-i_{b}\right)+12\left(i_{a}-i_{c}\right)=0 \longrightarrow 11 i_{a}-5 i_{b}-6 i_{c}=12\),
  • \(\operatorname{KVL}(b): 24 i_{b}+4\left(i_{b}-i_{c}\right)+10\left(i_{b}-i_{a}\right)=0 \longrightarrow-5 i_{a}+19 i_{b}-2 i_{c}=0\),
  • \(\operatorname{KVL}(c): 12\left(i_{c}-i_{a}\right)+4\left(i_{c}-i_{b}\right)+4 i_{x}=0 \longrightarrow i_{a}+i_{b}=2 i_{c}\),

using \(i_{x}=i_{a}-i_{b}\). At this stage, the equations must be solved using a method such as substitution. For example, using \(i_{a}=2 i_{c}-i_{b}\), we have

\(\begin{aligned}22 i_{c}-11 i_{b}-5 i_{b}-6 i_{c}=&-16 i_{b}+16 i_{c}=12 \longrightarrow-4 i_{b}+4 i_{c}=3, \\&-10 i_{c}+5 i_{b}+19 i_{b}-2 i_{c}=24 i_{b}-12 i_{c}=0.\end{aligned}\)

The final equation states that \(i_{c}=2 i_{b}\); hence, we have \(i_{b}=3 / 4 \mathrm{~A}, i_{c}=3 / 2 \mathrm{~A}\), and \(i_{a}=\) 9/4 A. Finally, we obtain

\(i_{x}=i_{a}-i_{b}=9 / 4-3 / 4=3 / 2 \mathrm{~A} .\)
