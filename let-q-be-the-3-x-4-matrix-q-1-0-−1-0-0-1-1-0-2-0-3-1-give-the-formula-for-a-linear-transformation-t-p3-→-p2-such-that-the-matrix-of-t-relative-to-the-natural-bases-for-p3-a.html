Guest Question.

Let Q be the (3 × 4) matrix

\(Q=\left[\begin{array}{rrrr}1 & 0 & -1 & 0 \\0 & 1 & 1 & 0 \\2 & 0 & 3 & 1\end{array}\right]\).

Give the formula for a linear transformation \(T: P _{3} \rightarrow P _{2}\) such that the matrix of T relative to the natural bases for \(P _{3} \text { and } P _{2}\) is Q.

Admin
Solution

Let \(B=\left\{1, x, x^{2}, x^{3}\right\} \text { and let } C=\left\{1, x, x^{2}\right\}\). Following the proof of Theorem 21, we form a subset \(\left\{q_{0}(x), q_{1}(x), q_{2}(x), q_{3}(x)\right\} \text { of } P _{2}\) by using the columns of Q. Thus

\(q_{0}(x)=(1) 1+0 x+2 x^{2}=1+2 x^{2}\)
\(q_{1}(x)=(0) 1+1 x+0 x^{2}=x\)
\(q_{2}(x)=(-1) 1+1 x+3 x^{2}=-1+x+3 x^{2}\)
\(q_{3}(x)=(0) 1+0 x+1 x^{2}=x^{2}\).

If \(p(x)=a_{0}+a_{1} x+a_{2} x^{2}+a_{3} x^{3}\) is an arbitrary polynomial in \(P _{3},\), then define \(T: P _{3} \rightarrow P _{2} \text { by } T(p(x))=a_{0} q_{0}(x)+a_{1} q_{1}(x)+a_{2} q_{2}(x)+a_{3} q_{3}(x)\). Thus

\(T(p(x))=\left(a_{0}-a_{2}\right)+\left(a_{1}+a_{2}\right) x+\left(2 a_{0}+3 a_{2}+a_{3}\right) x^{2}\).

It is straightforward to show that T is a linear transformation. Moreover, \(T(1) =q_{0}(x), T(x)=q_{1}(x), T\left(x^{2}\right)=q_{2}(x), \text { and } T\left(x^{3}\right)=q_{3}(x)\). It follows that Q is the matrix of T with respect to B and C.
