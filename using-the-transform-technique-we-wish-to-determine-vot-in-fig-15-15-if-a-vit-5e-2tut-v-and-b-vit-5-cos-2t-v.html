Guest Question.

Using the transform technique, we wish to determine \(v_{o}\)(t) in Fig. 15.15 if (a) \(v_{i}(t) = 5e^{-2t}\)u(t) V and (b) \(v_{i}\)(t) = 5 cos 2t V.

Admin
Solution

a. In this case since \(v_{i}(t) = 5e^{-2t}\)u(t) V, then

\(V_{i}(ω) = \frac{5}{2 + jω}\) V

H(ω) for the network is

H(ω) = \(\frac{R}{R + jωL}\)
= \(\frac{10}{10 + jω}\)

From Eq. (15.45),

Eq. (15.45): \(V_{o}(ω) = H(ω)V_{i}(ω)\)

\(V_{o}(ω) = H(ω)V_{i}(ω)\)
= \(\frac{50}{(2 + jω)(10 + jω)}\)
= \(\frac{50}{8}( \frac{1}{2 + jω}-\frac{1}{10 + jω})\)

Hence, from Table 15.2, we see that

\(v_{o}(t) = 6.25[e^{-2t}u(t) – e^{-10t}u(t)]\) V

b. In this case, since \(v_{i}\)(t) = 5 cos 2t,

\(V_{i}\)(ω) = 5πδ(ω – 2) + 5πδ(ω + 2) V

The output voltage in the frequency domain is then

\(V_{o}(ω) =\frac{50π[ δ( ω- 2) + δ(ω + 2)]}{(10 + jω)}\)

Using the inverse Fourier transform gives us

\(v_{o}(t) = \mathscr{F}^{-1} [V_{o}(ω)] =\frac{1}{2π}\int_{-∞}^{∞}50π\frac{δ(ω – 2) +δ(ω+ 2)}{10 + jω}e^{jωt}\) dω

Employing the sampling property of the unit impulse function, we obtain

\(v_{o}(t) = 25 (\frac{e^{j2t}}{10 + j2} +\frac{e^{-j2t}}{10 – j2} )\)
= 25 \(( \frac{e^{j2t}}{10.2e^{j11.31°}} +\frac{e^{-j2t}}{10.2e^{-j1.31°}} )\)
= 4.90 cos (2t – 11.31°) V

This result can be easily checked using phasor analysis.
