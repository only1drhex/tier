Guest Question.

The helical coil extension spring of is subjected to a dynamic loading from 1.5 to 5 lbf. Estimate the factors of safety using the Gerber failure criterion for (a) coil fatigue, (b) coil yielding, (c) end-hook bending fatigue at point A of Fig. 10–7a, and (d) end-hook torsional fatigue at point B of Fig. 10–7b.

Admin
Solution

A number of quantities are the same as in : \(d = 0.035 in, S_{ut} = 264.7 kpsi, D = 0.213 in, r_{1} = 0.106 in, C = 6.086, K_{B} = 1.234, (K)_{A} = 1.14, (K)_{B} = 1.18, N_{b} = 12.17 turns, L_{0} = 0.817 in, k = 17.76 lbf/in, F_{i} = 1.19 lbf\), and \((τ_{i})_{uncorr }\)= 15.1 kpsi. Then

\(F_{a} = (F_{max} − F_{min})/2 = (5 − 1.5)/2 = 1.75 lbf\)
\(F_{m} = (F_{max} + F_{min})/2 = (5 + 1.5)/2 = 3.25 lbf\)

The strengths from include \(S_{ut} = 264.7 kpsi, S_{y} = 198.5 kpsi, and S_{sy} = 119.1 kpsi\). The ultimate shear strength is estimated from Eq. (10–30) as

\(S_{su} = 0.67S_{ut}\) (10–30)

\(S_{su} = 0.67S_{ut} = 0.67(264.7) = 177.3 kpsi\)

(a) Body-coil fatigue:

\(τ_{a} =\frac {8K_{B} F_{a}D}{πd^{3}} =\frac {8(1.234)1.75(0.213)}{π(0.035^{3})} (10^{−3}) = 27.3 kpsi\)

\(τ_{m} =\frac {F_{m}}{F_{a}}τ_{a} =\frac {3.25}{1.75}27.3 =50.7 kpsi\)

Using the Zimmerli data of Eq. (10–28) gives

\(S_{sa} = 35 kpsi (241 MPa) S_{sm} = 55 kpsi (379 MPa)\) (10–28)

\(S_{se} =\frac {S_{sa}}{1 − \left( \frac {S_{sm}}{S_{su}}\right)^{2} }= \frac {35}{1 − \left( \frac {55}{177.3}\right)^{2}} = 38.7 kpsi\)

From Table 6–7, p. 299, the Gerber fatigue criterion for shear is

\((n_{f} )_{body} =\frac {1}{2} \left( \frac {S_{su}}{τ_{m}}\right)^{2} \frac {τ_{a}}{S_{se}} \left[-1+\sqrt { 1 + \left(2 \frac {τ_{m}}{S_{su}} \frac {S_{se}}{τ_{a}}\right)^{2} }\right]\)

\(=\frac {1}{2} \left( \frac {177.3}{50.7}\right)^{2} \frac {27.3}{38.7} \left[-1+\sqrt { 1 + \left(2 \frac {50.7}{177.3} \frac {38.7}{27.3}\right)^{2} }\right] = 1.24\)

Fatigue factor of safety

\(n_{f} =\frac {1}{2} \left(\frac {S_{ut}}{σ_{m}} \right)^{2} \frac {σ_{a}}{S_{e}} \left[ -1+ \sqrt{ 1+ \left(\frac {2σ_{m} S_{e}}{S_{ut} σ_{a}} \right)^{2} } \right] σ_{m} > 0 \)

(b) The load-line for the coil body begins at \(S_{sm} = τ_{i}\) and has a slope \(r = τ_{a}/(τ_{m} − τ_{i} )\). It can be shown that the intersection with the yield line is given by \((S_{sa})_{y} = [r/(r + 1)](S_{sy} − τ_{i} )\). Consequently, \(τ_{i} = (F_{i}/F_{a})τ_{a}\) = (1.19/1.75)27.3 = 18.6 kpsi, r = 27.3/(50.7 − 18.6) = 0.850, and

\((S_{sa})_{y} =\frac {0.850}{0.850 + 1}(119.1 − 18.6) = 46.2\) kpsi

Thus,

\((n_{y})_{body} =\frac {(S_{sa})_{y}}{τ_{a}} =\frac {46.2}{27.3} = 1.69\)

(c) End-hook bending fatigue: using Eqs. (10–34) and (10–35) gives

\(σ_{A} = F \left[ (K)_{A} \frac {16D}{πd^{3}} +\frac {4}{πd^{2}}\right]\) (10–34)

\((K)_{A} =\frac {4C^{2}_{1} − C_{1} − 1}{4C_{1}(C_{1} − 1)} C_{1} =\frac {2r_{1}}{d}\) (10–35)

\(σ_{A} = F_{a} \left[ (K)_{A} \frac {16D}{πd^{3}} +\frac {4}{πd^{2}}\right]\)

\(= 1.75 \left[ 1.14 \frac {16(0.213)}{π(0.035^{3}) }+\frac {4}{π(0.035^{2})} \right] (10^{−3}) = 52.3\) kpsi

\(σ_{m} =\frac {F_{m}}{F_{a}} σ_{a} =\frac {3.25}{1.75}52.3 =97.1\) kpsi

To estimate the tensile endurance limit using the distortion-energy theory,

\(S_{e} = S_{se}/0.577 = 38.7/0.577 = 67.1\) kpsi

Using the Gerber criterion for tension gives

\((n_{f} )_{A} =\frac {1}{2} \left( \frac {S_{ut}}{σ_{m}}\right)^{2} \frac {σ_{a}}{S_{e}} \left[-1+\sqrt { 1 + \left(2 \frac {σ_{m}}{S_{ut}} \frac {S_{e}}{σ_{a}}\right)^{2} }\right]\)

\(=\frac {1}{2} \left( \frac {264.7}{97.1}\right)^{2} \frac {52.3}{67.1} \left[-1+\sqrt { 1 + \left(2 \frac {97.1}{264.7} \frac {67.1}{52.3}\right)^{2} }\right]= 1.08\)

(d) End-hook torsional fatigue: from Eq. (10–36)

\(τ_{B} = (K)_{B} \frac {8FD}{πd^{3}}\) (10–36)

\((τ_{a})_{B} = (K)_{B} \frac {8F_{a}D}{πd^{3}} = 1.18 \frac {8(1.75)0.213}{π(0.035^{3})}(10^{−3}) = 26.1\) kpsi
\((τ_{m})_{B} =\frac {F_{m}}{F_{a}}(τ_{a})_{B} =\frac {3.25}{1.75}26.1 = 48.5\) kpsi

Then, again using the Gerber criterion, we obtain

\((n_{f} )_{B} =\frac {1}{2} \left( \frac {S_{su}}{τ_{m}}\right)^{2} \frac {τ_{a}}{S_{se}} \left[-1+\sqrt { 1 + \left(2 \frac {τ_{m}}{S_{su}} \frac {S_{se}}{τ_{a}}\right)^{2} }\right]\)

\(=\frac {1}{2} \left( \frac {177.3}{48.5}\right)^{2} \frac {26.1}{38.7} \left[-1+\sqrt { 1 + \left(2 \frac {48.5}{177.3} \frac {38.7}{26.1}\right)^{2} }\right] = 1.30\)
