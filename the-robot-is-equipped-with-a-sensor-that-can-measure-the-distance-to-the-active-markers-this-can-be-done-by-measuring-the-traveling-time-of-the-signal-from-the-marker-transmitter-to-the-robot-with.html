Guest Question.

The robot is equipped with a sensor that can measure the distance to the active markers. This can be done by measuring the traveling time of the signal from the marker (transmitter) to the robot with the receiver. There are three active markers placed at known locations: \(m_{1}\) =
\(\left[x_{m 1}, \gamma_{m 1}\right]^{T}\) = \([0,0]^{T}\) , \(m_{2}\) =\(\left[x_{m 2}, \gamma_{m 2}\right]^{T}\) = \([5,3]^{T}\), and \(m_{3}\)= \(\left[x_{m 3}, \gamma_{m 3}\right]^{T}\) = \([1,5]^{T}\)

At a current robot position r = \([x,y]^{T}\) measured distances are d1 = 3.2016 m, d2 = 3.0414 m, and d3 = 2.6926 m as illustrated in Fig. 5.14.
What is the current robot position r?
This can again be solved by some optimization algorithm similarly as in with PSO or using some analytical solution.

Admin
Solution

The task of the trilateration algorithm is to find unknown position x,
y so that the relations in Eq. (5.70) are valid. This can be done with
PSO similarly as in , where the implementation needs to be
modified to include relations (5.70) in computation of the criteria function.

\(\begin{aligned}d_{1}^{2} &=\left(x_{m 1}-x\right)^{2}+\left(y_{m 1}-y\right)^{2} \\d_{2}^{2} &=\left(x_{m 2}-x\right)^{2}+\left(y_{m 2}-y\right)^{2} \\d_{3}^{2} &=\left(x_{m 3}-x\right)^{2}+\left(y_{m 3}-y\right)^{2}\end{aligned}\)

The solution of Eq. (5.70) can also be done analytically. Many
different algorithms can be implemented; here, a simple solution is given.
Subtracting the third equation in Eq. (5.70) from the first and the second one results in

 

\(\begin{aligned}&d_{1}^{2}-d_{3}^{2}=\left(x_{m 1}-x\right)^{2}-\left(x_{m 3}-x\right)^{2}+\left(y_{m 1}-\gamma\right)^{2}-\left(\gamma_{m 3}-\gamma\right)^{2} \\&d_{2}^{2}-d_{3}^{2}=\left(x_{m 2}-x\right)^{2}-\left(x_{m 3}-x\right)^{2}+\left(\gamma_{m 2}-\gamma\right)^{2}-\left(\gamma_{m 3}-\gamma\right)^{2}\end{aligned}\)

By rearranging

 

\(\begin{aligned}&2\left(x_{m 3}-x_{m 1}\right) x+2\left(y_{m 3}-\gamma_{m 1}\right) y=d_{1}^{2}-d_{3}^{2}-x_{m 1}^{2}+x_{m 3}^{2}-\gamma_{m 1}^{2}+y_{m 3}^{2} \\&2\left(x_{m 3}-x_{m 2}\right) x+2\left(y_{m 3}-\gamma_{m 2}\right) y=d_{2}^{2}-d_{3}^{2}-x_{m 2}^{2}+x_{m 3}^{2}-y_{m 2}^{2}+y_{m 3}^{2}\end{aligned}\)

we obtain linear equations in terms of x and y, which can be written in
matrix \(A r=b\) where

 

\(A=\left[\begin{array}{ll}2\left(x_{m 3}-x_{m 1}\right) & 2\left(\gamma_{m 3}-\gamma_{m 1}\right) \\2\left(x_{m 3}-x_{m 2}\right) & 2\left(\gamma_{m 3}-\gamma_{m 2}\right)\end{array}\right]\)

and

 

\(b =\left[\begin{array}{l}d_{1}^{2}-d_{3}^{2}-x_{m 1}^{2}+x_{q 3}^{2}-y_{m 1}^{2}+\gamma_{m 13}^{2} \\d_{2}^{2}-d_{3}^{2}-x_{m 2}^{2}+x_{m 3}^{2}-\gamma_{m 2}^{2}+y_{m 3}^{2}\end{array}\right]\)

from where an unknown position is obtained:

\(r =A^{-1} b\)

The correct solution is r =\([2,2.5]^{T}\) , and Matlab script implementation is given in Listing 5.8.
