Type II, Determine Flowrate

GIVEN Air at a temperature of 100 °F and standard pressure flows from a clothes dryer. According to the appliance manufacturer, the 4-in.-diameter galvanized iron vent on the clothes dryer is not to contain more than 20 ft of pipe and four 90º elbows.

FIND Under these conditions determine the air flowrate if the pressure at the start of the vent pipe, directly downstream of the dryer fan, is 0.20 in. of water.

Admin
Solution

Application of the energy equation (Eq. 8.21) between the beginning of the vent pipe, point (1), and the exit of the pipe, point (2), gives

 

\(\frac{p_{1}}{\gamma}+\alpha_{1} \frac{V_{1}^{2}}{2 g}+z_{1}=\frac{p_{2}}{\gamma}+\alpha_{2} \frac{V_{2}^{2}}{2 g}+z_{2}+h_{L}\) (8.21)

 

\(\frac{p_{1}}{\gamma}+\frac{V_{1}^{2}}{2 g}+z_{1}=\frac{p_{2}}{\gamma}+\frac{V_{2}^{2}}{2 g}+z_{2}+f \frac{\ell}{D} \frac{V^{2}}{2 g}+\sum K_{L} \frac{V^{2}}{2 g}\) (1)

 

where \(K_{L}\) for each elbow is assumed to be 1.5. In addition, \(V_{1}=V_{2} \text { and } z_{1}=z_{2}\). (The change in elevation is often negligible for gas flows.) Also, \(p_{2}=0, \text { and } p_{1} / \gamma_{H_{2} O}=0.2 \text { in., or }\)

 

\(p_{1}=(0.2 \text { in. })\left(\frac{1 ft }{12 in .}\right)\left(62.4 lb / ft ^{3}\right)=1.04 lb / ft ^{2}\)

 

\(\text { Thus, with } \gamma=0.0709 lb / ft ^{3} \text { (see Table B.3) and } V_{2}=V \text { (the air }\) velocity in the pipe2, Eq. 1 becomes

This is done easily by an iterative scheme as follows.

It is usually simplest to assume a value of f, calculate V from Eq. 2, calculate Re from Eq. 3, and look up the appropriate value of f in the Moody chart for this value of Re. If the assumed f and the new f do not agree, the assumed answer is not correct—we do not have the solution to the three equations. Although values of either f, V, or Re could be assumed as starting values, it is usually simplest to assume a value of f because the correct value often lies on the relatively flat portion of the Moody chart for which f is quite insensitive to Re.

Thus, we assume \(f=0.022\), approximately the large Re limit for the given relative roughness. From Eq. 2 we obtain

 

\(V=\left[\frac{945}{6.0+60(0.022)}\right]^{1 / 2}=11.4 ft / s\)

 

and from Eq. 3

 

\(\operatorname{Re}=1860(11.4)=21,200\)

 

\(\text { With this Re and } \varepsilon / D \text {, Fig. } 8.20 \text { gives } f=0.029 \text {, which is not }\) equal to the assumed solution \(f=0.022\) (although it is close!).

We try again, this time with the newly obtained value of \(f=0.029, \text { which gives } V=11.0 ft / s \text { and } \operatorname{Re}=20,500 . \text { With }\) With these values, Fig. 8.20 gives \(f=0.029\), which agrees with the assumed value. Thus, the solution is \(V=11.0 ft / s \text {, or }\)

 

\(Q=A V=\frac{\pi}{4}\left(\frac{4}{12} ft \right)^{2}(11.0 ft / s )=0.960 ft ^{3} / s\)

 

COMMENTS Note that the need for the iteration scheme is because one of the equations, \(f=\phi(\operatorname{Re}, \varepsilon / D)\), is in graphical form (the Moody chart). If the dependence of f on Re and \(\varepsilon / D \text { is }\) known in equation form, this graphical dependency is eliminated, and the solution technique may be easier. Such is the case if the flow is laminar so that the friction factor is simply \(f=64 / \operatorname{Re}\). For turbulent flow, we can use the Colebrook equation rather than the Moody chart. Thus, we keep Eqs. 2 and 3 and use the Colebrook equation (Eq. 8.35a) with \(\varepsilon / D=0.0015\) to give

 

\(\frac{1}{\sqrt{f}}=-2.0 \log \left(\frac{\varepsilon / D}{3.7}+\frac{2.51}{\operatorname{Re} \sqrt{f}}\right)\) (8.35a)

 

\(\begin{aligned}\frac{1}{\sqrt{f}} &=-2.0 \log \left(\frac{\varepsilon / D}{3.7}+\frac{2.51}{\operatorname{Re} \sqrt{f}}\right) \\&=-2.0 \log \left(4.05 \times 10^{-4}+\frac{2.51}{\operatorname{Re} \sqrt{f}}\right)\end{aligned}\) (4)

 

From Eq. 2 we have \(V=[945 /(6.0+60 f)]^{1 / 2}\), which can be combined with Eq. 3 to give

 

\(\operatorname{Re}=\frac{57,200}{\sqrt{6.0+60 f}}\) (5)

 

The combination of Eqs. 4 and 5 provides a single equation for the determination of f

 

\(\begin{aligned}\frac{1}{\sqrt{f}}=&-2.0 \log \left(4.05 \times 10^{-4}\right.\\&\left.+4.39 \times 10^{-5} \sqrt{60+\frac{6.0}{f}}\right)\end{aligned}\) (6)

 

By using a root-finding technique on a computer or calculator, the solution to this equation is determined to be \(f=0.029\), in agreement with the above solution which used the Moody chart.

Note that unlike the Alaskan pipeline example () in which we assumed minor losses are negligible, minor losses are of importance in this example because of the relatively small length-to-diameter ratio: \(\ell / D=20 /(4 / 12)=60\).

The ratio of minor to major losses in this case is \(K_{L} /(f \ell / D)=\) \(6.0 /[0.029(60)]=3.45\). The elbows and entrance produce considerably more loss than the pipe itself.
