FIND Show that only some of the shaft power into the air is converted into useful effects. Develop a meaningful effi-ciency equation and a practical means for estimating lost shaft energy.

Solution

We use the same control volume used in . Application of Eq. 5.82 to the contents of this control volume yields

 

\(\frac{p_{\text {out }}}{\rho}+\frac{V_{\text {out }}^{2}}{2}+g z_{\text {out }}=\frac{p_{\text {in }}}{\rho}+\frac{V_{\text {in }}^{2}}{2}+g z_{\text {in }}+w_{\substack{\text { shaft } \\ \text { net in }}}-\text { loss }\) (5.82)

 

\(\frac{p_{2}}{\rho}+\frac{V_{2}^{2}}{2}+g z_{2}=\frac{p_{1}}{\rho}+\frac{V_{1}^{2}}{2}+g z_{1}+w_{\substack{\text { shaft } \\ \text { net in }}}-\text { loss }\) (1)

 

As in Example 5.26, we can see with Eq. 1 that a “useful effect” in this fan can be defined as

 

\(\frac{\partial}{\partial t} \int_{ cv }\left( W + V _{ cv }\right) \rho d \forall=0\) (5.26)

 

\(\text { useful effect }=w_{\substack{\text { shaft } \\ \text { net in }}}-\text { loss }\)

 

\(=\left(\frac{p_{2}}{\rho}+\frac{V_{2}^{2}}{2}+g z_{2}\right)-\left(\frac{p_{1}}{\rho}+\frac{V_{1}^{2}}{2}+g z_{1}\right)\) (2) (Ans)

 

other words, only a portion of the shaft work delivered to the air by the fan blades is used to increase the available energy of the air; the rest is lost because of fluid friction.

A meaningful efficiency equation involves the ratio of shaft work converted into a useful effect 1Eq. 22 to shaft work into the air, \(W_{\text {shaft net in }}\). Thus, we can express efficiency, \(\eta\), as

 

\(\eta=\frac{w_\text { shaft net in }-\operatorname{loss}}{ w _{\text {shaft net in }}}\) (3)

 

However, when Eq. 5.54, which was developed from the momentof- momentum equation (Eq. 5.42), is applied to the contents of the control volume of Fig. E5.19, we obtain

 

\(w_{\text {shaft }}=-\left(\pm U_{\text {in }} V_{\theta \text { in }}\right)+\left(\pm U_{\text {out }} V_{\theta \text { out }}\right)\) (5.54)

 

\(\frac{\partial}{\partial t} \int_{ cv }( r \times V ) \rho d \forall+\int_{ cs }( r \times V ) \rho V \cdot \hat{ n } d A=\sum( r \times F )_{\begin{array}{c}\text { contents of the } \\\text { control volume }\end{array}}\) (5.42)

 

\(w_{\substack{\text { shaft } \\ \text { net in }}}=+U_{2} V_{\theta 2}\) (4)

 

Combining Eqs. 2, 3, and 4, we obtain

 

\(\begin{aligned}\eta=&\left\{\left[\left(p_{2} / \rho\right)+\left(V_{2}^{2} / 2\right)+g z_{2}\right]\right.\\&\left.-\left[\left(p_{1} / \rho\right)+\left(V_{1}^{2} / 2\right)+g z_{1}\right]\right\} / U_{2} V_{\theta 2}\end{aligned}\) (5) (Ans)

 

Equation 5 provides us with a practical means to evaluate the efficiency of the fan of . Combining Eqs. 2 and 4, we obtain

 

\(\begin{aligned}\text { loss }=& U_{2} V_{\theta 2}-\left[\left(\frac{p_{2}}{\rho}+\frac{V_{2}^{2}}{2}+g z_{2}\right)\right.\\&\left.-\left(\frac{p_{1}}{\rho}+\frac{V_{1}^{2}}{2}+g z_{1}\right)\right]\end{aligned}\) (6) (Ans)
