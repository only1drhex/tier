Guest Question.

Solve the one-dimensional boundary-value problem -Φ”= \(x^{2} \), 0 ≤x ≤ l, subject to Φ(0) = 0 =Φ(1). Use the finite difference method.

Admin
Solution

First, we obtain the finite difference approximation to the differential equation Φ” \(= -x^{2},\) which is Poisson’s equation in one dimension. Next, we divide the entire domain 0≤x≤1 into N equal segments each of length h ( =1/N) as in Figure 14.8(a) so that there are ( N + 1) nodes.

\(-x_{0}^{2}=\frac{d^{2}\Phi }{dx^{2}}|_{x=x_{0}}\simeq \frac{\Phi (x_{0}+h)-2\Phi (x_{0})+\Phi (x_{0}-h)}{h^{2}}\)
or
\(-x_{j}^{2}=\frac{\Phi _{j+1}-2\Phi _{j}+\Phi _{\xi -1}}{h^{2}}\)
Thus
\(-2\Phi _{j}=-\chi _{j}^{2}h^{2}-\Phi _{j+1}-\Phi _{j-1}\)
or
\(\Phi _{j}=\frac{1}{2}(\Phi _{j+1}+\Phi _{j-1}+x^{2}_{j}h^{2})\)

Using this finite difference scheme, we obtain an approximate solution for various values of N. The MATLAB code is shown in Figure 14.9. The number of iterations ni depends on the degree of accuracy desired. For a one-dimensional problem such as this, ni = 50 may suffice. For two- or three-dimensional problems, larger values of ni would be required (see later: Table 14.1). It should be noted that the values of Φ at end points (fixed nodes) are held fixed. The solutions for N = 4 and 10 are shown in Figure 14.10.
We may compare this with the exact solution obtained as follows. Given that \(d^{2}_{\Phi }/ dx^{2} =-x^{2}\), integrating twice gives
\(\Phi =-\frac{x^{4}}{12}+Ax+B\)

where A and B are integration constants. From the boundary conditions,
\(\Phi (0)=0\rightarrow B=0\)
\(\Phi (1)=0\rightarrow 0=-\frac{1}{12}+A\) or \(A=\frac{1}{12}\)

Hence, the exact solution is \(\Phi= x( 1 – x^{3}) /12,\) which is calculated in Figure 14.9 and found to be very close to case N = 10.
