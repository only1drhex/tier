Guest Question.

A gravity retaining wall shown on the left retains a sandy backfill with \(\phi^{\prime}=34^{\circ} \text { and } \gamma=\) \(18 kN / m ^{3} \text {. Analyze the stability of the wall. } \gamma_{\text {concrete }}=24 kN / m ^{3} \text {. }\)

Admin
Solution

The free body diagram is shown on the right of the above figure.

Since the backfill is inclined, Equation 10.12 will be used for computing \(K_{A}\).

 

\(K_{A}=\cos \beta \frac{\cos \beta-\sqrt{\cos ^{2} \beta-\cos ^{2} \phi^{\prime}}}{\cos \beta+\sqrt{\cos ^{2} \beta-\cos ^{2} \phi^{\prime}}}\) (10.12)

 

\(K_{A}=\cos \beta \frac{\cos \beta-\sqrt{\cos ^{2} \beta-\cos ^{2} \phi^{\prime}}}{\cos \beta+\sqrt{\cos ^{2} \beta-\cos ^{2} \phi^{\prime}}}=0.9848 \times \frac{0.9848-\sqrt{0.9848^{2}-0.8290^{2}}}{0.9848=\sqrt{0.9848^{2}-0.8290^{2}}}=0.294\)

 

\(H=6000+500 \tan 10=6088 mm\)

 

\(P_{A}=0.5 K_{A} \gamma H^{2}=0.5 \times 0.294 \times 18 \times 6.088^{2}=98.1 kN \text { per m width }\)

 

\(P_{P}=0\)

 

\(\text { Let's tabulate the values of } W_{i}, x_{i} \text {, and } W_{i} x_{i} \text { as shown on page } 408 \text {. }\)

 

Block No. \(\text { Weight } W_{i}( kN \text { per } m )\) \(x_{i}( m )\) \(W_{i} x_{i}( kN – m \text { per } m )\)
1 \(0.5 \times 1 \times 6 \times 24=72.0\) 0.667 48.0
2 \(1.0 \times 6 \times 24=144.0\) 1.500 216.0
3 \(0.5 \times 0.5 \times 6.0 \times 24=36.0\) 2.167 78.0
4 \(0.5 \times 0.5 \times 6.088 \times 18=27.4\) 2.333 63.9
\(\Sigma W_{i}=279.4 kN \text { per } m\) \(\Sigma W_{i} x_{i}=405.9 kN – m \text { per } m\)

 

\(\text { Let's take } \delta \text { as } 2 / 3 \phi^{\prime} \rightarrow \delta=22.7^{\circ}\)

 

Applying Equation 14.2:

 

\(F_{\text {sliding }}=\frac{\text { Maximum resistance available }}{\text { Driving force }}=\frac{P_{P}+S_{\max }}{P_{A} \cos \beta}\) (14.2)

 

\(F_{\text {sliding }}=\frac{P_{P}+S_{\max }}{P_{A} \cos \beta}=\frac{0+(98.1 \sin 10+279.4) \tan 22.7}{98.1 \cos 10}=1.28<1.5\)

 

\(S_{\max } \text { above was obtained from Equation 14.1. }\)

 

\(S_{\max }=\left(P_{A} \sin \beta+\sum_{i=1}^{n} W_{i}\right) \tan \delta\) (14.1)

 

Applying Equation 14.5:

 

\(F_{\text {overturning }}=\frac{\text { Resisting moment }}{\text { Driving moment }}=\frac{\left(\sum_{i=1}^{n} W_{i} x_{i}\right)+P_{P} \frac{h}{3}+P_{A} \sin \beta b}{P_{A} \cos \beta \frac{H}{3}}\) (14.5)

 

\(F_{\text {overturning }}=\frac{\left(\sum_{i=1}^{n} W_{i} x_{i}\right)+P_{P} \frac{h}{3}+P_{A} \sin \beta b}{P_{A} \cos \beta \frac{H}{3}}=\frac{405.9+0+98.1 \sin 10 \times 2.5}{98.1 \cos 10 \times 2.029}=2.29>2\)

 

Bearing pressures:

Applying Equation 14.6:

 

\(R=\left(\sum_{i=1}^{n} W_{i}\right)+P_{A} \sin \beta\) (14.6)

 

\(R=\left(\sum_{i=1}^{n} W_{i}\right)+P_{A} \sin \beta=279.4+98.1 \sin 10=296.4 kN \text { per } m\)

 

\(S=98.1 \times \cos 10=96.6 kN \text { per } m\)

 

Inclination of the load to vertical:

 

\(\alpha=\tan ^{-1} \frac{S}{R}=\tan ^{-1} \frac{96.6}{296.4}=18.1^{\circ}\)

 

Substituting in Equation 14.9:

 

\(\bar{x}=\frac{\left(\sum_{i=1}^{n} W_{i} x_{i}\right)+P_{P} \frac{h}{3}+P_{A} \sin \beta b-P_{A} \cos \beta \frac{H}{3}}{R}\) (14.9)

 

\(\begin{aligned}\bar{x} &=\frac{\left(\sum_{i=1}^{n} W_{i} x_{i}\right)+P_{P} \frac{h}{3}+P_{A} \sin \beta b-P_{A} \cos \beta \frac{H}{3}}{R} \\&=\frac{405.9+0+98.1 \sin 10 \times 2.5-98.1 \cos 10 \times 2.029}{296.4}=0.852 m\end{aligned}\)

 

\(\therefore \text { eccentricity } e=1.25-0.852=0.398 m <B / 6 \ldots \text { lies within the middle third. }\)

 

\(q_{\max }=\frac{Q}{B}\left(1+\frac{6 e}{B}\right)=\frac{296.4}{2.5}\left(1+\frac{6 \times 0.398}{2.5}\right)=231.8 kPa\)

 

\(\text { Bearing capacity calculations: } B^{\prime}=B-2 e=2.5-2 \times 0.398=1.70 m\)

 

\(\text { Plane-strain correction: } \phi^{\prime}=1.1 \times 34=37.4^{\circ} \rightarrow N_{q}=45, N_{\gamma_{\text {, Meyerhof }}}=57\)

 

\(\text { Shape factors: } s_{q}=s_{\gamma}=1\)

 

\(\text { Depth factors: Since } D_{f}=0, d_{q}=d_{\gamma}=1\)

 

Inclination factors:

 

\(i_{\gamma}=\left(1-\frac{\alpha}{\phi}\right)^{2}=\left(1-\frac{18.1}{37.4}\right)^{2}=0.27\)

 

Applying Equation 12.7:

 

\(q_{\text {ult }}=s_{c} d_{c} i_{c} c N_{c}+s_{q} d_{q} i_{q} \gamma_{1} D_{f} N_{q}+s_{\gamma} d_{\gamma} i_{\gamma} 0.5 B \gamma_{2} N_{\gamma}\) (12.7)

 

\(q_{\text {ult gross }}=s_{\gamma} d_{\gamma} i_{\gamma} 0.5 B \gamma_{2} N_{\gamma}=0.27 \times 0.5 \times 1.70 \times 18 \times 57=235.5 kPa\)

 

\(\text { Since } D_{f}=0, q_{\text {ult, net }}=235.5 kPa\)

 

\(\therefore F_{\text {bearing capacity }}=\frac{q_{\text {ult }}}{q_{ max }}=\frac{235.5}{231.8}=1.02<3.0\)

 

The wall is not safe with respect to sliding and bearing capacity; it is safe against overturning failure.
