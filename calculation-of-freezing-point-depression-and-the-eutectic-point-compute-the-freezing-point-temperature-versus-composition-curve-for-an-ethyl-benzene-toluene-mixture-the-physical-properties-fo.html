Guest Question.

Calculation of Freezing-Point Depression and the Eutectic Point

Compute the freezing-point temperature versus composition curve for an ethyl benzene–toluene mixture. The physical properties for this system are given in the table.

Toluene Ethyl Benzene
\(\delta( cal / cc )^{1 / 2} \text { at } 25^{\circ} C\) 8.9 8.8
\(\underline{V}^{ L }( cc / mol ) \text { at } 25^{\circ} C\) 107 123
Normal melting point (K) 178.16 178.2
Heat of fusion (J/mol) 6610.7 9070.9
\(C_{ P }^{ S }( J / mol K )\) 87.0 105.9
\(C_{ P }^{ L }( J / mol K )\) 135.6 157.4
Admin
Solution

The solubility parameters for toluene and ethyl benzene are nearly equal, so that the liquid mixture can be considered to be ideal \(\left(\gamma_{1}, \gamma_{2}=1\right)\). Since we do not know a priori whether pure toluene or pure ethyl benzene will freeze out as the solid component from a given mixture, the calculational procedure we will follow is to first assume that toluene appears as the solid component and compute the freezing-point temperature \(T_{f}\) of all toluene–ethyl benzene mixtures.

The calculations will then be repeated assuming ethyl benzene appears as the solid phase. The freezing point of a given mixture, and the solid that appears, will then be determined by noting which of the two calculations leads to the higher freezing point, as it is that solid that will freeze out first.

If we assume that toluene freezes out, \(T_{f}\) is found from the solution of

 

\(\ln x_{1}=-\frac{6610.7}{8.314}\left(\frac{178.16-T_{f}}{178.16 T_{f}}\right)-\frac{48.6}{8.314}\left(1-\frac{178.16}{T_{f}}+\ln \frac{178.16}{T_{f}}\right)\)

 

where x is the mole fraction of toluene. Since \(\Delta C_{ P }\) is so large, none of the terms in this equation can be neglected. The results are given in the following table. Similarly, if we assume that the solid precipitate is ethyl benzene, the mixture freezing point is computed from

 

\(\ln x_{1}=-\frac{9070.9}{8.314}\left(\frac{178.2-T_{f}}{178.2 T_{f}}\right)-\frac{51.5}{8.314}\left(1-\frac{178.2}{T_{f}}+\ln \frac{178.2}{T_{f}}\right)\)

 

where x is now the mole fraction of ethyl benzene. The results of this calculation are also given in the following table. Both sets of freezing points are plotted in Fig. 12.3-1.
