Guest Question.

Derive the interpolation function of a corner node in a cubic serendipity element.

Admin
Solution

First we note that the polynomials used for rectangular serendipity elements should not contain terms under the cone of Figure 9.2.5. For the element under consideration, the polynomial form is given in Eq. (9.2.32). Now consider node 1 of Figure 9.2.8. The function \(\psi _{1}\) must vanish on lines \(\zeta \) = 1 and \(\eta \) = 1. In addition, it should vanish at nodes 2, 3, 5 and 7. For any corner node, the interpolation function is of the form

\(\psi_{i}(\xi, \eta)=\left(1+\xi_{i} \xi\right)\left(1+\eta_{i} \eta\right)\left(a_{i}+b_{i} \xi^{2}+c_{i} \eta^{2}\right)\) i = 1, 2, 9 or 12

where \(\left(\xi_{i}, \eta_{i}\right)\) denote the local coordinates of the corner nodes. For node 1, we have \(\zeta _{1}\) = −1 and \(\eta _{1}\) = −1, and \(\psi _{1}\) has the form

\(\psi_{1}(\xi, \eta)=(1-\xi)(1-\eta)\left(a_{1}+b_{1} \xi^{2}+c_{1} \eta^{2}\right)\)

We must determine the constants \(a_{1},b_{1}\), and \(c_{1}\) using the conditions,

\(\psi_{1}(-1,-1)=1\), \(\psi_{1}\left(-\frac{1}{3},-1\right)=0\), \(\psi_{1}\left(-1,-\frac{1}{3}\right)=0\)

These conditions give the relations

\(a_{1}+b_{1}+c_{1}=\frac{1}{4}, a_{1}+\frac{b_{1}}{9}+c_{1}=0, a_{1}+b_{1}+\frac{c_{1}}{9}=0\)

whose solution is: \(a_{1}=-10 / 32, b_{1}=c_{1}=9 / 32\) , and the interpolation function \(\psi _{1}\) becomes

\(\psi_{1}(\xi, \eta)=\frac{1}{32}(1-\xi)(1-\eta)\left[-10+9\left(\xi^{2}+\eta^{2}\right)\right]\)

For a node intermediate to the corner nodes, the interpolation functions take a different form. For nodes 2 and 3, for example, \(\psi \) must vanish at \(\zeta \) = −1, \(\zeta \) = 1 and \(\eta \) = 1:

\(\psi_{i}(\xi, \eta)=\left(1-\xi^{2}\right)(1-\eta)\left(a_{i}+b_{i} \xi\right), \quad i=2 \text { or } 3\)

and for nodes 5 and 7, \(\psi \) must vanish at \(\zeta \) = 1, \(\eta \) = −1 and \(\eta \) = 1.

\(\psi_{i}(\xi, \eta)=(1-\xi)\left(1-\eta^{2}\right)\left(a_{i}+b_{i} \eta\right), \quad i=5 \text { or } 7\)

The constants \(a_{i}\) and \(b_{i}\) are to be determined using the interpolation property. As an example, consider node 2. We have

\(\psi_{2}(\xi, \eta)=\left(1-\xi^{2}\right)(1-\eta)\left(a_{2}+b_{2} \xi\right)\)

The \(a_{2}\) and \(b_{2}\) are to be determined from the conditions,

\(\psi_{2}\left(-\frac{1}{3},-1\right)=1, \quad \psi_{2}\left(\frac{1}{3}, -1\right)=0\)

which give \(a_{2}=-b_{2} / 3=9 / 32\) . Hence,

\(\psi_{2}(\xi, \eta)=\frac{9}{32}\left(1-\xi^{2}\right)(1-\eta)(1-3 \xi)\)

 

 

Eq. (9.2.32):\(\psi_{i}=\psi_{i}=c_{1}+c_{2} \xi+c_{3} \eta+c_{4} \xi \eta+c_{5} \xi^{2}+c_{6} \eta^{2}+c_{7} \xi^{2} \eta+c_{8} \xi \eta^{2}+c_{9} \xi^{3}+c_{10} \eta^{3}+c_{11} \xi^{3} \eta+c_{12} \xi \eta^{3}\)
