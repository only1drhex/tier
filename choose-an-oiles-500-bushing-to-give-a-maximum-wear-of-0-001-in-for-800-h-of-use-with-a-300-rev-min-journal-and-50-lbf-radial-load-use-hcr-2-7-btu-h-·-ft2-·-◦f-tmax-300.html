\(w = f_{1} f_{2}K \frac {4}{π} \frac {F}{DL} \frac {πDN t}{12} =\frac {f_{1} f_{2}K FN t}{3L}\) (12–32)

\(L =\frac{f_{1} f_{2}Kn_{d} FNt}{3w} =\frac{1(1)0.6(10^{−10})2(50)300(800)}{3(0.001)} = 0.48 in\) (1)

From Eq. (12–38) with \(f_{s}\) = 0.03 from Table 12–9, \(\hbar_{CR} = 2.7 Btu/(h · ft^{2} · °F), and n_{d} F for F\),

\(T_{f} \dot{=} T_{∞} +\frac{10π f_{s} FDN}{J \hbar_{CR}(2πDL/144)} = T_{∞} + \frac {720 f_{s} FN}{J \hbar_{CR}L}\) (12–38)

\(L\dot{=} \frac {720 f_{s}n_{d} FN}{J \hbar_{CR}(T_{f} − T_{∞}) }=\frac{720(0.03)2(50)300}{778(2.7)(300 − 70)} = 1.34 in\)

The two results bracket L such that 0.48 ≤ L ≤ 1.34 in. As a start let L = 1 in. From Table 12–13, we select D = 1 in from the midrange of available bushings.

Trial 1: D = L = 1 in.

Eq. (12–31):

\(P_{max} =\frac {4}{π} \frac {F}{DL}\) (12–31)

\(P_{max} =\frac{4}{π} \frac{n_{d} F}{DL} =\frac{4}{π} \frac{2(50)}{1(1)} = 127 psi < 3560 psi (OK)\)

\(P =\frac {n_{d} F}{DL} =\frac{2(50)}{1(1)} = 100 psi\)

Eq. (12–29):

\(V =\frac {πDN}{12}\) (12–29)

\(V =\frac{πDN}{12} =\frac{π(1)300}{12} = 78.5 ft/min < 100 ft/min (OK)\)

\(PV = 100(78.5) = 7850 psi · ft/ min < 46 700 psi · ft/ min (OK)\)

Our second estimate is L ≥ 0.48(1.64) = 0.787 in. From Table 12–13, there is not much available for \(L = \frac {7}{8}\) in. So staying with L = 1 in, try D = \(\frac {1}{2}\) in.

Trial 2: D = 0.5 in, L = 1 in.

\(P_{max} =\frac{4}{π} \frac{n_{d} F}{DL} =\frac{4}{π} \frac{2(50)}{0.5(1)} = 255 psi < 3560 psi (OK)\)

\(P =\frac {n_{d} F}{DL} =\frac{2(50)}{0.5(1)} = 200 psi\)

\(V =\frac{πDN}{12} =\frac{π(0.5)300}{12} = 39.3 ft/min < 100 ft/min (OK)\)

Note that PV is not a function of D, and since we did not change L, PV will remain the same:

PV = 200(39.3) = 7860 psi · ft/min < 46 700 psi · ft/min (OK)

From Table 12–9, \(f_{1}\) = 1.34, L ≥ 1.34(0.48) = 0.643 in. There are many \(\frac{3}{4}\) -in bushings to select from. The smallest diameter in Table 12–13 is D = \(\frac{1}{2}\) in. This gives an L/D ratio of 1.5, which is acceptable according to Eq.(12–33).

0.5 ≤ L/D ≤ 2 (12–33)

Trial 3: D = 0.5 in, L = 0.75 in. From trial 2, V = 39.3 ft/min does not change.

\(P_{max} =\frac{4}{π} \frac{n_{d} F}{DL}=\frac{4}{π} \frac{2(50)}{0.5(0.75)} = 340 psi < 3560 psi (OK)\)

\(P =\frac {n_{d} F}{DL} =\frac{2(50)}{0.5(0.75)} = 267 psi\)

PV = 267(39.3) = 10 490 psi · ft /min < 46 700 psi · ft /min (OK)

Select any of the bushings from the trials, where the optimum, from trial 3, is D = \(\frac{1}{2}\) in and L = \(\frac{3} {4}\) in. Other factors may enter in the overall design that make the other bushings more appropriate.
