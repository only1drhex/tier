Guest Question.

Relay Control of a Linear System

Use the describing function method to analyze the nonlinear control system shown in Figure 11.25 with the controller characteristic given in Figure 11.26.

Admin
Solution

The nonlinear controller is a “bang-bang” controller. In control parlance, the controller is called an ideal relay: it is either “full on” in one direction or “full on” in the other direction. This kind of control action can occur with a very high gain saturating amplifier.

When the input of the ideal relay is greater than zero, its output is 1. When the input is less than zero, its output is −1. Thus, the operation of the relay can be expressed mathematically as

 

\(u(t)=\left\{\begin{array}{r}1 \quad \text { for } \quad e(t)>0 \\-1 \quad \text { for } \quad e(t)<0\end{array}\right.\)

 

If the input to the relay is a sinusoid \(e(t)=e_{0} \sin \omega t\) the output is a square wave with the amplitude shifting between 1 and −1 every \(t=\pi p / \omega, p=1,2\) The period of the square wave is \(2 \pi / \omega\).

Since a square wave is an odd function, the Fourier coefficients \(a_{p}=0, p=0,1,2, \ldots\) and thus the Fourier series of the output of the relay can be written as

 

\(u(t)=\sum_{p=1}^{\infty} b_{p} \sin p \omega t\)

 

The fundamental harmonic of \(u(t)\)is \(b_{1} \sin \omega t=u_{1} \sin \omega t\) where

 

\(\begin{aligned}b_{1} &=\frac{\omega}{\pi} \int_{-\pi / \omega}^{\pi / \omega} y(t) \sin \omega t d t \\&=\frac{2 \omega}{\pi} \int_{0}^{\pi / \omega} \sin \omega t d t=-\left.\frac{2}{\pi} \cos \omega t\right|_{0} ^{\pi / \omega} \\&=\frac{4}{\pi} .\end{aligned}\)

 

Hence, the describing function of the ideal relay is

 

\(N\left(e_{0}\right)=\frac{u_{1}}{e_{0}} \angle 0=\frac{4}{\pi e_{0}} .\)

 

The transfer function of the linear plant model is

 

\(G(s)=\frac{16}{s(s+2)^{2}}\)

 

which has two poles at \(s\) = −2 and one pole on the imaginary axis. The plant is marginally stable. Setting \(s=i \omega\)

 

\(G(i \omega)=\frac{16}{i \omega(i \omega+2)^{2}}=\frac{-64 \omega+16\left(\omega^{2}-4\right) i}{\omega\left(\omega^{2}+4\right)^{2}}\)

 

The imaginary term vanishes at ω =2 making \(G(i \omega)\) real with the value −1/16. Setting this equal to −1/\(N\) = \(-\pi e_{0} / 4\) gives the amplitude of the limit cycle as \(e_{0}\) = 4/π. The amplitude of the fundamental harmonic of the plant output is

 

\(\frac{4}{\pi}|G(i 2)|=\frac{4}{\pi}\)

 

The filtering hypothesis is satisfied in this example. The Fourier series of the output of the relay is

 

\(\frac{4}{\pi}\left(\sin \omega t+\frac{1}{3} \sin 3 \omega t+\frac{1}{5} \sin 5 \omega t+\cdots\right)\)

 

Thus, for example, the amplitude of the third harmonic of the plant output is

 

\(\frac{4}{3 \pi}|G(i 6)|=\frac{1}{45} \frac{4}{\pi}\)

 

The higher harmonics are highly attenuated compared with the fundamental harmonic.
