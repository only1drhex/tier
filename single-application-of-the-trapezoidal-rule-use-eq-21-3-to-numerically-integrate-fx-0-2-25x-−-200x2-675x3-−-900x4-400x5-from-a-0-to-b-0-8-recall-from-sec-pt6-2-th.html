Guest Question.

Single Application of the Trapezoidal Rule

Use Eq. (21.3) to numerically integrate

\(I=(b-a) \frac{f(a)+f(b)}{2}\) (21.3)

\(f(x) = 0.2 + 25x − 200x^2 + 675x^3 − 900x^4 + 400x^5\)

from a = 0 to b = 0.8. Recall from Sec. PT6.2 that the exact value of the integral can be determined analytically to be 1.640533.

Admin
Solution

The function values

\(f(0) = 0.2\)

 

\(f(0.8) = 0.232\)

can be substituted into Eq. (21.3) to yield

\( I=(b − a)\frac{f(a) + f(b)}{2}\) (21.3)

\( I\cong 0.8\frac{0.2 + 0.232}{2}= 0.1728 \)

which represents an error of

\( E_{t}= 1.640533 − 0.1728 = 1.467733\)

which corresponds to a percent relative error of \(\varepsilon _{t} = 89.5\)%. The reason for this large error is evident from the graphical depiction in Fig. 21.6. Notice that the area under the straight line neglects a significant portion of the integral lying above the line.
In actual situations, we would have no foreknowledge of the true value. Therefore, an approximate error estimate is required. To obtain this estimate, the function’s second derivative over the interval can be computed by differentiating the original function twice to give

\(f^{\prime \prime }(x) = −400 + 4050x − 10,800x^2 + 8000x^3\)

The average value of the second derivative can be computed using Eq. (PT6.4):

\(Mean=\frac{\int_{a}^{b}{f(x)dx} }{b-a} \) (PT6.4)

\(\overline{f}^{\prime \prime } =\frac{\int_{0}^{0.8}{(−400 + 4050x − 10,800x^2 + 8000x^3)dx} }{0.8 − 0} = −60\)

which can be substituted into Eq. (21.6) to yield

 

\( E_{t}=-\frac{1}{12}f^{\prime \prime }(\xi )(b-a)^{3} \) (21.6)

\(E_{a}=-\frac{1}{12} (−60)(0.8)^3 = 2.56\)

which is of the same order of magnitude and sign as the true error. A discrepancy does exist, however, because of the fact that for an interval of this size, the average second derivative is not necessarily an accurate approximation of\(f^{\prime \prime }(\xi )\). Thus, we denote that the error is approximate by using the notation \(E_a\), rather than using \(E_t\).
