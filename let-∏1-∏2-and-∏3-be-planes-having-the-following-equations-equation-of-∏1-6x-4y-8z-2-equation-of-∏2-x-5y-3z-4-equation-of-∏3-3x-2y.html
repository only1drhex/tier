Guest Question.

Let \(\Pi_{1}, \Pi_{2}, \text { and } \Pi_{3}\) be planes having the following equations:

\(\text { Equation of } \Pi_{1}: \quad 6 x+4 y+8 z=2\)
\(\text { Equation of } \Pi_{2}: \quad x+5 y+3 z=4\)
\(\text { Equation of } \Pi_{3}: \quad 3 x+2 y+4 z=9\)

(a) Determine if any two of these three planes are parallel.

(b) Find the equation of the plane parallel to \(\Pi_{1}\), passing through \(P_{0}=(1,2.1)\).

Admin
Solution

(a) Let \(n _{1}, n _{2}, \text { and } n _{3}\) denote the normals for \(\Pi_{1}, \Pi_{2}, \text { and } \Pi_{3}\), respectively. Then

\(n _{1}=\left[\begin{array}{l}6 \\4 \\8\end{array}\right], \quad n _{2}=\left[\begin{array}{l}1 \\5 \\3\end{array}\right], \quad n _{3}=\left[\begin{array}{l}3 \\2 \\4\end{array}\right]\).

Clearly, \(n _{1}=2 n _{3}, \text { and so } n _{1} \text { and } n _{3}\) are parallel vectors. Therefore, \(\Pi_{1}\) and \(\Pi_{3}\) are parallel planes.

(b) Any plane parallel to \(\Pi_{1}\) has a normal equal to \(n _{1}\). Thus, using Eq. (1) \(A\left(x-x_{0}\right)+B\left(y-y_{0}\right)+C\left(z-z_{0}\right)=0\), we find the equation of the plane parallel to \(\Pi_{1}\) and containing \(P_{0}=(1,2,1)\) has equation
6x + 4y + 8z = 22
or, after canceling a common factor of 2,

3x + 2y + 4z = 11.
