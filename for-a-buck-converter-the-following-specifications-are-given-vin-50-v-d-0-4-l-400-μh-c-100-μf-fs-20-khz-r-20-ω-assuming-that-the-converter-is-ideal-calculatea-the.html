Guest Question.

For a buck converter, the following specifications are given:

\(V_{in} = 50 V, D = 0.4, L = 400 \mu H, C = 100 \mu F, f_s = 20 kHz, R = 20 \Omega\).

Assuming that the converter is ideal, calculate:

a) The average output voltage.

b) The maximum and minimum inductor current values.

c) The output voltage peak-to-peak

Admin
Solution

a) Assuming continuous inductor current the average output current is:

\(\overline{V_o} = V_{in} D = (50)(0.4) = 20 V\)

b) The maximum and minimum magnetizing inductance current values are:

\(I_{L,max} =\overline{V_o}[\frac{1}{R} + \frac{1-D}{2Lf_s}] = 20 [\frac{1}{20} + \frac{1-0.4}{2(400)(10)^{-6}.20(10)^3}] = 1 + \frac{1.5}{2} = 1.75 A\)

 

\(I_{L,min} =\overline{V_o}[\frac{1}{R} – \frac{1-D}{2Lf_s}] = 1 – \frac{1.5}{2} = 0.25 A\)

From the above value of \(I_{L,min}\) the initial assumption, that the inductor current is continuous, is satisfied.

c) The output voltage peak-to-peak ripple can be found from

\(\frac{\Delta V_o}{\overline{V_o}} = \frac{1-D}{8LCf^2} = \frac{1-0.4}{8(400)(10)^{-6}20000^2} = 0.00469 = 0.469 \%\)

or

\(\Delta V_o = (20)(0.00469) = 0.0938 V\) peak-to-peak
