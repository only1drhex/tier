Guest Question.

Determine the composite rate of return for the cash flows in Table 7-6

TABLE 7-6 Cash Flow and Cumulative Cash Flow Sequences,
Cash Flow, $ Cash Flow, $
Year Net Cumulative Year Net Cumulative
0 0 0 6 500 -350
1 200 +200 7 400 +50
2 100 +300 8 300 +350
3 50 +350 9 200 +550
4 -1800 -1450 10 100 +650
5 600 -850
Admin
Solution

A review of Table 7-6 indicates that the nonconventional cash flows have two sign changes and the cumulative cash flow sequence does not start with a negative value. There are a maximum of two \(i^{*}\) values. To find the one \(i^{\prime }\) value, develop the net-investment series \(F_{0}\) through \(F_{10}\) using Equation [7.6] and c = 15%.

\(F_{t} = F_{t-1}(1 + i) + C_{t}\) [7.6]

\( F_{0} = 0\)
\( F_{1}= \$200\) (\( F_{1}> 0\) , use c)
\( F_{2}= 200(1.15) + 100 = \$330\) (\( F_{2}> 0\), use c)
\( F_{3} = 330(1.15) + 50 = \$429.50\) (\( F_{3}> 0\), use c)
\( F_{4} = 429.50(1.15) – 1800 = \$- 1306.08\) (\( F_{4}< 0\), use \(i^{\prime }\))
\( F_{5}= – 1306.08(1 +i^{\prime }) + 600\)
Since we do not know if \( F_{5}\) is greater than zero or less than zero, all remaining expressions use \(i^{\prime }\).

\( F_{6} = F_{5}(1 + i^{\prime }) + 500 = [-1306.08(1 +i^{\prime }) + 600](1 +i^{\prime }) + 500\)
\( F_{7}= F_{6}(1 +i^{\prime }) + 400\)
\(F_{8} = F_{7}(1 + i^{\prime }) + 300\)
\(F_{9} = F_{8}(1 + i^{\prime }) + 200\)
\(F_{10} = F_{9}(1 + i^{\prime }) + 100\)

To find \( i^{\prime }\) , the expression \( F_{10} = 0 \) is solved by trial and error. Solution determines that \( i^{\prime } = 21.24\%.\) Since \( i^{\prime } > MARR\), the project is justified. In order to work more with this exercise and the net- investment procedure, do the case study in this chapter.

Comment
The two rates which balance the ROR equation are \( i^{*}_{1}= 28.71\%\) and \( i^{*}_{2} = 48.25\%\). If we rework this problem at either reinvestment rate, the \( i^{\prime } \) value will be the same as this reinvestment rate; that is, if \(c = 28.71\%, \) then \( i^{\prime }= 28.71 \%. \)
