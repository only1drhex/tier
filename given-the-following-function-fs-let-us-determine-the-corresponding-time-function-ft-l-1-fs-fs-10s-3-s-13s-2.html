Guest Question.

Given the following function F(s), let us determine the corresponding time function f(t) = \(\mathscr{L}^{-1}\) [F(s)].

F(s) = \(\frac{10(s + 3)}{(s + 1)^{3}(s + 2)}\)

Admin
Solution

Expressing F(s) as a partial fraction expansion, we obtain

F(s) = \(\frac{10(s + 3)}{(s + 1)^{3}(s + 2)}\)

= \(\frac{K_{11}}{s + 1 }+\frac{K_{12}}{(s + 1)^{2}} +\frac{K_{13}}{(s + 1)^{3}} +\frac{K_{2}}{s + 2}\)

Then

\((s + 1)^{3}F(s) \mid_{s=-1} =K_{13}\)
20 = \(K_{13}\)

\(K_{12}\) is now determined by the equation

\(\frac{d}{ds}[(s + 1)^{3}F(s)]\mid_{s=-1} = K_{12} \)

\(\frac{-10}{(s + 2)^{2}}\mid_{s=-1} = -10 = K_{12} \)

In a similar fashion \(K_{11}\) is computed from the equation

\(\frac{d^{2}}{ds^{2}}[(s + 1)^{3}F(s)]\mid_{s=-1} = 2K_{11} \)

\(\frac{20}{(s + 2)^{3}}\mid_{s=-1} = 20 = 2K_{11} \)

Therefore,

10 = \(K_{11}\)

In addition,

\((s + 2)F(s) \mid_{s=-2} =K_{2}\)
– 10 = \(K_{2}\)

Hence, F(s) can be expressed as

F(s) = \(\frac{10}{s + 1} – \frac{10}{(s + 1)^{2}} +\frac{20}{(s + 1)^{3}} – \frac{10}{s + 2}\)

Now we employ the transform pair

\(\mathscr{L}^{-1}[\frac{1}{(s + a)^{n+1}} ]=\frac{t^{n}}{n!}e^{-at}\)

and hence,

f(t) = \((10e^{-t} – 10te^{-t} + 10t^{2}e^{-t} – 10e^{-2t})\)u(t)
