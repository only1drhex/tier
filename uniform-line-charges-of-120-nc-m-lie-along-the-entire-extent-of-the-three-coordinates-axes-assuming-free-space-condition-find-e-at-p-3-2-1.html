Guest Question.

Uniform line charges of 120 nC/m lie along the entire extent of the three coordinates axes. Assuming free space condition, find E at P(-3, 2, -1).

Admin
Solution

Since all line charges are infiniely-long, we can write:
\({ E }_{ P }\) =\( \frac { { { \rho }_{ 1 } } }{ 2\pi { \epsilon }_{ 0 } } \left[ \frac { { R }_{ xP } }{ { { \left| { R }_{ xP } \right| }^{ 2 } } } +\frac { { R }_{ yP } }{ { \left| { R }_{ yP } \right| }^{ 2 } } +\frac { { R }_{ zP } }{ { \left| { R }_{ zP } \right| }^{ 2 } } \right]\)
where \({ R }_{ xP }, { R }_{ yP }\) and \({ R }_{ zP }\) are the normal vectors from each of the three axes that terminate on point P . specifically, \({ R }_{ xP }\) = (-3, 2, -1) – (-3, 0, 0) = (0, 2, -1) , \({ R }_{ yP }\) = (-3, 2, -1) – (0, 2, 0) = (-3, 0, -1), and \({ R }_{ zP }\) = (-3, 2, -1) – (0, 0, -1) = (-3, 2, 0) . Substituting these into the expression for \({ E }_{ P }\) gives
\({ E }_{ P }\) = \(\frac { { { \rho }_{ 1 } } }{ 2\pi { \epsilon }_{ 0 } } \left[ \frac { { 2a }_{ y }-{ a }_{ z } }{ 5 } +\frac { { -3a }_{ x }-{ a }_{ z } }{ 10 } +\frac { { -3a }_{ x }+2{ a }_{ y } }{ 13 } \right] \) =\( -1.15{ a }_{ x } + 1.20{ a }_{ y } – 0.65{ a }_{ z }\) kV/m
