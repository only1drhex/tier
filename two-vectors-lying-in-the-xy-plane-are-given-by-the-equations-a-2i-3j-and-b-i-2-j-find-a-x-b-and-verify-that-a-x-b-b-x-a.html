Guest Question.

Two vectors lying in the xy plane are given by the equations \( A =2 \hat{ i }+3 \hat{ j } \text { and } B =-\hat{ i }+2 \hat{ j } \text {. Find } A \times B \text { and verify } \) that A × B =-B × A.

Admin
Solution

Using Equations 11.7a through 11.7d, we obtain

\( \hat{ i } \times \hat{ i }=\hat{ j } \times \hat{ j }=\hat{ k } \times \hat{ k }=0 \) (11.7a).

\( \hat{ k } \times \hat{ i }=-\hat{ i } \times \hat{ k }=\hat{ j } \) (11.7d).

\( A \times B =(2 \hat{ i }+3 \hat{ j }) \times(-\hat{ i }+2 \hat{ j }) \)

\( =2 \hat{ i } \times 2 \hat{ j }+3 \hat{ j } \times(-\hat{ i })=4 \hat{ k }+3 \hat{ k }=7 \hat{ k } \).

(We have omitted the terms containing \( \hat{ i } \times \hat{ i } \text { and } \hat{ j } \times \hat{ j }\) because, as Equation 11.7a shows, they are equal to zero.)

\( \text { We can show that } A \times B =- B \times A \text {, because } \) \( B \times A =(-\hat{ i }+2 \hat{ j }) \times(2 \hat{ i }+3 \hat{ j }) \)

\( =-\hat{ i } \times 3 \hat{ j }+2 \hat{ j } \times 2 \hat{ i }=-3 \hat{ k }-4 \hat{ k }=-7 \hat{ k } \).

\( \text { Therefore, } A \times B =- B \times A \text {. } \)

\( \text { As an alternative method for finding } A \times B \text {, we could } \)\( \text { use Equation 11.8, with } A_{x}=2, A_{y}=3, A_{z}=0 \text { and } B_{x}=-1 \)\( B_{y}=2, B_{z}=0 \):

\( A \times B =\left(A_{y} B_{z}-A_{z} B_{y}\right) \hat{ i }-\left(A_{x} B_{z}-A_{z} B_{x}\right) \hat{ j }+\left(A_{x} B_{y}-A_{y} B_{x}\right) \hat{ k } \) (11.8).

\( A \times B =(0) \hat{ i }-(0) \hat{ j }+[(2)(2)-(3)(-1)] \hat{ k }=7 \hat{ k } \).
