Guest Question.

(a) Using Poisson’s summation formula, obtain \(\hat{X}(j \omega) \text { if } x(t)=\cos \omega_{0} t\). (b) Repeat part (a) for \(x(t)=u(t) e^{-t}\).

Admin
Solution

(a) From Table 6.2

 

\(X(j \omega)= F \cos \omega_{0} t=\pi\left[\delta\left(\omega+\omega_{0}\right)+\delta\left(\omega-\omega_{0}\right)\right]\)

 

Hence Eq. (6.45a) gives

 

\(\hat{X}(j \omega)=X_{D}\left(e^{j \omega T}\right)=\frac{1}{T} \sum_{n=-\infty}^{\infty} X\left(j \omega+j n \omega_{s}\right)\) (6.45a)

 

\(\hat{X}(j \omega)=\frac{\pi}{T} \sum_{n=-\infty}^{\infty}\left[\delta\left(\omega+n \omega_{s}+\omega_{0}\right)+\delta\left(\omega+n \omega_{s}-\omega_{0}\right)\right]\)

 

The amplitude spectrum of ˆx(t) is illustrated in Fig. 6.9a.

(b) From Table 6.2, we have

 

\(X(j \omega)= F \left[u(t) e^{-t}\right]=\frac{1}{1+j \omega}\)

 

Since

 

\(x(0+)=\lim _{t \rightarrow 0}\left[u(t) e^{-t}\right]=1\)

 

Eq. (6.45b) gives

 

\(\hat{X}(j \omega)=X_{D}\left(e^{j \omega T}\right)=\frac{x(0+)}{2}+\frac{1}{T} \sum_{n=-\infty}^{\infty} X\left(j \omega+j n \omega_{s}\right)\) (6.45b)

 

\(\hat{X}(j \omega)=\frac{1}{2}+\frac{1}{T} \sum_{n=-\infty}^{\infty} \frac{1}{1+j\left(\omega+n \omega_{s}\right)}\)

 

The amplitude spectrum of \(\hat{x}(t)\) is plotted in Fig. 6.9b for a sampling frequency \(\omega_{s}=15\) rad/s.
